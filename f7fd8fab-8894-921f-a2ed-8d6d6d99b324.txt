　　在这绝境之中，楚风眠没有被毁灭，反而是借助这始祖天龙之爪，无数圣龙秘境老古董联手围攻的力量之下，凝聚了不朽真身。
<br/>　　本来在这始祖天龙之爪的轰击之下，楚风眠只是勉勉强强摸到了不朽真身的门槛，可依然无法踏入。
<br/>　　可就是最后这些圣龙秘境老古董，为了彻底杀了楚风眠，将楚风眠逼入到了更加的绝境之中去，在这毁灭之中，才让楚风眠察觉到了不朽真身真正的奥义。
<br/>　　楚风眠重塑了肉身，在这一次次肉身被毁灭之下，楚风眠的身上的气息，非但是没有任何的减弱，反而是比起之前，还要更加强横。
<br/>　　这几乎是不可能的事情。
<br/>　　像是对九劫古帝，甚至是半步天人境的强者来说，他们被打破肉身，都可以瞬间之间重新凝聚肉身。
<br/>　　可每一次凝聚，都会让实力跌落，损失精血。
<br/>　　一点是肉身被打破的次数多一点，可能会直接跌落境界。
<br/>　　可现在楚风眠被打破了无数次的肉身，居然是非但没有损失任何一点点的力量，反而是身上的气势，更进一步，
<br/>　　这就是不朽真身的强横之处。
<br/>　　一旦是凝炼出了不朽真身，便是哪怕被打爆在多次的肉身，都可以恢复过来，而且不会有着任何一点点的损伤。
<br/>　　这就是为何无数强者，哪怕是天人境的强者，都想要凝炼不朽真身的原因。
<br/>　　“这小子居然没死？而且气息还更强了，到底怎么回事？”
<br/>　　“此人被打爆了这么多次肉身，不死也要境界跌落了，怎么可能气息还会更强？”
<br/>　　凌山，法理古帝等人看向楚风眠的眼神之中都是震惊无比，根本无法理解眼前这一切。
<br/>　　不朽真身，他们最多也只有在古籍之中听说过，天下之间肉身的极致，便是不朽。
<br/>　　可他们却从未见过。
<br/>　　“不用惧怕，这小子肯定是动用了什么秘术，在强撑，我们再度催动始祖天龙之爪，灭杀此子！”
<br/>　　凌山长啸一声，也算是稳定住了所有人的心思，只见他一股灵力，再度注入那始祖天龙之爪之中，打算催动始祖天龙之爪的力量，来再一次轰杀楚风眠。
<br/>　　“还想要催动始祖天龙之爪？这始祖天龙之爪，不是你们有资格可以得到的，给我拿过来吧！”
<br/>　　突然间楚风眠的身形动了，他的身形猛然向着那始祖天龙之爪飞了过去。
<br/>　　唰的一下。
<br/>　　楚风眠的身形便是来到了那始祖天龙之爪的面前，一把对着始祖天龙之爪抓了过去。
<br/>　　这始祖天龙之爪的厉害楚风眠已经是领教过了，他上一次可以撑住始祖天龙之爪的轰击，也是依靠了神树的力量。
<br/>　　那神树虚影，至少抵挡住了始祖天龙之爪七成以上的力量，才让楚风眠不至于被直接轰杀。
<br/>　　不然的话，刚刚那始祖天龙之爪一爪之下，楚风眠就直接被轰杀了，当场湮灭，就连一点点的反抗机会都没有。
<br/>　　虽然现在凌山等人，已经是没有多少的灵力了，不可能催动出刚刚那般强横的攻击来。
<br/>　　但是这始祖天龙之爪，对楚风眠依然有着极大的威胁，既然如此，楚风眠就先夺取了这始祖天龙之爪再说。
<br/>　　而且在见识了这始祖天龙之爪的强大之处后，楚风眠身上的龙皇血脉，对着始祖天龙之爪已经是更加的渴望了起来。
<br/>　　这些圣龙秘境老古董，不过是以一种手段强行催动了始祖天龙之爪之上一部分的禁制，来催动这始祖天龙之爪。
<br/>　　论起手段，楚风眠可不会比起这些圣龙秘境老古董之下。
<br/>　　而且楚风眠更是拥有龙皇血脉，比起那些圣龙秘境老古董，要更加容易催动这始祖天龙之爪。
<br/>　　楚风眠一股灵力，打入到了这始祖天龙之爪之中，刹那之间便是看到了，在这始祖天龙之爪之中，一道大阵被刻印在了其中。
<br/>　　这一道大阵，笼罩了始祖天龙之爪三分之一的部分，显然是以外力，打入其中的。
<br/>　　那些圣龙秘境老古董，便是凭借这一道大阵的力量，才可以催动这始祖天龙之爪来进行攻击。
<br/>　　这种手段，倒是跟那荒鎏金炼化剑鼎的手段，极为相似。
<br/>　　不过这始祖天龙之爪的力量，要比起剑鼎还是要强大的多。
<br/>　　所以哪怕是他们这么多名九劫古帝，半步天人境强者联手催动，才只能够是爆发出始祖天龙之爪力量的十分之一。
<br/>　　“吞天秘术，给我吞！”
<br/>　　楚风眠现在也不着急得到这始祖天龙之爪，只要是让这些圣龙秘境老古董，无法催动这始祖天龙之爪，无法对楚风眠在造成威胁。
<br/>　　楚风眠便是可以先斩杀了这些圣龙秘境老古董，在来慢慢研究这始祖天龙之爪的奥秘。
<br/>　　吞天秘术，被打入其中，居然没有遭到一点点的反抗。
<br/>　　这始祖天龙之爪，已经是失去了灵性，跟剑鼎不同，毕竟剑鼎属于灵器，其中拥有这剑道之主的烙印。
<br/>　　所以哪怕是剑道之主已经是陨落了，有着烙印的存在，那荒鎏金依然是无法炼化剑鼎，只能够以让剑鼎沉睡的手段，来控制楚风眠。
<br/>　　但是这始祖天龙之爪，却是始祖天龙的一部分，始祖天龙陨落，这始祖天龙之爪也就成了真正的无主之物。
<br/>　　理论上任何人都可以将这始祖天龙之爪炼化，不过始祖天龙之爪之中蕴含的力量，比起寻常天人境强者，还要强大的多。
<br/>　　这些圣龙秘境老古董，根本没有炼化始祖天龙之爪的能力，所以才只能够用阵法来进行催动。
<br/>　　如果这始祖天龙之爪之中还拥有意识的话，那这些圣龙秘境老古董拼了命，也别想要催动。
<br/>　　这些圣龙秘境老古董可以将阵法打入其中，楚风眠也可以用吞天秘术，将这阵法打破。

<br/>　　整个阵法，被这吞天秘术吞噬一瞬，便是破碎大半。
<br/>　　“不好！”
<br/>　　“阻止他！”
<br/>　　凌山，法理古帝等人脸色严峻，他们对付楚风眠，凭借的最大底牌，就是这始祖天龙之爪。
<br/>　　一旦是他们失去了这始祖天龙之爪，那就根本失去了跟楚风眠对抗的资本，必死无疑。
