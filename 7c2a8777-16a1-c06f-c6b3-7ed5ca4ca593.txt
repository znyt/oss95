　　夜色越来越深，营地中的篝火开始一点一点的熄灭。
<br/>　　天已经快要亮了，但等待中的攻击却依旧没有到来。
<br/>　　怎么回事？埋伏带兵埋伏在沈城西南的副将忍不住开时焦急起来，时不时将目光投向东方已经开始隐隐有些发白的天空。
<br/>　　而就在此时，正在仰头看向天空的副将突然愣了一下，随后飞快的趴到地上，将耳朵贴向地面。
<br/>　　一阵阵微弱的震动从地面上传来，似野兽的咆哮一般。
<br/>　　接着，那震动越来越急，渐渐的已经不用将耳朵贴在地面也可以听到那咆哮的水声。
<br/>　　来了，终于来了，靺鞨人的攻击终于来了。
<br/>　　副将先是如释重负般松了口气，但很快又紧张起来，对身边的亲卫打了几个手势，示意他们马上做好应付洪水的准备。
<br/>　　虽然他们已经躲到了城池的另一面，但水火无情，天知道那些靺鞨人到底为他们准备了多少水，如果那洪水真的特别大，一座土城明显起不到阻挡一切的作用。
<br/>　　而且除了防洪，他们还需要做好进攻的准备，因为大水过后，靺鞨人必然来攻，他们还要给靺鞨一个大大的惊喜。
<br/>　　……
<br/>　　而此时的沈城西北，黑齿常之的脸上也露出了一抹淡淡的微笑。
<br/>　　为将者，当上知天时，下知地理，中晓人情！靺鞨人虽然学了一些大唐的兵法，但他们终究学的还是不深，百密一疏在河床那里留下了一丝破绽，而就是这一丝破绽决定了他们的败亡。
<br/>　　现在一切只看天意，不知道他们派出多少人来打伏击，若是全军尽出，只怕这沈阳将会是他们最后的一站。
<br/>　　……
<br/>　　积蓄了三天的洪水翻滚着急冲而下，带起无数泥沙。
<br/>　　原本清澈而温和的河水变成浑浊而疯狂，顺着河床吞噬着一切敢于阻拦它的事物。
<br/>　　无数在河边饮水的动物被滔天的大水卷起，成为河水的一部分，无数河边的植物顺水而飘，河上的简单的桥梁被瞬间吞噬，一切的一切都在大水到来的时候变的那么脆弱。
<br/>　　终于，洪水来到了唐军营地的前面，它咆哮着，嘶吼着，没有任何犹豫的疯狂扑向营地中的一切。
<br/>　　沉得的拒马被河水高高卷起又重重抛下，然后变成一堆知明堂的东西，营地中的篝火，帐篷，所有与大地无关的东西都被那咆哮的河水所吞噬。
<br/>　　……
<br/>　　这一切看的副将额头冷汗涔涔，如果不是黑齿常之看出河水的水位不对，提前察查靺鞨人的计划，估计现在一万骑兵已经剩不下几个人了吧。
<br/>　　想到这里，副将不由将目光投向北方，那是黑齿常之的方向，这位领军主将的任务是负责阶段敌军的后路，将那些准备趁火打劫的靺鞨人彻底留下，而他的任务则是堵住靺鞨人的前方，让他们无路可逃，最后将这些人围死在这片满是泥泞的河床之上。
<br/>　　但是，为什么大水已经过去，该来的信号却迟迟没有到呢？难道是将军那边出了什么问题？
<br/>　　久等之后，眼看天色已经大亮，北方的天空却依旧没有任何信号烟升起的，这让副将不由开始担心起来。
<br/>　　但，军令如山，就算是他紧张的恨不能用头往地上的泥水里狠狠撞上那么百十下，也依旧不敢离开现在的位置，他必须等，等那三个深绿色的信号烟。
<br/>　　……
<br/>　　而此时此刻，靺鞨人也是焦急万分，大祚荣也在不断的骂娘。
<br/>　　千算万算他们忘了计算大水过后道路泥泞的问题，看着战马在满是泥水的河岸上艰难跋涉，而唐军的大营却还在十里之外，大祚荣真的很想给自己来两个耳光。
<br/>　　不行了，不能再拖了，再拖下去唐军一定会回过神来，到时候好好的一场埋伏就会打成遭遇战，或者追击战，想到这里大祚荣对身后的骑兵吼道：“快，再快一点，把不重要的东西全都丢掉，轻装前进，快！”
<br/>　　“快快快，没听到首领的命令么，把没用的东西行礼都丢掉。”
<br/>　　“丢掉丢掉，只要把那些唐军都杀死，有很多好东西给你们捡，到时候只怕捡都捡不过来，你们的那些破毡子还要它干嘛。”
<br/>　　随着大祚荣的一声令下，下面立刻有人催促起来，皮鞭与刀背的敲打下，很多靺鞨人丢掉了绑在战马上的行礼，留着下随身的刀箭。
<br/>　　行军的速度再次提高，但提高的并不多，可是这已经是极限了，毕竟战马不是战船，你不能指望它在水里跑的比地上还快。
<br/>　　……
<br/>　　望远镜中，黑齿常之已经可以看到靺鞨人的影子，望着那些在泥水中跋涉的敌人，他的嘴角习惯性的翘了起来，同时对身后的亲卫打了一个手势：“命令部队，准备进攻！三发绿色信号弹准备。”
<br/>　　“是！”亲卫早就已经等的不耐烦了，听到他的命令之后几乎是用跳的离开。
<br/>　　时间不大，这片不大的土丘后面，已经响起了队部集合的声音。
<br/>　　……
<br/>　　等待还在继续，不过看着黑齿常之已经举起的右手，五千唐军的肾上腺素已经开始飙升，尽管他们并不知道肾上腺素是什么东西，但总是每一个人都很兴奋。
<br/>　　此时此刻，好像每一个呼吸的时间都是那么漫长，急的让人恨不能跳起来，跑到那些靺鞨人身后推他们一把，让他快一点，再快一点。
<br/>　　“全军突击，杀……”就在众人急的差点把眼珠子瞪出来的时候，远远的河岸边靺鞨人终于动了，随着一声命令，近五万的靺鞨骑兵猛的提高了马速，疯狂的冲向飘满‘尸体’的唐军大营。
<br/>　　而就在同时一刻，黑齿常之的右臂狠狠的落了下来：“发信号，全军随本将，杀！”
<br/>　　“杀！杀！杀！”“哗……”
<br/>　　衣甲碰撞，杀声震天！
<br/>　　早就已经等的红了眼的五千唐军奋力的跳上马背，刀出鞘，弩上弦，跟在黑齿常之的身后冲出了土丘，向着靺鞨人的身后杀了过去。
<br/>　　而与此同时，三发副将期待已久的绿色信号烟冲天而起，昭示着决战的开始。
