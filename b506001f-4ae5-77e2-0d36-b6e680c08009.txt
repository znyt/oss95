<br/>    燕山，沙海之外，身背六把刀的无疆看向燕山，露出一抹欣赏的神色，“还真是一把好刀啊，可惜，他的师父控制的太严格了，即便是让他掌握大君的力量，他也是能够掌控的住的吧？”
<br/>    
<br/>        “虽然这样会让他彻底入魔，意识变得嗜杀，但是这样，不正是我想要的吗？”
<br/>    
<br/>        “他该拥有的大君的力量！”
<br/>    
<br/>        无疆踏步，进入沙海之中，风沙并没有像是对待妖王那般想要搅碎他，而是散开向两边，给他让出一条路来，无疆笑了笑，道：“看来风妖虽然实力不怎么样，但是记性还是很好的。既然你主动让开路来，我这一次，就放你一马吧。”
<br/>    
<br/>        无疆道完，继续前进。
<br/>    
<br/>        不过一个时辰，他已经到了燕山的脚下，他可以发现很快有血衣剑卫发现了他，众多血衣剑卫露出警戒的神色，看向他，“你是何人？”
<br/>    
<br/>        无疆微笑，“鄙人，魔宗大宗主，无疆！”
<br/>    
<br/>        众多血衣剑卫惊慌不已，他们虽然没有见过无疆，但是还是能够感受到无疆的威势，能够如此轻松通过沙海，来到这里的，绝对是真正的魔宗大宗主。
<br/>    
<br/>        “不知道大宗主来我燕山作甚？”
<br/>    
<br/>        无疆背负双手，泰然自若，“我此行来，对不住燕山，所以我不杀你们。而我此行的目的，是要见一个人。”
<br/>    
<br/>        一个血衣剑卫问道：“你要见何人？”
<br/>    
<br/>        无疆淡淡道出一个名字，“李素衣！”
<br/>    
<br/>        众多血衣剑卫将消息禀报了回去，凌仙和李素衣也是得知了这个消息，凌仙道：“魔宗大宗主要见你，你知道是什么情况吗？”
<br/>    
<br/>        李素衣摇摇头，道：“我并未和魔宗大宗主有任何关系，也没有见过，我也不知道是什么事。二师兄，我应该怎么办？”
<br/>    
<br/>        凌仙戒指中的凌家老祖，忽然在凌仙的意识中告诉凌仙：“外面来的那人，已经无限接近仙人了，在这苍天之下，恐怕没有任何一人是他的对手。”
<br/>    
<br/>        凌仙在心中问道：“那我们怎么办？”
<br/>    
<br/>        凌家老祖道：“既然他要见李素衣，你拦不住，你也没有任何办法，而且他此行来并没有杀人，不如让他见李素衣一面。”
<br/>    
<br/>        凌仙对着李素衣道：“魔宗大宗主来到此处，并没有杀人，说不定并没有敌意，你不妨和他见上一面。”
<br/>    
<br/>        李素衣也只能点头道：“那好吧。”
<br/>    
<br/>        在燕山之下的一处别院中，李素衣推开院门，可以看得到一个背着六柄刀的男人，“你找我？”
<br/>    
<br/>        无疆回头，露出一抹欣赏之色，“还真是一柄好刀啊！”
<br/>    
<br/>        李素衣有些疑惑，静静等待着无疆接下来的话。
<br/>    
<br/>        无疆笑道：“你这柄刀，还在鞘中之时，我就知道你是一柄好刀，我是一个爱刀之人，所以，我想要从燕山取走你这柄刀。”
<br/>    
<br/>        李素衣想要走，却是半步都动弹不得，那是无疆的威势，让他一步都动弹不得，无疆围着李素衣行走，一边走一边道：“我知道你修行的是魔宗功法，而且是最为邪恶的《白骨饕餮》，同样的，我也知道你身后的这柄刀，那是血海魔刀，一柄真正的神兵。”
<br/>    
<br/>        “我虽然没有见过你的师父，但是我不得不说，他对你很好，将《白骨饕餮》传授给你，而且传授的很好，将血海魔刀交给你，这在我们魔宗来说，简直是不可能的事。在魔宗，只有大君，才配拥有这柄刀。”
<br/>    
<br/>        “你的实力可以突飞猛进，我也能够感受到，你能够掌控住大君的力量，只是意识会有些被魔念扰乱，不过这对于魔来说简直太正常了。”
<br/>    
<br/>        “所以，我要带你走，让你成为真正的大君，让你真正有资格拥有这柄刀。”
<br/>    
<br/>        无疆伸手，将僵住不能动的李素衣扛在自己的肩上，无疆回头看了一眼燕山，“你的师父教你教的很好，磨刀磨的很出色，我没有经过他的同意便带走你，不问自取是为偷，既然是我不对在先，那我就不杀任何人。”
<br/>    
<br/>        无疆离开，凌仙可以看得到，但是却没有任何办法。
<br/>    
<br/>        凌仙看向无疆离开的身影，无力跪倒在地，“师父，徒儿无能.......”
<br/>    
<br/>        一天之后，无疆在魔宗落下，然后将李素衣丢到了一座监牢中，李素衣并不知道这个监牢是做什么，在监牢后，有着一扇门通向另一座监牢，在其中同样是关着一个人，李素衣看向这一座座监牢的最后，那是巫三起。
<br/>    
<br/>        此时的巫三起，已经是悟道境巅峰。
<br/>    
<br/>        无疆站在监牢之外，朝着李素衣道：“每天，我会打开你附近牢笼之间的通道，而这个牢笼之中，只能有一个人活下来的时候，我才会给予你们食物，若是两个人，那么你们就只能等着饿死。”
<br/>    
<br/>        一天之后，无疆看着浑身鲜血的李素衣，将一只烤鸡丢了进去，李素衣伸手，接住烤鸡，一块一块撕扯着放进自己的嘴里。
<br/>    
<br/>        无疆看向李素衣身旁的尸体笑了笑，“很不错嘛，不过还没有特别好。”然后无疆伸手一指巫三起，“我可以看得出来你对他的杀意，所以我抓了他来，你应该知道，他是悟道境巅峰，而你只是入道境巅峰，如果你不施展《白骨饕餮》吞了这些无辜的入道境和悟道境提升修为。”无疆对于无辜两个字咬的很重，“那么你最后面对他的时候，只有死路一条。”
<br/>    
<br/>        李素衣可以感受的到巫三起的目光，他上一世，就是死在了巫三起的手里，这一世，难道又要死在巫三起的手中吗？
<br/>    
<br/>        不，我不能死在巫三起的手里！
<br/>    
<br/>        可是我这么做的话，师父一定会很失望的吧.......
<br/>    
<br/>        三天之后，无疆再次来到牢笼之前，伸手指了指李素衣身前的那些魔修，“可以供你吞的人，可不多了，你没有机会犹豫了。”
<br/>    
<br/>        又是一天过去，无疆再次来到牢笼，李素衣握着血海魔刀，他的身旁却是没有了另外那个魔修的尸体，无疆笑了笑，“你愿意吞下这些无辜的修行者了？”
<br/>    
<br/>        李素衣缓缓抬头，一双眸子红的如同在滴血，“魔修，何来无辜？”
