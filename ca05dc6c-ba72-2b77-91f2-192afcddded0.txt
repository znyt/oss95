        真的是苏漫雪吗？
<br/>
<br/>        落以柔一步一步朝床边靠近，两只水灵的大眼睛，目不转睛地盯着睡在床上的这个女人。
<br/>
<br/>        这个女人侧卧着，身段玲珑有致，腰肢特别细，不施粉黛的睡颜，倒是天生丽质。
<br/>
<br/>        再仔细一看，她又觉得这个女人不像是苏漫雪。
<br/>
<br/>        “不要走！”童筱颖突然睁开双眼，一声呓语坐起身来。
<br/>
<br/>        落以柔被她这一声尖叫给吓了一跳，一脚往后一踩，结果重心不稳，双手在半空中呈划船状。
<br/>
<br/>        童筱颖见状，立即伸手，抓住了落以柔的手腕，拉了她一把。
<br/>
<br/>        岂料，落以柔的身子因为惯性，直接往童筱颖的身上扑去。
<br/>
<br/>        两个女人，一上一下，倒回了床上。
<br/>
<br/>        落以柔的一只手，直接压在了童筱颖肚子的左侧。
<br/>
<br/>        那个地方缺了一颗肾，这突然袭来的重量，让童筱颖痛得倒吸了口冷气，鼻息里全是一股淡淡的令她熟悉的薄荷香。
<br/>
<br/>        “对、对不起！对不起！”落以柔见童筱颖皱着眉头，连忙从童筱颖身上起来，站在床边不停地向她九十度鞠躬，嘴里也不停地道歉。
<br/>
<br/>        童筱颖捂着肚子，吃痛地坐起身来，向落以柔摆了摆手，但脸色却有些苍白：“没事。”
<br/>
<br/>        “你……”落以柔站直了身子，见童筱颖的脸色不太好，于是凑过前去，弱弱地关问道，“要不要紧？要不，我送你去医院。总裁夫人？”
<br/>
<br/>        “你叫我什么？”童筱颖抬眸诧异地看着落以柔。
<br/>
<br/>        落以柔赧然一笑，抬起手来比划着讷讷地说道：“傅总跟我说，你是总裁夫人，你……”
<br/>
<br/>        “……”童筱颖尴尬地抚着额头，又呲牙又是咬嘴唇，完全不知道该怎么回答这个女孩的话。
<br/>
<br/>        “总裁夫人，你好，我叫落以柔，是郑助理的秘书。”落以柔顿时落落大方地向童筱颖伸出手来。
<br/>
<br/>        童筱颖放下手来，看了落以柔一眼，礼貌性地跟落以柔握了握手，嘴角挤着笑容，自我介绍道：“你好，我叫童筱颖。”
<br/>
<br/>        童筱颖？！
<br/>
<br/>        这个名字怎么这么耳熟呢？
<br/>
<br/>        落以柔在心底思忖了一会儿，但是一时半会儿也想不起来。
<br/>
<br/>        不过，她果然不是苏漫雪，只不过是跟那个苏漫雪长得很像。
<br/>
<br/>        “你真的是总裁夫人？”落以柔仍旧有些难以接受这个事实，哪怕是傅夜沉亲口说的。
<br/>
<br/>        童筱颖被她这么问得尴尬不已，而且，不知道是不是因为另一半“出轨”让她“出”出了敏感神经，她总觉得，这女孩和傅夜沉之间的关系不简单。
<br/>
<br/>        不然，这女孩身上怎么会跟傅夜沉身上有一模一样的薄荷香气。
<br/>
<br/>        不会又要她去斗“小三”吧？！
<br/>
<br/>        一个季思妍就已经让她精神错乱又崩溃了，再来一个，她可真的吃不消啊！
<br/>
<br/>        童筱颖想想都觉得自己的心好累。
<br/>
<br/>        不过，幸好，她不喜欢傅夜沉，也不会让自己去爱上傅夜沉。而傅夜沉娶她，只不过是因为她长得像苏漫雪罢了。
<br/>
<br/>        所以，傅夜沉要是婚内出轨了什么的，她正好有理由跟他离婚。
<br/>
<br/>        “你怎么不说话呢？”落以柔接着问道。
<br/>
<br/>        童筱颖这才回过神来，不假思索地回答：“大哥说是就是吧！”
<br/>
<br/>        “大哥？！”落以柔惊愕，“你叫傅总，大哥？”
<br/>
<br/>        “对，他是我前夫的亲哥哥，我当然叫他大哥啊！”童筱颖淡定地笑了笑。
<br/>
<br/>        “前夫？你结过婚？那你现在跟傅总……”一时间，这信息量实在是太大了，落以柔表示自己有些难以消化。
<br/>
<br/>        童筱颖微笑着点了点头，却有些难以启齿地补充道：“二婚！”
<br/>
<br/>        “你……”落以柔欲言又止，简直不敢置信。
<br/>
<br/>        一个女人先嫁给弟弟，离婚后又嫁给前夫的哥哥，这不是心机婊是什么？
<br/>
<br/>        “你是为了报复你前夫，让你前夫管你叫‘大嫂’，膈应你前夫，所以你就利用傅总吗？你这么做，对傅总太不公平了！”落以柔咬牙切齿，义愤填膺。
<br/>
<br/>        报复前夫？利用大哥？
<br/>
<br/>        童筱颖眨巴着眼睛，看着落以柔一脸懵。
<br/>
<br/>        她从未想过还可以这样去为了报复一个人而利用另一个人！
<br/>
<br/>        “那个，你冷静一下，你可能对我有什么误解。但是，我现在不想跟你解释。我得走了，就此拜拜哈！”童筱颖咧着嘴，从床上下来，四下看了看，发现自己的手机和钱包就放在床头柜上，于是连忙拿起自己的东西，只身离开。
<br/>
<br/>        这次，她要回云城去住，住满两年，她就可以理直气壮地回来跟大哥离婚了。
<br/>
<br/>        落以柔看着童筱颖离去的背影，心里莫名其妙地感到很窝火。
