　　这就好像扔了一颗手榴弹出去，手榴弹炸不死对方，已经够恐怖的了，居然被对方抓住之后，就不爆炸了！
<br/>　　陈晓宇还是第一次经历这种事，一时间目瞪口呆，下一秒钟，一股强大额力量，居然将符印朝着自己推了过来，在身前点碎，符印所凝聚的灵力，在一瞬间爆发，将陈晓宇震飞出去，砰的一声摔倒在地上。
<br/>　　陈晓宇爬起来，目瞪口呆地望着碧清。
<br/>　　碧清没有追击，只是歪着头，邪邪地看着他。这副样子……在叶少阳看来，简直像极了苏沫。
<br/>　　“喂，师兄，你没事吧？你这没发挥实力啊，继续来怼她！”叶少阳过去把陈晓宇扶起来，面对碧清，恨恨地说道：“你可不要小看我师兄，他是没有准备好，有本事你让他好好准备一下！”
<br/>　　“滚你妈的！”陈晓宇捂着胸口大骂。
<br/>　　碧清却对他伸出一只手，简单地吐出两个字：“再来！”
<br/>　　陈晓宇快哭了，哀声吼道：“我不是他师兄！”
<br/>　　碧清却不管他，人影一闪，冲了过去。
<br/>　　陈晓宇的猴王想要援手，冲上去挡在前面。
<br/>　　不过碧清只是分神了一下，手指一抬，一道白气飞出，将猴王全身裹住，动弹不得。不过可能也是觉得它太弱的缘故，也没杀它，就这么困在空中，路线不变，继续朝陈晓宇冲来。
<br/>　　边上，卢晓清发动了突袭。
<br/>　　碧清左手一挥，又是一道白气飞出，挡在他面前，卢晓清双手结印，奋力一击，居然将白气堪碎，再一看，碧清已经到了陈晓宇面前，援手是来不及了。
<br/>　　“师兄，上啊，别怂！”叶少阳后退了几步，陈晓宇也知道跑不掉了，只好苦着脸迎敌，这一次用上了十成修为，在身前结了一道坚固的结界，这结界没有一点攻击性，完全是防守用的——他已经怕了。
<br/>　　他也不是傻比，刚才那一回合的交锋，已经让他认识到自己跟碧清的差距，要不是叶少阳玩阴的坑他，他根本就不想出这个头，眼看着碧清又来，以及怂了的他只好用尽全力，布置了这么一个结界，希望能挡住碧清这一击，赢过一点脸面。
<br/>　　碧清张开五指，重重地拍打在陈晓宇布置的结界上，然后，僵持了有五秒钟左右，陈晓宇以为自己顶住了，刚要松口气，碧清突然一笑，只听得咔嚓一声，一道裂缝从结界的中心出现，然后迅速开裂，随后一声巨响，结界崩塌，陈晓宇再次倒飞出去……
<br/>　　碧清另只手轻轻横扫，将猴王也摔在地上。
<br/>　　“师兄，师兄你没事吧！”叶少阳冲过去，十分关切地低头俯瞰陈晓宇。
<br/>　　陈晓宇被摔得浑身像散架了一样疼，在地上翻来覆去，想要开口骂叶少阳，但是疼的说不出话，在地上直哼哼，只好拿眼睛使劲瞪叶少阳，恨不得把他吞了。
<br/>　　“师兄，你为什么不出全力，我都跟你说了，她很厉害，绝对有资格做你的对手，你完全不用留情的！”
<br/>　　“滚！”陈晓宇喉咙里喷出一口血，也不知道是因为受伤还是被叶少阳给气的。
<br/>　　碧清本来已经想放过陈晓宇了，听见叶少阳这么说，又把目光移到陈晓宇身上，轻声道：“再来。”
<br/>　　陈晓宇要疯了，当场泪流满面，“我真的不是他师兄啊，我不是你对手……别总针对我行不行啊啊啊！”猛然转头看着大伙，“你们别都看着啊，一起上啊！”
<br/>　　妙心等人也冲了上去，不过不是为了帮陈晓宇，而是大敌当前，他们决定一起上，灭了眼前这个不明来历的强大邪物。
<br/>　　面对众人的围攻，碧清毫不在意，双臂张开，抬头望着头顶上空那两道龙气，喃喃道：“是时候了。”
<br/>　　当下轻喝一声，双手轻轻往回拉，两道龙气突然脱离了轨道，分别落在她两只手中，成为茫茫的两团。碧清摊开双手，氤氲迅速散开，迅速充斥了整个空间。大伙一下子如坠云中，视线模糊，除了茫茫白色，什么都看不见了，凭着感觉冲到之前妙心站立的位置，但是她人已经不在那里。
<br/>　　人呢？
<br/>　　大伙茫然四顾，心中十分紧张。
<br/>　　就在这时，头顶上方有光线闪烁起来，大伙抬头看去，是那之前悬浮在空中的五团灵光：金、绿、褐、白、红，不断旋转、扩大，落在众人身边，互相交织起来，形成了一个奇怪的图案。如果有人从高空俯瞰下去，现在的五色灵光组成的图案就好像一个五角星，而叶少阳一干人等，就在“五角星”最中间的方格里。
<br/>　　那些鬼魂失去了束缚之后，也都纷纷觉醒，数以百计的鬼魂，落在“五角星”中，纷纷朝众人扑过来。
<br/>　　“大家背靠背，互相照顾！”
<br/>　　在妙心的提醒下，大伙背靠背，一直朝外，各持法器，抵挡这些恶鬼的进攻。
<br/>　　“叶少阳，你个畜生！！”陈晓宇想要跟叶少阳拼命，被卢晓清拉着，塞到自己身后，“就你现在这德行，能打得过谁，你抓紧调息，我们先顶着！”
<br/>　　陈晓宇不甘心地恨恨瞪了叶少阳一眼，叶少阳一边应敌，冲他笑道：“师兄你赶快调息，一会人家说不定还要找你决斗，毕竟你是我们中间最强的一个！”
<br/>　　“你混蛋！”陈晓宇暴跳如雷，不过也有点担心碧清找到自己，也不敢再耽误，躲在卢晓清身后开始调息。
<br/>　　妙心狠狠瞪了叶少阳一眼，“闹够了没有！我们都要死在这里，你还有心情坑他！”
<br/>　　“未必就死吧。”叶少阳环视左右，说道：“这五道光是什么东西？”
<br/>　　“应该是显形后的地气水精……本来应该是我祖先用来封印她的地载阵，不过好像已经被她参透玄机，反而控制住了，现在这是……五行阵法吧？”
<br/>　　“是五行阵，不过五行阵只是一个大致说法，五行气息纵横交织，千变万化，不知道这具体是什么阵，我很纳闷，这女人为什么会懂这个。”
