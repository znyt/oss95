　　第431章
<br/>　　和珅的目光有些咄咄逼人起来，他一直是个有野心的人，之所以跟着程大雷，是因为觉得，程大雷能给他想要的东西。
<br/>　　当某一日，他明白程大雷不能给他想要的东西时，定然会拂袖离去。就算靠首领光环，怕也只能令他回首看上那么一两眼，却并不能挽留他离去的脚步。
<br/>　　只有志同，方能道合。但很显然，和珅与程大雷的志向不同。
<br/>　　程大雷沉默着，和珅也沉默着，黑色的碑林中，只有赫赫长风刮过，刘悲小口啜饮着酒。
<br/>　　最后是和珅率先打破沉默，他低下头：“大当家是否还没有考虑清楚，我们可以再等等。”
<br/>　　“什么叫没考虑清楚，我是担心贸然行动，陷入危险。”程大雷感觉到了和珅话里的质疑，冷哼一声道：“凡事小心为上，让军策府去研究，拿出一份具体周全的计划来，注意保密。”
<br/>　　“是！”和珅干脆的答应，告辞离开。
<br/>　　程大雷明白，和珅筹划这件事怕已不止一两天，从一开始，他的野心就不是一座蛤蟆城可以满足的。接下来，军策府会全速运转，几天后，就有一份具体周祥的计划摆在程大雷面前，等程大雷最后抉择。
<br/>　　可是，程大雷自己没有这样的野心呐。
<br/>　　他长叹一口气，卸去和珅带来的压力，整个人坐在那里，便得有气无力起来。
<br/>　　伪装成某种自己不是的人，还真是疲惫呐。
<br/>　　只是，当摆在某个位置上后，有些事便身不由己，所有力量都催促着你往前走。
<br/>　　和珅的行动速度很快，三天后，一份详细周密的计划就摆在程大雷面前。这是军策府所有谋士群策群力的结果，安排好了程大雷可能担忧的所有事情。
<br/>　　计划如此周密，定然是由和珅主导，某些残忍狠毒的谋略，定然是出自毒计军师吴用的手笔。
<br/>　　程大雷翻看这份计划时，和珅就坐在程大雷对面，眸子里隐隐有光芒闪动。
<br/>　　“大当家，你看这份计划怎样，有没有什么不周全的地方？”
<br/>　　和珅言外之意，是等待程大雷的夸赞。
<br/>　　“周全，周全极了，该考虑到的地方都考虑到了。”程大雷叹口气。
<br/>　　“如此的话，那我们就可以开始行动了。先由白元飞带人潜入凉州城，我们从城外发起总攻，这件事必须快，以快打慢，事情才能万无一失。”和珅如一柄摁在弓弦上的剑，已经快要按捺不住。
<br/>　　该来的终究是来了，和珅步步紧逼，程大雷必须认真考虑这件事。
<br/>　　拿下凉州城，进而拿下整个凉州，让凉州变为自己的领地。让自己这个琴川之主进一步，成为整个凉州之主。
<br/>　　这是最优的发展策略，如今的蛤蟆城已经有吞下整个凉州的实力。
<br/>　　程大雷之所以犹豫，还是有些嫌麻烦。一座蛤蟆城，自己是可以管理好的，但整个凉州，便会有更多麻烦的事情出现。
<br/>　　“大当家……”
<br/>　　程大雷恍然回过神，见柳芷出现在面前，他奇怪的看着对方，这种议事的场合，她本不该出现的。
<br/>　　“高飞虎和黄三元想要见大当家，让我来问问大当家，有没有时间见他们。”
<br/>　　“见，见，为什么不见，还通报什么，直接进来就是。”
<br/>　　程大雷如蒙大赦，赶紧向和珅递了个眼神，言外之意是：等见完了他们，再聊咱们的事。
<br/>　　黄三元和高飞虎并肩走进来，看到和珅也在，高飞虎道：“大当家，没有打扰你们吧，如果你们有正事，我们可以等等再聊。”
<br/>　　高飞虎和黄三元有些紧张，他们虽然曾经在青牛山与程大雷并肩作战，但相隔三年，已经不能以从前的眼光看程大雷。
<br/>　　现在的蛤蟆城要人有人，要粮有粮，程大雷已经是一城之主。高飞虎二人难免忐忑，身份不同了，规矩和排场也和以前不一样。所以，今天二人来见程大雷，都按照规矩来，生怕哪一步做得不对。
<br/>　　程大雷白了二人一眼，道：“不打扰不打扰，你们不来找我，我也要去找你们，山寨里安排得怎样。”
<br/>　　高飞虎一帮人现在已经安排妥当，老弱妇孺等不能参战的人，安排好了住处。除掉这些，高飞虎手下有四五千能打仗的弟兄，成立虎字军，由高飞虎指挥。
<br/>　　现在正在做的，是给他们建造房屋，分发衣物，另外最主要的事情，就是进行艰苦的训练。
<br/>　　这帮人毕竟是草莽之辈，难免有些不守军纪的地方，先用严酷的训练锤炼掉他们身上的浮气，免得日后有不好的事情发生。
<br/>　　双方说了会儿话，高飞虎和黄三元身上的紧张才渐渐散去，变得自然起来。
<br/>　　程大雷这才开口询问：“你们今天找我是什么事呐，是不是有什么安排不周到的地方？”
<br/>　　“安排都挺周到的，我们来，是因为来的路上听到一些消息，我们想了想，觉得应该向大当家汇报。”
<br/>　　这几天，高飞虎和黄三元也是忙碌不堪，有许多事情要处理。事情刚告一段落，他们就来向程大雷汇报，很显然这件事很重要。
<br/>　　“关于哪里的消息？”程大雷。
<br/>　　“是长安城的。”黄三元。
<br/>　　程大雷一直在收集关于长安城的情报，不过，他的消息来源，主要是靠江湖传言这个技能。但因为是江湖传言么，有些消息似是而非，具体情况程大雷也不知道。但高飞虎是从江南一路过来的，路上遇到许多人，肯定掌握着第一手消息。
<br/>　　“长安那边怎样了？”程大雷问。
<br/>　　“那个……是这样的。”黄三元琢磨着用词：“正义教打下长安城后，各路诸侯联手攻打正义教，最后正义教没有撑过去，带着败兵狼狈逃向幽州。”
<br/>　　“嗯，这件事我已经知道了。然后呢……”
<br/>　　“然后各路诸侯齐聚长安城，商量该由谁继承帝位。将军府支持李乐天，相府支持李行哉，江南李星支持……反正乱七八糟，有资格继承帝位的人，竟多达十几位。最后吵得不可开交，谁也说服不了谁。”
<br/>　　“然后呢？”
<br/>　　“然后他们就商量下来主意，谁杀死林少羽为明帝复仇，谁继承帝位。”
<br/>(本章完)
