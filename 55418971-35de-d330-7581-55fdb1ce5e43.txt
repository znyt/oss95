　　埋骨之地内的丧尸数以亿计，单纯的一路推进根本不现实，所以狼牙战团的想法便是引流，将一批批的丧尸引向那个圆圈，以与丧尸攻城战相反的战线，打出一场同样是防守与拉锯战的模式。
<br/>　　而那个将埋骨之地包围的巨大圆圈，就是第一战线！
<br/>　　这项计划很大胆，但却出奇的符合当下情况，虽然这个圆圈太大不可想象，狼牙战团的人都无法将之完全包围，只能占据一个个点，在由侦查队来回游走确认相互间的联系。
<br/>　　在战斗机数量有限的情况下，这一战也干脆放弃了空战，将所有战斗都设计在地面。
<br/>　　但埋骨之地的大量飞行丧尸又不得不重视，于是第一次的占领圆圈以及接下来的战场改造，才显得尤为重要。
<br/>　　从占据基础地理位置到人员全部到达，一共耗时两天一夜，紧接着马不停蹄的就开始了战场改造，由全体狼牙战团的十万人动手，深得狼牙建筑工程队的真传，整个十万大军已经全部都是新人类，力气不是普通的建筑工程队成员能比，一起动工的速度和效果尤其可怕，这条圆形战线近乎一刻一个样的在变化，不出半天时间这条圆形战线就被大改。
<br/>　　战线由圆圈内到外高度递减，呈现出一个缓和的坡度，同时地面下方的地道打的四通八达，连通了整个圆圈的所有据点，方便联络的同时更接应快速。
<br/>　　上方有着厚厚的土地加大着承受力，中空的地方也用了无数的树桩顶·住，哪怕战斗机集体十几架冲下来，也无法一次性将其冲开，表面上方更是铺满了掩饰物，看上去就像最为普通的空地，根本察觉不到丝毫。
<br/>　　而那些来自于科技部和上设计的新型武器，就被安置在各个定点中，隐藏在周围的大·片枯草里。
<br/>　　圆圈又再次被挖深形成迷宫般的走道，与圆圈内的埋骨之地地面衔接，这样哪怕丧尸群引不开集体涌来，也一时半会儿挤不进来，分散在各个点的小队就有足够的时间慢慢解决它们。
<br/>　　至于引流的最重要一环，自然就是射击队负责的丧尸引诱剂，如何扩散和控制也是尤其需要慎重对待的一项，必须将每一波引来的丧尸都分散到不同的区域才行。
<br/>　　当十万大军聚集在圆圈处后，三战队也正式归队，只是杀羽和黑芒的双队长不在，一下子都让众人相当困扰，只能交由副队长领队。
<br/>　　三战队的任务最重，也是最凶险的一环，他们需要负责引来丧尸群的超级丧尸和丧尸首领，确保智力丧尸第一时间解决或被缠住，免得影响战役进程。
<br/>　　第三天，除了还没回归的徐锋和江凌若，所有准备就绪。
<br/>　　何峰最后望了一眼升起的日出，在寒冬里发起了埋骨之战的正式进攻口令：“射击队准备，引流！”
<br/>　　刷刷刷！
<br/>　　已经扩张到两千人的射击队成员从各个方向窜出，奔向了指定的道路和地点，前方很长的一段道路里都是安全区，但他们要去的地方是埋骨之地中心，引丧尸群出动再打爆丧尸引诱剂分流。
<br/>　　没有侦查队的辅助，没有精英队作为后盾，更没有三战队在前方开路，射击队的独立战斗虽凶险，但狼牙战团任何一支队伍都足够成熟。
<br/>　　更何况，这是他们有史以来最重大的一场战役，更是一向颠覆世界的斗争！
<br/>　　所有人都准备好了，埋骨之地，要么埋葬整个丧尸种族的老巢，要么就将自己埋葬在这里！
<br/>　　——————
<br/>　　就在埋骨之战正式打响的时候，远在东方某个海洋孤岛上的楚涵，在恶言相向以死相逼等众多手段都用过之后，依然没有放弃针扎，这一天在沙滩上写下了三个巨大的字母：SOS。
<br/>　　紧接着，人消失不见无影无踪整整一周！
<br/>　　这一情况高少辉和旺财都没当回事，每日例行前来空投食物和水，旺财也没有感受到楚涵的任何状况。
<br/>　　可在一周之后，高少辉第一次将天照3000的高度降到最低，贴地飞行，这么一飞就发现那些空投的食物，竟然一份不少的放在原处，怎么投下来的现在就还是怎么样。
<br/>　　也就是说楚涵整整一周都粒米未沾滴水未进！
<br/>　　这情况一下子就将高少辉吓个半死，当场将天照三天停在空地，拎着旺财就走出机舱寻找楚涵。
<br/>　　“不吃不喝一周，哪怕楚涵等阶再高也扛不住，他到底想干嘛？”高少辉快速行走中抱怨道。
<br/>　　旺财还有点懵，再次确认了一下与楚涵的联系，道：“他还在岛上没错，我能感受的到，身体状况也并不是病危。”
<br/>　　“啥玩意儿？”高少辉满脸怒气：“你只能感受的到是不是病危？”
<br/>　　“只有危机状况才有警告啊！”旺财也火气直冒，“还不是你非不让他去阴阳谷。”
<br/>　　“他去了就是死！”高少辉脱口而出，吼道：“他要是十阶我绝对不拦着！”
<br/>　　两人争吵间，整个孤岛都空空荡荡没有一点声音，尤其是当这两人将整座小岛都逛遍也没发现楚涵身影时，终于彻底慌了……
<br/>　　这时候的楚涵，却一个人站在一个地道之中，深埋在岛屿之下。
<br/>　　这是他在孤岛上转悠时无意发现的，所以便留下了SOS的信号先一步前来，并不是故意要戏耍高少辉和旺财。
<br/>　　只是他已经在这通道等了三天也没见到高少辉和旺财赶来，当他想回地面上时却发现回去的路竟然被封死，两旁亮起了像是烛火一样的照明物，却又与普通的烛火有着本质的不同。
<br/>　　饥饿都还好，因为这湿漉漉的海岛底下通道里，长着很多不知名的植物，直接让他度过了一周。
<br/>　　于是索性，楚涵便一个人独自朝前摸索，走过了不知道多长的通道，绕了好几道弯一路向下，四天的路程貌似都已经深入到海底。
<br/>　　古怪和莫名的指引让他感觉很奇怪，也让他在这一路上心慢慢平静下来，将阴阳谷暂且放下，也将埋骨之地的那件大事暂且放下。
