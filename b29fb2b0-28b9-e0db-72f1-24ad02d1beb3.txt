　　老者冰冷的话语堪堪落下。
<br/>　　嗖…嗖…嗖…
<br/>　　远方原本疾驰而来的一道道身影，已瞬间而至。
<br/>　　那是一个个身披胄甲的兵士。
<br/>　　胄甲之上，六条张牙舞爪的狂龙图案，映衬其上。
<br/>　　这数十武者，清一色的圣皇境巅峰。
<br/>　　为首几人，竟达武道大能层次。
<br/>　　“六龙卫。”萧逸只看了一眼，便认出这群武者。
<br/>　　六龙卫，六龙地域域主亲卫。
<br/>　　人数不多，但个个皆是精锐。
<br/>　　传闻，历代六龙卫，皆是龙家强者所组成，拱卫域主，巡守整个六龙地域。
<br/>　　当然，萧逸也并不感到什么惊讶。
<br/>　　这些大城，或地域，其城主、域主，都会有自己培养的亲卫，也就是所谓的城卫队。
<br/>　　而六龙卫，便是其中精锐中的精锐。
<br/>　　“易霄。”这时，那老者冷眼盯着萧逸。
<br/>　　一身气息，完全锁定了萧逸周遭。
<br/>　　“还不速速散去火焰？”
<br/>　　萧逸摇了摇头，“我没空与你们废话。”
<br/>　　萧逸的目光，看向下方府邸，手中火焰，瞬间轰出。
<br/>　　“你…”老者脸色一怒，一个闪身，瞬间接下了火焰。
<br/>　　轰…
<br/>　　老者一掌拍出，掌中元力惊人，竟是瞬间轰散了火焰。
<br/>　　“易霄，你好大的胆子。”老者勃然大怒。
<br/>　　“胆敢无视我六龙地域域规？”
<br/>　　嗖…嗖…嗖…
<br/>　　周遭数十六龙卫，已不再是包围，而是封锁了萧逸周遭所有范围。
<br/>　　“我只是在击杀邪修。”萧逸看了眼老者。
<br/>　　“哪来的邪修？”老者皱眉质问道。
<br/>　　“下方。”萧逸淡漠道。
<br/>　　老者看了眼下方府邸，皱了皱眉，看向身旁一个六龙卫，“下方可是张家府邸？”
<br/>　　“禀长老。”身旁六龙卫点点头，“是。”
<br/>　　“嗯。”老者点点头，看向萧逸，“张家，乃是我六龙地域有数的大家族之一，何来邪修？”
<br/>　　“整个张家，上下皆是邪修。”萧逸淡漠地看了眼老者。
<br/>　　老者冷声质问道，“证据呢？”
<br/>　　“等我屠了张家，自会搜出证据。”萧逸回答道。
<br/>　　“那就是没有证据了。”老者冷声道，“空口白话，你说是便是？”
<br/>　　“是。”萧逸淡漠地点了点头。
<br/>　　“你…”老者脸庞一抽。
<br/>　　“我没兴趣与你们废话。”萧逸话音落下，身影一动。
<br/>　　然而，身影才刚动。
<br/>　　嗖…嗖…嗖…
<br/>　　一道道元力锁链，顷刻漫天而来。
<br/>　　锁链，玄妙蜿蜒。
<br/>　　几乎只是一瞬间，数十根元力锁链，便已封锁了萧逸周遭一米范围。
<br/>　　“嗯？”萧逸眉头一皱。
<br/>　　“好精妙的配合。”
<br/>　　这些锁链，并非真的是锁链，只是元力而成。
<br/>　　但在萧逸的感知中，这些锁链内蕴含的元力，异常浑厚澎湃，比真正的圣器锁链还要坚硬得多。
<br/>　　锁链，玄妙配合下，竟宛若大阵。
<br/>　　而且，数十六龙卫出手的速度极快，竟是一瞬间便布下了这元力锁链大阵。
<br/>　　不难看出，六龙卫确实训练有素。
<br/>　　他们单打独斗的实力，或许和一般的圣皇境巅峰差不多。
<br/>　　但配合起来，绝对是一加一大于二的效果。
<br/>　　萧逸暂时未动，只是看向老者。
<br/>　　“易某只问一次，六龙卫，还有你，可是要保邪修？”
<br/>　　“保邪修？”老者一愣，随后一惊，“胡说八道。”
<br/>　　“那为何拦易某？”萧逸的语气，异常冰冷。
<br/>　　“无凭无据，你说张家是邪修便是邪修？”老者的语气，同样强硬。
<br/>　　“易霄，老夫听过你的名头。”
<br/>　　“但你在其他地域大开杀戒，搅得各地域风声鹤唳，动荡不停，是你的事。”
<br/>　　“可在我们六龙地域，你还没资格如此。”
<br/>　　“六龙地域，有六龙地域的规矩；若你敢继续放肆，便休怪老夫不客气。”
<br/>　　“那便是谈不拢了。”萧逸摇了摇头，眼眸一冷。
<br/>　　他拖着一身重伤，没来得及养伤，便要立刻追击，便是要逼毒袍出来。
<br/>　　他没兴趣在这里浪费时间。
<br/>　　若逼不了毒袍出来，他便短时间内屠尽六龙地域各大邪修分部。
<br/>　　只要毒袍还在六龙地域内，他便能短时间内寻出。
<br/>　　“滚。”萧逸身躯一震。
<br/>　　漫天紫炎，倾泻而出。
<br/>　　他甚至不必祭出兽手兽脚。
<br/>　　仅仅是星幻手套覆于手中，五色圆环凝聚而出。
<br/>　　看似坚硬的元力锁链，还有这锁链大阵，瞬间在火焰之下燃成虚无。
<br/>　　他与毒袍交手，不使用世间强悍火焰；是因为他和毒袍本身的差距太多，火焰起不了功效罢了。
<br/>　　可这些六龙卫，区区一群圣皇境巅峰，想拦他？
<br/>　　笑话罢了。
<br/>　　轰…一声爆响。
<br/>　　数十六龙卫的封锁，瞬间溃散。
<br/>　　紫晶灵炎，随元力锁链蔓延，直击六龙卫而去。
<br/>　　一众六龙卫脸色一变，连忙撒手。
<br/>　　嘭…又是一声爆响。
<br/>　　萧逸出手，紫晶灵炎蔓延而出，也是他们想撒手就撒手的？
<br/>　　火焰爆发，数十六龙卫，顷刻炸得灰头土脸。
<br/>　　不过，未受伤。
<br/>　　萧逸暂时留手一分，以他的控火能力，足以做到想杀便杀，想伤便伤。
<br/>　　“放肆。”老者，亦瞬间出手。
<br/>　　萧逸瞥了一眼，左手，兽手瞬间凝聚。
<br/>　　仅仅一拳。
<br/>　　轰…
<br/>　　老者直接被轰退千米，一口腥血溢出。
<br/>　　“易某本职乃是击杀邪修，再敢阻碍，也休怪易某不客气。”
<br/>　　说罢，萧逸瞬间出手。
<br/>　　刚要落下张家。
<br/>　　轰…
<br/>　　空气中，一道爆响。
<br/>　　一个中年人，凭空而现。
<br/>　　中年人，一拳轰出。
<br/>　　“噗。”萧逸只来得及堪堪打出一拳抵挡，接过，却是被一拳反震，轰飞千米，且一口腥血喷出。
<br/>　　“龙啸。”萧逸稳下身影，直视面前中年人。
<br/>　　整个六龙地域，能一拳败他的，也就只有龙啸。
<br/>　　再者，他现今仍旧重伤在身，自是接不住龙啸一拳。
<br/>　　“易霄，你好大的威风。”龙啸一现身，便是面容冰冷。
<br/>　　“对我城主府的长老还有六龙卫不客气？”
<br/>　　“就凭你？”
<br/>　　嗖…龙啸的身影，瞬间消失在原地。
<br/>　　萧逸，只觉一花，随后腹部一疼，已是一口腥血喷出。
<br/>　　轰…
<br/>　　萧逸的身影，重重被轰下地面。
<br/>　　地面，被轰出一个巨坑，扬起漫天灰尘。
<br/>　　待得灰尘散落，萧逸缓缓站起，眉宇间，已是一阵煞白。
<br/>　　刚才龙啸的一拳，未伤他要害，但却轰断了他数根骨头。
<br/>　　“易霄。”龙啸，居高临下地俯视着萧逸。
<br/>　　“我不管你这等天骄妖孽，在别的地域有何等猖狂，又何等的肆无忌惮。”
<br/>　　“但在我六龙地域，你还不够格。”
<br/>　　“今日，念你是猎妖殿分殿主，我饶你一命，只是小惩大诫一番。”
<br/>　　“再有下次，无视我六龙地域规矩，那便不是断几根骨头那么简单了。”
<br/>　　“哼。”
<br/>　　说罢，龙啸冷哼一声。
<br/>　　冷哼声，直直传入萧逸耳中。
<br/>　　“噗。”萧逸体内顿觉一阵气血翻涌，又是一口腥血喷出。
<br/>　　“滚吧。”龙啸一掌扇出。
<br/>　　萧逸直接被轰飞万米。
