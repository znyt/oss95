　　听闻司空炎之言，苏莫无奈的摇了摇头，不过，只要对方没有太过火，他也不会说什么。
<br/>　　此时此刻，这小世界之中，人数众多，修为也是高低不等。
<br/>　　不说龙族之人，就是各种奴仆，修为也是高低不等。
<br/>　　有些奴仆的修为，非常的低，甚至不到真神之境。
<br/>　　但是，有些奴仆的修为，却是非常的可怕，仅从气息来判断，修为都有六七品之高。
<br/>　　当然，这些有六七品修为的奴仆，也都不再年轻，均是中年之态，显然均是活了上万年不止。
<br/>　　“什么修为的奴仆，都能来争夺宝物？”苏莫转头，向龙腾问道。
<br/>　　这些奴仆之中，这么多强者，修为低的人怕是没有什么机会。
<br/>　　“均是七品修为以下的奴仆，修为超过七品的奴仆，已经成为外族，不会来此争夺宝物！”龙腾摇了摇头道。
<br/>　　言罢，他又道：“不过，你放心，争夺宝物不会因为修为高而容易，具体情况族中会安排的！”
<br/>　　对于具体怎么发放宝物，龙腾也不是很清楚，因为据他了解，每次都不同，这样所有人都没有提前得知规则，就不会提前应对，相对而言就更公平。
<br/>　　“龙腾，最近你可是春风得意啊！”
<br/>　　一道轻笑声响起，只见一名身穿华服，面容俊朗的龙族青年，向龙腾走了过来。
<br/>　　此人正是上次在奴务府碰到的龙洌，此人上次的修为，和苏莫相同，现如今居然也达到了三品中级，以及和苏莫相当。
<br/>　　龙洌的身边，跟着两名女子，均是奴仆的身份。
<br/>　　其中一女，便是上次龙洌新收的玄冰族奴仆，身材娇小，皮肤白嫩，模样很是娇美。
<br/>　　此女修为不高，只有二品低级的修为。
<br/>　　不过，另外一女却是修为挺高，达到了三品高级。
<br/>　　此女，是一名人族女子，穿着一声黑衣，身材高挑，面容同样娇美，但是面容寒霜，非常的冷艳。
<br/>　　“龙洌，你拥有两名娇美的奴仆！我哪有你春风得意啊！”龙腾面带嗤笑之色。
<br/>　　对于这个龙洌，龙腾是避而远之，此人是龙族的一位顶级妖孽，也是神域的万古天骄之一。
<br/>　　虽然此人的本体，不是五爪金龙，是稍次一些的紫血天龙，但是对方是祖龙的后裔，拥有祖龙血脉，天赋极高。
<br/>　　当然，祖龙血脉并不是一种血脉之力，而是身上流淌着稀薄的祖龙之血。
<br/>　　当然，作为万古天骄，龙腾对此人本应该很重视，但他心中却是非常的不屑。
<br/>　　因为这个龙洌喜好女色，喜爱收姿色娇美的女.奴，这些女.奴明面上是奴仆，但实际上均是玩物而已。
<br/>　　据说这些年，龙洌收过一百多个奴仆，有些已经死了，有些赠送给了别人，如今只剩下了这两个奴仆。
<br/>　　不过，龙洌虽然喜欢收女奴仆，但也不会只看姿色，而是也要看天赋。
<br/>　　据说，龙洌收的奴仆，每一个都天赋不凡，有的甚至堪比妖孽天才。
<br/>　　“哈哈！”
<br/>　　听闻龙腾之言，龙洌朗声大笑，继续道：“奴仆而已，当然是收看着顺眼的！”
<br/>　　言罢，龙洌大量了一番司空炎和苏莫，道：“你这两个奴仆，天赋不简单啊，我没记错的话，他们比上次在奴务府之时，修为都提升了！”
<br/>　　“当然，没有一点本事，我龙腾也不会收到麾下！”龙腾面色淡淡的说道。
<br/>　　“我这两位奴仆冰儿和凝雪，天赋不凡，根本不是你这两个奴仆可比！”龙洌指了指身边娇小玄冰族女子和黑衣女子，淡然一笑说道。
<br/>　　“那就拭目以待吧！”龙腾沉声说道。
<br/>　　“呵呵！”
<br/>　　龙洌轻笑一声，随即看了看冰儿和凝雪，道：“冰儿、凝雪，等会若是有机会，领教一番龙腾的这两位小奴。”
<br/>　　“是，主人！”冰儿和凝雪两人，均是立刻应声。
<br/>　　“哈哈！”
<br/>　　随即，龙洌轻笑一声，便带着两女，转身离开，走向别处。
<br/>　　“这个龙洌，真是嚣张！”司空炎面色阴沉，低声说道。
<br/>　　“龙洌倒还好，龙族有近二十位万古天骄，其中有几人轻易不能得罪！”龙腾摇了摇头说道。
<br/>　　苏莫和司空炎闻言，沉默不语，他们自然不想得罪龙族的万古天骄。
<br/>　　但若是别人想揉捏他们，他们也不会无动于衷。
<br/>　　“走，我们去那边！”
<br/>　　少倾，龙腾示意，然后带走苏莫和司空炎，向不远处走去。
<br/>　　那里，聚集了几位龙族之人，以及几位年轻的奴仆。
<br/>　　这群龙族之人共有四人，全是龙族的妖孽天才，龙洌亦是在其中。
<br/>　　另外三人，居然全部是五爪金龙，让得苏莫微微一惊。
<br/>　　这三人，是两男一女，个个气息强大，气质不凡，头生纯金色龙角。
<br/>　　两名男子一高一矮，高个男子面容刚毅冷峻，修为是三品高级，矮个男子脸庞圆润，修为二品低级。
<br/>　　而那名龙族女子，修为最高，达到了四品低级。
<br/>　　龙族之中，虽然有不同的龙种，但是，头上的龙角基本上就能看出属于什么品种。
<br/>　　每一种龙的龙角都不同，比如五爪金龙，头生金色龙角、紫血天龙头生紫色龙角，独角银龙头生银色独角。
<br/>　　不过，这也不绝对，有的两种不同的龙族，龙角相差不大。
<br/>　　比如，黑暗魔龙和普通的黑龙，同样是黑色的龙角，但是，两种龙族的龙角，也有细微的差别，对龙族熟悉之人，能轻易的分辨。
<br/>　　这三位龙族，苏莫一眼就能分辨，全部是五爪金龙，龙族之人顶级的存在。
<br/>　　“龙腾，好久不见！”
<br/>　　“龙腾，没想到你也有空来看奴仆夺宝！”
<br/>　　见龙腾走来，高个龙族青年和女子，纷纷开口招呼，目光打量着龙腾，以及龙腾身后的苏莫和司空炎。
<br/>　　“哼！龙腾，别来无恙！”
<br/>　　矮个青年却是冷哼一声，面色颇为冷淡，他前段时间，别龙腾击败，很没有面子，自然不会对龙腾客气。
<br/>　　“诸位，你们不是同样有空！”龙腾笑了笑，并未在意矮个龙族青年的态度。
<br/>　　“闲来无事，我这个奴仆是我当初千挑万选的，被我调教的不错，今日我来看看他的表现！”高个龙族青年笑着说道，他的身后只有一位奴仆。
<br/>　　这位奴仆面色古板，身材极为高大，约有一丈左右，全身肌肉虬扎，皮肤犹如石质。
