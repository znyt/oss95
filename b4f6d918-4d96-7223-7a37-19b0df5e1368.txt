她，陆寒霆的未婚妻，是怎样一个人？
<br/>
<br/>        前段时间在办公室里陆司爵曾经问她，想不想问一问这个未婚妻，那时她可以潇洒的转身离开，可是现在她终于忍不住的问。
<br/>
<br/>        她想问一问取代她位置的这个女孩，将要拥有她的陆先生的这个女孩，究竟是怎样一个人。
<br/>
<br/>        陆司爵似乎早已经料到她会问出来，“她是首富家的千金，漂亮，优雅，有学识，有涵养，跟阿霆很般配。”
<br/>
<br/>        “那…她喜欢陆先生么？”这是夏夕绾最关心的问题。
<br/>
<br/>        陆司爵点头，“喜欢，很喜欢，阿霆离开帝都城这些年，她一直在等着阿霆回来，不管阿霆是怎样一个人，依然执着的爱着他。”
<br/>
<br/>        夏夕绾觉得这是她想要的答案了，这个女孩儿很好很好，她应该开心的，可是她勾起唇角时，还是有大颗大颗的泪珠砸落了下来。
<br/>
<br/>        一下子，她又笑又哭的像个傻子。
<br/>
<br/>        陆司爵上了车，几辆吉普车疾驰而去，带着陆寒霆慢慢的消失在了视线里。
<br/>
<br/>        夏夕绾就站在原地，看着吉普车离开，远去，她伸出小手捂住自己的脸，滚烫的泪珠在指尖穿梭。
<br/>
<br/>        她缓缓蹲下身，纤柔的身体抱成一团，哭的整个人不能自已，陆先生，再见，再也不见。
<br/>
<br/>        这么一刻的夏夕绾就像是一个被遗弃的孩子。
<br/>
<br/>        ……
<br/>
<br/>        等了一会儿，赵婶伸手，去扶夏夕绾，满脸的心疼，“绾绾小姐，我们回屋去吧，外面凉。”
<br/>
<br/>        手刚碰到夏夕绾时，夏夕绾直接倒在了赵婶的怀里。
<br/>
<br/>        赵婶大惊失色，“绾绾小姐！绾绾小姐，你怎么了？”
<br/>
<br/>        夏夕绾是医者，她知道自己不行了，现在全身的毒素已经侵入了心脉，她一只脚跨入鬼门关了。
<br/>
<br/>        可是，她并不想死。
<br/>
<br/>        夏夕绾回顾了自己短暂的一生，其实她还有很多事情没有做，首先她就不是一个好女儿，她愧对妈咪。
<br/>
<br/>        夏夕绾突然想起了李文清院长那边的宝箱，是妈咪留下来的那个宝箱。
<br/>
<br/>        “赵婶，我要去枢密研究院，现在就送我去找李文清院长，快！”夏夕绾紧紧的抓住了赵婶的手。
<br/>
<br/>        赵婶点头，“好，绾绾小姐，我们现在就去。”
<br/>
<br/>        ……
<br/>
<br/>        一辆黑色保姆车疾驰在了路上，一路往枢密研究院的方向开去。
<br/>
<br/>        赵婶驾车的，夏夕绾坐在后座上，身边是几个照顾她饮食起居的婶儿们。
<br/>
<br/>        “老赵，后面有车跟着我们！”
<br/>
<br/>        赵婶透过后视镜看了一眼，后面的确有一辆车在跟着。
<br/>
<br/>        赵婶轻蔑的勾了一下唇，“护好小姐，我甩开他们！”
<br/>
<br/>        说完，赵婶踩下油门加速，变道，黑色保姆车灵活的滑进车流，在大岔路口那里消失不见。
<br/>
<br/>        红灯已经亮了，后面的车紧急刹车，尖锐的刹车声无比的刺耳，驾驶座上的黑衣人迅速拿出手机打电话，“喂，先生，跟丢了，夏小姐身边的那个婶子相当警觉，身手不凡，她们绝对不是凡人。”
<br/>
<br/>        很快，那端传来了陆司爵低沉的嗓音，“村子那里呢？”
<br/>
<br/>        “黑色保姆车开出来的时候，他们全部撤离了，现在村子里面空无一人。”
