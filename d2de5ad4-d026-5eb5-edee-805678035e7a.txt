　　两日的时间转瞬即过，出发的终于日子到了。
<br/>　　这一日，大清早，苏莫便离开了火元峰，来到了神武学府的大门外。
<br/>　　此刻，在学府的大门前，已经聚集了不下数百人，熙熙攘攘，人声鼎沸。
<br/>　　“苏兄，这里！”人群中的谢彬看到苏莫，顿时摆手呼喊了一声。
<br/>　　苏莫转头看去，发现谢彬和数十名皇祖圣朝派系的人聚集在一起，他便抬脚走了过去。
<br/>　　苏莫的到来，引起了所有人的关注，实在是他最近几个月名声太过响亮了。
<br/>　　真圣榜第一的名头，这可不是开玩笑！
<br/>　　各大派系的众人，望着苏莫的目光，也是各不相同。
<br/>　　有羡慕、有不屑、有不服、还有敌视。
<br/>　　羡慕自然是羡慕苏莫的天赋，以及真圣榜上的排名。
<br/>　　不屑则是不屑苏莫的作风，上次外界的人前来挑战，都辱骂上门了也不敢冒头，实在是让人不耻。
<br/>　　不服气的人，则是不服苏莫能排在真圣榜第一，
<br/>　　就算苏莫天赋妖孽，但修为并不高，排在真圣榜第一实在是太夸张了。
<br/>　　而且，所谓的四生战魂，两种特殊体质，不过都是推测而已，根本没有人真正的见过。
<br/>　　所以，有很多人不服苏莫的排名。
<br/>　　但是，这一看之下，发现苏莫居然突破到了武帝境六重境界，众人还是暗暗心惊，这种修炼的速度真是惊人。
<br/>　　那些敌视苏莫的人，自然便是太史世家派系的人了，苏莫杀了如此多太史世家派系的天才，这个仇不可谓不深。
<br/>　　“各位，早！”苏莫来到皇祖圣朝派系的人聚集之处，向众人微笑抱拳。
<br/>　　皇祖圣朝派系的人不多，只有三十余人，谢彬算是修为最低的人了。
<br/>　　不过，此刻的谢彬已经不是武帝境四重修为，而是突破到了武帝境五重的修为。
<br/>　　“苏兄早！”众人均是向苏莫回礼。
<br/>　　“苏兄，我观你达到了武帝境六重，这次你有没有把握在真圣碑留名？”谢彬眸中闪亮好奇之色，这是他比较关心的问题。
<br/>　　至于苏莫与巫族之间的事，其它人根本不知道，所以谢彬只是以为苏莫去真圣界，是要的真圣之气，并且在真圣碑留名。
<br/>　　“我又不清楚真圣界的具体情况，哪里来的什么把握！”苏莫摇头说道，他虽然自信不必任何人弱，但能不能在真圣碑留名，他还真的是不清楚。
<br/>　　“呃~~！”谢彬无语，你可是真圣榜第一，就算没有把握，也不能说的这么直接吧！
<br/>　　“你呢？”苏莫笑问道。
<br/>　　“我啊！”
<br/>　　谢彬闻言一笑，道：“我就是去走个过场，见见世面！”
<br/>　　谢彬可不认为他能在真圣榜上留名，但这种观看整个古灵星河天才争锋的机会，他可不愿意错过。
<br/>　　两人闲聊之间，学府门外的人越聚越多，任云都作为武帝境九重的天才，自然也来了。
<br/>　　“苏兄！”任云都来到苏莫的面前，向苏莫微微抱拳。
<br/>　　“任兄！”苏莫回礼。
<br/>　　任云都转头看了看四方，随即压低声音道：“苏兄，据我所知，这一次真圣界之行，有很多人在等着看你的笑话呢！”
<br/>　　“是吗？”
<br/>　　苏莫闻言轻笑一声，无奈的道：“他们想看尽管看便是！”
<br/>　　他对此毫不在意，这些想看他笑话的人，无非就是不服他被排在真圣榜第一罢了。
<br/>　　“苏兄，话不能这么说，这些人不服你被排在真圣榜第一，想要看你的笑话，你就应该用结果狠狠打他们的脸，让他们无话可说。”任云都说道。
<br/>　　就在此时，一道略带不屑的声音响了起来。
<br/>　　“呵呵！那也要他有这个实力才行啊！”
<br/>　　闻听此言，苏莫和任云都不约而同的循声望去，便看到尤天韩带着一帮太史世家派系的人走了过来。
<br/>　　苏莫立刻眼眸一眯，眸中冷光闪烁。
<br/>　　踏！踏！踏！
<br/>　　尤天韩直接走到了苏莫的面前，阴冷的目光与苏莫对视在一起，敌意不言而喻。
<br/>　　这一刻，整个学府门外都安静了下来，所有人都看向尤天韩和苏莫，现场的气氛显得剑拔弩张。
<br/>　　“苏莫，等到了真圣界，你就会知道你这个真圣榜第一不过是个笑话！”尤天韩嘴角噙着一丝笑意，淡淡的说道。
<br/>　　“我是不是笑话，你没有资格评判吧？”苏莫冷冷的说道。
<br/>　　“呵呵，我的确是没有资格评判！”
<br/>　　尤天韩微笑点头，继续道：“但事实会证明一切，到时候你就会知道，你与我们这些荣登真圣榜的人，差距是如何之大！”
<br/>　　尤天韩眸中带着戏虐之色，什么四生战魂？什么两种特殊体质？别说尚不能确定，就算真是这样又如何？
<br/>　　苏莫的修为他也看在眼里了，武帝境六重，修炼速度的确是惊人，但这个修为还是太低了。
<br/>　　这一次真圣界之行，巫人王绝对不会让对方活命，他也不会让对方活命。
<br/>　　“是吗？”
<br/>　　苏莫轻笑一声，不置可否的耸了耸肩，道：“那好，到时候我就看看，你尤天韩有什么值得自傲的地方，我和你的差距有多大！”
<br/>　　“呵呵，你会看清的！”尤天韩自信一笑，随即转身离开。
<br/>　　很多弟子听闻尤天韩和苏莫的对话，均是面露微笑，这一次真圣界之行，的确是苏莫证明自己的机会。
<br/>　　若是苏莫表现的太差，不仅是有辱真圣榜第一的名头，更是会让神武学府丢进了脸面。
<br/>　　若是苏莫真的表现极佳，顺利在真圣碑留名，那真圣榜第一的名头，才能真正坐的安稳。
<br/>　　不过，以苏莫目前的修为来说，这个可能性太低了。
<br/>　　“太嚣张了！”谢彬咬牙切齿低声的说道。
<br/>　　“苏兄，你要努力在真圣碑留名，用事实狠狠的打他的脸！”任云都说道。
<br/>　　“嗯！”苏莫轻轻的点了点头，但他完全没有当回事，尤天韩虽强，还不被他放在眼里，他的威胁是巫族主脉的天才。
<br/>　　真圣榜的榜单上，姓巫的人可是有好几位，排名最高的一位叫巫殇，名列真圣榜第八位。
<br/>　　苏莫心中清楚，大祭司既然想要在真圣界解决自己，肯定不会指望天冥星支脉的那些天才，定然是会让主脉的天才出手。
<br/>　　总之，这一次真圣界之行，他不仅要在真圣碑留名，还要解决巫族的杀机，度过这次危局。
