　　自从来到这个世界，他一直压抑，举目无亲，孤立无援，连法力也只剩下一成……这种孤独又委屈的感觉，在叶少阳心中压抑了太久，今天总算是见到了萧逸云，并且亲人相认，又被他这么问，叶少阳真的是有点忍不住，在他面前爆发了情绪。
<br/>　　叶少阳跟萧逸云之前对话声音很低，老王和老陈两人虽然一直在密切注意这边，但并不知道他们在说什么，此刻见到叶少阳歇斯底里的样子，都吓了一跳，内心更加懵逼了。
<br/>　　萧逸云对叶少阳的表现也是有点吃惊，他虽然有些动摇，但是一时还是有点不能接受叶少阳是从未来穿越过来的，毕竟，这件事实在有点挑战他的认知。
<br/>　　他现在对叶少阳的态度，最多也就是半信半疑，因此对叶少阳歇斯底里的状况，表现得很平静，淡淡说道：“假如你真是从未来穿越二来，那么，你来阴司做什么？”
<br/>　　“找你！告诉你这一切！”叶少阳不假思索地回答。
<br/>　　萧逸云怔了一下，道：“就算我相信了你，之后呢，我能做什么，或者说，你让我做什么？”
<br/>　　叶少阳道：“我不知道。我只是希望，你能相信我，跟我一起想办法，让我能够回到自己的世界里去，我不想在这呆着，我回去还有非常非常重要的事情要办。”
<br/>　　萧逸云迟疑了一下，回答道：“我只是天子殿一个押司，我想不到什么办法。”
<br/>　　“想不到，就一起想！”
<br/>　　“为什么要帮你？”
<br/>　　“因为，你是我兄弟！”叶少阳看着，态度十分严肃，也十分诚恳，“一百年后的你知道，我这人从来不喜欢求人，但是这次我真的真的是没有办法了，我在这里举目无亲，一个熟人都没有，如果不是躲在千斤坠里，我连阴司都到不了，我需要你的帮助！”
<br/>　　萧逸云看着他，表情也变得凝重起来。
<br/>　　叶少阳回望着他，用力点了点头。
<br/>　　萧逸云迟疑了半晌，说道：“我不知道要怎么做。”
<br/>　　叶少阳听他这么说，明白他对自己说的那些，已经相信了几分，说道：“我想见见崔府君，他是阴天子，对这种事情，或许会有办法。”
<br/>　　萧逸云一听，犹豫起来，随后说道：“那你在这等着吧，我亲自去找府君大人，看他怎么说。”
<br/>　　走了两步，又想起什么，转过身来，望着叶少阳说道：“你之前说，我在一百年后，跟你是兄弟？”
<br/>　　叶少阳点点头，道：“能为对方慷慨赴死的那种兄弟。”
<br/>　　萧逸云心中一动，看了他一会，转身进了天子殿。
<br/>　　叶少阳就在外面惴惴不安地等着，心里思索着待会见到崔府君之后应该怎么说，想到这个世界的崔府君，跟自己那个世界的不是同一人，这种感觉很是奇怪。
<br/>　　叶少阳正在胡思乱想着这些东西，突然平地里生出一股阴风，从天子殿的大门里刮出来，沿途经过那几个功曹鬼差的身边，没造成任何影响，反而直奔叶少阳而来。
<br/>　　叶少**本猝不及防，立刻被这股阴风卷起来，朝着城外方向飞去。叶少阳在空中想要调集法力抵抗，才发现飓风的强度，别说自己只有以往十分之一的法力，就算是全盛时期，也绝对抵抗不了这股强烈的阴风。
<br/>　　“你不在六道之中，生死簿上无名，非我阴司所辖，亦无权入我阴司！去你来处，勿要再擅长阴司！”
<br/>　　一个充满威严的声音，在叶少阳耳边响彻，震惊之余，他立刻认出这是崔府君的声音，这时候他人已经飘到了酆都城外，前方形成了一个虚空裂缝，他被阴风直接吹了进去，短暂的穿梭之后，他回到人间，魂魄回到自己的躯体内。
<br/>　　叶少阳睁开眼睛，自己还在赵四家后面那块高粱地里，强烈的阳光刺的他睁不开眼，崔府君的话还萦绕在他耳边。叶少阳心中一片惨然。
<br/>　　阴司，天子殿。
<br/>　　崔府君站在后花园里，在一片树丛中望着南方的天空。萧逸云站在他身边，也跟他一样望着那个方向。
<br/>　　叶少阳刚从那个方向穿过虚空，被阴风送回到人间。
<br/>　　崔府君道：“你稍后通知黄泉大道上的守卫，让他们加强防范，切不可再让这个人进来。”
<br/>　　萧逸云木讷地点点头。
<br/>　　崔府君没有回头，淡淡说道：“你想知道什么？”
<br/>　　萧逸云犹豫了一下，说道：“大人，这个人说的，有没有可能是真的？”
<br/>　　“从一百年后穿越时空而来？”崔府君转过身，看着他，说道，“这样的事情，从来就没有发生过。”
<br/>　　“可是，他说的那些事情……”
<br/>　　“读心之术，古来有之，你以为有些事你没有说出去，别人就不会知道？”
<br/>　　萧逸云沉吟着不说话。
<br/>　　崔府君道：“他非我三界六道之内的生灵，不管他从哪里来，他与我等之间，绝不可产生任何关联，不然可能会造成无法预料的后果。”
<br/>　　萧逸云怔了一下道：“什么无法预料的后果？”
<br/>　　“既然是无法预料，我又怎知。”崔府君说完，停顿了一下，接着说道，“三界六道之外的事情，连我也是无法推演。”
<br/>　　萧逸云内心一片困惑，没有作声。
<br/>　　两人沉默了片刻，崔府君说道：“你去办差去吧，切记，不可去人间找他。”
<br/>　　萧逸云道：“弟子遵命。”
<br/>　　躬身行了一礼，转身走出了天子殿。
<br/>　　从天子殿出来，他的目光仍然望着叶少阳消失的方向，脑海中叶少阳的影子挥之不去，怔怔地想：难道，这个人真的是从一百年后来的？难道他真的是自己将来的……兄弟？
<br/>　　“能为对方慷慨赴死的兄弟。”叶少阳的这句话，在他脑海中萦绕不去。
<br/>　　在高粱地里收拾好东西，叶少阳在太阳下静坐了好久，把事情整个想了一遍。
<br/>　　崔府君既然用一道阴风把自己送到人间，显然他一定是听了萧逸云的讲述，然后决定不帮自己，也不见自己，所以才会用这么干脆的方式将自己扔走。
