　　　　在温城市中心最豪华的地段，有一座气派不凡的巨大别墅。
<br/>　　别墅前处刻着两个精雕玉琢的大字“白宅”，这便是本市最大名鼎鼎的隐藏家族，掌权者跺一跺脚，都能让这片土地颤抖几下。
<br/>　　所谓的社会，看似只有一个，但在特殊人的眼中，其实就有两个社会，一个现代社会，一个江湖社会，你也可以称之为修武界。
<br/>　　在修武界，人类可以做到飞檐走壁，开山断石，拥有普通人无法相信的力量。
<br/>　　白家，就是汇聚了大量觉醒修武者的一个大家族，屹立不倒，里面子弟众多，都拥有一些神奇的能力。
<br/>　　此时，这栋大别墅的内部，藏着一处古老而庄严的宅院，有一伙子聚集在一户外院的门前，议论纷纷。
<br/>　　只见一位身穿制服的丰满美女走了过来。
<br/>　　“二小姐”
<br/>　　来人不是别人，正是从温城大学出来的白楚艳。
<br/>　　“少主在里面正在发火，我等都不敢进去了”
<br/>　　白楚艳微微的蹙眉：“发生什么事情了，能惹我小弟生气”
<br/>　　“周秦半个月前死在了白镇，少主派人去查了很久，总算查出来了”
<br/>　　“就是那个可以超控蚊虫的觉醒者？”白楚艳想起来了，轻蔑的摇了摇头：“死了就死了，不过是一个外姓，小弟何须那么在意”
<br/>　　一位手下开口：“听说周秦帮少主办事，出去前，少主将自己心爱的勾魂笛借给了周秦，现在周秦活不见人死不见尸，少主的勾魂笛也就没了”
<br/>　　“什么？”白楚艳大吃一惊：“小弟竟然将勾魂笛借给外人，简直胡闹”
<br/>　　那可是一件杀伤力极大的灵器，运用的好，可以杀人于无形，笛声拂过，草木皆兵。
<br/>　　白楚艳没有心情跟这些人多说什么，直接迈开修长的腿，朝着里面进去。
<br/>　　别看白家外面是一栋豪华的巨大别墅，但里面却藏着古色古香的庄园，按“左青龙、右白虎、前朱雀、后玄武”的地理特征，形成了这方“旺气之所在”的宝地。
<br/>　　庄园之内，只见一位身材伟岸，肤色古铜，五官轮廓分明而深邃的年轻男子负手而立，他幽暗深邃的冰眸子，显得狂野不拘，冷淡无波的看着前面一位单膝下跪的属下回报。
<br/>　　“少主，属下无能，我们明明得到了蛇王石，既然半路杀出了一只修炼有成的猫精，蛇王石被那该死的猫精一口吞了下去，然后就一溜烟的跑掉了”
<br/>　　古铜色男子浑身散发这一种骇人的气息，别看年纪不大，但却做到了不怒自威，他便是隐藏世家白家的未来继承人，白君然。
<br/>　　天生的修武天才，天赋异禀，三岁便身体觉醒，年方二十，已经快要突破三阶生灵的地步。
<br/>　　白君然听着属下的汇报，顿时不轻不重的冷哼一声，那单膝下跪的属下脸色骤变，诚惶诚恐起来：“属下该死，把自己办砸了”
<br/>　　“你是该死，区区一块石头也能弄丢”白君然脸上噙着一抹邪恶：“一只猫精也敢抢本少的东西，看来是活腻了”
<br/>　　那名属下道：“少主，猫精吞了蛇王石，它隐藏不了的，在给属下一次机会，属下定然将猫精给找出来”
<br/>　　“你自己下去领家法吧，人只有一条命，机会也只有一次，你的机会已经没了”
<br/>　　此言一出，那名属下惊恐不已，机会没了的，等于命也没了。
<br/>　　白家屹立不倒，家法森严，领家法等于去死。
<br/>　　“少主，求你在给属下一次机会吧，属下一定将猫精给少主找回来”
<br/>　　白君然幽暗深邃的冰眸子淡淡的瞥了他一眼：“消失吧”
<br/>　　那名属下浑身一颤栗，不敢在多言，只能乖乖的退下。
<br/>　　“猫精？有意思，别被本少逮到，否则本少会亲自给它解刨，取出蛇王石”
<br/>　　白君然也是流年不利，他小小年龄，天赋异禀，已经开始做突破三阶生灵的准备。
<br/>　　一般的生物，突破成三阶生灵，那犹如山鸡变凤凰，会有一番脱胎换骨的变化，生命力大幅度的旺盛，寿命暴涨，可以这样说吧，三阶的生灵活个几百年，不成问题。
<br/>　　但是二阶生物，就算你在强，你顶多活个百年，那么就必死无疑。
<br/>　　唯有突破三阶，成为生灵，才是王道。
<br/>　　三阶生灵，凌驾普通生物之上，可谓脱去凡胎，但是肉眼凡胎想成为三阶生灵是非常的苛刻，动物成精，突破三阶，要承受天雷天火的洗礼，而人类同样如此。
<br/>　　然而，天雷天火只是最后一步，前提是你要先拥有这个资格。
<br/>　　人类修武者修炼，需要各种宝物的辅助，这次白君然野心很大，他要一步到位，为了突破三阶生灵，他做了很多的准备。
<br/>　　有三样东西可以助他双脚直接迈三阶，第一种，是需要一头三阶生灵的内在金丹（动物类），但凡三阶生灵，身体之中都会孕育出一颗金丹出来，乃是生命精华所在。
<br/>　　三阶生灵，无比强大，也只有白家这种底蕴，才能得到一颗三阶生灵的金丹。
<br/>　　另外两样到是简单一点，他派一位蚊虫掌控者，用变异蚊子的手段，为他准备一千颗血精，人类血之精华。
<br/>　　第三样，就是蛇王石，相传是一条曾经正道蛇王之宝物，可以帮人修为大涨。
<br/>　　这三样，按照白君然的计划，此时都应该在他自己手中了。
<br/>　　可计划赶不上变化，他派出去的蚊虫掌控者意外的死在外面，活不见人，死不见尸，最可气的是，还将他借与的勾魂笛给弄丢了。
<br/>　　勾魂笛可是宝贝，白君然想想就心疼。
<br/>　　还有就是蛇王石，明明派人千辛万苦的弄到手，半路却杀出一条猫精，这都什么跟什么。
<br/>　　莫非老天都在阻他成为三阶生灵？
<br/>　　白君然幽暗深邃的冰眸子光芒大亮，一副我命由我不由天的霸气散发出来。
<br/>　　猫精，你灵智大开，却如此的蠢，跟我白君然作对，你以为吞了蛇王石藏起来，我就找不到你？
<br/>　　吞了蛇王石，你所到之处，将群蛇涌动，那里出现大数量的蛇，你就必然暴露。
<br/>　　白君然暗暗思忖，并不担心找不回蛇王石。
