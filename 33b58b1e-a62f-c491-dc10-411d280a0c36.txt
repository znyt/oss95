　　第256章
<br/>　　夜来，程大雷在一片小湖旁边歇脚。二人走了这一路，皆是风尘扑扑，程大雷倒还是耐得住，不过李婉儿从小养尊处优可有些受不了。
<br/>　　程大雷先洗了洗身子，又在岸边换好衣服，过程中李婉儿一直躲在马车里。
<br/>　　“好了，你可以出来了。”程大雷换好衣服后喊。
<br/>　　“你该不会偷看吧。”李婉儿抱着衣服，忐忑不安道。
<br/>　　“什么话，我是那样的人么，我在长安青楼都不逛。”
<br/>　　“不行，你也进马车里，我看着马车，你一露脑袋我就能看见你。”
<br/>　　“有这个必要么，我觉得我们已经到了互相信任，坦诚相见的地步。”
<br/>　　程大雷说着话，还是上了车，他没有忘记自己的剑，一直在手中拎着。
<br/>　　李婉儿看了看左右，芳草萋萋，天蓝如洗，倒是一副好景色。她将靴袜褪下，将玉足点进水里，湖水真凉，她结结实实打了个寒颤。
<br/>　　不过如此却带来些清凉，正好洗去连日来的疲惫。
<br/>　　她一点一点褪掉衣服，身子浸入水中，警惕的盯着马车，没发现程大雷的动静，方才略微放心。
<br/>　　马车里，程大雷用剑挑开一道缝隙，撇撇嘴，心里想着：她就这么相信马车的气密性么？完全看得见好不好。
<br/>　　“程大雷……”
<br/>　　外面忽然响起李婉儿的声音，程大雷还以为被发现了，做贼心虚的他问道：“怎么了，需要我给你递浴巾么？”
<br/>　　“没，没什么，这里太空了，要是没个人说话，感觉就剩下自己似的。”
<br/>　　原来没被发现，程大雷又把剑撩起来。
<br/>　　“聊什么呀？”
<br/>　　“其实也没什么好聊的，对了，你为什么要用斧头，拿着也不好看，带着也不方便。”
<br/>　　“你懂什么，我那是橙武，还带附加属性，给座金山也不换啊。”
<br/>　　“橙武？”李婉儿摇摇头，用手撩起水花：“有时候你会说些奇奇怪怪的话，我也听不懂。”
<br/>　　“哈……”
<br/>　　李婉儿抿嘴笑笑，还想在说什么，忽然见程大雷拎着一柄剑从马车中跃出来。
<br/>　　李婉儿立刻把身子沉入水中，只有一颗脑袋冒在外面。
<br/>　　“你个混蛋，我就知道你会……”
<br/>　　程大雷抖手丢过来一件袍子，冷着脸道：“藏好，有人来了。”
<br/>　　李婉儿一惊，还没回过神来，就听到一个声音响起。
<br/>　　“程当家果然好耳力，佩服佩服。”
<br/>　　一身着书生儒衫的人出现，他手中拿一柄纸扇，一步三摇，像个文人墨客一样。只不过他身上的袍子十分宽大，好像穿了一件别人的衣服。而他一张脸，酒糟鼻，蛤蟆嘴，鼻梁上一颗黑痣。
<br/>　　“在下杀人书生，给程当家见礼。”
<br/>　　杀人书生任德笛，是江湖上有名的杀手，其人手段残忍，明明大字不识一个，却好附庸风雅，就连程大雷在长安也听过她的名头。
<br/>　　崔相的人。
<br/>　　“这几天我碰见很多人，每个人出现都是一副吃定我的样子，然后……”程大雷耸耸肩。
<br/>　　“程当家可以试试，我会不会和他们一样。“任德笛已经拔出腰间长剑。
<br/>　　“不着急，等人齐。”程大雷道。
<br/>　　“等人齐，还有别人？”任德笛有些吃惊。
<br/>　　“哈哈，姓程的，你果然有两下子，俺老袁没有看错你。”
<br/>　　一个汉子从草丛中站起来，铁塔般身材，头发却是奇怪的红色。
<br/>　　“赤发鬼袁祖。”任德笛把他认了出来。
<br/>　　赤发鬼袁祖，辽山一带水贼之首，此人的了得之处在于，他向来独来独往，可辽山十八家水寨却都听他的，因为没人打得过他。
<br/>　　李婉儿躲在水里，忐忑不安的看着这一幕，对方有两个人，却不知程大雷能不能挡住。
<br/>　　只见程大雷看着远处，无可奈何道：“都已经来了，您也别藏着掖着了，出来吧。”
<br/>　　“和尚与我说过你，你果然有些出乎预料。”
<br/>　　只见远处草丛中缓步出来一人，手中擎一柄黑伞，看年纪在四十岁左右。
<br/>　　晴天打伞，行者周霜。
<br/>　　和尚，道士，屠夫，赌鬼，醉汉……共同组成三十六龙卫，而面前晴天打伞的行者周霜也是三十六龙卫之一。
<br/>　　三人出现，各占一方，程大雷身后是湖水，湖水之中是李婉儿。
<br/>　　如今杀人书生要杀自己，赤发鬼对杀不杀自己无所谓，要抢公主，行者要带走公主，顺便杀自己，而程大雷要带公主离开。
<br/>　　如果以四人为点画线，会发现四人之间既有联系，也有矛盾。
<br/>　　程大雷感觉有些头疼，其他三人的情况也差不多。四人都属于强者，但绝没有强到无视其他人存在的地步。于是四人互相提防，互相警惕。
<br/>　　“要不，咱们来一圈麻将，谁赢了谁说了算。”程大雷想出一个主意。
<br/>　　“程大雷，你一路走来，已杀死江湖十一位高手，如你这样的凶徒，今日我绝不能让你留在世上。”行者。
<br/>　　程大雷撇撇嘴，不想搭理他。
<br/>　　“人家要杀他，他难道不出手，等着别人来杀么。”杀人书生道：“你这个人脑袋有些不清楚呐。”
<br/>　　随后，杀人书生转向程大雷：“只要你本事比我强，今天能杀了我也可以。”
<br/>　　“俺和他们不一样，俺不杀你。”赤发鬼道：“让我把公主带走就行，俺娘让我把公主带回去做老婆。”
<br/>　　程大雷的脑袋越发疼了，他忽然一指赤发鬼：“先杀他。”
<br/>　　看着不明所以的二人，程大雷道：“鱼龙卫不会让公主落到山贼手里，我想崔相也不希望这一幕出现吧。”
<br/>　　“你这小子，你不是东西啊，咱们可是同行啊。”
<br/>　　“同行才是冤家呢。”程大雷扑向赤发鬼。
<br/>　　“好！”
<br/>　　行者周霜道了一声，扑向赤发鬼，杀人书生桀桀笑着，也扑向赤发鬼。
<br/>　　四人相逢，各亮兵器。周霜的铁伞合在一起，束成一道枪，扎的是……程大雷。
<br/>　　程大雷的剑刺的是杀人书生，杀人书生剑打得也是程大雷，口中还高喊：
<br/>　　“杀了赤发鬼。”
<br/>　　赤发鬼拿一把大刀站在那里有些懵：
<br/>　　“我该打谁啊。”
<br/>　　他看到没人攻击周霜，得，我就打你吧。挥着大刀砍向周霜。
<br/>
