　　听闻天火尊者的警告，慕清霄微微点头，却也没有多说，下面有些什么东西，他清楚的很。
<br/>　　陀舍古帝洞府，确实不是他一个小小斗皇能染指的，当然，几年后就不一定了。
<br/>　　下潜的时候，他保持着正常的速度，但上游的时候，并不必在意太多，速度自然就快了数倍。
<br/>　　片刻后，距离地面千米的岩浆世界中，空间一阵扭曲，一道身影就凭空出现在其上。
<br/>　　抬头往了一眼上空，取出容器，慕清霄手中已经出现一枚乳白色的陨落心炎，轻轻一推，任由它漂浮到岩浆中。
<br/>　　“前辈，可否帮晚辈在此处设下一道空间结界。”
<br/>　　说着，慕清霄又道：“我取走了陨落心炎，天焚炼气塔就会停止运行，这枚异火算是给他们的补偿吧。”
<br/>　　“你小子倒是有心，也罢，不过老夫的力量本就虚弱，空间领域将异火隐藏几年就已经是极限了。”
<br/>　　“那就有劳前辈了。”
<br/>　　虚幻的灵魂自雪白色纳戒中升起，只见天火尊者双手结印，一股压迫感就自四周包围了过来......
<br/>　　最后，形成一个小型的空间结界，将陨落心炎隐藏在其中。
<br/>　　此时，慕清霄依旧能感觉到，陨落心炎就在面前，但肉眼却看不见，手也摸不着，当真奇妙。
<br/>　　“好了，以老夫现在的力量，小型的空间结界，还是难不倒我的。”
<br/>　　“多谢。”
<br/>　　天火尊者微微一笑，道：“不必客气，老夫也是有求于你，互帮互助罢了。”
<br/>　　当陨落心炎隐藏好，慕清霄抬头看了一眼上方，心念一动，手指在纳戒上一抹，魂剑就已经出现在了手中。
<br/>　　“两个老家伙，都一个多时辰了，居然还守着，真是有耐心，看来是不可能平静的离开了。”
<br/>　　“两名斗宗，你小子不会是硬闯进来的吧？”
<br/>　　天火尊者脸色有些古怪，九星斗皇就敢硬闯，要知道斗宗与斗皇可不是一个层次，而且还是两位。
<br/>　　在他看来，慕清霄要么是有底气，要么就是个彻彻底底的疯子，但看到他一脸自信的模样，显然是前者。
<br/>　　“需要老夫帮忙吗？虽然我现在只是一缕灵魂，但要对付低级斗宗，还是有些办法的。”
<br/>　　闻言，慕清霄嘴角抽了抽，这糟老头果然留有后手，若是自己先前起了歹念，恐怕也不会太好过吧？
<br/>　　后者毕竟是斗尊，手段自然不一般。
<br/>　　“两名斗宗而已，我既然能闯进来，自然也能脱身，就不劳烦前辈了。”
<br/>　　慕清霄的语气极为平淡，却又充满了自信，说罢，便取出一件黑袍，灵气包裹着将身形包裹住。
<br/>　　旋即，空间一阵扭曲，便自原地消失。
<br/>　　...
<br/>　　...
<br/>　　此时，天焚炼气塔最底层，两位长老盘坐在上空，但心神却停留在岩浆表面。
<br/>　　“老苏，都快两个时辰了，怎么一直不见动静，难道陨落心炎学乖了？”
<br/>　　“谁知道呢，等到明日，如果在没有动静，我们就去请那位，让他来重新布置封印，暂时先守着吧。”
<br/>　　就在两人交谈之际，平静的岩浆表面，忽然荡起一阵涟漪，顿时，两位长老同时睁开双眼，目光凝重。
<br/>　　“来了，一定不能让它冲出塔外。”
<br/>　　话语刚落，死寂的塔底，一道细微的声音陡然响起，空间一阵扭曲，一道黑影就已经站在了岩浆表面上。
<br/>　　望着突兀出现的黑袍人，两位长老皆是瞳孔一缩，对视一眼，作出战斗的准备。
<br/>　　苏长老也不废话，直奔主题，语气沉重，道：“你们是哪个势力的，难道不知道这样做的后果吗？”
<br/>　　由于忌惮慕清霄的实力，苏长老也不敢贸然出手，而是试探起对方的来历，哪怕现在留不住他，往后也要清算这笔帐。
<br/>　　慕清霄沉默不语，抬起魂剑，一剑斩下，恐怖的剑气就脱剑而出，直接向苏长老席卷而去。
<br/>　　剑气速度极快，凌厉无比，苏长老眼中满是凝重，斗气自体内狂涌而出，手掌中斗气流转，直接迎向剑气。
<br/>　　“轰轰...!”
<br/>　　两股力量相撞，使得整个天焚炼气塔都动荡起来，平静的岩浆表面剧烈动荡起来，岩浆拍岸......
<br/>　　在这一瞬间，空间一阵扭曲，慕清霄就向塔顶掠去。
<br/>　　庆长老神色凝重，怒道：“狂徒，吃老夫一掌。”
<br/>　　望着紧追不舍的庆长老，黑袍之中，出现两颗血月一般的双眼，三颗黑色勾玉妖异至极，渐渐融合。
<br/>　　“月读！”
<br/>　　霎时间，庆长老目光迎上这对瞳孔，精神一阵恍惚，随后目光就变得呆泄起来，身形也停滞在了空中。
<br/>　　眨眼间，慕清霄的身影就消失不见。
<br/>　　此时，苏长老已经冲出剑气余波，一只胳膊上的衣袖变得破烂不堪，望着慕清霄离去的方向，目光沉重无比。
<br/>　　见庆长老停留在空中，苏长老一闪就来到身前，察觉到他呆泄的目光，厉喝道：“醒来！”
<br/>　　庆长老浑身一阵，双眼恢复清明，但背后已经出现一阵冷汗，吞咽了口唾沫，目光惊惧。
<br/>　　“刚才你怎么突然就停下了？”苏长老有些不解的询问道。
<br/>　　“血红色的双眼，勾玉，非常邪门，我只是对上一眼，精神就恍惚起来......”庆长老回过神来，结结巴巴的说道。
<br/>　　苏长老听的一愣一愣的，完全不明白，这老家伙在说什么东西。
<br/>　　“老苏，我刚才好像中了灵魂攻击，很强，若非你及时惊醒，我自己想要破开，估计需要不少时间。”
<br/>　　“灵魂攻击？难怪...”
<br/>　　难怪刚才他见庆长老，目光呆泄。
<br/>　　今天的事情，很多地方都令人费解，封印被破，却不见陨落心炎暴动，神秘的黑袍人，似乎又没有对他们下死手的意思......
<br/>　　片刻后，庆长老无奈的摇了摇头。
<br/>　　“他所施展的手段，我都未曾见过，刚才的剑气速度极快，威力堪比地阶中级斗技，还有恐怖的灵魂攻击，看不出是哪一方势力的。”
<br/>　　苏长老苦笑一声，今天的事情，完全出乎他们的预料，敌人显然是有备而来啊。
<br/>　　“现在情况有些玄乎，我在这里守着，你快去请那位，这件事情，已经完全脱离了我们能掌控的范围。”
<br/>　　“我这就过去，就是不知道，陨落心炎情况如何。”
