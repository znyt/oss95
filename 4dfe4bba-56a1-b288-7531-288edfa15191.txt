　　褚彦一脸悻悻，被苏莫怼了一下之后，也不再多言。
<br/>　　在他看来，苏莫肯定是得到了什么玄奇神药，但不愿意说出来而已。
<br/>　　“段惊天，你的天赋的确不凡，但是这一次去轮回古城，俱是六宫的天才，妖孽很多，你会大开眼界的！”骆妤脸色清冷的说道。
<br/>　　或许是看不惯苏莫，骆妤看到苏莫修为飚射，就想要打压一番。
<br/>　　“是吗？有哪些妖孽？”苏莫不置可否的问道。
<br/>　　“我们人王宫就不必说了，其它五宫的天才，并不比我们差，尤其是，这一次有可能冷无敌也会去？”骆妤沉声道。
<br/>　　“冷无敌？”苏莫闻言惊讶，这不可能吧？难道有人猜到他要去？
<br/>　　“当然！”
<br/>　　骆妤点了点头，继续道：“冷无敌的身份，虽然没有人知道，但有人猜测，此人有可能其它几宫隐藏的妖孽，这一次有可能去轮回古城！”
<br/>　　言语之中，骆妤的美眸中也充满了期待之色，数年之前，她亲眼见识了冷无敌的妖孽，不知道对方如此达到了什么地步？
<br/>　　苏莫闻言，微微的松了口气，原来，只是有人猜测而已。
<br/>　　这倒是他太过敏感了，根本不可能有人猜到他的身份。
<br/>　　冷无敌这个名字，只有昙花一现而已，以后都不会再出现了。
<br/>　　“冷无敌！”褚彦闻言，暗暗的攥了攥拳头。
<br/>　　若是这一次，真的能碰到冷无敌，他一定要一雪前耻。
<br/>　　“呵呵，若是他真的去轮回古城，怕是威名不保了！”姬月松轻轻笑了笑。
<br/>　　当初在天浑第一斗场，他是因为修为太高，而没有上场领教冷无敌。
<br/>　　但是，这次在轮回古城，可不是战台切磋，他修为高就是天大的优势。
<br/>　　苏莫闻言，心中冷笑，他不惧任何人，就算是姬月松，也不足为虑。
<br/>　　不过，这个姬月松，天赋的确是逆天，不到百岁，修为居然达到了二品高级。
<br/>　　“月松哥，这一次轮回古城之行，你会是最耀眼的存在！”骆妤笑着说道。
<br/>　　她对于姬月松，很有信心，整个六宫之中，能与其争锋之人，屈指可数。
<br/>　　就算是那个冷无敌来了，但是，因为修为的差距，也不太可能和姬月松争锋。
<br/>　　“我也对月松大哥有信心！”沁儿笑着说道，她们都清楚姬月松的实力。
<br/>　　前段时期，姬月松在天浑第一斗场，可是连败其它五宫的妖孽。
<br/>　　苏莫面色淡淡，没有多言，静静的等待。
<br/>　　姬月松，也会是他的踏脚石。
<br/>　　当然，他不会小觑姬月松，对方的实力，绝对不是浪得虚名。
<br/>　　广场周围，聚集了很多弟子，均是在观察这一次去轮回古城之人。
<br/>　　嘈杂的声音，此起彼伏。
<br/>　　在众人的议论声之中，又有几名弟子，陆续来到了广场中央，和众人汇合。
<br/>　　苏莫打量了一番所有人，貌似，二品修为的天才很多。
<br/>　　但是，修为达到二品高级的，除了姬月松之外，只有一人。
<br/>　　此人，是一名女子，身材高挑，穿着一身淡红色的长裙，容颜不算绝美，但也颇有紫色，给人一种冷眼之感。
<br/>　　从众人的交谈之中，苏莫知晓了，此女名为林无霜，也是一名顶尖妖孽，实力和姬月松相差无几。
<br/>　　时间缓缓流逝，约莫半个时辰之后，广场上聚集了约莫六七十人。
<br/>　　每一位，都是百岁成神的天才，每一位都是非常的不凡。
<br/>　　嗡~~
<br/>　　这时，空间一阵轻微波动，一位身材不高的中年人，突兀的出现在了广场上。
<br/>　　“拜见玄天长老！”
<br/>　　“见过长老！”
<br/>　　“拜见父亲！”
<br/>　　众人看清此人，纷纷抱拳行礼，因为来人，正是玄天长老。
<br/>　　苏莫同样向玄天长老抱拳，看来，这一次去轮回古城，由对方带领了。
<br/>　　“人到齐了，去轮回古城！”
<br/>　　玄天长老环视众人，目光在苏莫的身上稍稍停顿，随即大手一挥，朗声说道。
<br/>　　众人闻言，纷纷心中惊讶，没想到这一次，居然玄天长老亲自带队。
<br/>　　玄天长老，在人王宫排名第三，又被称为是三长老，地位崇高。
<br/>　　以往，前去轮回古城，多是由宫内的九长老或者十长老带队。
<br/>　　而这一次，居然由玄天长老带队，难道这一次去轮回古城有什么不同吗？
<br/>　　“走吧！”
<br/>　　玄天长老招呼一声，转身向宫外走去，仿佛缩地成寸一般，奇快无比。
<br/>　　而后，众人纷纷跟上。、
<br/>　　很快，众人便离开了人王城，飞天而起，向轮回古城而去。
<br/>　　轮回古城，距离颇远，就在八卫城所在之地。
<br/>　　一路无话，众人飞行了约莫半个月之后，便飞越了八卫城，临近了轮回古城。
<br/>　　轮回古城，雄伟壮阔，充满了岁月的沧桑。
<br/>　　这里，是轮回天尊曾经居住的地方。
<br/>　　轮回古城，不对外开放，就是六宫的弟子，不在特殊的时机，都没有机会进入其中。
<br/>　　古城的上方，笼罩着淡淡的氤氲，阻挡了一切，连神识都无法探查。
<br/>　　嗖嗖嗖!
<br/>　　在玄天长老的带领之下，众人纷纷落在了轮回古城那高大数十丈的青铜巨门的之前。
<br/>　　此时此刻，轮回古城门前，空无一人，寂静无比。
<br/>　　“其它五宫之人还未到来，大家自便吧！”玄天长老朗声说道。
<br/>　　“是，玄天长老！”
<br/>　　众人纷纷应声，随即，众弟子或盘膝打坐，或三三两两的闲聊了起来。
<br/>　　苏莫深深的吸了口气，随即便走到不远处，盘膝坐了下来。
<br/>　　这一次，他将直面面对其它五宫之人，心中难免有些忐忑。
<br/>　　毕竟，他身份很容易暴露，若是玄天长老不庇护他，那就会危险万分。
<br/>　　好在，他隐藏的极深，体内大量的混沌源力，一直处于分解的状态。
<br/>　　吞噬元神，则是在丹田深处，被浑厚无比的五行源力，层层封印。
<br/>　　想必，一般的强者，根本不可能认出他是轮回之身。
<br/>　　踏！踏！踏！
<br/>　　这时，脚步声响起，沁儿走到了苏莫的身边坐下。
<br/>　　“段惊天，你告诉我，你和我父亲之间，是不是有啥秘密？”沁儿面不改色，用神念向苏莫询问。
<br/>　　她心中好奇不已，上次他父亲让她通知段惊天出发的时间。
<br/>　　这一次，又亲自带队。
<br/>　　她猜测，她父亲这次亲自带队，可能也是因为这个段惊天。
<br/>　　她实在想不明白，这到底是因为什么？
<br/>　　这让她，对于眼前的青年，充满了好奇。
