　　两人疾驰，不足半天，便来到南林城。
<br/>　　南林城位于最南边。
<br/>　　风景宜人，气候四季如春。
<br/>　　在此居住，必是一件惬意之事。
<br/>　　两人来到秦家。
<br/>　　‘啾，啾…’
<br/>　　一声声鸣叫，这是警戒的意思。
<br/>　　十数个武者，飞跃而出。
<br/>　　萧逸和柳嫣然两人，是陌生人，而且气息不低。
<br/>　　秦家族人警戒也是正常。
<br/>　　不多时，又是十数武者飞出。
<br/>　　南林秦家，是出了名的团结。
<br/>　　“来者何人？”为首的一个洞玄境武者问道。
<br/>　　“萧逸。”萧逸淡淡一笑。
<br/>　　“萧逸？”那秦家族人想了想，疑惑道，“可是少家主在裂天剑派时的好友？”
<br/>　　“正是，秦飞扬在吗？”萧逸问道。
<br/>　　“不在。”秦家族人摇摇头。
<br/>　　“去哪了？我有要事找他。”萧逸再次说道。
<br/>　　“无可奉告。”秦家族人摇摇头。
<br/>　　“这…”萧逸一愣，他听闻，秦家人，向来好客友善。
<br/>　　忽然，秦家内，一道人影飞跃而出。
<br/>　　来人正是秦家家主，秦飞扬的父亲。
<br/>　　一位半步破玄武者。
<br/>　　“家主。”秦家族人纷纷行礼。
<br/>　　“免礼。”秦家家主摆摆手。
<br/>　　“两位小友，不知找我儿飞扬何事？”秦家家主拱了拱手，礼貌问道。
<br/>　　萧逸回答道，“是这样，再过几个月，剑派弟子便要挑选出10人前往裂天剑宗。”
<br/>　　“秦飞扬正是其中之一。”
<br/>　　“便过来找他。”
<br/>　　“前往裂天剑宗？”秦家家主一惊，道，“可是炎武王国的武道圣地？”
<br/>　　“正是。”萧逸点点头。
<br/>　　“嘶。”秦家家主倒吸一口凉气。
<br/>　　“北山郡裂天剑派，已经许多年无人前往了。”
<br/>　　秦家，虽没有破玄境武者。
<br/>　　但绝对是北山郡的大势力之一，传承多年。
<br/>　　自然的，一些秘辛是知道的。
<br/>　　“不对。”秦家家主忽然道，“飞扬并没有领悟你们剑派的传承石碑。”
<br/>　　“怎么有资格去剑宗呢？”
<br/>　　“你刚才说挑选出10人。”
<br/>　　“难道说，北山郡数百年没有出过的剑主出现了？”
<br/>　　萧逸笑笑，从怀中拿出剑主令。
<br/>　　“剑主令。”秦家家主顿时认了出来，连忙行礼。
<br/>　　“参见北山剑主。”
<br/>　　“秦叔叔不必多礼。”萧逸侧身闪过，没有受这一礼。
<br/>　　“现在能否告诉我，秦飞扬去哪里了？”
<br/>　　“可以。”秦家家主笑道，“不过飞扬他不会去裂天剑宗了。”
<br/>　　“他已经离开北山郡，去了雪翼雕一族修炼。”
<br/>　　说着，秦家家主从怀中拿出一个乾坤袋。
<br/>　　道，“这是飞扬离开前，让我交给你的东西。”
<br/>　　“哦？”萧逸一愣，问道，“他什么时候走的，我看能否把他追回来。”
<br/>　　在萧逸看来，进入裂天剑宗修炼的机会，不可多得。
<br/>　　但秦家家主却道，“走了差不多有两个月了吧。”
<br/>　　“就在他从裂天剑派回来没几天，就走了。”
<br/>　　“走前，疯了一样，也不像以前那么不正经了，很认真地说要刻苦修炼。”
<br/>　　“还说什么他相信你不会就这样死去，听得我糊里糊涂。”
<br/>　　“然后留下来这一袋东西，让我有机会就交给你。”
<br/>　　“啧啧。”秦家家主笑道，“也不知这孩子怎么想的。”
<br/>　　“你堂堂北山剑主，怎么会死去。”
<br/>　　说着，秦家家主摇了摇头。
<br/>　　“不得不说，裂天剑派，真的是一个教弟子的好地方。”
<br/>　　“飞扬那种吊儿郎当的性格，竟也能被你们门派长老，调教得如此认真。”
<br/>　　“额。”萧逸讪笑两声。
<br/>　　当日，离开剑派。
<br/>　　秦飞扬，一句话也没说。
<br/>　　没想到，却是坚信着萧逸没有死。
<br/>　　萧逸问道，“雪翼雕一族在哪里？比得过裂天剑宗吗？”
<br/>　　“若是比不过，我现在去将秦飞扬找回来。”
<br/>　　秦家家主还未说话。
<br/>　　柳嫣然抢先道，“在极寒之地。”
<br/>　　“雪翼雕一族，向来神秘，坐落于雪峰之巅。”
<br/>　　“那里被极寒罡风所围绕。”
<br/>　　“等闲之辈，就是地元境武者都无法前往。”
<br/>　　“只有雕之一族，有御风本领；或者雪翼雕一族，出来迎接，才能进入。”
<br/>　　“萧逸你去不了的。”
<br/>　　“若真要和裂天剑宗比的话，自然是裂天剑宗更强得。”
<br/>　　“只不过，秦飞扬是金翼雕武魂。”
<br/>　　“他在雪翼雕一族修炼，应该是更好的。”
<br/>　　“原来如此。”萧逸点点头。
<br/>　　“既然如此，那便不打扰了。”萧逸对着秦家家主拱拱手，便欲离去。
<br/>　　“哦？不多留几天吗？”秦家家主挽留道。
<br/>　　“南林城这边，风景宜人，风光很好。”
<br/>　　“谢秦叔叔好意，只是我还有要事，下次再来叨扰。”萧逸拱手道。
<br/>　　“既是如此，便不勉强了。”秦家家主点点头。
<br/>　　两人一番客套，萧逸与柳嫣然御空飞离。
<br/>　　这时，几个秦家族人来到家主身边，问道，“家主，雪翼雕一族，可是秘密，轻易不能跟外人说的。”
<br/>　　“对于普通人是秘密，对于强者，算不了什么。”秦家家主摇摇头。
<br/>　　“再说了，刚才那萧逸，乃是北山剑主。”
<br/>　　“去裂天剑宗，只有10个名额，他肯给一个名额飞扬。”
<br/>　　“有好处也知道来找飞扬。”
<br/>　　“证明，他们不是坏人。”
<br/>　　秦家家主笑呵呵说道。
<br/>　　……
<br/>　　另一边，两人御空飞行了半天，回到了裂天剑派。
<br/>　　“嫣然，你便先在剑派里等我，到时候我们一起出发前往王都。”萧逸说道。
<br/>　　“你不留在剑派吗？”柳嫣然问道。
<br/>　　“我要回家族，家族里还有要事等我回去处理。”
<br/>　　柳嫣然脸色顿时一阵失望，但也点了点头。
<br/>　　……
<br/>　　不多时，萧逸回到萧家。
<br/>　　陪了众长老几天，便又开始了他的修炼。
<br/>　　所谓山中无岁月，修炼不知时。
<br/>　　眨眼间，四个月便过去了。
<br/>　　距离前往王都，还有一个月。
<br/>　　这四个月来，萧逸无时无刻都在刻苦修炼。
<br/>　　但，修为却不怎么见长。
<br/>　　因为，每每修炼得来的真气，都会被冰鸾剑‘夺走’。
<br/>　　金丹雏形，万分之一都得不到。
<br/>　　另外，之前北山慕容家从各盛宝商行送来的四级妖兽精血，内丹。
<br/>　　萧逸也一一用了。
<br/>　　修为没进展。
<br/>　　肉体力量却增长了些。
<br/>　　原本6滴修罗之力，现在成了8滴。
<br/>　　只要再凝聚出两滴，达到10滴。
<br/>　　自身肉体力量，便能突破地元境。
<br/>　　“呼。”萧逸从修炼中醒来，吐出一口浊气。
<br/>　　“还有最后一个月，得抓紧时间了。”
<br/>　　萧逸似乎早有计划，一步步在做着他要准备的事。
<br/>　　停下修炼后，他再次离开了萧家。
