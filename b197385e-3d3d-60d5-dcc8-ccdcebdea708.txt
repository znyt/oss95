　　云岚宗的到来，对于慕清霄来说，不过是个小插曲，自那以后，萧炎修炼起来更是拼命，经过一番商议后，将修炼地点设在了萧家的后山。
<br/>　　慕清霄虽不在意别人的想法，但也不喜欢被人指指点点，毕竟他跟萧炎待在屋子里可不是一天两天了，而且传播速度很快，萧家人尽皆知。
<br/>　　不知道的，都以为他们两个大男人在屋子里做什么不为人知的事情。
<br/>　　半年的时间匆匆而过。
<br/>　　清晨，薄薄的淡白雾气笼罩着后山山顶，清风徐徐，带来一阵凉爽。
<br/>　　后山山顶一处隐秘的小树林中，萧炎一如既往的在此处刻苦修炼，准备迎接半个月后的成人仪式。
<br/>　　而慕清霄赫然已经成为一位二品炼药师，至于更高级的良药知识，也已经牢记在脑海中，成为高级炼药师也只是时间问题。
<br/>　　药尘欣慰的点了点头，道：“清霄，仅仅一年的时间，你就炼药的基础技能全部掌握，往后的成绩必然会超越为师，在你离开前，还有一样东西要传授于你。”
<br/>　　话语刚落，药尘就用灵魂力量，一缕白色光芒钻入慕清霄的脑海中。
<br/>　　慕清霄眼中满是震惊，药尘传授于他的东西，绝对是至宝啊，珍贵程度根本不能用价值来衡量：“谢师尊。”
<br/>　　“去吧。”
<br/>　　慕清霄跪地磕了三个响头，与萧炎简单的道别后，毅然转身离去。
<br/>　　药尘望着慕清霄的背影，叹了口气，其实他清楚，别说一个小小的加玛帝国，哪怕整个斗气大陆，恐怕都留不住自己这个徒儿。
<br/>　　......
<br/>　　......
<br/>　　烈日炎炎，炽热的高温，将泥土地面晒得裂开一条条缝隙。
<br/>　　踏在坚硬的泥土上，顿时有一股热浪自脚底涌进身体，使得路上行人大汗淋漓之余，不断的咒骂这鬼天气。
<br/>　　宽敞的黄土路上，一袭白衣青年，腰间挂着一块翠绿色佩饰，风度翩翩，尽显贵气，看起来估计是某个大型家族的大少爷。
<br/>　　青年与周围的画面格格不入，路上行人大汗淋漓，但他的身上却不见一丝汗水，也是因此，暗中有些不怀好意之人也打消了劫财的念头。
<br/>　　半刻钟后，慕清霄抵达最靠近魔兽山脉的一所小镇，小镇名为青山镇，因为临近魔兽山脉的缘故，又被称之为魔兽小镇。
<br/>　　小镇之中，最多的人群，自然是那些成日在刀口舔血的佣兵。
<br/>　　他们成群结队，互相抱着臂弯，在街道上口沫横飞，肆无忌惮的讨论着小镇中哪里的女人最有味道，哪里的酒最烈，哪里的魔兽最凶狠......
<br/>　　行走在由青石铺就而成的街道上，慕清霄神骏的容貌格外显眼，自然引来不少奇异的目光，不时还有女子花痴似得尖叫声，但他都没有理会，而是自顾自的走在街道上。
<br/>　　街道两边，有着不少商店，而且因为地利的缘故，生意还颇为火爆，慕清霄饶有兴致的打量着玲琅满目的商店。
<br/>　　慕清霄对武器铠甲那些铍铜烂铁没有多大兴趣，至于商人们口中的珍惜药材，倒是能够让他练习炼药术。
<br/>　　走进宽敞的店铺中，其中人流颇为不少，店员也是极其的忙碌，根本没有空闲招呼走进店铺的慕清霄。
<br/>　　他也不在意，店铺内的药材他都清楚，不用店员介绍，也清楚药效。
<br/>　　店员招待完别的顾客，见慕清霄穿着华贵，满脸堆笑的迎了上来，开始滔滔不绝的介绍起来。
<br/>　　但他在店铺中转了一圈，只找到几样药材，想要炼制三品丹药，缺少一样辅助药材，看来也只能碰运气了。
<br/>　　慕清霄指了指两样药材，道：“这两样药材给我来三十份。”
<br/>　　闻言，店员脸上挂满了献媚的笑容，道：“先生，请稍等，我立刻去给您取。”
<br/>　　片刻后，店员将三十份药材全部取来，道：“先生，一共是一千二百金币。”
<br/>　　微微点头，慕清霄取出储存空间中的紫色卡片，其中存有五千金币，倒是够用一段时间了。
<br/>　　付款过后，慕清霄就将材料收进储存空间中，自从他成为炼药师后，才发现这行业简直就是无底洞，天赋不错的还好，若是天赋差的，估计连自己的内裤都给赔进去。
<br/>　　“哇塞，你们快看，是小医仙啊！”
<br/>　　“真是好漂亮啊，啧啧，你们看她那腰细的.....”
<br/>　　“白痴，你想死啊？青山镇大半的佣兵都被小医仙救治过，当心被别人听见，割了你的舌头！”
<br/>　　站在店门口，两名男子正低声交谈，当其中一人说出些调戏的话语后，他的同伴急忙一把将他扯住，低声骂道。
<br/>　　两人的交谈自然多不过慕清霄的耳朵，似乎也是察觉到周围射来不怀好意的目光，男子脸色变得煞白，满脸尴尬的跟在同伴身后逃了出去。
<br/>　　小医仙，慕清霄自然不陌生，斗破的女主之一，视线顺着人群的目光望去，很快就看到一位身着白裙的女子身影。
<br/>　　随着人群的扩散，慕清霄终于看清，远处被众人簇拥的女子面貌。
<br/>　　女子身穿一套淡白色的衣裙，容貌倾国倾城，哪怕是比起紫嫣她们，也是不逞多让。
<br/>　　最让人眼前一亮的是，她那挂着淡然微笑的脸颊，透发着一股清新空灵的气质，这股与众不同的气质，顿时使她的魅力大幅度上升。
<br/>　　慕清霄视线打量着小医仙，最后停在那被一条绿带束着的柳腰上，望着那不足盈盈一握的柳腰，哪怕比起雅儿也丝毫不差，瞳孔中顿时闪过些许惊艳。
<br/>　　不过惊艳归惊艳，慕清霄也不是饿中色鬼，脑海中灵光一闪，忽然想到了些什么，随后离开商店。
<br/>　　走进青山镇一家热闹的客栈，小二就献媚的走了上来，恭敬道：“客观，想要吃点什么，我们客栈的材料都是有附近佣兵提供的魔兽肉，肉质鲜美……”
<br/>　　“那就来一份魔兽肉，在来一壶美酒，开一间上等客房一并送进来。”
<br/>　　“好嘞，客官请稍等，这就去给您准备。”
<br/>　　待店小二开好上等客房，将酒肉送了进来，慕清霄冲了个澡，然后喝着美酒，思考着接下来的路程。
