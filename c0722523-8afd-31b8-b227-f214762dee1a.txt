　　赢了比赛，苏重对作为裁判的长老行礼，然后就直接离开。
<br/>　　七脉会武参与者众多，今年这次人员尤其多。第一天比试，苏重只安排了一场。
<br/>　　他比完之后，就没他什么事了。苏重也不着急回去，就在通天峰上四处乱转，去各个擂台看别人比武。
<br/>　　青云门弟子众多，每个人天赋才情不同，个人炼制法宝就有差别。苏重可是见识了不少奇异法宝。当然，比起他那个不靠谱六师兄的骰子来，这些所谓奇异法宝还是相形见绌。
<br/>　　也是田不易宽容，要是遇到个严苛师傅，真得把它给活劈了。
<br/>　　转悠半天，见识了不少神异法诀，苏重看得目眩神池。
<br/>　　这才意识到，这里确实是个修真世界。以前他穿越的位面或许有超凡力量，但相比起来，还是诛仙世界更加多彩。
<br/>　　看着这些人在擂台上飞纵腾挪，苏重想到了一个问题，他还不会飞！
<br/>　　“必须尽快解决。”他可不想每次赶路都让别人带着。
<br/>　　想到这里他就不再闲逛，直接返回通天峰主殿附近的擂台。
<br/>　　各峰峰主大都在这里观看，田不易和苏茹也在这里。这会儿台上比武的是田灵儿。
<br/>　　对方是落霞峰一位不知名师兄。别看田灵儿年纪小，却遗传田不易和苏茹高绝资质。小小年纪修为高深。手里有苏茹给的法宝。一交手就占据上风，取胜只是早晚的事情。
<br/>　　“师傅师娘，我回来了。”苏重走到田不易身边道。
<br/>　　“尔旦回来了，怎么样，长见识了吧。咱们青云底蕴深厚，天才弟子到处都是。各种法诀层出不穷。平日里整天对着你那几个没正行的师兄，见不到其他风景，这次收获不小吧？”田不易笑呵呵。他对这个妖孽弟子一向宽容。
<br/>　　对比赛结果他问都没问，就怕弟子输了，脸上不好看。
<br/>　　“师傅，你不用担心，我赢了。”苏重怎么看不出来田不易故意转移话题。
<br/>　　“赢啦？！”田不易顿时惊喜起来。他当然希望自家弟子赢。
<br/>　　“好好好！不愧是我田不易的徒弟。”
<br/>　　苏茹在一边直翻白眼：“尔旦能赢那是他自己努力，关你什么事，我可没见你教过尔旦什么本事。”
<br/>　　田不易是个妻管严，丝毫不敢反抗。而且弟子赢了，他更不在乎。
<br/>　　正说着话，张小凡也走了回来。手里攥着漆黑短棍，神色轻松。
<br/>　　“师傅师娘，我回来了。”张小凡恭敬行礼。
<br/>　　“好好，回来就好。小凡啊，长见识了吧……”刚才说给苏重的话，田不易一字不差的又重复了一边。听得苏重直翻白眼。
<br/>　　“师傅，我赢了。”张小凡看着木讷，可不是真的傻。一样看出来师傅在转移话题。一句话就打断了田不易的喋喋不休。
<br/>　　“啥，你也赢啦？”田不易大喜：“不愧是我田不易的徒弟！哈哈……”
<br/>　　他最小的两弟子，本来没报什么希望，竟然全都漂亮的赢了比试。
<br/>　　虽然后面比试更加困难，但只要不是第一场就被刷下来，他就已经满足了。
<br/>　　“师傅，能教我们御剑飞行吗？”苏重趁着田不易高兴开口求教。
<br/>　　张小凡也达到了御物，只是一直没说。现在正好一块学了。
<br/>　　“御剑飞行？你没有法宝怎么御物？”田不易打量两个弟子，突然就看到了张小凡手里的短棍。
<br/>　　“小凡，你能御物啦？”他虽然早有察觉，却还是惊讶不已。
<br/>　　苏重庚金剑气攻击犀利，赢得比赛他不稀奇。参加比赛的，可大都是能御物的弟子。张小凡能赢，必然也会御物。
<br/>　　“这烧火棍还是个宝贝？”今天你惊喜有点儿多。他对这东西有印象，自从后山除了个绝地影响之后，张小凡就开始带着这根短棍。他怎么可能没注意。
<br/>　　“好，我教你们。”田不易当场就开始讲授御剑飞行。就连赢得比赛，高高兴兴跑回来邀功的的亲闺女都不理了。
<br/>　　御剑飞行看似神秘，但青云门以太极玄清道为根基，道法都是一脉相承。只要达到御物，练出本命法宝，知道秘诀后很快就能上手。张小凡只听完一遍就若有所得。
<br/>　　苏重被却不由苦笑。他练的可不是太极玄清道啊。
<br/>　　“尔旦，可是有什么难处？”师娘苏茹到底心细，一眼就看出苏重异常。
<br/>　　“师娘，我练得可是庚金剑气啊。”苏重无奈摊手。
<br/>　　“对，我怎么把这茬给忘了？”田不易拍着额头懊恼：“徒弟你不用担心，咱们青云家大业大，可不只有御剑飞行一门飞遁法门。千年来，练就奇怪法宝的师祖们不知凡几，总有些其他飞遁法门。稍后我去给你找一份。”
<br/>　　田不易这话倒不是安慰苏重，青云底蕴深厚，要找出一份飞腾秘法还真不难。
<br/>　　“师傅费心。”苏重只能等待田不易结果。
<br/>　　之后几天，苏重每天早晨和众位师兄弟去抽签，然后参加属于自己的比赛。基本上都是一天一场。
<br/>　　七脉会武几十年才举办一次，时间并不仓促，反而有意拖延。为的就是让平日里不怎么见面的各峰弟子相互交流，增加青云团结意识。
<br/>　　苏重庚金剑气迅捷锋锐，每次比赛几乎都是一道剑气把对手轰飞。
<br/>　　不过随着时间推移，遇到的对手越来越强，苏重不得不开始提高剑气威力。最近一场，他甚至动用了聚合剑气！
<br/>　　“虽然花了些功夫，好在终于进入八强比赛了。”苏重走下擂台心里在琢磨这次七脉会武。
<br/>　　按照原本轨迹，七脉会武的前四强将会下山历练。他们会前往滴血洞查探魔教余孽。而那里就藏着苏重的目标——第一卷天书。所以他必定要跟随同行。
<br/>　　苏重修炼的不是玄清道，根本就没有御物一说，就不能用寻找法宝的借口下山。只要得了四强名头，他就能名正言顺介入其中。
<br/>　　“不知道明天的对手会是谁？”苏重皱着眉头。
<br/>　　比试到了今天，留下的都不是易于之辈。小竹峰陆雪琪看着是个柔弱姑娘，实际上可是掌握着御雷真诀的狠人，而且她还真敢放！
<br/>　　真要她对上，苏重肯定要费一番手脚。
<br/>　　“还有龙首峰齐昊！”
<br/>　　苍松道人虽然最后判入魔门，但他教的几个徒弟却有真本事。这些年来，除了通天峰大师兄萧逸才，一直都是齐昊领先众人。
<br/>　　大竹峰大师兄宋大仁，不止一次败在对方手上。
<br/>　　“都是难啃的骨头啊。”
<br/>　　就连看似木讷，侥幸进入八强的张小凡，都有着噬魂这种大杀器，明天的比赛可见艰难。
