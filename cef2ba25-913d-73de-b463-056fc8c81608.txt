<br/>				这一段时间，陈默手中各种符箓还有阵基都有了小小的储备，一时间他自己都感觉有种成为地主的感叹。<br/>　　<br/>　　不容易啊，每天的坚持才换来如此的收获，都激动的想感谢各种tv了。<br/>　　<br/>　　对于爽肤水的热销，还有蔬菜的热销，他并没有过多的去干预，因为今天晚上，乾坤珠第二层禁制，就剩余最后一个禁制了。<br/>　　<br/>　　只要解开这个禁制，那么乾坤珠将面临着一次扩大，还有陈默希望的灵兽养殖区。<br/>　　<br/>　　所以，从来没有紧张过的陈默，却突然现今天的自己有些心不在焉，但是为了最后一个禁制的法决手势，陈默晚饭都没有去吃，给大家说了一声后，就在练功室里坐下，并开始静心凝神。<br/>　　<br/>　　现在的练功室，已经被他施放了一套聚灵阵的阵基，灵气是外面的好几倍！还有隔音阵和防护阵等，可以说这个练功房是他施放了很多阵法，绝对一等一的安全和隔音。<br/>　　<br/>　　当然，由于这里是集装箱房子，属于临时房子的一种，虽然建筑年限能够使用几十年，但是对于陈默的修炼来说，却并不是太过保险，所以陈默现在依然是只能在晚上修炼，白天露个面。<br/>　　<br/>　　因为在白天，如果谁要来找他，尤其是自己亲近之人，随便推门进来，就会生一些意外，不是自己修炼时候被打断，就是防御阵法起作用，让来人受伤。<br/>　　<br/>　　等他的别墅建成以后，就可以更加的保险一些。地下室几百平米的大笑，已经被他规划好，进门的地方都是厚重的钢制保险门。然后里面进行了全部的精装修，一部分用来储藏酒类，一大部分用来做练功室。<br/>　　<br/>　　等到陈默静下心来，先是从乾坤珠内取出一滴灵液，然后口含在内，一时间从口中的灵气缓缓滋润着全身。这是他害怕在施放禁制的时候，害怕自己的真元不足，能够及时的补充真元。<br/>　　<br/>　　然后运功，将位于丹田之上的乾坤珠显现出来。一时间，乾坤珠悬浮在陈默的胸口正前方，并且缓缓旋转着，莫名的光晕从白色乾坤珠上闪过，让其显得古朴，神秘和璀璨。<br/>　　<br/>　　陈默盯着悬浮的乾坤珠，等到过了好一会，才渐渐抬起手，按照乾坤珠内传给他的禁制手法，开始对着乾坤珠施放第二层的禁制法决。<br/>　　<br/>　　这是第二层最后一个节点，也是解开第二层禁制的最后一个禁制，包含1o8个手势。<br/>　　<br/>　　第二层的手势法决还不算是太难，如果再往后，那么每个禁制都会越来越难，就算是不难的第二层禁制，都是陈默夜以继日的练习后，才在这几个月里完成的。<br/>　　<br/>　　可想而知，第三层的禁制，陈默估计没有一年的时间，禁制根本打不开。<br/>　　<br/>　　好在，乾坤珠虽然没有完全认他为主，但是自从第一层禁制打开后，他已经得到了乾坤珠的认可，自然有一些好处，就是禁制解不开也没有什么，不会反噬到他的身上，最多就是浪费时间而已。<br/>　　<br/>　　只要乾坤珠的信息保密，只要没有什么人能够从他这里将乾坤珠抢过去，只要他能活的长久，那么他就能有时间慢慢解开乾坤珠的禁制。<br/>　　<br/>　　对此，陈默是悻然的。但是想到夜殇师父，他的内心又有些无法描述的感觉。<br/>　　<br/>　　因为乾坤珠，夜殇师父才来到了蓝海星。<br/>　　<br/>　　也是因为它，夜殇师父才不甘心的死亡。<br/>　　<br/>　　也是因为它，夜殇师父的传承才落到的陈默的头上。<br/>　　<br/>　　陈默不知道是要感谢乾坤珠，还是要恨它。乾坤珠虽然是个物品，但是却是他的机缘，但也因为这个，夜殇却失去了生命，所以说他的内心是无法描述的。<br/>　　<br/>　　现在，法决施展的渐多渐慢起来。由于1o8个手势，是从简单到难，所以越到后面也就越难。如果施展的时候出一点的错误，那么绝对是前功尽弃。<br/>　　<br/>　　每一次成功的法决释放，将真元以正确的手势输入到乾坤珠内，就可以见到一个光晕闪过，并且随着手势法决的增加，输入的真元增多，乾坤珠内第二层最后一个禁制检点的缓缓变淡，乾坤珠也别的开始璀璨夺目起来。<br/>　　<br/>　　好在，陈默练功的房子，并没有窗户，所以从外面是看不到房子里有些变得夺目的光芒，悬浮在空中缓缓旋转的乾坤珠。<br/>　　<br/>　　到最后一个法决，也是最后一个手势，在配合着他的真元下，缓缓对着乾坤珠施放，另外就是他的神识，也随之进入乾坤珠内，这是将禁制解开以后，把他自己的神识录入第二层上。<br/>　　<br/>　　幸好在前面的时候，陈默口中含了一滴灵液，要不然他的真元还真的在后面就会不足。好在他准备充足，在真元不继的时候，立刻将灵液咽下，补足丹田中的真元。<br/>　　<br/>　　神识录入到乾坤珠的第二层禁制上，顿时让其变得更加的光芒璀璨，在乾坤珠的表明浮现出一些奇怪的图形，犹如符箓的符文一样，瞬间变大后破碎，将真个房间都呈现出点点光芒，稍稍一会后，光点才渐渐消失。<br/>　　<br/>　　这个景象就如同在深夜施放礼花一样，在漆黑的空中爆闪，点亮一片黑暗，之后就是星星点点的光点，将夜空渲染的无比璀璨。<br/>　　<br/>　　这是解开乾坤珠第一层禁制所没有的，好在陈默位于练功房中，还加了很多阵法保护，这种景象并没有让第三者看见，这也是表明乾坤珠的第二层禁制完全解开了。<br/>　　<br/>　　对于这样的景象，陈默也是无语。这要是在修真星球，没有什么遮挡的话，那么刚才的景象绝对会招来牛鬼蛇神，自己死都不知道怎么死！<br/>　　<br/>　　闪过光华以后，乾坤珠瞬间闪亮了一下，一道信息瞬间传入到了陈默的脑海中，也让他的脑海一懵，有点反应不过来。<br/>　　<br/>　　乾坤珠传完信息后，一下子就消失，然后在陈默的丹田位置显现，可能是这里比较舒服吧，又开始缓缓转动，与自己的丹田遥相呼应。<br/>　　<br/>　　乾坤珠在他的丹田上方缓缓转动，每转一圈，就有一丝灵气融入丹田，壮大丹田的真元，还有一丝若有若无的气息，随之而上进入识海之中，强壮他的灵魂。<br/>　　<br/>　　刚才的信息有些多，也让陈默有些木木的。在乾坤珠消失的时候，还担心了一下，好在只是进入到丹田上方，也就放下心来。<br/>　　<br/>　　这个时候陈默才渐渐恢复过来。乾坤珠的信息传输的有些多，但是大部分都是第三层禁制的法决手势，依然是1o个禁制，36个节点，每个节点1o8个手势，但是却比第二层难了一些，另外就是乾坤珠内增加的功能说明。<br/>　　<br/>　　心神沉入其乾坤珠内，顿时一阵舒服的感觉，另外就是感觉乾坤珠与自己的联系又紧密了一分。并且，此时乾坤珠内的环境有了很大的变化。<br/>　　<br/>　　先就是正中心的白玉石台，变大变宽了一些，而且台阶高度也变大了些，不过台阶的阶数倒是没有变化，依然是36阶白玉台阶。<br/>　　<br/>　　青龙盘柱白玉雕像也变大了些，尤其是龙爪中的龙珠，以前是大约直径半米左右，现在增大不少，目测也有8o多厘米了。并且从乾坤珠传来的信息中，灵液的产量变大了一些，每天增加到了4滴。<br/>　　<br/>　　现在乾坤珠内空间内的灵气更上一层楼，变得比以前多了些。还有溪水，也从一米多变成了两米。<br/>　　<br/>　　另外就是空间高度，比以前要增加不少，就好像天地之间增加了不少的距离，以前只有几百米的距离，现在变得有近千米的距离，虽然天空之上依然的白雾，但是空间的增加，可也让这里变得空旷不少。<br/>　　<br/>　　还有就是种植区域的使用面积，从1ooo亩大小，变成了1oooo亩，依然是黑色的土地，不过这个区域的扩大，倒是让陈默有些小小的惊讶了，竟然扩大了这么多。<br/>　　<br/>　　突然，陈默目光一凝，他现在乾坤珠新出现的区域中，有一个巨大的白色身影。<br/>　　<br/>　　如果感觉不错，请推荐和收藏一下，谢谢大家哈！<br/>　　<br/>　　7<br/>　　<br/>　　//<br/>　　<br/>　　天才一秒记住本站地址：。手机版阅读网址：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
