　　天星地域，某地。
<br/>　　夜晚的星辰，繁星密布。
<br/>　　不少武者，抬头瞩目，不时赞叹几声。
<br/>　　忽然，一道流光在天际疾速划过。
<br/>　　流光的速度，极快；但也正是这极快的速度下，在天空白云上划出一条流线长痕。
<br/>　　长痕周遭，稀薄的夜幕之云冲散大半。
<br/>　　高空天幕，这唯美的星光夜色，陡然被‘打破’。
<br/>　　那像极了一副美景之图，忽然被人一剑划烂，过分至极。
<br/>　　“什么人？”
<br/>　　“混账。”
<br/>　　“会御空飞行了不起吗？看老子把他轰下来。”
<br/>　　下方，本在瞩目星辰的武者，一个个脸色愤然。
<br/>　　似是感觉到这一道道不善的愤怒目光，天际上的流光，陡然消散无踪。
<br/>　　……
<br/>　　天际之上，一道身影借着夜色，无声无息地瞬间降落。
<br/>　　身影，正是萧逸。
<br/>　　“呼。”萧逸轻呼出一口气，摇了摇头。
<br/>　　刚才那道流光，自然是疾速赶路的他。
<br/>　　不过没想到，他的赶路竟然引来那么多不善的目光。
<br/>　　不错，他已经到达天星地域了。
<br/>　　离开金光险地后，他一路疾驰，不过数天时间便已出了金光险地，加之横跨天索府周遭十数地域范围，最后来到了天星地域。
<br/>　　天星府，正好便在天索府旁边，彼此隔了15个地域。
<br/>　　这样的路途，事实上并不近。
<br/>　　不过，萧逸的一月历练，修为提升了不少，自是速度大增。
<br/>　　而最重要的是，他借助星光降临的力量，星光加身，强行提升了速度。
<br/>　　不过并没有8万5千道星光悉数降临。
<br/>　　毕竟那是极其耗费元力的事。
<br/>　　他只大概降临了6万道左右，大概等同圣皇境六重的实力。
<br/>　　萧逸松了一口气，也下意识地看了眼天空。
<br/>　　一看之下，也不禁为这美景赞叹一声。
<br/>　　整个天空，宛若一块黑幕。
<br/>　　而这黑幕之上，繁星密布，交映成辉。
<br/>　　天星地域这边的星光，远比其它地域密得多，也耀眼得多。
<br/>　　十八府的地域，虽强弱差不多，在中域的名声也差不多。
<br/>　　但若论哪一府的地域最美，当属天星地域。
<br/>　　天星地域的繁星夜色，哪怕放眼整个中域，那都是有名的美景之一。
<br/>　　“确实很美。”萧逸自语一声，随后，收回了看向天空的目光。
<br/>　　他没时间为这些美景所停留。
<br/>　　天星地域这边，有不少武者修习星光一道。
<br/>　　也是因为这个地域每到入夜便繁星璀璨，故星光一道的武者在此修炼效果极好。
<br/>　　据萧逸所知，传闻，这里也出现过星幻之火的传说。
<br/>　　这也是萧逸特地选择来天星地域历练的原因之一。
<br/>　　星幻之火，作为最神秘的世间强悍火焰，绝对是能让他火之一道实力大增的东西。
<br/>　　不过，无论是否有星幻这个的消息。
<br/>　　他来此历练，即便是单纯修炼，也对他的星幻剑道大有裨益。
<br/>　　嗖…
<br/>　　萧逸一个闪身，御空飞起。
<br/>　　他并未飞至高空，免得又惹来那些欣赏美丽星光的武者的谩骂。
<br/>　　他只是在半空快速飞行着。
<br/>　　现今他深处的范围，已然进入天星地域了，不过只是边缘范围。
<br/>　　十八府之一的天星府，便坐落在这个地域的中心处。
<br/>　　不过，萧逸并非要前往天星府或者那些城池，而是要前往天星险地。
<br/>　　天星险地，乃是天星地域以及周遭十数地域中，最为闻名的险地。
<br/>　　其危险程度，与金光地域差不多。
<br/>　　一路横跨天星地域，出了天星地域，再飞行大概两个地域的距离，便是人迹罕至的天星险地了。
<br/>　　大半天后。
<br/>　　恰好是第二日的清晨时分，萧逸赶至了天星险地。
<br/>　　如他所料，入目所视，乃是一个辽阔无边的妖兽森林。
<br/>　　萧逸闪身落下，刚要进入其中历练，忽然…
<br/>　　“嗯？”萧逸停下了脚步，眉头一皱。
<br/>　　天星险地之外，此时正聚集着大量武者。
<br/>　　有年轻天骄，也有修为深厚的老者，不下千人。
<br/>　　观这群的气息，最弱的都在圣王境三重以上。
<br/>　　“这么多武者？”萧逸面露疑惑之色。
<br/>　　据他所知，天星险地之内，可没有什么特殊之处，也没有吸引武者的地方，故平日里人迹罕至。
<br/>　　不同于金光险地，金光险地内，每至入夜，便金光飘荡，妖兽吞服这些金光，将实力大增。
<br/>　　武者，也会在此时斩杀妖兽，已获得内丹，炼制金光丹。
<br/>　　每一粒的金光丹，价值不菲，故终年吸引大量武者进入其中。
<br/>　　而天星险地这里，一来凶名在外，二来毫无好处，故只会有单纯为了历练而来此的少量武者。
<br/>　　可现在，这天星险地之外，竟然出现了那么多的武者，而且还聚集在一起。
<br/>　　萧逸略感疑惑，身影一闪，混入了这群武者之中。
<br/>　　“兄台，这里发生了什么吗？”萧逸来到一个看起来还算脸色平和的年轻人身旁，轻声问道。
<br/>　　这个年轻人，看着忽然出现的萧逸，显然吓了一跳。
<br/>　　待得看到萧逸只是个和他年龄相仿的年轻人，这才松了口气。
<br/>　　“兄台，你不知道？”年轻人疑惑地问道。
<br/>　　萧逸摇了摇头。
<br/>　　“哦。”年轻人笑笑，道，“兄台肯定是别的地域的武者吧。”
<br/>　　“否则，绝不可能不知晓这里之事。”
<br/>　　“哦？”萧逸面露疑惑之色，看起来似乎还是比较有名的事。
<br/>　　年轻人笑笑，道，“天星禁地，就要开启了。”
<br/>　　“自然的，我们天星地域，还有附近地域修习星光一道的武者，都赶来此了。”
<br/>　　“哦？开启天星禁地？”萧逸微微皱眉。
<br/>　　“嗯。”年轻人一副得意之色，泛泛而谈，道，“天星禁地开启，数年一度。”
<br/>　　“天星禁地，本是天星险地内天星府指明的禁地。”
<br/>　　“不过，天星府不仅没有欺我们这些普通武者，反而每数年一度的开启都会派遣强者，护送我们进入。”
<br/>　　“在天星禁地之内，我们这些修习星光一道的武者，修炼一日，可抵外头百日之功。”
<br/>　　“自然的，这几天附近各地域的武者都来了。”
<br/>　　“原来如此。”萧逸点了点头，面露恍然。
<br/>　　据他所知，天星府虽是十八府之一，以是中域霸主势力之一，但其风评极好。
<br/>　　在天星地域，以及周遭十数地域内，很受各方武者敬重。
<br/>　　“对了，兄台你也是修习星光一道的武者吗？”年轻人问道。
<br/>　　“若不是，你进不去天星禁地的。”
<br/>　　萧逸笑笑，指间凝聚出一道微弱星光。
