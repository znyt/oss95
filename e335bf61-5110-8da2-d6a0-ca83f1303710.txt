　　时间匆匆而逝，苏莫等待了五天，也没有等到巫天御出现。
<br/>　　地灵塔的空间中人来人往，换了一批又一批。
<br/>　　苏莫心中暗叹，这样等下去也不是办法，武帝境的武者，有的时候一闭关都是几个月，甚至是几年。
<br/>　　虽然巫天御不可能在修炼室中待几年，但待个几个月却是很有可能的。
<br/>　　就在此时，苏莫眼角余光瞥见了一个熟悉的面孔，只见一名高高瘦瘦、脸颊狭长的青衫青年，来到了地灵塔的第一层空间中。
<br/>　　苏莫见到此人，顿时双眉一挑，因为此人他曾经见过，正是当初去苍穹世界试炼的巫族天才之一。
<br/>　　此人还曾经被秦不死擒拿，后来在巫梵尊者的声波之下，被从秦不死的手中震飞了出去，逃得了一条性命。
<br/>　　而此人，现在的修为也达到了武帝境一重的境界。
<br/>　　苏莫心念电转，立刻在脑中搜索此人的信息，他搜魂了两名巫族之人，得到的记忆非常的多。
<br/>　　很快，苏莫便找到了此人的一些信息，此人名为巫长远！
<br/>　　因为此人也算是巫族的一名小天才，所以很多人都认识此人。
<br/>　　苏莫嘴角勾起了一丝轻笑，随即长身而起，向对方走了过去，他或许能从此人口中，套出一些青璇的消息。
<br/>　　“长远兄，恭喜修为大进！”苏莫走到了巫长远的身边，向对方抱拳笑道。
<br/>　　“你是……？”巫长远闻言一怔，打量了苏莫一番，面带疑惑之色。
<br/>　　他与巫尘并不相识。
<br/>　　“我是巫尘！”苏莫笑着说道。
<br/>　　“巫尘？”
<br/>　　巫长远思索了一番，随即恍然道：“你就是巫玉儿的未婚夫吧？”
<br/>　　巫长远不认识巫尘，但却是认识巫玉儿，对巫尘也是有所耳闻。
<br/>　　“正是！”苏莫点了点头。
<br/>　　“你有什么事吗？”巫长远面色淡淡的问道，并没有表现出多少殷切之色。
<br/>　　因为据他所知，巫尘就是个没有任何背景的家伙，天赋也不怎么样，虽然比他修为高，但也是因为比他年长十余岁。
<br/>　　“呃~~在下听闻长远兄上次去苍穹世界试练，受了不轻的伤，特来一问！”苏莫心中急转，随即轻笑着问道。
<br/>　　“哼！多谢你关心了！”巫长远听闻苏莫提到苍穹世界试炼一事，顿时冷哼一声。
<br/>　　此事，是他的耻辱，也是天冥星巫族一脉的耻辱。
<br/>　　他们堂堂巫族的一众天才，被一个土著杀得七零八落，此事已经成了巫族其它几脉的笑料。
<br/>　　言罢，巫长远自顾自的走到前方空旷之处，盘膝坐了下来，等待空余的修炼室。
<br/>　　“长远兄，你放心吧！这个仇迟早会报的！”
<br/>　　苏莫再次来到了对方的身边，盘膝坐下，继续道：“以我族的能量，要不了多久，就能找到那个苏莫，到时候定然将他碎尸万段！”
<br/>　　“哎！只可惜不能亲手斩杀这个卑微的土著！”
<br/>　　听闻‘巫尘’提到苏莫之名，巫长远满脸忿恨之色，继续道：“现在那个土著若是来到我的面前，我只需一招，便能将他镇杀！”
<br/>　　苏莫闻言心中冷笑，若不是在这巫族之内，他一指头就能戳死对方。
<br/>　　对方这点实力，在他眼中狗屁都不是，对方也就只能在这里吹吹牛.逼，过过嘴瘾了！
<br/>　　“那是，以长远兄现在的实力，镇杀此人轻而易举！”心中虽然不屑，但苏莫面上却是丝毫没有表现出来。
<br/>　　随即，苏莫见对方沉默不语，不想交谈陷入僵局，急忙继续道：“此人逃不出我族的手掌心，迟早会死，倒是我族灭了苍穹世界，屠灭了其内的所有人，也算是为那九百余名族人报仇了！”
<br/>　　“报个屁，苍穹世界那些土著都是贱命，就算再多十倍、百倍，也抵不上我族陨落的那些天才的命珍贵！”巫长远不屑的说道。
<br/>　　苏莫闻言，顿时心中杀机升腾，苍穹世界的人都是贱命？巫族之人高贵？
<br/>　　他真恨不得一拳将对方轰成渣渣！
<br/>　　不过，苏莫还是立刻压下了心中的杀意，在这巫族之内，必须保持理智。
<br/>　　“确实！”苏莫轻轻的点了点头。
<br/>　　“哎！这么长时间了，也不知道这个土著逃到了哪里？居然现在都没有查到！”巫长远叹了口气。
<br/>　　苏莫闻言，眼珠子一转，随即轻笑道：“听说那苏莫有朋友被巫天御擒拿了，其实我族可以用此女要挟对方现身！”
<br/>　　“哼！你懂个屁！巫娴姐和另外两名同族在对方手中，如何能要挟的了？”巫长远冷哼道。
<br/>　　“说的也是！”
<br/>　　苏莫恍然的点了点头，而后，仿似随意的问道：“那苏莫的那位朋友怎么样了？不会被巫天御直接杀了吧？”
<br/>　　“这你都不知道？”巫长远闻言，满脸诧异的望着苏莫。
<br/>　　此事早已人尽皆知，面前的家伙居然不知道？
<br/>　　“呵呵！我整天闭关，对于这件事了解的并不多！”苏莫微笑着说道。
<br/>　　“原来如此！”
<br/>　　巫长远恍然，没有丝毫怀疑，因为他压根不可能想到，苏莫会来到黑耀城，来到他的面前。
<br/>　　“那个女子乃是个奇女子，已经被大祭司带走了！”巫长远说道。
<br/>　　“什么？”苏莫闻听此言，顿时心中一震，青璇居然被大祭司带走了？
<br/>　　一瞬间，他的心沉到了谷底，天冥星巫族的大祭司，那可是武圣境界的强者，青璇落在对方的手中，他如何能救的出来？
<br/>　　“你怎么了？”巫长远见苏莫脸色不对，略显诧异的问道。
<br/>　　苏莫顿时回过神来，立刻压制心中的波动，左右看了一番，小心翼翼的问道：“我可是听闻，那女子体质特殊，容颜倾城，特别适合双修，难不成大祭司看上了？”
<br/>　　巫长远见苏莫贼头贼脑的模样，顿时无奈一笑，摇头道：“你最好还是少胡言乱语的好，这话若是传到了大祭司的耳中，你有十条命也不够杀得！”
<br/>　　“嘿嘿！我只是有些好奇而已！”苏莫尴尬的笑道。
<br/>　　“好奇会害死你！”
<br/>　　巫长远摇了摇头，再次打量了一番苏莫，告诫道：“你在族中一没背景，二没实力，八卦之心倒是奇重，若是不收敛，迟早会惹祸上身！”
<br/>　　言罢，巫长远便闭上了双眼，静静的打坐调息。
<br/>　　而苏莫则是脸色沉了下来，心中阴郁到了极点，他感觉自己这一次怕是要失望而归了。
<br/>　　他，根本没有能力，从大祭司的手中救走宏青璇。
