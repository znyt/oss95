　　第469章
<br/>　　背靠大河，前方已被帝国军围死。从程大雷的角度望过去，当真是黑压压一片。事实上，这支临时组成的剿匪军人数并不多，无非就那么一万余人。但程大雷的兵力更惨……
<br/>　　不足二百。
<br/>　　这要是打起来，怕会全军覆没。唯一的出路，大概就是从身后的大河逃生，河宽三百步，程大雷自己脱身是没有问题的，不过带着徐神机和银眸，再加上这二百义字军兄弟……
<br/>　　“大当家，不若你自己逃命去吧，留得青山在，不怕没柴烧，日后东山再起，再为我们报仇。”徐神机。
<br/>　　“咦……”程大雷怔了怔，却是没料到徐神机竟会说出这番话。这可不是他的人性，莫非这徐神机是其他人假冒的。
<br/>　　单雄信和白平也同时道：“大当家，你先走，日后记得替我们报仇就好。”
<br/>　　“言之有理，言之有理。”徐神机大声赞同，悄悄默默对程大雷道：“你带我一个人走，应该没问题吧？”
<br/>　　程大雷恶狠狠瞪了他一眼，确认眼前这徐神机是真的。
<br/>　　胜败乃兵家常事，接受失败，及时止损是一名合格指挥者应有的素质。曹操有割须弃袍，刘备就更别提了，逃跑的时候连老婆都可以不要……
<br/>　　但程大雷，他从来都不是什么合格指挥者。归根结底，一山贼而已，要杀便杀，要恨便恨，同样，要死的时候，也得死。
<br/>　　“都不必说了，我带你们冲杀出去，能活几个算几个，都活不了……”程大雷恶狠狠的盯着薛丁山：“临死之前，我也得杀了你。”
<br/>　　“大当家！”二百义字军同时立在程大雷身后。
<br/>　　“废话就不用说了，有时间想着怎么能从这里活着离开吧。”
<br/>　　在程大雷思考对策时，帝国军也没有闲着。不断缩小包围圈，持续向前推进，另外不知谁出的馊主意，帝国军发出对程大雷的一声声谩骂。
<br/>　　比如『你已经是秋后的蚂蚱，蹦跶不了多久』，『快快放下武器，在薛将军的裤裆下钻过去，可以饶你不死』……
<br/>　　不用想也知道，这馊主意一定是薛丁山出的。兵者诡道也，以正合，以奇胜。老实说，薛丁山的诡计成功刺激到了程大雷。本来他就看薛丁山不顺眼，听到这话，更是三尸神暴跳，五灵豪气腾空。
<br/>　　程大雷紧紧攥着大斧头，眉毛一下一下的有力跳动。他暗下决定，今天如果不杀这薛丁山，自己下半辈子就不姓程。
<br/>　　当然，若杀不了薛丁山，估计自己的下半辈子就到此为止。
<br/>　　“全军准备！”程大雷举起斧头：“今日死战！”
<br/>　　相比程大雷的亢奋，帝国军表现得很淡定。毕竟，程大雷已经是瓮中之鳖，手到擒来。
<br/>　　陆承不得不对这薛丁山刮目相看，这员小将究竟是哪里冒出来的，只听问祖籍是京州，之前却是没听说过。不过这刚一露脸，就给了陆承震惊，决不可小觑此人，日后他的成就绝对不小。
<br/>　　“薛将军，今日击败程大雷，必然是大功一件，日后薛将军飞黄腾达，可莫忘了陆某。”
<br/>　　薛丁山顿了顿，忽然笑道：“陆军师哪里话，今天若杀了程大雷，你我功劳各居一半，没有陆军师出谋划策，事情怎可如此顺利。”
<br/>　　陆承眼睛一亮，这薛丁山比想象中会来事儿，并不是个不通人情事故的人。与这样的人搭档，过程总是令人愉悦的。
<br/>　　“薛将军果然名不虚传，轻易便将程大雷逼至绝境，此战过后，定是天下扬名。”
<br/>　　此刻，程大雷大吼一声，领着大军冲杀过来。
<br/>　　薛丁山微微眯眼，握紧手中的银戟：“谨慎些，程贼已经穷途末路，只能铤而走险，他的人头价值万金，莫让别人抢走了。”
<br/>　　陆承重重的点点头，他心中自然明白这一点。各路诸侯各怀心事，铆足了劲儿从这场乱局中捞资本，不然的话，若大家都是一条心，杀林少羽也不会那么困难。
<br/>　　如今状况，第一防的是程大雷逃脱，第二，是一定要抢到程大雷的脑袋。
<br/>　　他那颗人头，万金的价格怕是挡不住，至少值一座城池。
<br/>　　在程大雷开始冲锋的同时，剿匪军也发起了冲锋，两方人马撞在一起，二百对一万，几乎可以说连个水花都砸不起来。
<br/>　　程大雷也知道是绝境，绝境便只能厮杀，他手握鬼面斧，挥出去便是人头和鲜血。银眸护着徐神机，赤发灵官单雄信的金顶枣阳槊也是杀得威风凛凛。
<br/>　　纵然三人如此武勇，陷入千军万马丛中，其实也翻不起多大浪花。他们三人暂时无生命危险，可二百义字军却接连死伤。
<br/>　　大势已去，程大雷带着他们，几乎已无可能逃脱。
<br/>　　“大当家，你先走吧！”白平大吼。
<br/>　　“废什么话，一起冲杀出去。”程大雷将面前的敌人结果。
<br/>　　“带着我们，大当家也走不脱，我们兄弟一条贱命，死了也不可惜。”
<br/>　　白平忽然吼了一声，举起手中的大刀，冲着残存的义字军吼道：“来，兄弟们，为大当家腾一条路。”
<br/>　　“杀！”
<br/>　　在白平的带领下，他们发出一阵阵狂吼，像飞蛾扑火一般，疯狂的扑向敌人。程大雷狠不得拍死白平，果然什么人带什么兵。正义教有一个算一个，怎么都和林少羽一样，关键时刻就寻死呢。
<br/>　　可不用程大雷拍，白平已经死了，挂在敌人的长矛上。眼前的确出现一条路，一条用尸体铺成的路。
<br/>　　“姓薛的，你给我去死！”
<br/>　　程大雷发了疯，无论如何，今天也要杀了这薛丁山。
<br/>　　薛丁山眼睛微微眯起，口中道了一声：“来得好！”
<br/>　　他身后，弓兵准备完毕，只要程大雷进入射程，便万箭齐发，要他来个万箭穿心。
<br/>　　恰此时，远处一阵马蹄声急。薛丁山下意识回头望去，乌压压都是骑兵，足有三千余人。
<br/>　　又来！
<br/>　　陆承道了一声，怎么又有神兵天降，来救程大雷突围。
<br/>　　但他看错了，这次明显和上次不同。这三千铁骑，人人统一黑甲，背后一只丑陋狰狞的癞蛤蟆，为首一员白袍小将，白盔白家白龙马，口中高喝一声：
<br/>　　“大当家莫慌，子龙来也。”
<br/>(本章完)
