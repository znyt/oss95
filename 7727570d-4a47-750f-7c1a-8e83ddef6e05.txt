　　苏莫的修为，早已达到了武皇境八重的巅峰，而且经过长时间的沉淀，根基已经非常的扎实了。
<br/>　　一直以来，都是资源限制了他的修为提升，不然他也不会被困在武皇境八重。
<br/>　　吞噬武魂催动，霎时之间，三十余枚神丹中的玄力，疯狂的倾泄而出，向苏莫涌去，完全将他的身体包裹了起来。
<br/>　　大混沌五行神诀运转，苏莫立刻开始极力炼化了起来。
<br/>　　这些神丹中的玄力，各不相同，纷杂无比，需要不断的炼化，才能被他所用。
<br/>　　轰隆隆！
<br/>　　无尽的玄力，如同长江大河，在苏莫全身经脉中奔流，他全身的气息立刻暴动了起来，开始冲击武皇境九重境界。
<br/>　　神丹中玄力绵绵无尽，为他的修炼，提供了足够的支撑。
<br/>　　远处，无数的苍穹神宫弟子，在遥望苏莫。
<br/>　　虚界神图中的空间并不大，所有人都能看到苏莫的修炼情况，不过并没有人敢去上前打扰。
<br/>　　甚至，百万人都非常的安静，鸦雀无声。
<br/>　　众人暗暗心惊，直接吸纳神丹中的玄力修炼，这可是闻所未闻的事情。
<br/>　　难怪苏莫修炼的速度如此逆天，原来还有这种不可思议的本领。
<br/>　　就这样，在无数人的观看之下，苏莫极力的吸纳玄力、炼化玄力，身上的气息不断的鼓荡。
<br/>　　一个时辰又一个时辰，足足十个时辰之后，三十多枚神丹中的玄力，被苏莫吸纳一空，全部化为了飞灰。
<br/>　　而这个时候，他身上的气息，仿若在水下涌动的巨兽，随时都会破水而出。
<br/>　　哗啦啦！
<br/>　　毫不犹豫，苏莫手臂一挥，数百亿极品灵石出现在他的周身，形成了一座大山。
<br/>　　灵石乍一出现，便纷纷爆裂，化为浓郁的灵雾，疯狂的涌动。
<br/>　　苏莫继续吞噬灵气，他体内的五枚神丹，不断的震动，强大的气息通过他的身体，传到体外，形成猛烈的狂风。
<br/>　　不过，在虚界的神图中，狂风并不能肆虐，全部被空间之力压制了下来。
<br/>　　无数苍穹神宫的弟子，望着那无尽的灵雾，个个目瞪口呆，居然还能这么修炼？
<br/>　　轰隆隆！
<br/>　　轰隆隆！！
<br/>　　不多时，苏莫的体内传出了一阵阵轰鸣之声，轰鸣声越来越强烈，如同天雷滚滚，震动了整片神图空间。
<br/>　　少倾，一股磅礴浩大的气息，从苏莫的身上升腾而起，弥漫八荒。
<br/>　　突破了！
<br/>　　所有苍穹神宫之人，无不是瞪圆了双眼，好快的速度！
<br/>　　武皇境高阶的修为，还能有这种突破修为的速度，实在是不可思议！
<br/>　　这简直比真灵境武者，突破修为还要容易啊！
<br/>　　苏莫突破了修为之后，并未结束修炼，继续吞噬灵气。
<br/>　　待所有灵气吞噬完之后，他再次取出无数的极品灵石，继续吞噬。
<br/>　　又过去了几个时辰，几乎将身上所有的极品灵石耗尽之后，苏莫才停止了修炼。
<br/>　　他已经一口气冲到武皇境九重的巅峰，实力再次大涨。
<br/>　　呼！
<br/>　　苏莫睁开了眼眸，张口吐出一口浊气之剑，他甚至想冲到准帝的境界，但显然不可能。
<br/>　　少倾，苏莫和众人打了声招呼，便离开了神图空间。
<br/>　　他在秦不死的宫殿中，一边盘膝修炼，巩固暴涨的修为境界，一边静等消息。
<br/>　　就这样，苏莫修炼了半个月之后，光头大汉和黑衣中年人来到了宫殿中。
<br/>　　“苏莫，大哥传来消息了！”光头大汉率先开口道。
<br/>　　“什么消息？”苏莫立刻停止了修炼，长身而起。
<br/>　　“巫族试炼者全部都集中在了一起，约有近七百人，目前正在西海的一座小岛上！”光头大汉说道。
<br/>　　“七百人？”苏莫闻言疑惑，不是只剩下了四百人左右了吗？哪来的七百人？
<br/>　　光头大汉见此，立刻说道：“之前还有三百名左右的巫族试炼者，分散去了各处，现在他们都全部汇合在了一起！”
<br/>　　“他们在那处小岛做什么？”苏莫又问道。
<br/>　　“据大哥传回来的消息，这些试炼者是隐藏在小岛上，看来不准备出来了，估计会一直等到试炼结束才会离开！”光头大汉道。
<br/>　　“很好，我们去灭了他们！”苏莫点了点头，眸中杀机爆闪，身上强大的气息涌动。
<br/>　　“你已经突破了？”黑衣中年人惊讶的声音响起，他到此刻才发现，苏莫居然已经突破了修为。
<br/>　　光头大汉亦是心中一惊，好快的速度，才不过半个月的时间，居然就突破了！
<br/>　　“我们走吧！”苏莫点了点头，不由分说，身形一闪便离开了宫殿，光头大汉两人连忙跟了上去。
<br/>　　随后，三人一起，联袂向西海而去。
<br/>　　……
<br/>　　无边无际的西海之上，三道身影划破长空，急速而行。
<br/>　　苏莫三人，仅用了三天时间，便来到了西海之上。
<br/>　　几个时辰之后，苏莫视线的尽头，数万里之外，出现了一座岛屿。
<br/>　　这座岛屿不大，只是一处无人的荒岛，方圆不过数百里，岛上是一片连绵起伏的山脉，山脉中古树参天，遮天蔽日。
<br/>　　哗啦！
<br/>　　就在此刻，前方响起破水之声，海水中冲出一道身影，正是秦不死。
<br/>　　苏莫三人见此，立刻飞到了秦不死的身前。
<br/>　　“苏莫，就是前面一座小岛，巫族试炼者全部隐藏在岛上！”秦不死沉声说道，他眸中精光一闪，一眼便看出了苏莫的修为已经突破了。
<br/>　　“他们知晓你的存在吗？”苏莫问道。
<br/>　　“应该不知晓，老夫有特殊的追踪之法，他们绝不会发现。”秦不死笃定的说道，他的追踪之法，乃是属于不死鸟的绝技，一般的武帝都不可能轻易发现他。
<br/>　　苏莫闻言点了点头，看来巫族之人是真的不准备试炼了，估计也是担心他杀来，便隐藏在了此地。
<br/>　　“稍后我独自出手，你们负责击杀逃跑之人！”苏莫没有浪费时间，对三人说了一句，便径直向前方的岛屿飞去。
<br/>　　他现在极为自信，也他现在的实力，完全能够横扫一众巫族试炼者了。
<br/>　　秦不死三人立刻跟上，他们对此没有意见，苏莫的实力本就强大，现在又突破了修为，更是到了难以想象的地步。
<br/>　　片刻之间，四人便跨越了数万里的距离，来到了小岛之上。
