　　战台上。
<br/>　　会武继续进行，仿佛永无止境一般，一名名年轻的武者上台比试，有的实力差距不大，需要百招才能分出胜负，但有的一招便能结束战斗。
<br/>　　那闪灵族的灵枫，瞬移到对手的身后，在对方没反应过来之前，便一招将对手击败。
<br/>　　啸天狼族的幽狼，扬天一声狼嚎，便将对手吓的脸色苍白，瑟瑟发抖，主动的认输了。
<br/>　　除此之外，御剑谷和雷云禽族，同样有一名妖孽人物。
<br/>　　御剑谷的卓剑豪，作为一名本命剑客，甚至都没有动用灵剑，只是随手一剑，就将对方劈飞了出去。
<br/>　　而雷云禽族的雷云皓，同样恐怖无比，口中吐出一个雷球，直接将对手烧成了焦炭，差点闹出人命。
<br/>　　每一个大势力，都有顶尖的妖孽人物，除了这些妖孽之外，还有一些人虽然稍弱一筹，但也算是非常厉害的天才。
<br/>　　比如黑炎族的炎火，啸天狼族的墨狼等人，虽然比之扬天成、炎澜天等妖孽，实力有所不如，但绝对不比林皇、唐古等人差。
<br/>　　不知过去了多久，第一轮战斗好像接近了尾声。
<br/>　　这种比赛规则，无论是挑战还是被挑战，战过一局之后，便无法再登场，只能在下一轮才能战斗了。
<br/>　　“苏莫啊！其实我们道宗，以往在七大势力之中，算是非常强的，但最近百余年来，没有招到天赋妖孽的弟子，所以门下弟子和其它势力的弟子之间，便渐渐的拉开了距离！”
<br/>　　苏莫的身边，冯光絮絮叨叨，摇头晃脑，面上充满了无奈。
<br/>　　“那为何不多招些天赋高的妖孽？”苏莫淡淡的问道。
<br/>　　冯光闻言无语，摇头道：“你以为天才是大白菜，到处都是，这岂是想招就能招的到的？”
<br/>　　“也是！”苏莫点了点头。
<br/>　　就在此时，蓦地，北方看台上，那名主持会武的褐袍老者站了起来，朗声高喝道：“第一轮挑战结束，下面第二轮挑战开始！”
<br/>　　苏莫闻言一怔，随即面色一僵，他还没上场呢，居然第一轮就结束了。
<br/>　　也罢！
<br/>　　随即，苏莫暗暗摇了摇头，既然如此，那就第二轮再上场吧！
<br/>　　少比一场，只是少了一分而已，没有大碍！
<br/>　　因为不可能有人全胜得满分，谁分数高，他便挑战谁。
<br/>　　“苏莫，其实你不该来观战！”冯光的声音继续响起。
<br/>　　“为何？”苏莫问道。
<br/>　　“你看到了我们道宗的窘境，难免会看低道宗！”冯光叹道，在他想来，苏莫看到道宗弟子在会武中一败涂地，难免会对道宗产生不好的印象。
<br/>　　苏莫闻言，无语的摇了摇头，随即沉声道：“冯长老，你放心吧！这次会武，道宗不会垫底，肯定会夺得第一名！”
<br/>　　“夺得第一名？”
<br/>　　冯光闻言一怔，随即苦笑一声，道：“好了，你也不用安慰我了，这是不可能的事情！”
<br/>　　他可不会认为道宗弟子能得第一名，这完全没有可能，别说第一名了，就算是第十名都没有什么可能了！
<br/>　　“绝对可能！”苏莫淡淡的说道，言语无比的笃定。
<br/>　　冯光见此，暗暗的摇了摇头，心中暗自腹诽，这苏莫是不是眼瞎了？
<br/>　　还得第一名？
<br/>　　道宗弟子完全无法和其它六大势力的弟子争锋，这么清楚的局面，难道还看不清么？
<br/>　　此时，真枯半圣等道宗长老、以及其他道宗的弟子们，都听到了苏莫的话语。
<br/>　　真枯半圣眉头一皱，其他弟子则是个个脸带怒色，对苏莫怒目而视。
<br/>　　“苏莫，你是在故意嘲讽吗？”林皇冷冷的说道，在他看来，如此的局面之下，苏莫还在说道宗弟子能得第一，明显就是在故意嘲讽。
<br/>　　“苏莫，你既然来观战，就老老实实的观战，不要胡言乱语！”吕罡轻喝道。
<br/>　　呃~~
<br/>　　苏莫神情一怔，转头环视众人，见所有道宗弟子都对他怒目而视，连真枯半圣都阴沉着脸望着他，顿时有些无语了。
<br/>　　暗暗摇了摇头，苏莫也不与众人争辩，事实会证明一切。
<br/>　　这个时候，战台上的第二轮挑战已经开始了。
<br/>　　那主持会武的褐袍老者，对于每个势力有多少名武帝境九重的弟子，心中一清二楚，所以战斗场数一到，他便宣布第二轮挑战开始，并不会遗漏任何一人。
<br/>　　但是，武帝境八重的苏莫，并没有被算在其中，显然没有被认为是参加会武的人。
<br/>　　一场战斗接着一场战斗，胜者意气风发，败者黯然神伤。
<br/>　　道宗的弟子，又有数十人上场，但获胜的只有三人，大多数都败了。
<br/>　　嗖！
<br/>　　又一场战斗之后，黑炎族的炎火上场了，直接就将矛头对准了吕罡。
<br/>　　“吕罡，我们一战的时候到了！”炎火目光如电，望在吕罡的身上，眸中战意沸腾。
<br/>　　这一战，再次点燃了武斗场的气氛。
<br/>　　“哈哈！好戏要开始了，吕罡若是败了，道宗三大顶尖天才，就全部败了！”
<br/>　　“吕罡若败，道宗基本上可以回去了，不用参加接下来的会武了！”
<br/>　　“这一战胜负难料啊！”
<br/>　　“是啊！吕罡的实力肯定不如炎澜天，但比之炎火，怕是伯仲之间！”
<br/>　　整个武斗场，均是议论纷纷，这一场战斗，虽然不是顶尖妖孽之间的巅峰一战，但关乎道宗在这次会武中的成绩，却也是让更多人来了兴趣。
<br/>　　吕罡缓缓的站了起来，双拳紧握，眸中同样战意沸腾。
<br/>　　“吕罡，你不能再败了！”
<br/>　　真枯半圣的声音，突然在吕罡的耳边响起，声音在充满了郑重。
<br/>　　在真枯半圣看来，吕罡可以失败，但只能败在炎澜天、扬天成等妖孽的手中，不能败在炎火的手中。
<br/>　　炎火在七大势力之中，只能算是顶尖的天才，并不算是妖孽天才，若是吕罡连炎火都打不过，那道宗的脸，真是要丢尽了。
<br/>　　吕罡闻言，心中感受到了一丝沉重的压力，随即他默默的点了点头，而后身形一闪，向站台上飞去。
<br/>　　身形落在战台上，吕罡和炎火，相隔千余里而立。
<br/>　　“吕罡，你不是很狂妄吗？现在就拿出你的实力，让我看看你有没有狂妄的资本！”炎火望着吕罡，嘴角勾起戏虐的笑容。
<br/>　　“相比于你，我已经非常谦逊了！”吕罡目光凌厉了起来，双拳之上玄力缭绕。
<br/>　　“你们道宗三大天才，以败其二，若你也败了，我觉得你们道宗可以退出会武了！”
<br/>　　炎火面带轻笑，眸中尽是揶揄之色，黑炎族和道宗一向不对路，所以他不会放过言语打击对方的机会。
<br/>　　“哈哈，的确是可以退出会武了！”
<br/>　　“顶尖天才都全败了，留下来还有什么意思？”
<br/>　　“又是一败涂地！”
<br/>　　“道宗就没有一个有实力的天才！”
<br/>　　不少黑炎族的弟子，听闻炎火的话，均是大笑了起来，满脸嘲笑之色。
<br/>　　“可恶！”
<br/>　　所有道宗弟子，均是心中暗怒，这些黑炎族的人，实在是太嚣张了。
<br/>　　但是，想到黑炎族之人的实力，众人均是满心的无奈，技不如人，只能忍受屈辱。
<br/>　　只能等它日实力强大了，再夺回颜面了。
<br/>　　真枯半圣、冯光长老等道宗的长老人物，个个脸色阴沉，非常的难看。
<br/>　　不过，他们也不能说什么，弱者永远都抬不起头，这是亘古不变的真理。
<br/>　　这些长老人物，此刻心中都有些后悔了，后悔接下这个差事，来到这里丢人现眼！
<br/>　　现在，他们只能祈求，祈求吕罡能大展神威，一举打败炎火，重振道宗气势！
