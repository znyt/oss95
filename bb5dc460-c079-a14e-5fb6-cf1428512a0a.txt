　　李浩然放下了手，这让秋亚子很是意外，感觉自己走也不是，打也不是，一时间说不出话来。
<br/>　　“让你们掌教来见我，我等的很烦，我耐心有限。”李浩然的语气还是那么轻，但是停在秋亚子耳朵里，却带着一种无法抗拒的压力。
<br/>　　秋亚子有些尴尬地盯着李浩然看了一会，然后转身，快步走进了通往后院的小门。
<br/>　　他一走，他的几个内门弟子就更尴尬了，本来几个人摩拳擦掌，想要教训一下眼前这个胆敢擅闯山门的家伙，结果……自己的师父秋亚子居然被一招击败，直接跑了，也没给他们任何关照，这几个内门弟子面面相觑，不知道该怎么办。只好也都傻站着不动。
<br/>　　李浩然根本没看他们，继续打开道经，看了起来。
<br/>　　过了有一盏茶的工夫，秋亚子回来了，这次来的不光是他自己，而是跟三个道士一起。
<br/>　　秋亚子的几个内门弟子，一看到他们四个人一起进来，当场震惊了。
<br/>　　这包括秋亚子在内的四个人，是桃花山掌教的四位弟子，不是监院就是住持，四人都是准天师牌位，平时各自收徒，明争暗斗，很少会有一起行动的时候。没想到为了一个不速之客，桃花山四大强者一起都出动了。不过，更加超过他们想象的还在后面：
<br/>　　这四人进门之后，立刻站成一排，中间让出了一条通道，一个身穿灰色道袍的老者缓缓走了过来。
<br/>　　“掌教大人！”
<br/>　　“师公！”
<br/>　　大伙吃惊地叫起来。他们知道，掌教大人的法力，离地仙境界也只差一步，最近恰逢乱世，干脆就闭门谢客了，在崇山之中闭关修炼。就算是他们这些内门弟子，也已经很久没见到这位老人家了。没想到居然为了面前这个贸然闯山的人出关了。
<br/>　　他们不知道的是，秋亚子在一招输给李浩然之后，立刻明白了自己跟他之间的差距，见他坚持要见掌教，也是没有办法。秋亚子心里明白，如果这件事掌教不出面，他们四大弟子一起上，怕也不是这个人的对手，没办法只好去闭关的山峰找掌教汇报。
<br/>　　掌教听说之后，也觉得此事蹊跷，不敢怠慢，召唤四大弟子一起过来了。
<br/>　　“贫道道号山河，不才摄桃花山掌教之位，适才听弟子说起，尊驾要见贫道，不知何事？”
<br/>　　这个年代新文化运动都结束了，普通人说话基本上都是白话，不过山河真人是前清过来的，又不怎么跟人接触，说话还是有点之乎者也的过去的味道。
<br/>　　李浩然目光从道经上收回来，落在山河真人身上，说道：“你是紫袍真君多少代传人？”
<br/>　　山河真人一怔，本能地看了一眼身边供奉的神像，最中间一尊就是紫袍真君，也就是桃花山第一代开派祖师。
<br/>　　“贫道的紫袍真君第十三代传人。”
<br/>　　“十三代，也不少了。”李浩然望着紫袍真君的神像，说道，“你们这神像不对，紫袍没有这么相貌堂堂，他是一只紫貂精，尖嘴猴腮，形容猥琐……”
<br/>　　此言一出，众人勃然变色。
<br/>　　山河真人涵养极好，听见他这么侮辱自己开派祖师，也忍不住了，摊开一只手，掌心紫气流动，是用符印召唤而来的紫薇天火，仿佛凭空生出，这一手，也算是他的绝学了。
<br/>　　李浩然瞥了他手掌一眼，摇头说道：“真是一代不如一代，紫袍当年位极灵仙，他的隔代传人，居然连地仙都不是……”
<br/>　　山河真人怒气冲冲地说道：“尊驾欺我可以，敢欺我祖宗，山人也敢一怒！”
<br/>　　当下抬起手掌，对着李浩然拍了过去。
<br/>　　李浩然轻轻抬起一只手，在空中横着抹了一下，灵力仿佛凭空生出，形成了一道结界。
<br/>　　山河真人的手掌打在上面，紫气火焰四溢而出，连身后一干弟子都为余波所动，定睛看去，李浩然纹丝不动，他面前的结界，在紫薇天火的炙烤之下，也是纹丝不动。
<br/>　　这一幕，深深震惊了山河真人。
<br/>　　“你难道不知道，紫袍真君的真身？”李浩然也有些意外，进而问道。
<br/>　　“你……你侮辱……”
<br/>　　“心随意动，紫气东来，若恒于咒，破于术，沉于符，起于山，未必有余，必出少阳而没于少阴……”
<br/>　　李浩然念到这里，住了口，望着山河真人。
<br/>　　山河真人听见他念的这段，顿时张口结舌，他身后的四大弟子也是张口结舌。
<br/>　　“你……你怎么会我桃花山内门心法口诀，你究竟是什么人？”山河道人吃吃地问道。
<br/>　　“我昔日在东华山下传道，紫貂听我传道三载，顿悟得道，这一段心法是我传于他，让他开宗立派。你桃花山镇山之宝翡翠紫金戒，也是我送与他，我是你祖师，今日见我，还不参拜？”
<br/>　　山河真人傻傻地看着他，嘴唇颤抖着：“您是……”
<br/>　　李浩然抬起左手，在空中划了几笔，有青气从指尖喷射，在他头顶上空形成了一只牛头的形状，灵气昂然，宛如神光显灵。
<br/>　　“您是……青牛祖师！”山河道人叫起来，“不可能，绝不可能！青牛祖师是数千年前的人……”
<br/>　　李浩然抬手如拈花状，中指轻轻一弹，头顶上空那尊青气所化的牛头，猛然间发出一声长鸣。
<br/>　　包括山河真人在内，所有人都感到心神震颤，双膝一软，一个个都跪在了李浩然的面前。
<br/>　　山河真人这才知道假不了了，对方真的是青牛祖师……
<br/>　　自己这桃花山的开派祖师紫袍真君，也不过是青牛祖师当年若干门徒之一，还不是最出色的那个。用一百年后的话说，只配有给青牛祖师提鞋的资格。
<br/>　　山河真人激动地痛哭起来，脑袋磕在地上，拜倒在李浩然脚下。
<br/>　　“不知祖师降临，弟子万死……”
<br/>　　李浩然脸上仍然是波澜不惊，说道：“带我去你的修所。”
<br/>　　山河真人的修所，位于桃花山最高的一座山峰上，并不是一般的密室，而是一个四面通风的小亭子，建立在悬崖顶上，从这里可以俯瞰桃花山的全景，风水走向尽收眼底。
