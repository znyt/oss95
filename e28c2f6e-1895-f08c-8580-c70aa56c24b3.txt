　　这嘶哑的声音，绝不正常，楚风眠知晓，这声音是一种故意修炼出来的声音。
<br/>　　实力稍弱一些的武者，听到这声音，都会直接头昏脑涨，不死也会疯狂。
<br/>　　不过如今鲜血魔尊等人，都是生死境巅峰的人物，听到这声音，只是感觉到有些不适，却并没有太多影响。
<br/>　　他们三人，一起便是踏入到了这宫殿之中来。
<br/>　　这宫殿，十分之大，在外面一眼看去都极为庞大，但是其中，更是另有乾坤，有着数十里的大小，居然是一处独立的空间。
<br/>　　曾经云羽天宫这等是无愧于九域巅峰的势力，居然是一座宫殿，都足以建立成如今这样另有乾坤的模样。
<br/>　　就在这宫殿的最中心，便是有着一座极为高大的王座，一道身影，正是坐在这王座上。
<br/>　　这人的身份，不需要任何思考，便是这所有地魔门弟子中的第一人，锁魂魔尊。
<br/>　　在楚风眠看向这锁魂魔尊的一刻，他的双眼，也是扫视着在场三人。
<br/>　　锁魂魔尊的目光，冰冷无比，一双眼睛极为清澈，似乎是足以看清任何一个人身上的所有底牌。
<br/>　　显然是这锁魂魔尊，也专门学习了一种独特的功法。
<br/>　　这扫视，在楚风眠的身上只停留了一下，便是离开，随后锁魂魔尊的目光便是看向了鲜血魔尊。
<br/>　　“鲜血，到底发生了什么事？”
<br/>　　锁魂魔尊开口，声音中便是蕴含有着无上威严，顿时鲜血魔尊的身躯，都是微微有些颤动起来，急忙开口道。
<br/>　　“禀告锁魂师兄，我在外面，寻找宝物的时候，突然碰到一个陌生的小子，这小子应该是刚刚进入到天羽神域的新人，我本来打算擒下他，献给锁魂师兄的话，可没想到这小子的实力极为强大，我都不是他的对手。”
<br/>　　鲜血魔尊惟妙惟肖的讲述起来。
<br/>　　“随后我便是发出玉符，然后杀狂，疯魔两位敢来，我们三人围攻，居然都不是那小子的对手，那小子手中的一把灵剑，锋利无比，我们远远不是他的对手，后来疯魔被斩杀，我跟杀狂才急忙逃了回来。”
<br/>　　鲜血魔尊一边说着，一边还展示这他胸口的伤痕，表示所言不虚。
<br/>　　“灵剑？据你所说，此人似乎是得到过颜古龙帝的洞府传承，这把灵剑，莫非是祖龙至尊剑？”
<br/>　　锁魂魔尊思考一下，眼神中都是露出了几分贪婪之色道。
<br/>　　“祖龙至尊剑，是龙族至宝，天级灵器，甚至是在无数天级灵器中，都位列上品的灵器。”
<br/>　　“在归墟天中，都不一定有着如此宝物，被一个小子得到，实在是太过浪费了，走，就由我去杀了他，夺取灵剑！”
<br/>　　锁魂魔尊，突然从王座上站了起来，他的身躯极为高大，足有三四米高，一站起来，俯视着众人。
<br/>　　“鲜血，杀狂，迷瞳，再次之前，你们先跟我进来，我们九魔城，来了一个贵人，你们也跟我一起来见一见。”
<br/>　　锁魂魔尊说完，直接向着宫殿深处走去，之留下他们三人。
<br/>　　“少主，这锁魂魔尊的宫殿深处，到底隐藏这什么，我也不知道，很有可能蕴含危险，怎么办？”
<br/>　　鲜血魔尊一道灵识，急忙悄悄对着楚风眠说道。
<br/>　　“无妨，进去就行，反正我也正好准备对付这锁魂魔尊，不管他到底要做什么，我都会让他死无葬身之地！”
<br/>　　楚风眠平静的开口道，他毫不犹豫，直接向着宫殿深处走去。
<br/>　　这宫殿，有着十几公里长，他们三人，无法动用遁光，一步一步走进去，也用了一段时间。
<br/>　　“轰！”
<br/>　　就在楚风眠身边，逐渐变得漆黑一片的时候。
<br/>　　突然无数禁制，就在他们三人的身边，纷纷出现，这禁制的目标，正是楚风眠跟鲜血魔尊二人。
<br/>　　至于那迷瞳魔尊，如今早已经是退后，似乎是早有准备，唯有他们二人，落入到了这禁制之中。
<br/>　　“锁魂师兄，怎么回事？”
<br/>　　楚风眠看到这景象，也是不动声色，有些惊讶的开口道。
<br/>　　“楚风眠，你不用装下去了，你的隐藏可以瞒过其他人，但是却不可能瞒得过我。”
<br/>　　就在楚风眠的开口之际，一道调笑的声音突然响起。
<br/>　　只见从黑暗之中，一道年轻身影，突然出现，这年轻身影，坐在宝车之中。
<br/>　　而站在他面前的，正是锁魂魔尊，如今一脸恭敬，为这年轻身影牵着马车，充当马夫。
<br/>　　“小魔主？”
<br/>　　楚风眠脸上显现出几分惊讶，随后他也索性不再隐藏，身形变化成真正的样子来，开口道。
<br/>　　“没想到你的速度，居然这么快。”
<br/>　　眼前这年轻身影，正是小魔主。
<br/>　　如今这一切，似乎都是由小魔主安排的。
<br/>　　“只能碰巧赶上罢了，差一点还真的让你，混进了我们九魔城中。”
<br/>　　小魔主看着楚风眠，脸上露出几分调笑道。
<br/>　　“你的胆子倒是够大的，居然是敢独自一人，混到我们九魔城来，如今倒是自投罗网。”
<br/>　　“小魔主英明，这小子的伪装，属下都没有看出。”
<br/>　　锁魂魔尊看着楚风眠，眼神中都是露出了几分杀意道。
<br/>　　“据说此人，在外面跟小魔主作对，如今属下就杀了此人，为小魔主先除一个麻烦。”
<br/>　　“少主？怎么办？”
<br/>　　鲜血魔尊的脸上露出了几分慌张。
<br/>　　本来计划之中，他们二人是打算接近到锁魂魔尊的时候，突然出手，出其不意直接将锁魂魔尊轰杀的。
<br/>　　可如今，却是楚风眠，落入到了陷阱之中来。
<br/>　　“少主？鲜血，你个废物，也敢来背叛我们九魔城？敢背叛九魔城的人，都只有死！”
<br/>　　说话之间，锁魂魔尊突然出手了，从他的手心中，一道漆黑铁索，突然出现，猛然一动，便是向着鲜血魔尊打来。
<br/>　　在这漆黑铁索的边缘，便是有着一座钩子，这钩子，似乎是足以将武者的魂魄，都给直接勾走。
<br/>　　“九幽锁！”
<br/>　　鲜血魔尊脸色大变，这锁魂魔尊一出手，便是动用了他的天级灵器，九幽锁。
<br/>　　九幽锁，不锁肉身，专锁魂魄，肉身的力量再大，在这九幽锁的面前，都没有任何的意义，根本无法抵挡。
<br/>　　鲜血魔尊连连后退，可是这九幽锁，已经是轰杀过来。
<br/>　　“在我的面前也打算杀人？可笑！”
<br/>　　站在一旁的楚风眠，突然冷哼一声。
