　　付元青显然对于此地颇为熟悉，他带着苏莫快速的前进，穿过了一片辽阔的平原，很快便来到了一处山脉地带。
<br/>　　这山脉占地极广，绵绵无绝，有无数低矮的山峰，已经茂盛的植被。
<br/>　　“阁下，这山脉之中，是道剑最多的地方，但也是争夺最为激烈的地方。”
<br/>　　伫立在山脉的边缘处，黑衣青年付元青凝视着苏莫，面色郑重的说道。
<br/>　　“以你的了解，此地是来的人最多的？”苏莫沉声问道，前面的那些考验，估计已经淘汰了一批人。
<br/>　　而能来到这一关的人，已经不是很多了，再加上分散各处，人应该不是很多了。
<br/>　　“起码有千人会陆续进入这山脉之中，非常危险！”付元青道，这一关的目标是道剑，所有人都会抢夺，只要碰到其他人，就难免会交手。
<br/>　　“你那知道，道剑一般都在什么地方？”苏莫沉声问道。
<br/>　　“这个就不得而知了！”
<br/>　　付元青摇了摇头，继续道：“道剑的出现，无迹可寻，有可能在任何地方，不过想要找到并不难，难的是如何避免其他人的抢夺！”
<br/>　　“按照你的意思说，我们找到道剑，也可能被别人抢去？”苏莫问道。
<br/>　　“是！”付元青点了点头，他知道苏莫很强，强大的过分。
<br/>　　但是，毕竟修为太低，遇到云沐、拓跋流明、千幻邪君等万古天骄，根本没有反抗之力。
<br/>　　“与其如此，我们只要找其他人就行了！”苏莫眸中精光一闪，既然是这种情况，那他还找什么道剑，只需要抢夺别人就行了。
<br/>　　“你是想要专门抢别人的道剑？”付元青眸光微闪，虽然苏莫实力很强，但是他却是并不看好。
<br/>　　因为这样的话，风险太大了，一旦遭遇强者，或者是多名强者围攻，那很可能陨落与此。
<br/>　　“你我一起！”苏莫郑重的说道，对方既然如此熟悉，他暂时不想要对方离去。
<br/>　　“啊！”付元青闻言，顿时脸色一苦，他跟着简直就是危险万分啊。
<br/>　　虽然他也是妖孽天才，但是有自知之明，以他现在的修为层次，能杀死他的人很多。
<br/>　　“走吧，带我去你认为有人或者有道剑的地方！”苏莫沉声说道。
<br/>　　“好吧！”付元青无奈的点了点头，但是，他的心中却是暗暗的决定，若是苏莫去抢夺别人，他一定要躲开，免得被人杀死。
<br/>　　随后，付元青带着苏莫，进入了山脉之中。
<br/>　　付元青虽然对于此地了解，但是，以往他也没有来过，只能按照他提前了解的讯息，去找寻道剑或者可能有其他人的地方。
<br/>　　“前方半刻钟的路程，有一座大峡谷，据说每一次万道剑冢开启，都有人在峡谷中得到道剑！”一边飞行，付元青一边指着前方说道。
<br/>　　“好，那我们就去大峡谷！”苏莫点了点头，这一次，他打定了注意，专门抢夺道剑。
<br/>　　不多时，他们两人便来到了大峡谷之中，这处峡谷长达数十里。
<br/>　　峡谷之中，两边岩壁之上，到处是一个个不大的洞穴。
<br/>　　这些洞穴很是幽深，不知道其内通向何处。
<br/>　　“这些洞穴之中，都可能有道剑，但是，也可能有其他人！”付元青道。
<br/>　　“嗯！”
<br/>　　苏莫点了点头，随即，他便落在了峡谷之中，找了一个地方，盘膝坐了下来。
<br/>　　“我们……就这样等吗？”付元青站在苏莫的身前，惊疑不定的问道。
<br/>　　“等！”
<br/>　　苏莫点了点头，他懒得去寻找，就在此地静等就是。
<br/>　　既然此地可能有道剑，自然有其他人来，说不定都在附近的洞穴之中，等有人出来，他再行动。
<br/>　　“好吧！”付元青无奈的叹了口气，随即，走到稍远的地方坐下。
<br/>　　他担心距离苏莫太近，万一苏莫遇到强敌，他可以立刻逃走，不至于被人杀死。
<br/>　　随即，俩人都不再说话，静静的等待了起来。
<br/>　　苏莫一边修炼，一边等待，他的修为已经颇为扎实了，距离突破的三品，也近在眼前了。
<br/>　　两人没有等待多久，便有人从不远处的一个洞穴之中出来了。
<br/>　　此人是为青年，各子不高，但很是壮硕，修为乃是三品高级，气息极为浑厚。
<br/>　　此人出来之后，看到峡谷之中盘坐的苏莫和付元青，顿时微微一怔。
<br/>　　随即，他轻笑一声，便大步而行，来到了付元青的前方。
<br/>　　“小子，有没有道剑，交出来我饶你不死！”矮壮青年冷视着付元青，沉声说道。
<br/>　　其没有找苏莫，因为，苏莫修为低微，在他看来苏莫应该是付元青的跟班。
<br/>　　付元青见此，面现惧色，随即目光望向了双眸紧闭的苏莫。
<br/>　　“你有没有道剑，交出来我饶你不死！”苏莫睁开了眼眸，淡淡的望着矮壮青年。
<br/>　　“嗯？”矮壮青年闻言，双眸一挑，大感有意思，一位二品的蝼蚁，居然敢出此言。
<br/>　　咻！
<br/>　　苏莫不待对方回答，立刻出手了，以指代剑，一剑疾刺而出，直击矮壮青年。
<br/>　　他深知，说再多的话都是无用，唯有绝对的实力，才能压块别人的意志。
<br/>　　一剑刺出，混沌剑气呼啸长空，恐怖的吞噬之力浩浩荡荡。
<br/>　　“什么？”矮壮青年见此，顿时心中大骇，只此一击，他便知晓苏莫的恐怖。
<br/>　　面对巨大的吞噬力量，他知道他逃离的速度，不可能快过剑气，所以立刻便出剑抵挡。
<br/>　　唰！
<br/>　　一剑出，浩大的剑气，化为了一柄巨大的石剑，瞬间和混沌剑气相击在了一起。
<br/>　　轰！
<br/>　　一声爆响，石剑瞬间崩溃，几乎是不堪一击，犀利的混沌剑气，瞬间便斩中了矮壮青年。
<br/>　　啊！
<br/>　　一声惨叫，响彻整片峡谷，矮壮青年直接被击飞了出去，撞击在不远处的岩壁上，染血喷洒。
<br/>　　好在，青年防御极强，身体仿佛化为了岩石，才保住了性命。
<br/>　　“付元青，搜搜他身上有没有道剑！”苏莫并未起身，淡淡的吩咐道。
<br/>　　付元青已经逃到了不远处，闻言，立刻前去查看。
<br/>　　矮壮青年，身受重创，再无反击之力，在付元青的一番搜索之下，身上的一柄道剑，被搜了出来。
<br/>　　“给你！”
<br/>　　付元青闪身来到了苏莫的身边，将道剑交给苏莫，望着面色平静的苏莫，他心中掀起惊涛骇浪。
<br/>　　眼前的青年，虽然修为很低，但是战力简直强的令人发指。
<br/>　　矮壮青年失去道剑之后，面上带着一丝不甘，随即急速离去。
<br/>　　“不错！”苏莫给了付元青一个赞赏的眼神，随即收起了道剑。
<br/>　　这柄道剑，弥漫这阴暗的气息，属于另外一种剑道。
<br/>　　现如今，他已经得到了两柄道剑，集齐十柄道剑，乃至更多的道剑，轻而易举。
<br/>　　嗖嗖嗖！！
<br/>　　或许是此地的动静，引起了其他人的注意，仅仅是过了几个呼吸的时间，便有道道犀利的破空声来临。
<br/>　　苏莫的眼眸，顿时一凝，因为，他感受到了慕容雷利的气息。
<br/>　　“很好！”苏莫的嘴角，勾起了一丝冷厉的笑容，此人上次追击他到极空城，现在该付出代价了。
<br/>　　他站了起来，目光透过峡谷，望向远处的天际。
<br/>　　苏莫的面色，稍稍凝重，因为来人是一群，他不得不慎重。
<br/>　　毕竟，慕容雷利是终极神王殿之人，和很多终极神王殿弟子在一起。
<br/>　　而终极神王殿之中，有一位四品修为的韩生，肯定无比的强大。
