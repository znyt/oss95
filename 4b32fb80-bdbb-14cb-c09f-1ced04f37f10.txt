　　感受到他掌心传来的温暖，美杜莎眸子闪烁一下，心中有些异样，脸颊上竟不可思议的浮现两抹红晕，芳心跳动频率加速。
<br/>　　“我们什么时候动手？”
<br/>　　望了一眼窗外的天色，慕清霄皱了皱眉，道：“在过两个时辰，夜晚方便动手，而且有利于你脱离他们的追击。”
<br/>　　微微点头，美杜莎望着他俊逸的面容，不禁陷入沉思，她的男人必须是强者，最起码要比她强。
<br/>　　否则，凭什么征服她。
<br/>　　而面前的青年，拥有绝强的实力，处事又小心谨慎，对自己的女人又温柔，真是找不出一丝的瑕疵。
<br/>　　...
<br/>　　...
<br/>　　随着时间的推移，夜幕降临，皎洁的月光倾洒而下，内院变得比白天清静不少。
<br/>　　卧室中，望着坐在桌前的佳人，慕清霄走上前，道：“仙儿，我有事要出去一趟。”
<br/>　　“你要去哪？”
<br/>　　听他说要离开，小医仙有些紧张，她早就习惯了待在慕清霄身边，没有他的地方，感觉不到一丝的安全感。
<br/>　　见她紧张的模样，慕清霄伸手捏住她小巧的鼻子，道：“就在学院里，你安心修炼就行了，我很快就回来。”
<br/>　　将他作怪的手从小鼻子上拍了开来，小医仙柳眉一皱，满含期待，道：“我想跟你一起去。”
<br/>　　“......你暂时帮不到我，乖乖呆在这里，听话。”慕清霄语气温柔的说道。
<br/>　　小医仙抿了抿粉唇，想要说些什么，却不知道如何开口。
<br/>　　她现在不过是一名小小的九星斗师，迦南学院中比她强的不知凡几，如何能帮上他的忙。
<br/>　　想到此处，美眸中闪过一丝不甘，一股浓浓的自卑感由心而生，或许自己根本没有资格与他在一起也说不定。
<br/>　　察觉到她的情绪波动，慕清霄轻轻的将她揽入怀中，道：“别太想多了，你可是我的女人，不比任何人差。”
<br/>　　闻言，小医仙感动至极，趴在他的怀中，幸福感犹然而生，使劲的点头。
<br/>　　“我一定努力修习，尽量能帮上你的忙。”
<br/>　　慕清霄俯下身子，一把吻住她的粉唇，小医仙玉手抓着他的衣服，笨拙的迎合，彼此交缠，深情拥吻。
<br/>　　片刻后，唇齿分离，捏了捏她粉嫩的脸颊，慕清霄瞳孔中浮现三颗勾玉。
<br/>　　勾玉融合，化作复杂的原子图案，随后在她身上设下一道极强的瞳术后，便转身离去，融入夜色之中。
<br/>　　...
<br/>　　...
<br/>　　离开居住区域后，慕清霄收敛全身的气息，周身空间一阵扭曲，便消失在原地。
<br/>　　片刻后，一颗巨树枝干上方，空间一阵扭曲，他的身影已经出现在其上。
<br/>　　此时的慕清霄，早已换上一袭黑袍，黑袍将全身包裹的一丝不露，站在巨树的阴影中，一般人根本察觉不到。
<br/>　　在视线前方数十米处，便是凹陷的地形，露出地面一截的天焚炼气塔矗立在黑暗之中。
<br/>　　由于白天修炼疲惫，八成的学员都选择返回，养精蓄锐后在来苦修，因此夜晚，塔内十分的幽静。
<br/>　　还没有到闭塔的时间，大门依旧敞开，慕清霄朝四周看了一眼，眨眼间，身影已经消失不见。
<br/>　　进入塔内，一股炽热席卷而来，慕清霄并未停留，闯过低级修炼室，向练气塔中央位置掠去。
<br/>　　在天焚炼气塔中，想要通往下一层，就必须通过护卫把守的大门，而且护卫都是斗王级别。
<br/>　　而且，每一层都有一位斗皇隐藏在暗处，所以想要穿过护卫与长老重重把守的大门，前往练气塔底层，完全是不可能的。
<br/>　　但天焚炼气塔中，有着一条唯一的通道，无人敢把守，也无人敢从那里前往最底层，但慕清霄就是个例外。
<br/>　　没错，那就是天焚炼气塔最中央处，深不见底的黑洞。
<br/>　　长老们都把守在黑洞周围，但第一层的杨长老，不过四星斗皇的实力，想要穿过他的防守，轻而易举。
<br/>　　只要穿过杨长老的防守，就能从黑洞中前往最后一层。
<br/>　　到达最后一层后，并不是说已经接触到了陨落心炎，在最后一层还设有一层强大的封印，只有破开封印，才能前往更深处。
<br/>　　而最后一层封印所在，肯定有迦南学院的严加防守，最起码也是斗皇巅峰的强者把守。
<br/>　　而这里，就需要在破开封印后，美杜莎暂时将守塔长老们引开，哪怕只有几秒的时间，慕清霄都能乘虚而入。
<br/>　　穿梭在修炼室之间，隐约能看见，在这个时间点，依旧有学员在其中苦修。
<br/>　　这时，美杜莎清冷悦耳的声音在耳边响起：“如果夺走陨落心炎，练气塔的功效，恐怕也会随之消失吧？”
<br/>　　闻言，慕清霄神秘一笑，道：“这可不一定呢。”
<br/>　　对于他的话语，美杜莎不明所以，接着道：“不管怎么说，陨落心炎丢失，迦南学院一定会追查到底，你想好对策了吗？”
<br/>　　“不必担心，早就想好对策了，我想迦南学院不仅不会追究，估计还会反过来感谢我。”
<br/>　　当然，慕清霄就是那种做好事不留名的人，感谢就不需要了。
<br/>　　若是他记得没错，练气塔底下除了一枚诞生灵智的陨落心炎，也就是冲击封印的陨落心炎。
<br/>　　还有着另一枚，刚成型没多久的陨落心炎，也就是说，塔底总共有两枚陨落心炎。
<br/>　　慕清霄只需要夺走其中一枚就行，至于另一枚，就留给迦南学院吧，这些老家伙可是挺难缠的。
<br/>　　“感谢你？”袖口中，淡紫色瞳孔微微一愣，仿佛自己听错了一般？
<br/>　　他居然说，夺走陨落心炎后，迦南学院会感谢他？将他杀之而后快还差不多吧？
<br/>　　几分钟后，慕清霄收敛气息，穿梭于修炼室之间，避过守塔的杨长老，最终来到了练气塔中央位置。
<br/>　　那迎面而来的炽热气息，使得美杜莎有些不舒服，但她却没有吭声，静静的趴在他的袖口中。
<br/>　　时间紧迫，慕清霄没有丝毫犹豫，纵身一跃就跃进黑洞中，旋即，一股高温扑面而来，手一挥，便将高温挡在外面。
<br/>　　除了最后一层，练气塔黑洞其他层面并没有设下封印，慕清霄以极快的速度掠下。
