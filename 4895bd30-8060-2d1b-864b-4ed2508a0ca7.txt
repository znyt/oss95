　　“是他？”
<br/>　　萧逸眉头轻皱，语气中，带着半分惊咦。
<br/>　　东方铁云，倒是一愣，“怎么，易霄副殿主你认识那位传奇？”
<br/>　　事实上，萧逸虽已不将毒袍放在眼中。
<br/>　　但毒袍那逃命的本事，他还是感到惊讶的。
<br/>　　毕竟，能将他这个高品炼药师都给瞒过去，成功诈死，这可不是普通手段。
<br/>　　而萧逸想起来，之前无论是东方铁云，还是狂妄如龙啸，在说起毒袍从那位传奇强者手中逃脱之事，都无不面露凝重与忌惮之色。
<br/>　　故萧逸好奇下，特意问了一句。
<br/>　　只是没想到，竟然是他。
<br/>　　此时，东方铁云看着萧逸。
<br/>　　萧逸笑笑，不语。
<br/>　　“好了，没别的事，我们三人便告辞了。”灸炎拱拱手，道。
<br/>　　“好。”东方铁云拱拱手，“他日有空，三位再来我六龙地域主殿，鄙人无任欢迎。”
<br/>　　龙啸亦拱拱手，不过，只看向灸炎，“灸炎殿主，慢走，不送。”
<br/>　　事实上，二人是来送行的。
<br/>　　不过，在龙啸看来，有资格让他送行的，也就只有灸炎一人罢了。
<br/>　　他看向萧逸的脸色，仍旧是冰冷。
<br/>　　萧逸也未理会。
<br/>　　他当然知晓龙啸是什么意思。
<br/>　　他毕竟夺了六龙地域三条龙形煞气，龙啸会有好脸色才怪。
<br/>　　不过，对于吸收了毒煞和血煞，萧逸可丝毫不觉有半分内疚。
<br/>　　之前，若非龙啸，他岂会受龙炎反噬，在湖底遭受痛苦折磨一月不说，还险些送命。
<br/>　　那些痛苦和折磨还好说，萧逸也不太在乎。
<br/>　　倒是他因此而把自己乾坤戒内的9成珍稀天材地宝都耗了个尽。
<br/>　　能让他萧逸看上眼的天材地宝，岂会是普通之物。
<br/>　　而最重要的是，他在湖底养伤一月，也白白浪费了他一月时间。
<br/>　　还有之前龙啸的处处阻挠和为难，愣是让他在六龙地域这里为了剿杀邪修，逼毒袍出来，而又浪费了近一月时间。
<br/>　　算起来，单单在六龙地域这里，就耗了他两月时间。
<br/>　　原本，他离开剑帝碑后，距离古帝之墓开启，就只剩下六月多些，不到七月的时间。
<br/>　　时间本就紧迫，他得捉紧实力击杀邪修，还有以此当作历练的提升修为与实力。
<br/>　　按他之前一开始的横扫速度，短短时间就不知道横跨了多少地域，剿杀了数之不尽的邪修分部。
<br/>　　而在六龙地域这里，却因邪修，生生让他浪费了整整两月时间。
<br/>　　至此，他剩下的提升实力的时间，最多不到两月。
<br/>　　他不修邪道、雷到和木之一道，奈何不了其它三条龙形煞气。
<br/>　　否则，他把另外三条龙形煞气一并抢了的心都有。
<br/>　　至于那条风煞，被雾妖夺了去。
<br/>　　萧逸同样不在意。
<br/>　　雾妖虽面容狰狞，却远比那些拥有好看的皮囊却心思龌蹉之辈好得多。
<br/>　　且雾妖，是一只陪在他身边的同伴。
<br/>　　雾妖想要那条风煞，那便要了。
<br/>　　龙啸若想抢回来，他萧逸也不介意在六龙地域再战一场。
<br/>　　之前，龙啸总说，六龙地域有六龙地域的规矩。
<br/>　　又说，规矩由他龙家定下，他龙家凌驾于规矩之上。
<br/>　　说白了，拳头大就是规矩。
<br/>　　既如此，萧逸便也凭此告诉他，如今，他龙家的拳头，压不住他萧逸。
<br/>　　嗖…嗖…嗖…
<br/>　　灸炎、唐音、三人，与东方铁云告辞一番，御空飞离。
<br/>　　不过片刻，三人已消失在远方。
<br/>　　……
<br/>　　“好快的速度。”萧逸皱眉看了眼灸炎。
<br/>　　此刻，他紫炎火翼疾速抖动，速度全开下，还是比灸炎慢了许多。
<br/>　　这就是传奇强者的速度，快得惊人。
<br/>　　“易霄副殿主，我带你飞一趟吧。”说罢，灸炎来到萧逸身旁。
<br/>　　灸炎一手捉住萧逸，一手捉住唐音。
<br/>　　嗖…恐怖的速度，宛若跨越空间般疾速。
<br/>　　灸炎，化作一道火光，穿梭于天际之中。
<br/>　　萧逸心头暗暗惊骇，却也暗暗思索。
<br/>　　看着灸炎，他不禁想起了承风殿主与横天殿主二人。
<br/>　　他们虽不是同一殿的武者，但皆是上古八殿中在总殿任职的主殿。
<br/>　　故三人有一定的可比性。
<br/>　　萧逸想起来，当初初见承风殿主与横天殿主二人时，只觉二人远比武道大能强大。
<br/>　　或许，二人是绝世之境。
<br/>　　可后来，萧逸的感知愈发强后，又隐隐觉得二人怕是不止这个实力。
<br/>　　特别是他踏过极致剑修的‘杀戮之路’时，这种感觉更为明显。
<br/>　　不过，后来许久未遇到这两人了，萧逸也不好判断。
<br/>　　今日，遇到这灸炎殿主，萧逸却陡然从灸炎殿主的气息强弱中，依稀能判断出横天殿主二人的修为了。
<br/>　　对，依稀。
<br/>　　在他的感觉中，横天殿主与承风殿主的气息，还有给他的感觉，竟是远远强过了现今灸炎殿主给他的感觉。
<br/>　　至于准确强了多少，萧逸却说不准。
<br/>　　所以，只能是依稀的判断。
<br/>　　但萧逸又可以确定，横天殿主二人，绝对比灸炎殿主给他的感觉要浓烈许多，强大许多。
<br/>　　“易霄副殿主。”
<br/>　　这时，灸炎的一声话语，打断了萧逸的思索。
<br/>　　“嗯，怎么？”萧逸问道。
<br/>　　“呵呵，没什么，只是想聊聊罢了。”灸炎笑笑。
<br/>　　“之前，我和唐音去追毒袍，还以为你又会跑得无影无踪。”
<br/>　　“还好你没走，不然我二人还真无法向总殿主交待。”
<br/>　　萧逸白了二人一眼，道，“怎么，猎妖殿要通缉我不成？”
<br/>　　“当然没有。”灸炎笑道。
<br/>　　萧逸没好气道，“那不就得了，易某无缘无故跑什么？”
<br/>　　“以防意外罢了。”灸炎和唐音二人笑笑。
<br/>　　“毕竟易霄副殿主你有前车之鉴。”
<br/>　　萧逸暗暗无奈。
<br/>　　这二人所指的，自然是大概一年前，猎妖殿发出的‘考核’邀请。
<br/>　　不过，那时他正以自己的正常的身份，去了风刹殿和修罗殿总殿参加考核之路。
<br/>　　而且，他也压根不知道猎妖殿总殿给他发了考核。
<br/>　　也是数月前，他在衡元地域时，听衡元地域的主殿主说起，才知晓此事。
<br/>　　……
<br/>　　时间渐渐过去。
<br/>　　一天半多些，三人来到了猎妖地域。
<br/>　　“易霄副殿主，猎妖地域，到了。”灸炎指了指前方。
<br/>　　“前方不远，便是猎妖殿总殿。”
<br/>　　“猎妖殿总殿。”萧逸看向远方，眼中尽是期待之色。
<br/>　　对于猎妖殿，萧逸有着远比其它殿更深的感情。
<br/>　　毕竟，他当年率先加入的，便是猎妖殿。
<br/>　　他当年，亦是以猎妖师的身份，初出茅庐。
<br/>　　猎妖殿总殿，这座大陆之上最古老的猎妖殿，他终于要见识一番了。
