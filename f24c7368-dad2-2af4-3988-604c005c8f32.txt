　　第741章
<br/>　　在渡口翘首以盼的几人赶忙站起，就见程大雷浑身湿漉漉的，头发还在往下滴着水。
<br/>　　“大当家，你是如何出来的？”刘发财问。
<br/>　　“哈，小小一座汝南城算什么，本当家只要略施小计，他们便拦不住我。”
<br/>　　“那是，那是……”刘发财道：“不过，大当家你是如何出来的？”
<br/>　　“现在不是说话的时候，追兵差不多快到了，咱们快些走。”
<br/>　　“是是。”
<br/>　　徐神机连忙答应一声，他没再打破砂锅问到底问到底。反正程大雷身上稀奇古怪的事情也不差这一件两件。
<br/>　　崔白玉上上下下打量程大雷一遍，心中也是百思不得其解。程大雷究竟想了什么法子，能从汝南城脱身，还可以做到毫发无伤。
<br/>　　程大雷将黑牛牵到船上，徐神机几人也先后上船，放开船锚，乌篷船顺流而下。
<br/>　　魏牧冲领着士兵出城的时候，已经失去了程大雷的踪迹。带着人在城外折腾了一夜，结果是一无所获，只能带着士兵回城。
<br/>　　而此时，程大雷已经乘船向东，经过了汝南城。
<br/>　　……
<br/>　　官道上，一少年郎牵着匹黑蹄斑点马，天气炎热，他走得汗流浃背，抓起酒囊咕咚咚灌了一口，仍旧解不开身上的燥意。
<br/>　　前面看到一棵大树，少年郎走到树下，准备歇一歇，等凉快些再赶路。
<br/>　　这少年正是从平野城悄悄溜出来，有平野虎之称的平野陈梦。从平野到汝南路途并不算近，这一路走来也是受了很多苦。陈梦心里也有些焦急，谁知道到了汝南城程大雷是死是活。假若自己到的时候，他已经死了，自己白费这场辛苦自不必说，也没有了露脸的机会。
<br/>　　一个少年对每一个出名的机会，都像飞蛾见火般趋之若鹜。失去了这次出头的机会，还不知下次要等到什么时候。
<br/>　　心里一会儿想着自己击败程大雷，天下扬名，天下美女蜂拥而来。一会儿又想着白白辛苦一场，竹篮打水一场空，心情又有些郁郁。
<br/>　　少年心性，朝夕万变。
<br/>　　正这时候，只见官道上一队人远远过来。他们骑着高头大马，身披黑甲，黑甲背心有鱼龙花纹。
<br/>　　陈梦皱起眉头：这应该是长安的鱼龙卫。于是他便多看了两眼，只见对方为首者是个年轻军官。披头散发，擎一杆长枪，披头散发，眉目竟极其英俊。
<br/>　　是谁呢，年纪轻轻就能统领鱼龙卫？
<br/>　　想到这里，陈梦有些不开心。大家都是同样年纪，凭什么你已经可以执掌天子近卫，而自己却仍旧是籍籍无名。
<br/>　　呸，肯定是王孙贵族，仗着父辈发迹的。
<br/>　　不过转念一想，自己也算是王孙贵族，在这一点上似乎没资格鄙视对方。
<br/>　　手不由自主的摸着斧头，想着自己要是和对方碰一碰，应该不至于打不赢对方。
<br/>　　恰好，那年轻军官也往他身上看了一眼，二人目光对视，都很冷漠，眼底却又都藏着火。
<br/>　　对方打马而过，没有停留。陈梦靠在大树上，手摩挲着斧头。心想自己这次只要击败程大雷，就算是正式出世，天下英雄有了自己这一号。日后，就不用把一个小小的鱼龙卫首领看在眼里了。
<br/>　　心里琢磨着，脑袋越来越沉，陈梦昏昏欲睡。
<br/>　　谁料那队官兵竟去而复还，为首那军官骑在马上问：“喂，娃娃，你是姓程么？”
<br/>　　这军官自然就是从长安出来的马超马孟起，他注意到陈梦还是因为对方的一双斧头。他没有见过程大雷，据说长得很丑，这少年不就是很丑么。据说是用斧头，但谁知道他用的是一柄还是两柄。
<br/>　　陈梦一听他称呼自己『娃娃』，就有些不开心。更生气的是对方的态度，本少爷是堂堂平野城城主的公子，你怎么就不能毕恭毕敬的下了马，称呼一声『爹爷』。竟然敢在马上对自己大呼小叫，眼里还有没有王法了。
<br/>　　他昂起头：“你是那个，我和你爷爷一个姓，你问这个干嘛？”
<br/>　　马孟起不怒反笑，看来是八九不离十，管你是不是正主，先打过再说。
<br/>　　“好娃娃，来来来，让你拿上兵器，骑上马，马爷爷教教你做人。”
<br/>　　陈梦长身而起，晃了晃肩膀：“来来来，小爷怕了你就是你养的。”
<br/>　　二人拉开架势，各擎兵器，马孟起冲身后一挥手：“你们都不需插手，让我好好教训教训他。”
<br/>　　关于马超，大家都知道他来历神秘，但谁也知道他是李乐天重用之人。日后必然有大好前程，而且本事惊人。
<br/>　　“马将军放心，弟兄们帮你压好阵，这小子跑不了。”
<br/>　　马超一擎长枪，冲陈梦点点头：“来。”
<br/>　　陈梦手持双斧，眼神变得犀利干净，他同样点点头：“来。”
<br/>　　二人双腿夹着马，互相绕着圈子，忽然眼神一炙热，向对方狂奔而去。
<br/>　　两马相错，结结实实的撞在一起，互相打成一团。陈梦擎两柄三十八斤大斧，但在他手中却轻若无物，使得是十八路砍柴斧法。
<br/>　　马超的招式更蛮横些，一杆铁枪在手里挑绕绷刺，攻得犀利无比。
<br/>　　二人本来都没看得起对方，打起来方才知道棋逢对手将遇良才，都是少年好胜，都想结结实实和对方碰一场。
<br/>　　打得越来越凶险，马超的随从渐渐看得目不转睛。他们佩服马超的武艺，同样知道对面的少年不简单。
<br/>　　就在这时候，从西边过来一行人。中间一辆马车，另外两人一人骑黑马，一人骑头猪鼻骡子。
<br/>　　这一行人自然是从汝南城过来的程大雷，他们弃船上岸后，就找了一辆马车，沿着官道过来。
<br/>　　看到官道旁的打斗，程大雷停住脚步，看得津津有味。
<br/>　　他忍不住查看二人的信息，确认对方是马超后，忍不住倒抽一口凉气。
<br/>　　原来是他。
<br/>　　更令他吃惊的，对面的少年竟然能和马孟起战个不相上下。但他的名字自己从未听说过，应该是这个世界的土著。
<br/>　　帝国还是有高人呐，自己也不可小觑了天下英雄。
