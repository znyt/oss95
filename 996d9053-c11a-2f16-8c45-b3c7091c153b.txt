　　“话说，我是不是也该弄一个分身之类的东西。如今自己一人在洪荒之中行走，很是不便，弄个分身也是极好的。而这株混沌灵根就是不错的选择，没有什么比这更好的了。”
<br/>　　元始看着混沌空间中的这株混沌灵根，如此想到。
<br/>　　想做就做，元始平复心情，端坐于巨树之前，默默静心，运转元神功法，凝聚元神之刀，强忍着元神受撕裂灵魂般的痛苦，将自己已经快要形成界域的元神斩出三分之一大小。同时，又将自己所有记忆复制一份交给这份元神，之后直接将此元神投入到巨树之中。通过那份元神与本体的联系，控制其与混沌灵根融合。
<br/>　　同时，两者一起运转《神元九转》，修复元神的同时，也炼化这混沌灵根。元神涌动，混沌之气也在缓缓融入巨树之中，帮助修复灵魂。
<br/>　　如此，时光匆匆，一元会十二万九千六百年已经过去！
<br/>　　混沌之气依旧在不断融入巨树之中，如今只剩四分之一了，混沌空间就快要到极限了。
<br/>　　“不行，再这样下去，因果莲台的混沌空间就快要破裂了，而且，这巨树也不会如此快的化形，需前往混沌一趟。”元始睁开双眼，看着眼前已经到了极限，欲摇摇欲坠的混沌空间，如是说道。
<br/>　　元始一步踏出混沌空间，再次回到不周之顶，看着眼前熟悉的景象，元始不再耽搁，快速向山下赶去。
<br/>　　路过九大罡层天幕，元始不再停留，顺着来时之路，飞向山下。不知是不是错觉，元始总觉得这不周山上的威压已经减轻了许多，不再像来时一般强大了。元始因为有事，所以并没有太过在意，况且，这些天地威压减轻再正常不过了。盘古并不能一直如此，总有一天这些威压会全部消散。
<br/>　　一路之上，元始发现不周山上已经不复之前光秃秃的、只有山石的景象了，现如今，山上已经有仙草灵植之类的生灵了，只不过还很稀少而已。这让元始不得不感慨，洪荒之中，时间真不值钱啊，他感觉并没有做多少事情，时间就已过去如此之久了。
<br/>　　一路之上，元始也遇到了一些先天三大族之中的麒麟一族的成员，期间发生一些不愉快的事情。不过，对上元始，只能算他们倒霉，元始可没有手下留情的想法，洪荒之中，不是你死就是我活，他才不管你是麒麟还是神龙，惹毛了他，不管是谁，都要付出代价。
<br/>　　麒麟，那又如何，元始会怕他们吗？以如今元始的实力，在这没有准圣、没有圣人的洪荒天地，已经不惧任何对手了。大家比的就是大罗金仙境界的积累与大道法则的领悟，对于这些，元始更不惧任何人。
<br/>　　此次前往混沌虚空，元始不再像之前游玩洪荒一样，一步一步，慢慢散步，而是运用上了飞行之术，虽然没有空间法则那般厉害，但也差不到哪里？一瞬间千万里，比起一个跟头十万八千里快了不知多少！
<br/>　　混沌虚空位于天之尽头，而不周神山位于洪荒最中心，想要到天之尽头，距离并不算短，要是平常的大罗金仙，需要飞数百年。但这对于元始来说，只是数十年的时间而已。
<br/>　　来到天之尽头，元始上了洪荒星空，如今还没有势力霸占星空，并不像后世天庭的势力范围，有星君常年驻守在此。
<br/>　　来到洪荒星空，入眼所见是一片茫茫的星河，这片星河之中并没有星辰之精，十分贫瘠，并没有什么利用价值。
<br/>　　元始也没有多做停留，直接向着混沌虚空而去。越靠近混沌，混沌之气就越多，刚靠近混沌元始并没有感觉什么，好歹自己的肉身也算比较强悍了。可是进入混沌之中，元始才发现事情并不是如此简单，这混沌之气无时无刻不在侵蚀着自己的真身，对比之前在不周山上的磨难还要强大许多。
<br/>　　不过，元始的玄黄不灭之体也不是吃素的，虽然它还没有小成，但毕竟修炼到第三重了，达到了准圣级别的强度了。这些混沌之气的侵蚀，元始还是能够忍受的。
<br/>　　这让元始不得不感叹，还是混沌魔神真身好啊，不用担心混沌之气的侵略，而且还混得如鱼得水。不过，那些如都已成过去了，自己的玄黄不灭真身大成之后，一点也不比混沌真身差。
<br/>　　经过不停的向着混沌方向飞去，元始终究还是到了洪荒与混沌的交界处，这地方好像天然的纽带一般，让洪荒之中没有大量混沌之气涌入，反而有不少混沌之气被转化为先天灵气进入洪荒天地。
<br/>　　元始知道这是因为盘古开天之时在洪荒最外围加了一层“防护罩”，它既能够阻止混沌之气的进入，也能防御混沌外敌，而且它并不妨碍洪荒之中的生灵前往混沌。
<br/>　　元始迈步踏入混沌之中，混沌还是一如既往，灰蒙蒙一片，无边无尽，苍茫无垠。感受到混沌之气包裹着自己的真身，元始既感到熟悉的气息，也感到有些难受，自己终究不是之前的那个混沌魔神了。
<br/>　　不过，这些都在元始承受范围之内，与想象中一样，他自己还是能够控制一些混沌之气的，不至于被它们同化。
<br/>　　快速向着混沌空间深处而去，元始并没有放出因果莲台与混沌灵根，而是继续往深处走去。直至走了不知多远，到了混沌深处，元始才放出因果莲台，任由它吸收此中混沌之气，弥补混沌空间。
<br/>　　“轰隆隆！”
<br/>　　因果莲台一出现在混沌之中，就发出剧烈震动，混沌之气形成一个巨大旋涡狂涌而来，注入到因果莲台之中。
<br/>　　靠！看着狂涌而来的混沌之气，元始连忙闪身进入元始印之中，再不躲开，自己就要被这狂暴的混沌之气冲刷的连渣渣都不剩了。先前还能承受，是因为混沌之气并不暴动，而如今，呵呵……
<br/>　　因果莲台之中，混沌灵根感受到源源不断的混沌之气，兴奋的摇摆起来，就要吸收这些混沌之气，但，它突然停了停。一道灵光闪现，就已消失在混沌空间。
<br/>　　等再次出现，巨树已经出现在混沌虚空之中，与莲台相隔数万里。顿时，两个巨大的混沌旋涡涌起，一个是因果莲台在吸收混沌之气，一个是巨树在吸收混沌之气。
<br/>　　如此，时间眨眼间便已过十个元会，百万余年转瞬即逝！
