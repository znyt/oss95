        &lt;h3&gt;第三卷第三百零三章  分析&lt;/h3&gt;
<br/>
<br/>        寒伯亲自去机场接安男一家人，然后送回杭州。
<br/>
<br/>        寒伯安从美洲回来的这一段时间一直都没有回寒家老宅，有事情都是和父亲电话联系，寒老爷子现在倒是对寒伯安上心了，一天不打电话他的心里就慌乱得很。
<br/>
<br/>        寒冰玉和落雪在美洲有惊无险，其间寒伯安也一直信息全无，陈子昂又带走了安男，让寒老爷子的心七上八下，他觉察出了不同寻常，暗中开始调查他身边的人。
<br/>
<br/>        苓希一直在他们家里呆着，表现得很乖巧，安男一走，苓希也就没有必要针对谁，况且有消息传来，寒伯安他们已经签了合约，苓希虽然没有功劳但是有苦劳，苓希受到了表扬，还是让在她寒家多呆一段时间，看看情况，另外让苓希将注意力放在寒伯安的助理李长卿的身上。
<br/>
<br/>        苓希觉得很奇怪，为什么要将注意力放在一个男助理身上？
<br/>
<br/>        寒伯安在西城陪了安男几天，飞回上海见了陆宇霆夫妇，这一切寒老爷子都是知晓的，只是不知道寒伯安有什么打算。
<br/>
<br/>        他们家的股份被寒老太爷直接给了俩个大的，老三只有很少的一些股份，这让寒老爷子心里一直觉得愧对老三，对苓如兰母子就多了一份宠爱，也对苓家照顾有加，苓家所做之事只要不太过，他都睁一只眼闭一只眼，原本苓如兰想将侄女苓希嫁给老大寒伯安，伯安不愿意，老爷子也是不太愿意，他希望给寒伯安找一个门当户对的人家，他和落家关系好，他不存在借力，但是到了寒伯安这一辈又当别论，寒伯安娶了安家的安男，虽然安家不是大富大贵之人，但是安家的关系网很大，安林曦的子弟遍天下，随便拉一个出来都是响当当的人物，所以寒老爷子是默许了的。
<br/>
<br/>        寒伯安接了安男没有回寒家老宅，寒老爷子心里很担心也很不舒服，便给寒伯安打电话：“阿琪，你这一段时间去哪里了？”
<br/>
<br/>        “美洲啊，从美洲回来就去了西城，现在在杭州呢。”寒伯安回来是给寒老爷子打过电话汇报过行程的。
<br/>
<br/>        “你不要和一些不三不四的人来往，安全重要。”寒老爷子叮咛着寒伯安，言语里透露着关心。
<br/>
<br/>        “西城有业务，在加上安男和陈子昂都在。”寒伯安也听出寒老爷子的关心，便出言解释。
<br/>
<br/>        “你什么时候回来？要不下午回来吃饭？”寒老爷子试探着问。
<br/>
<br/>        “我今天下午就不回去，下午陪安男他们，明天下午回去。”寒伯安也是刚到安家，正在和安林曦说话。
<br/>
<br/>        安林曦是经济学家，代凉是社会学家，俩个大儒正在和寒伯安讨论社会形势和经济走向。
<br/>
<br/>        “就目前的态势所见，京城吴家、陆家、张家看似一体，实则支离破碎，他们每个人都有自己的想法，因为他们都认为他们的孩子很优秀，而且他们手中都掌握着大部分经济。
<br/>
<br/>        就拿原来的陈家，在一夜之间分崩离析，被落家和你们家吞噬，如果没有外界的帮助，陈家也只是损失一些经济利益，而不是整个覆灭，如今的陈家真的是回天无力了，而陈家和吴家关系很好，所以陈家之事肯定会有外界的作用。”代凉对寒伯安说。
<br/>
<br/>        “落家和寒家之前从来都不卷入任何势力之争，现在的落家和吴家即将打破平衡。寒家将会是被拉拢的对象，如果拉拢不成或许毁灭。”安林曦淡淡的插话。
<br/>
<br/>        “吴家和陆家是姻亲，却是旁支，而张家和陆家是妥妥的姻亲。还有，落白已经回归张家，落白是养子，落雪是亲生女儿，所以你就处在一种尴尬的境地，左不得右不得。”代凉看着寒伯安。
<br/>
<br/>        “无论你选择哪一家你都是被打击的对象，而陈子昂和你的处境基本差不多，她还有张家的俩个孩子，但有一点，陆家是不会允许陈子昂的孩子回归张家的，如果陆家聪明是会利用陈子昂的，相反他们是会打击陈子昂的。”安林曦开口。
<br/>
<br/>        “陈子昂是不会和陆家、张家站在一起的。陈子昂有现在的成就完全是落家的功劳，说白了，没有落家就没有陈子昂的今天，所以陈子昂将会是第一个被打击的对象。”寒伯安微微一笑，接过安林曦的话。
<br/>
<br/>        “陈子昂的崛起让很多人望尘莫及，她虽然和落家、张家都有关系，但是他们的关系却又若隐若现，尤其是和落家，极少来往，这么几年从来不曾说他们有过任何来往，落家也不曾对她施过援手，所以她算是一株独秀。木秀于林，风必摧之。”安男有些忧心的插话。
<br/>
<br/>        “张家会想办法保护陈子昂的。”寒伯安笑着对安男说。
<br/>
<br/>        “在强大的利益面前张家也不可能会保护陈子昂的，如果想要保护当初就不会放弃。”安男自从知道陈子昂是她的亲戚之后，心就无形的向陈子昂靠近。
<br/>
<br/>        “张函已经不是原来的张函了。”寒伯安安慰着安男。
<br/>
<br/>        “凭张函一己之力能对抗陆家和张家？”安男不相信张函能保护好陈子昂，这么多年了他们都看着呢，张函还不是一如既往的无能为力。
<br/>
<br/>        “陈子昂不会需要任何人的帮助，她自己就可以，陈子昂的智慧和谋略不在你们之下，况且她的三姐是一个重要的人。”安林曦淡淡的一笑。
<br/>
<br/>        “爸，你是说潘禺？”安男像发现新大陆一样兴奋。
<br/>
<br/>        “所以，你们没有必要担心别人，你们要考虑你们自己的路怎么走。”代凉看着眼前的一双女儿，他们怕是要搅入一场无形的斗争了。
<br/>
<br/>        这个社会都是瞬息万变的，如何能走端走正对他们来说都是一场考验。
<br/>
<br/>        安男和寒伯安相视一笑，轻轻的将手握在一起，此生既然选择相守，无论前路如何，我们都会相携而行，不离不弃。
<br/>
<br/>        安林曦和代凉都是明白人，他们所担心和分析就是事实，社会在发展，各种关系错综复杂，谁能在这个大染缸里面独善其身？只不过每个人的选择不一样而已。

<br/>        寒伯安拥着安男在安家偌大的院子里踏着漫天星辰，乘着月色漫步而行，这一段时间他东奔西走没有时间好好陪陪安男，此时月光正好，夜色正浓，有亲爱的她陪着真好。
<br/>
<br/>        自从有了你，世界更美丽，自从有了你，生活更有意义，幕昏晨起，拥你浅笑入寐，看你淡扫蛾眉，听你嘤嘤耳语，抚你春桃盛开，携你流连四季。
<br/>
<br/>        我们看天看地看沧海，听风听语听鸟鸣，行一路颠沛流离，品一生花开四季，一生有你不分离。
