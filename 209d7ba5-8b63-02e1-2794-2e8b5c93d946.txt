        &lt;h3&gt;弟三卷第一百九十七章  傅海生辞职&lt;/h3&gt;
<br/>
<br/>        傅海生手中的项目已经开始了，他没有想到陈子昂公司没有中标，在他的心中倾向于陈子昂，这个倒不是说他对陈子昂旧情未了，而是实实在在的陈子昂他们的方案做得好。
<br/>
<br/>        陈子昂他们的方案做得近乎完美，一本方案八百多页，方方面面都考虑到了，八个人在洛城呆了半年，将他们单位调查的清清楚楚，对于他来说，单位想要发展，必须要考虑到后期的发展方向，现在是信息化大爆发的年代，他们所做的一切都要为后面的发展做好铺垫，洛城俩个项目同一天招标，可惜的是陈子昂的公司一个都没有中标。
<br/>
<br/>        看着陈子昂他们公司做的方案，傅海生舍不得将手从书中抽离，方案里阐述的思路清晰，技术过硬，让他爱不释手，可是就这样完美的方案竟然不能中标，让他感觉不可思议，不知道那些专家是如何看待问题的。
<br/>
<br/>        当呼延临庆顺利中标，到他们单位来洽谈后期的合作时，他才恍然大悟，这一切都是夏颜在里面搞得鬼。
<br/>
<br/>        夏颜，现在真的是能力强大，不但将西城的信息化界玩弄股掌之间，还将手伸到了周边。
<br/>
<br/>        傅海生看着那厚厚的方案，长叹一声，他的愿望、他的抱负怕也只是黄粱一梦，就像他可笑的人生一样，终究是一场空。
<br/>
<br/>        他叫来了当初负责和陈子昂公司人员一起做方案的负责人：“严华，我看了这次中标方的方案，这个方案里很大一部分都是沿用这个公司的方案，我想知道是怎么回事。”
<br/>
<br/>        吕严华：“这个不是夏总告诉我，说是您安排的吗？”
<br/>
<br/>        傅海生：“夏总？那个夏总？”
<br/>
<br/>        吕严华：“您老婆啊，她当时说您要求将这一家的方案给现在的中标方，而且这一次招标都是夏总提前安排好了的。”
<br/>
<br/>        傅海生：“她给你承诺的是什么？”
<br/>
<br/>        吕严华：“她给我说，等您调到更高的位置上，您现在的这个位置就是我的。”
<br/>
<br/>        傅海生：“你觉得这两家方案谁家的好？”
<br/>
<br/>        吕严华：“当然是您手中拿的这一家，只是我人微言轻，我能怎么办呢？”
<br/>
<br/>        傅海生：“你今天倒是说了一句实话，恐怕不是你人微言轻吧，是你想坐现在的这个位置吧？没问题，我现在就给你写推荐信，但是，我要告诉你的是，这次招标我不承担任何责任，和我没有任何关系，我并不知道夏颜的任何操作。”
<br/>
<br/>        吕严华：“这个.......”
<br/>
<br/>        傅海生：“你们和胡安娜是不是已经达成同样的协议了？”
<br/>
<br/>        吕严华：“我并没有接触胡安娜，都是您老婆夏总联系的，这个和我没有关系，而我过去也都是按照夏总的指示在办事情。”
<br/>
<br/>        傅海生：“呵呵呵呵，真是好本事！”
<br/>
<br/>        傅海生可以叫停项目，但是叫停项目也是需要理由的，一切的顺序和流程都是合规的，他没有理由叫停项目。
<br/>
<br/>        他请了长假，递交了辞职信，回家带孩子去了。
<br/>
<br/>        孩子这几年一直都是党桂芳在管着，周宏生的生意越来越好，忙起来的时候，周宏生也会叫党桂芳帮帮忙，这样党桂芳就顾不上家里，傅海生就当起了家庭煮男，负责俩个孩子的吃喝拉撒，而且也将党桂芳夫妇的生活包揽起来了。
<br/>
<br/>        过年的时候两家人都是在一起。
<br/>
<br/>        周宏生：“我说海生，你最近不上班在家里管孩子怕也不是个事吧，家里有保姆，你还是回去上班去。”
<br/>
<br/>        傅海生：“我已经递交了辞职申请，这些年有些累了，想休息一段时间。”
<br/>
<br/>        夏颜：“你说什么？你已经提交了辞职申请？你怎么这么没有出息啊？”
<br/>
<br/>        傅海生：“人各有志，你有出息不就行了。”
<br/>
<br/>        党桂芳：“海生不如回来帮着宏生？我感觉我帮不了宏生，我带孩子是可以的，做生意就不是我的料。”
<br/>
<br/>        周宏生：“哎呀，海生啊，你要是真的不上班了，回来，咱兄弟二人好好的干一番事情，我们现在的公司啊，如火如荼，后期还有大的动作，我这人书读得少，干别的可以，抉择上还是摇摆不定，怎么样啊？”
<br/>
<br/>        夏颜：“那海生还不如去呼延临庆的公司，工作还对口。”
<br/>
<br/>        傅海生：“我已经厌倦了信息化上的事情，宏生，那我就来给你打工了。”
<br/>
<br/>        周宏生：“说什么话呢，我能挖到傅大才子可是我的荣幸。来，兄弟，以后啊，我们两家就是一家，干杯。”
<br/>
<br/>        夏颜无可奈何，不过她现在并不会过多关心傅海生的工作，她和傅海生就是维持着表面的关系，说白了，傅海生现在就是她的一个面子工程，现在她的身份和地位对于傅海生来说都是无法企及的，如此的不上台阶的人不要也罢。
<br/>
<br/>        如果她和傅海生离婚，傅海生会不会去找陈子昂？
<br/>
<br/>        陈子昂和江家大少没有可能，也被张函抛弃，如果她和傅海生离婚了，傅海生是不是就自由了？
<br/>
<br/>        想到傅海生会和陈子昂在一起，夏颜的心火被点燃了，这一生他们都不可能在一起！
<br/>
<br/>        傅海生并不知道夏颜心里所想，他也没有再去想是不是能和陈子昂在一起，自从上次聚会之后，傅海生将他对陈子昂的那份炽热的感情深深的埋藏起来，他傅海生配不上陈子昂，在所有的权衡利弊之后，他并没有选择和陈子昂站在一起，而是屈辱的选择了亲情，选择了夏颜和他的父亲，他没有去揭穿一切，而是相信陈子昂选择了原谅他们，既然陈子昂已经原谅了，那么他也就保存沉默。
<br/>
<br/>        夏颜：“这样也好，海生在家里可以照看孩子，你们在工作上需要我帮忙的给我说，家里就靠你们了。”
<br/>
<br/>        党桂芳：“颜颜，你也别说那样的话，我们都是十几年的老同学了，那么见外干什么呢。你们做你们的事情，我来给咱把娃管好。我这人啊，读书不行，但是做家务没有问题。想起读书的时候，我这个脑子笨的，陈子昂给我一道题讲几遍，我就是不会。”
<br/>
<br/>        几个人沉默了一下，周宏生笑着说：“陈子昂是谁？穿越过来的怪物，你能比得上？你还是好好的做我周宏生的糟糠之妻。”
<br/>
<br/>        夏颜：“看你们俩个人酸的，孩子都这么大了。”
<br/>
<br/>        党桂芳：“我们是酸的，你们是甜的，刚好酸甜搭配！”
<br/>
<br/>        傅海生辞了公职，在周宏生的公司当起了副经理，至于以后是什么样子，走一步看一步吧。
