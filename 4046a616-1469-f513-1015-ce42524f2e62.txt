　　苏莫离开地灵塔之后，便回到了巫尘的住处，在一间修炼室中盘膝而坐，继续修炼。
<br/>　　对于巫烈和巫玉儿，他根本无需多想，几招之内便能镇杀。
<br/>　　只不过，在杀那两人之时，一定不能暴露自己的身份。
<br/>　　首先，他的本命灵剑阵，以及大苍穹剑法等都不能使用。
<br/>　　因为这些招数一旦使用，会被巫天御、巫长远等去过苍穹世界的人认出来。
<br/>　　他只能不用任何武技，或者是用一些早年学习过的普通的武技。
<br/>　　但即便如此，他镇杀巫烈和巫玉儿两人，也没有多大的问题。
<br/>　　因为修炼了混天魔功，这具身体的修为实力已经暴增了近三倍，再配合上他有无数的武魂和战魂，战胜巫烈两人轻而易举。
<br/>　　当然，这么多武魂能不使用就尽量不使用，免得被那些去过苍穹世界的人认出来。
<br/>　　还有吞噬战魂，这些武魂和战魂就算要用，也要暗中使用，免得被人看出端倪。
<br/>　　苏莫继续修炼混天魔功，他也没有使用灵晶，这这样吸纳天地灵气修炼。
<br/>　　他身上魔气鼓荡，越发的浑厚强大，强烈的魔威充斥在整座修炼室之中。
<br/>　　他距离功法彻底转化成功，还有一段距离，初步估计起码需要半个月的时间，不过他也不急。
<br/>　　只要在少司选拔开始之前，功法能转化完毕，然后再修炼精深少许，就可以了。
<br/>　　而距离少司选拔，还有近一年的时间，时间非常的充裕。
<br/>　　还有就是，混天魔功有不少强大的手段，他之后也要抽空修炼一番。
<br/>　　时间匆匆而逝，日落日升，一天时间转瞬即过。
<br/>　　……
<br/>　　清晨，朝阳初生，苏莫便离开了住处，向黑耀城的武斗场走去。
<br/>　　由于搜魂了巫尘的记忆，他对于武斗场也并不陌生，这是黑耀城巫族人决斗的地方。
<br/>　　武斗场内，可以生死斗、赌斗、甚至是兽斗，是黑耀城一处充满血腥之地。
<br/>　　没过多久，苏莫便来到了武斗场。
<br/>　　武斗场仿佛是一座城中之城，占地方圆十数里，从大门走进去便能看到高高的战台，以及四方一层层的环形看台。
<br/>　　这是一处足以容纳数十万人的武斗场，规模相当宏大。
<br/>　　“火焰狮烧死它！”
<br/>　　“白甲兽将它撕成碎片！”
<br/>　　“吼！吼！吼！！”
<br/>　　武斗场内喧嚣震天，看台上坐满了密密麻麻的巫族之人，呼喝声、呐喊声不绝于耳。
<br/>　　而此刻，战台之上正有两只凶兽在撕斗，一只是全身燃烧这火焰的雄狮，还有一只是似猿非猿，全身布满白色鳞甲的凶兽。
<br/>　　两只凶兽均是高达十丈，凶威盖世，在方圆近十里的战台上不停的撕斗。
<br/>　　这是兽斗，也算是赌斗的一种，观战之人可以下注。
<br/>　　这两只凶兽都有武帝境的实力，每一次碰撞力量均是强大无比，让得战台震动，气浪汹涌，卷动四方。
<br/>　　不过，战台布置了加固阵法，上空也布置了一个巨大的阵法气罩，将冲击完全隔绝。
<br/>　　苏莫刚一走进武斗场，便发现看台上落下不少目光，全部都落在了他的身上。
<br/>　　很多人都已经在等着他了！
<br/>　　昨日一事已经在黑耀城传开了，虽然没有造成什么大的轰动，但也吸引了不少人的兴趣。
<br/>　　毕竟，巫尘的畜生行径，非常让人不齿。
<br/>　　苏莫目光转动，扫视四方，很快便看到了巫烈和巫玉儿，两人均是眼眸森寒的望着他。
<br/>　　巫烈的嘴角，该挂着戏虐的笑容。
<br/>　　除了巫烈和巫玉儿之外，巫天御、巫长远等昨日在地灵塔的人，几乎全部都来了。
<br/>　　苏莫面无表情，在看台上找了一处空位，飞身坐了上去。
<br/>　　兽斗还在继续，他自然要等此兽斗结束之后才能上场。
<br/>　　苏莫并没有等待太久，仅仅是半刻钟之后，两兽分出了胜负，那头白甲兽被火焰雄狮烧成了焦炭。
<br/>　　战台很快便被一名巫族中年长老清理干净，胜利的火焰雄狮也没能幸免，被长老挥手斩杀，一并处理了。
<br/>　　然后，整个武斗场短暂的安静了下来，暂时无人比斗，众人的目光均是在苏莫、巫烈和巫玉儿三人身上徘徊。
<br/>　　显然，众人都在等待三人的决斗。
<br/>　　嗖！
<br/>　　破空声响起，身形高大的巫烈，穿着一声赤色的长袍，飞身落在了站台上。
<br/>　　“畜生巫尘，下来受死吧！”巫烈的目光直视苏莫，冷冷的喝道，声音中蕴含浓郁的杀机。
<br/>　　苏莫面色冷漠，长身而起，一步踏出，同样落在了战台上，与巫烈相隔十里互望。
<br/>　　武斗场中寂静无声，好戏要开始了，这可是两名族人死斗，可不比之前的兽斗。
<br/>　　不过，没有人看好‘巫尘’，毕竟巫尘虽然天赋不错，但可没有听说有越级战斗的实力。
<br/>　　“巫尘，看到刚才的白甲兽了吗？你等会也是同样的下场！”巫烈轻笑一声，眸中杀机爆射，璀璨夺目。
<br/>　　他也没有多言，话音一落，铁拳一握，拳上赤色光芒耀眼，充满着炽烈的高温。
<br/>　　“且慢！”就在巫烈准备出手之时，苏莫突然开口制止。
<br/>　　“怎么？现在知道怕了？不过已经晚了！”巫烈闻言顿时嗤笑一声，满脸的讥讽之色。
<br/>　　他还以为苏莫是怕了呢！
<br/>　　苏莫没有理会巫烈，转头看向依旧坐在看台上的巫玉儿，朗声道：“你也下来吧！我好一并解决了！”
<br/>　　苏莫此言一出，所有人都是一怔，随即纷纷惊讶了起来。
<br/>　　不是吧？巫尘居然要一对二？
<br/>　　巫尘有这个实力吗？
<br/>　　哪来的底气？
<br/>　　脑子坏了吧？
<br/>　　众人不太相信，巫尘越级战斗的情况下，还能以一对二！
<br/>　　巫烈脸色一沉，对方是看不起他吗？蝼蚁般的实力居然还想以一对二！
<br/>　　巫玉儿的俏脸也是一呆，她完全没有想到巫尘会如此狂妄，要以一对二！
<br/>　　她心中杀机涌动的同时，亦是充满了疑惑。
<br/>　　因为她很了解巫尘，以前的巫尘谨小慎微，低调无比，与现在这个巫尘完全是判若两人。
<br/>　　不过，现在她也来不及多想，既然对方要找死，她也不介意尽快除掉这个祸端。
<br/>　　嗖！
<br/>　　巫玉儿身形一闪，便飞落在了战台之上，和巫烈并肩而立。
<br/>　　“巫尘，你人面兽心，这一次我不会手下留情！”巫玉儿望着苏莫，俏脸冷漠如霜。
<br/>　　“别说没用的了，你们两个到地狱去苟合吧！”苏莫淡淡的说道，到了这个时候，对方居然还在演戏，真是不知所谓。
<br/>　　他之所以要一对二，而不是一个一个的解决，就是想要在最短的时间内解决两人。
<br/>　　免得杀死巫烈之后，让巫玉儿识破他的身份，或者是让对方心生惧意，不敢一战。
<br/>　　话音一落，苏莫不在多言，率先出手了。
