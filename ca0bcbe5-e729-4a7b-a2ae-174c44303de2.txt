　　苏莫见到那名黄泉魔宗的强者进入大河中，并未担心。
<br/>　　此人进入河中，要么是想在河中藏身，要么是担心苏莫追击，想要从河中逃离。
<br/>　　毕竟，河水可以很大限度遮掩气息，让得苏莫难以追击。
<br/>　　苏莫立刻加快速度追了过去，少倾之后，便也飞到了那条大河之畔。
<br/>　　轻笑一声，苏莫毫不犹豫，纵身飞进了河水中。
<br/>　　这条河很大，宽约数百里，长不知几许，河水很深足有近千丈。
<br/>　　苏莫进入河中，还没落到湖底，就发现了那名黄泉魔宗强者的踪迹。
<br/>　　此人倒是颇为谨慎，并未在河底藏身，而是在河底中急速潜行，已经潜行到了近两万里之外。但还是没有逃脱苏莫幽冥魔瞳武魂的视线。
<br/>　　由于在河水中赶路，速度稍稍受阻，苏莫和对方的距离又拉近了不少。
<br/>　　“再谨慎也无用！”苏莫冷笑一声，急速向对方追去。
<br/>　　这名黄泉魔宗的强者，在河底潜行的时间并不长，约莫一个多时辰之后，便身形上潜，从河水中飞了出去。
<br/>　　苏莫也立刻身形上潜，脑袋浮出了水面，随即他便看到对方向北方飞去。
<br/>　　此人倒是真的谨慎，北方可不是黄泉魔宗所在的方向，此人向北方而去，可见其也是为了预防苏莫追击。
<br/>　　苏莫见此立刻离开了河中，再次向对方追击而去。
<br/>　　这名黄泉魔宗的强者，是一名中年人，身形消瘦，身上的气息非常的阴森，仿佛是厉鬼一般，估计是鬼罗殿之人。
<br/>　　此人离开大河之后，并未急速赶路，显然已经有些放松下来，认为苏莫不会追来了。
<br/>　　毕竟，他们一共有七人，不可能这么巧合的追他。
<br/>　　而且，这么长时间了，他应该彻底安全了。
<br/>　　又飞行了一个时辰之后，这名黄泉魔宗的强者飞进了一处山脉中，而后身形降落在了一个山谷中。
<br/>　　而此刻，苏莫已经追到了此人万里之内，事实上他已经放慢了速度，免得追的太近，被对方发现。
<br/>　　因为苏莫不准备直接与对方交手了，而是看看有没有机会偷袭，如此的话杀死对方将变的简单，也能保留住对方的神丹。
<br/>　　当然，还有最主要的原因，那就是他的大三宝术时效已过，目前他不仅实力回落，而且因为身体变的较为虚弱，实力还有些倒退！
<br/>　　所以，直接交战的话，他杀死对方的成功率比较的低，对方是武皇境九重的超级强者，手段肯定不少。
<br/>　　唯有偷袭，他才能最大可能的杀死对方。
<br/>　　唰！
<br/>　　苏莫立刻身形下落，同样落在了山林之中。
<br/>　　随即，他极力收敛气息，向对方所在的山谷潜行而去。
<br/>　　不多时，苏莫便接近了山谷。
<br/>　　他没有用灵识查探，免得被对方发现，只是远远的观看。
<br/>　　山谷中，那名黄泉魔宗的强者，盘坐在谷中一块大石上，张口服下几枚丹药，赫然是在疗伤。
<br/>　　“疗伤？”苏莫顿时心中了然，他当时破开天煞冥魔阵之时，黄泉魔宗的七人应该是受到了影响，全部受了内伤。
<br/>　　远远的，苏莫藏身于一片荆棘丛中，身上的气息收敛到了极致，他在暗自思索偷袭的办法。
<br/>　　可以肯定的是，他本身的实力绝对要高于此人，但也强的极为有限，此人虽然受了内伤，但他也较为虚弱，两人目前的实力估计半斤八两。
<br/>　　“怎么偷袭呢？”苏莫有些犯愁了，对方是武皇境九重强者，感知必定敏锐，他一出手对方肯定就会发现。
<br/>　　对方若一心想逃的话，他目前很难留下对方。
<br/>　　思索了片刻之后，苏莫眼眸一亮，心中立刻有了办法。
<br/>　　对方这次是用莫须有的异宝来吸引他上当，那他就用同样的方法，来吸引对方。
<br/>　　虚界神图！
<br/>　　用虚界神图吸引对方，然后他藏身在神图空间中，便能实行绝杀的偷袭。
<br/>　　舍不得孩子套不着狼，这种方法虽然有些风险，但却是成功率非常高的方法。
<br/>　　苏莫曾经藏在虚界神图中偷袭过黑龙老祖，一击得手，所以对于此种方法，他也算是有经验了。
<br/>　　随即，苏莫立刻缓缓后退，远离了这名黄泉魔宗的强者。
<br/>　　他身形如电，在山林中穿梭，不多时便找到了一只四级黑虎妖兽。
<br/>　　这只黑虎体长五丈有余，威风赫赫，正在自己的地盘上闲逛，却不知道有一名人类已经靠近了它千丈之内。
<br/>　　苏莫小心翼翼的靠近了黑虎千丈之内，躲在一颗参天古树之后。
<br/>　　黑虎由于等级较低，根本发现不了苏莫的存在。
<br/>　　“就你了！”苏莫目光望着前方的黑虎，嘴角勾起一丝笑意，既然要引那名黄泉魔宗的强者上钩，他自然不能将虚界神图随便扔。
<br/>　　他要利用这只黑虎，让那名黄泉魔宗强者，误以为虚界神图是黑虎从某处得到的。
<br/>　　随即，苏莫将虚界神图取出，并且从储物玉镯中取出了一根帝级长枪。
<br/>　　他将长枪指向天穹，微微注入一丝金属性玄力，长枪立刻爆发出刺目的光芒，直冲天穹。
<br/>　　刺目的光芒照耀八方，方圆数万里之内，任何人或妖兽都能立刻发现。
<br/>　　于此同时，苏莫另一只手，将虚界神图一把扔向了不远处的黑虎。
<br/>　　这一切做完，几乎在不到一个眨眼的时间之内，苏莫意念一动，瞬间便进入了虚界神图中。
<br/>　　而虚界神图，划过一道优美的弧线飞向了黑虎。
<br/>　　此刻，这只黑虎铜铃般大小的眼眸中，还有些发怔。
<br/>　　它正在闲逛，便发现侧方一道金光直冲天穹，它还没反应过来，便又见到一个物体向它抛飞了过来。
<br/>　　黑虎定晴一看，发现并不是遭到了攻击，而是一个不知名的物体，它并没有感觉到什么危险气息。
<br/>　　嘭！
<br/>　　虚界神图跌落在地，落在了黑虎的身前。
<br/>　　黑虎打量着眼前的虚界神图，硕大的眼眸中露出惊奇之色，它虽然不知虚界神图是何物，但能感觉到神图上的气息，相当的不凡。
<br/>　　这只黑虎毕竟是四级妖兽，灵智已经不低了。
<br/>　　黑虎打量了一番虚界神图，随即张嘴叼起，而后向远处急速飞掠而去。
