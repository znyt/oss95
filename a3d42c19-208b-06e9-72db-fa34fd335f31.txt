　　翌日清晨，卢云便再次来到了苏莫的住处，将他收集好的炼器材料，以及一个帝级的炼器熔炉，交给了苏莫。
<br/>　　炼器材料是足够炼制二十柄重渊斧的材料，一份不多一份不少。
<br/>　　这一次，卢云从中赚取了三成的利润。
<br/>　　要知道十件帝级宝器，价值非常的大，而炼制的材料却并没有多贵，若不是还有一件炼器熔炉，卢云足以赚六成以上的利润。
<br/>　　这一点，苏莫心中非常清楚，但为了自己的目的，一点钱财不足挂齿。
<br/>　　拿到了材料以后，苏莫便进入了炼器室之中，准备炼器。
<br/>　　不得不说，苏莫的胃口非常的大，要知道，他以前可是连皇级兵器都没有炼制过，而现在居然要一步登天，直接炼制帝级兵器。
<br/>　　炼器室中。
<br/>　　硕大的炼器熔炉摆放于房间的中央，地上整齐的摆放着二十份炼器材料。
<br/>　　而苏莫正盘膝坐在地上，面色凝重无比。
<br/>　　“欧炼子前辈说过，我的精神力天赋超绝，是天生的炼器师，希望真的如此！”苏莫深深的吸了口气，而后闭上了眼眸。
<br/>　　他要在脑海中，将重渊斧的炼制步骤，从头到尾再梭理几遍，争取少犯错误。
<br/>　　苏莫的心中还是有些底气的，以他的精神力天赋，只要能牢牢的按照炼器传承中的步骤，稳步炼制下去，不出现纰漏，炼成重渊斧不是太大的问题。
<br/>　　良久之后，苏莫再次睁开了双眼，开始炼制重渊斧。
<br/>　　唰！
<br/>　　手掌一挥，炉盖飞起，十数种材料飞到了炼器熔炉之中。
<br/>　　而后，苏莫立刻盖上炉盖，开始生火。
<br/>　　狂暴的火属性玄力，涌入火口，瞬间化为了熊熊烈焰。
<br/>　　熊熊烈焰煅烧着十数种材料，发出滋滋的声响，少倾之后，几乎是每隔半个呼吸的时间，苏莫便要放入一种或数种材料。
<br/>　　不多时，苏莫便已经放入了一百余种材料。
<br/>　　这些材料的熔炼，极为讲究，火候无论是深一分还是浅一分，都会导致炼制失败。
<br/>　　而且，基本上每一种材料的熔炼火候，都不尽相同，非常考验炼制者的控制力。
<br/>　　火焰的威力，要达到什么水平，什么时候该强、什么时候该弱，需要牢牢的控制住，一丝一毫的差别，也会导致失败。
<br/>　　滋滋滋~~~
<br/>　　在一阵滋滋声响中，毫无意外，苏莫第一次炼制很快便失败了。
<br/>　　这是因为其中一种材料熔炼过了头，导致彻底崩盘。
<br/>　　呼！
<br/>　　苏莫深吸口气，他没有急着再次炼制，而是仔细回味刚才的炼制过程，寻找自己失败的原因。
<br/>　　失败并不可怕，可怕的是不知道为何会失败！
<br/>　　他需要的是进步，是一次比一次进步，免得重蹈覆辙，同样的问题，他不能失败两次！
<br/>　　半个时辰后，苏莫清理了一番炼器熔炉，再次开始了第二次炼制。
<br/>　　这第二次炼制的过程，比第一次稍稍顺利一些，他将三百六十余种材料全部熔炼了。
<br/>　　但是，却是因为最后两种材料，熔炉的火候稍浅，导致熔炼的器胚溶液，品质达不到要求，再次以失败告终。
<br/>　　苏莫再次休息了大半个时辰，仔细反省自己的错误，随后再次开始了炼制。
<br/>　　第三次，失败！
<br/>　　第四次，失败！
<br/>　　第十二次，失败！
<br/>　　第十三次，失败！
<br/>　　第十四次，失败！
<br/>　　第十五次，成功！
<br/>　　整整三天三夜，苏莫不间断的炼制，终于在第十五次之时，成功了！
<br/>　　呼！
<br/>　　拿起炉中已经彻底冷却的战斧，苏莫仔细的打量了起来。
<br/>　　此斧是一种短斧，斧柄只有两尺出头，斧头很大，如同一张小案几一般。
<br/>　　此斧通体漆黑如墨，斧刃较钝，并不锋利。
<br/>　　但是，此斧却非常的重，简直是沉重无比，拿在手中让苏莫感觉，他拿的不是一件兵器，而是一座大山。
<br/>　　重，便是此斧最大的特点，这是因为此斧的炼制材料中，有不小一百种，都是沉重无比的特殊材料。
<br/>　　“品质很低！”苏莫呢喃自语，此斧的品质虽然达到了帝级下品，但品质却是非常的差，算是帝级下品兵器中垫底的存在。
<br/>　　不过，就算品质再低，此斧终究是帝级下品宝器。
<br/>　　“只要成功了就行！”苏莫面上露出了一丝笑容，只要成功了，他便能加入炼器师公会。
<br/>　　这样他不仅安全无忧，到时候说不定还能请公会中强者出手，对付那两名巫族的尊者。
<br/>　　片刻之后，苏莫继续炼制，他还有五份材料，不能浪费了。
<br/>　　而且，他只成功了一次，还是不太熟练，还需多磨练一番。
<br/>　　十数个时辰之后，所有炼器材料便被苏莫消耗一空。
<br/>　　那最后的五份材料，他只失败了一次，又成功了四次，熟练度大大的增加了。
<br/>　　随后，苏莫也没有耽搁时间，立刻离开了房间，前去寻找柴管事，准备参加考核。
<br/>　　他没有知会秦不死，只是只身前往。
<br/>　　对于苏莫来说，自然是越早通过考核，成为公会的炼器师越好。
<br/>　　……
<br/>　　炼器师公会之外，十余里处。
<br/>　　一家豪华的客栈之中，巫梵和巫尹两人站在窗口处，遥望着不远处的炼器师公会。
<br/>　　“怎么回事？这两个土著进入炼器师公会已经五日了，怎么还不出来？”
<br/>　　巫梵脸色阴沉的说道，因为不敢用灵识查探炼器师公会，他现在也是有些急了。
<br/>　　“他们不会住在炼器师公会不出来了吧？“巫尹苍老的脸色同样不太好看。
<br/>　　若是真是如此的话，那就麻烦了！
<br/>　　“这样吧！你先带族中子弟回天冥星！我进炼器师公会中看看！”思索了一番，巫梵说道。
<br/>　　他们前来追杀苏莫和秦不死，那些在苍穹世界参加试炼，而侥幸逃回来的族中子弟，也被他们用空间宝器带在了身上。
<br/>　　因为苍穹世界入口所在的那颗废星，距离他们天冥星支族极远，没有他们的带领，那些族中子弟，怕是三五年都无法回到天冥星。
<br/>　　“这样也好！”巫尹闻言点了点头，试炼结束这么久了，若是再不回去，族中怕是也该急了。
<br/>　　只不过，这次回族之后，怕是族中高层要雷霆震怒了，死了九百余名天才子弟，整个支族估计都要炸开了！
<br/>　　但是，此事也瞒不住，他们迟早要回族交代。
<br/>　　“那你见机行事吧！千万不能让那两个土著逃了！”巫尹嘱咐道。
<br/>　　“放心吧！任他们有千般手段，也逃不出我的手掌心！”
<br/>　　巫梵自信的说道，这份自信来源于实力，若是两个土著能从他手中逃了，那他也不配成为武尊境的尊者了！
<br/>　　“嗯！还有，最好不要在炼器师公会中动手！”巫尹再次叮嘱了一句，炼器师公会势力太大，他们巫族也不敢得罪。
<br/>　　而且，他们只是巫族的一条支脉，并不是强大的巫族主脉。
<br/>　　言罢，巫尹身形一闪，便消失在了房间中。
<br/>　　而后，巫梵瞥了一眼不远处的炼器师公会，眸中杀机一闪而逝，随即，他的身形亦是瞬间消失无踪。
