        &lt;h3&gt;第三卷第八十二章  寒伯安见丈母娘&lt;/h3&gt;
<br/>
<br/>        寒伯安和安男从国外回来，俩个人一起去了安家，安家原本祖籍河北，只是后来安男父亲和母亲已经是业界泰山北斗人物，他们在工作的地方居住下来。
<br/>
<br/>        寒伯安不是第一次来安家，当然这一次就更加亲切了，他已经成功的将安男从未婚妻转变成妻子。
<br/>
<br/>        安家的住所给人一种平实而精致的感觉，显得自然、轻松、休闲、质朴，庭院的亲水平台、泳池、回廊相结合，呈现一种乡村风情的生活格调，显得活泼而不单调。
<br/>
<br/>        挑高的门厅和气派的大门，圆形的拱窗和转角的石砌，浪漫与庄严的气质相结合。尖塔形斜顶，抹灰木架与柱式装饰，自然建筑材料与攀附其上的藤蔓相映成趣，经典而不落时尚，古典、开朗两相宜。
<br/>
<br/>        安男挽着寒伯安的胳膊走进庭院，安男的父亲和母亲正在廊下下棋，绿蔓成荫，柔风轻拂，寒伯安和安男静静的站着看着老人将一盘棋下完。
<br/>
<br/>        安母喜笑颜开，拍了拍手:“今天你下厨。”
<br/>
<br/>        可是一抬头发现安男和寒伯安站在旁边：“你们什么时候回来的？迟不回来早不回来，偏偏等今天回来。”
<br/>
<br/>        “好了，好了，我今天下厨。来，伯安，坐。”安父也不理安男直接指着凳子让寒伯安坐下。
<br/>
<br/>        “爸、妈、”寒伯安开口。
<br/>
<br/>        “嗯？在一起了？”安林曦看着一对璧人问。
<br/>
<br/>        “是的。”寒伯安微笑着回答。
<br/>
<br/>        “男男啊，你这是修成正果了？”安母代凉轻轻的将女儿的头发扎起来，笑着说。
<br/>
<br/>        “妈！”安男红着脸。
<br/>
<br/>        “老头子，今天饶过你，我和安男做饭去。”代凉开心的就像是一个孩子。
<br/>
<br/>        “妈，要不，我们出去吃？”寒伯安征求俩个老人的意见。
<br/>
<br/>        “为什么要出去吃？我才不去呢。”代凉微笑着拉着女儿的手，女儿走了这些天，她也想念得紧。
<br/>
<br/>        “你们两个聊着，等会饭好了叫你们。”他们家是有保姆的，只是安父和安母做饭会是俩个人之间的游戏。
<br/>
<br/>        偶尔下厨为对方做一顿自己亲手做的饭菜，心里也是莫名的幸福。
<br/>
<br/>        “想好了？”安林曦问寒伯安，安男和寒伯安谈了七年。
<br/>
<br/>        “想好了！”寒伯安笑着说。
<br/>
<br/>        “我们安家崇尚一生一世。还有，安家没有滔天的权势，没有堆积的钱财。”安林曦看着寒伯安的眼睛。
<br/>
<br/>        “安家有的是虚怀若谷的胸襟和源远流长的家风。寒家也是一生一世一双人。”寒伯安看坦然的看着安林曦微微的笑着。
<br/>
<br/>        安家就此一女，爱若掌上明珠，安家这一处院子还是单位分的，安林曦已经在这里住了几十年了，当然，安家不但有保姆还会有安保人员，这也就是寒老爷子没有强迫寒伯安的原因。
<br/>
<br/>        安家的孩子是能配得上他们寒家的，而且两个孩子还是同学。
<br/>
<br/>        安林曦听寒伯安如此说，便也放下心来，寒家也是名门望族，几个孩子都是人中龙凤，而且寒伯安做事情稳重，他们已经谈了这么多年，能走在一起也是经过深思熟虑的。
<br/>
<br/>        “还准备办婚宴吗？”安林曦直接就问出来了。
<br/>
<br/>        “这就是来和您商量的。”寒伯安此次回来就是商量结婚的事情。
<br/>
<br/>        “一切从简吧，我当时和她妈妈结婚的时候就一张床，一个水壶、请同事们吃了顿饭。”安林曦笑着说。
<br/>
<br/>        “哈哈哈，男男现在可不愿意一张床，一个水壶，我也不愿意。”寒伯安否决了安林曦的一切从简，好歹他是寒家的人，一生就结一次婚，虽然提前进入流程了，但是该走的程序还是得走，他要给安男一个浪漫而美好的婚礼。
<br/>
<br/>        “那就随你们。”安家的人不是迂腐之人，年轻人的事情年轻人做主，况且寒伯安也只是打着商量的口气告诉他一声而已。
<br/>
<br/>        “好的。”寒伯安松了一口气，毕竟是老丈人，还得事事请教，只要是不反对就好。
<br/>
<br/>        安男和母亲在厨房里忙碌着，别看安男是安家的掌上明珠，但是也和她妈妈一样，偶尔会露上一手，饭菜也是做得极好的。
<br/>
<br/>        代凉经常教育女儿：“你可以雇保姆，但是你必须有能做一手好菜的本领。偶尔下厨给自己或者给家人一个惊喜。”
<br/>
<br/>        代家也只有她这一个女儿，不过她不是代家亲生的，而是代凉的父亲当初在河北的一个朋友的孩子，当时那户人家里遭受巨变，将女儿留给代家，带着儿子远走他乡，从此便音讯全无。
<br/>
<br/>        那家人走的时候留给了代凉一只手镯，原本是给代家抚养代凉的生活费，但是代家也是书香门第，是不会拿着小代凉唯一的信物度日的。
<br/>
<br/>        于是在代凉结婚的那天将情况一五一十的讲给代凉听，希望代凉以后能找到她的亲人。代凉也想着要去找，但是茫茫人海去哪里找？况且就凭一个手镯怕也是找不到的。
<br/>
<br/>        母女俩个聊着私密的话题，很快八菜一汤也就准备好了。
<br/>
<br/>        饭桌上，安男的手镯碰撞了一下餐具，安林曦：“那个可是你妈妈的信物，虽然也是要给你的，但是你也不能这样碰撞。”
<br/>
<br/>        “妈妈的东西都是好的。”安男有些委屈，在爸爸的心目中，妈妈永远都是第一位。
<br/>
<br/>        “那是当然，要不然那来的你？”安林曦秀恩爱的水准也可以算得上泰山北斗了。
<br/>
<br/>        “听男男说这是一个有故事的手镯？”寒伯安适时的岔开了父女俩的话。
<br/>
<br/>        “嗯，你妈妈亲生父母留下的。”安林曦回答着寒伯安的问题。
<br/>
<br/>        “现在还想找吗？”寒伯安想帮帮忙。
<br/>
<br/>        “我还有一个哥哥，但是找不到在哪里。”代凉语气有点忧伤，不过在代家她也是受尽宠爱，要不然她那能有现在幸福的日子。
<br/>
<br/>        “现在信息发达了，我们可以找一找试试。”寒伯安对代凉说。
<br/>
<br/>        “本来是一对，我哥哥一只，我一只，只是这个手镯太值钱了，我一直不想让它曝光。”代凉说出了心里的忧虑。
<br/>
<br/>        “我来试试吧，我就拍一张照片就可以，也许会有希望。”
<br/>
<br/>        “那就让伯安试试吧。”
<br/>
<br/>        江俞轩他们家是搞房地产的，认识一些考古人生，而那些人士有可能会认识一些收藏家，如果代凉的哥哥缺钱的话有可能会卖掉，如果卖掉了，找到源头就可以找到代凉的哥哥。
<br/>
<br/>        寒伯安准备将手镯拍下来，发给江俞轩。
<br/>
<br/>        原本几个人是应该要商量婚事的，却被一个手镯带偏了主题，但是看着代凉那希冀的眼神和安男开心的笑容，寒伯安心里也荡漾着一股暖流。
<br/>
<br/>        安男给寒伯安夹了一筷子菜，柔声说：“妈妈的事情就辛苦你了。能找到舅舅你就是家里的大功臣。”
<br/>
<br/>        “老婆的事情是大事情，我要爱妻如命，唯老婆命是从。”寒伯安也回应了一下安楠，全程无视旁边还有俩个老人。
<br/>
<br/>        秀恩，爱这件事情谁都会！
