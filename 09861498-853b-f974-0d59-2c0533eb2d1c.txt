        &lt;h3&gt;第三卷第二百六十五章  人生百态&lt;/h3&gt;
<br/>
<br/>        陈家的家宴充满了欢乐，因为找到了失散已久的亲人，大家在这一个晚上都感觉到了浓浓的亲情，人们的心里都感觉到了一种满足感和幸福感，包括局外人江俞轩和张倩楠。
<br/>
<br/>        喜乐安康是多么的平凡又是多么的难能可贵啊。
<br/>
<br/>        整个宴会洋溢在一种温情里，老实巴交的农民陈元良和他的妻子冉灵卉坐在一起，眼睛笑得眯了缝，他们找到了失散已久的妹妹，他们的孩子一个个的都长大成人，他们此时应该无憾，儿孙自有儿孙福，今后的路怎么走，他们也无能为力，只是上天护佑，让他们都能平平安安。
<br/>
<br/>        对于张函，他们已经原谅了他，他们已经活了这么久，什么事情看不透？再见张函，他们知道，这个孩子心里是有他们的女儿的，只是造化弄人，张函和陈子昂情深缘浅。
<br/>
<br/>        张函来给两位老人敬酒的时候，陈元良说：“小函，人生不能重来，要珍惜现在，展望未来，不要活在过去里，你应该有更美好的将来，子昂已经是你的过去式，你应该向前看。孩子是你们的孩子，日子得你们自己过，希望你们都各自幸福。”
<br/>
<br/>        “谢谢。”张函动了动嘴，用口型喊了句爸爸，终究是没有叫出口，老人家的一句：“你们各自幸福”就将他和陈子昂的关系分得清清楚楚，虽然心里很疼，张函却很感激陈家的大度。
<br/>
<br/>        江俞轩给陈爸爸敬酒的时候，陈元良说：“总是听子建说你有多优秀，我这耳根子都听腻了，今天一见果然不错，伯伯祝你幸福。”
<br/>
<br/>        寒伯安和安男喊着舅舅给陈元良敬酒的时候，陈元良笑着说：“我们陈家的孩子真的优秀，连找的女婿都这么优秀。”
<br/>
<br/>        安林曦在旁边听得只翻白眼，安男是他的女儿好不好，怎么就成了陈家的孩子。
<br/>
<br/>        陈元良笑眯眯的对安林曦说：“妹夫，你别不高兴，凉儿可是我们陈家的人，你说男男不是我们陈家的人是谁的人?”
<br/>
<br/>        “你说的对，包括我都是陈家的人，男男肯定是陈家的人了。”安林曦被陈元良看破了心思便回了一句，这一句惹得大家哈哈哈大笑。
<br/>
<br/>        快乐总是很短暂，因为陈元良生病着，晚宴就结束的比较早一点。
<br/>
<br/>        陈子昂想多陪陪父亲，便随父亲一起回到了医院。
<br/>
<br/>        张倩楠对江俞轩说：“我还有工作没有做完，这几天要赶快将手中的工作做完，好返回北京，我就不随你一起回去了。”
<br/>
<br/>        江俞轩略显尴尬的看了一眼大家：“好吧，我送你回去。”
<br/>
<br/>        “也行，哥，你走不走？俞轩送我去单位，我晚上还得加班，我们一起走吧。”张倩楠喊张函一起走。
<br/>
<br/>        潘青松问张函：“你们明天回北京？”
<br/>
<br/>        “是的，明天我们就走了，这里的事情就交给你了，记得要保护好自己，别一根筋。”张函叮咛潘青松。
<br/>
<br/>        “你这话说的，要我像你一样当缩头乌龟？”潘青松嗤之以鼻。
<br/>
<br/>        “三姐夫，张函一直是你认识的那个张函，也是陈子昂最初认识的那个张函，从来都未曾改变过，改变的只是环境，环境让我们物是人非。三姐夫请保重，代我向三姐问好，就说张函敬重三姐和陈家的人！”张函说完转身离去。
<br/>
<br/>        潘青松张了张口，什么话也说不出来，他怎么感觉张函有一种很悲壮的样子，似乎有一种壮士一去不复返感觉。
<br/>
<br/>        陈子霖开车将安男一家送回酒店，对代凉说：“姑姑，明天我来带着你和姑姑还有安男妹子在西城好好转转，你们也不用着急，休息好，我九点钟来接你们。”
<br/>
<br/>        “子霖，我想到你们老家去看看。”代凉想去看看哥哥曾经生活的地方。
<br/>
<br/>        “行，那我明天就带你回去看看，只是这几天家里没有人，都在西城。”陈子霖说。
<br/>
<br/>        “你父亲什么时候做手术？”代凉又问陈子霖。
<br/>
<br/>        陈子霖：“三天之后，子昂有从北京请专家过来。”
<br/>
<br/>        “那你明天上午早点过来接我，我去医院陪陪你父亲，明天下午我去你们老家。”
<br/>
<br/>        “行，姑姑您说了算。”
<br/>
<br/>        看着陈子霖开车离开，代凉叹了口气，转身发现安男和寒伯安还站在身边，嫌弃的说：“你们怎么还站在这里？俩个二百瓦的灯泡太耀眼了，林曦，我们走。”
<br/>
<br/>        代凉伸出手，拉着安林曦回了自己的房间。
<br/>
<br/>        寒伯安一把抱起安男：“老婆，我们被嫌弃了。”
<br/>
<br/>        安男被寒伯安抱着走进房间，嘴里不停的说：“放我下来，放我下来，成何体统。”
<br/>
<br/>        突然她就说不出口了。
<br/>
<br/>        西城的夜在这个夜晚，显得温暖而多情。
<br/>
<br/>        “我在美洲的那些天，没有给你任何信息你会不会很怪我？”寒伯安拥着安男轻声的问。
<br/>
<br/>        “第一天晚上有些怪你，不过你的那个前女友一来，我就不怪你。”安男笑着说。
<br/>
<br/>        “为什么？”寒伯安放开安男，好奇的盯着安男的眼睛。
<br/>
<br/>        “因为我傻呗。”安男调皮的一笑。
<br/>
<br/>        “就喜欢你傻傻的样子，一辈子都是相信我的。”寒伯安用指头轻轻的点了一下安男的鼻尖。
<br/>
<br/>        “冰玉和落雪出了一点事情，当时我的心里很乱，而且我也被人威胁，脱不开身，我怕有人对你不利，就更不敢和你联系，只是希望你能明白。安男，你要做好准备，和我在一起不光光是平安幸福，还有凶险和无奈。”寒伯安低声说。
<br/>
<br/>        “冰玉和落雪出了什么事情？”安男吃惊的问。
<br/>
<br/>        “你不知道吗？你这些天没有看新闻？”寒伯安皱着眉头。
<br/>
<br/>        “我一到寒家老宅就被阿姨拉着吃吃喝喝，说说笑笑，再加上你的前女友一来，感觉我上卫生间都会被监视，还看什么新闻啊。”安男说的是实情，第一天被苓如兰拉着和七大姑八大姨在一起，后来苓希来了，她完全就是被监视起来的样子。
<br/>
<br/>        “你是怎么想起来联系陈子昂的？”
<br/>
<br/>        “长卿给我打电话了，而你连个信息都没有，再加上你的前女友真的在监视我，让我感觉有些不对劲，想来想去，也找不到合适的人，于是我就给子昂打电话求助，子昂真的是太聪明了。当天晚上就将我带回了北京。”
<br/>
<br/>        “就你们俩个人吗？”寒伯安问。
<br/>
<br/>        “没有，还有陈子昂的外甥，潘禹。”
<br/>
<br/>        寒伯安微微一愣神：“难怪你们能轻松的离开寒家老宅，没有人敢动潘禹的。”
<br/>
<br/>        “你也认识潘禹吗？”
<br/>
<br/>        “不认识，但是知道这个人。”
<br/>
<br/>        ........
<br/>
<br/>        陈子昂随着父亲回到病房，心里很疼，她知道父亲的病，从北京请了专家，如果手术好了父亲能活三年五载的，如果不好也就是几个月的时间，她陪父亲的时间不多。
<br/>
<br/>        她让父亲坐下，拿来洗脚盆，为父亲洗着脚，之前想着父亲身体还好，可是当看到父亲那骨瘦如柴的脚时，陈子昂泪水直流。
<br/>
<br/>        “子昂，你别哭，爸爸老了，生老病死都是人之常态，我呢总算不枉来到这个世界一趟，有你们这么多可爱孝顺的孩子，我这心啊，无憾了。”爸爸伸手摸了摸陈子昂的头。
<br/>
<br/>        “爸......“陈子昂泣不成声，想一想十二岁的她还经常吊在父亲的脊背上，被父亲背着跑来跑去，心，一阵阵的揪疼。
<br/>
<br/>        “子昂，现在我也见到你姑姑了，只是你和你六哥我这心里还是有些不放心。”父亲叹了口气。
<br/>
<br/>        “爸，我找到我六哥了。”陈子昂不忍心看着父亲悲伤。
<br/>
<br/>        “子昂啊，你六哥和你三姐一样令人骄傲。”
<br/>
<br/>        “爸，你都知道了？”陈子昂抬头看着父亲，充满了震惊。
<br/>
<br/>        “你六哥还好吧？”父亲看着陈子昂问。
<br/>
<br/>        “他很好，只是目前还有任务没有完成，所以不能回来见您。您要等着他回来。”陈子昂低声回答。
<br/>
<br/>        “好！我等他回来！等着我的六儿回来！”
<br/>
<br/>        父女俩眼里都含着泪却看着彼此微笑，冉灵卉看着父女俩的互动，也是眼眶湿润，一言不发。
