<br/>				陈默在他的诊室中，将小瓶子里的小虫子好好观察了一遍，但是却没有看出是什么目的。<br/>　　<br/>　　这个小昆虫有些打破他的生物知识，不过好在他也没有较真什么的。因为他本身就是个BUG，修真都有了，那么出现个不同的小虫子，也不算什么。<br/>　　<br/>　　再说了，星球上几百万种的小生物，难道每一个他都要清楚么？就算是昆虫学家都不敢保证的事情，所以该如何就如何！不知道就不知道吧，也不算是什么。<br/>　　<br/>　　陈默为自己偷懒找到了很好的借口，所以将手中的小虫子，做了几层保险，套了两层的玻璃瓶不说，最后还装进不锈钢杯中。对于小虫子，他也不敢保证这个玻璃会不会被啃食透，所以就在外面套了个不锈钢。<br/>　　<br/>　　等回到别墅，就看见袁若珊大马金刀的坐在客厅沙发上，正在看无脑抗日神剧！<br/>　　<br/>　　“你喜欢看这类节目？”陈默有些好奇的疑问道。<br/>　　<br/>　　“还行吧！就是看个热闹!反正都是不用脑子想，算是解压吧！”袁若珊说道。<br/>　　<br/>　　陈默也就点点头，这类电视也就这样子，还真的不能深究的一些东西，不过看着热闹倒是挺好的。<br/>　　<br/>　　“说的也是！”<br/>　　<br/>　　将不锈钢杯子递给袁若珊。<br/>　　<br/>　　“我觉得这个东西还是你拿回去好好研究一下，看看到底是什么东西。以后要是研究出来了，就告诉我一下！”<br/>　　<br/>　　“这里面是那个小虫子？”<br/>　　<br/>　　“嗯！”<br/>　　<br/>　　“在这里里面会不会跑出来？”<br/>　　<br/>　　袁若珊是见过这个小虫子的能耐的，所以就有些担心的问道。<br/>　　<br/>　　“应该不会跑出来吧！这都封闭了好几层。再说了这个也就是个小虫子，应该没有什么可能！”<br/>　　<br/>　　陈默有些不确定的说道，想想后，接着说道：“我看你还是最好让专业人士运送，这样也好减少其中的危险性！”<br/>　　<br/>　　“不错！我就联系人来运送这个东西！”<br/>　　<br/>　　袁若珊说完，就开始联系人运送。对于这个小虫子，还是少接触为妙。<br/>　　<br/>　　等联系完以后，就一起在客厅里看着电视。突然，陈默想到今天晚上要如何睡觉啊！<br/>　　<br/>　　现在就要快夜里十点了，按说在村里的话，早就应该睡觉了。<br/>　　<br/>　　“哪个，今天晚上你在哪里睡觉啊？”<br/>　　<br/>　　“你这里不是有很多房间么，随便一个就成。”<br/>　　<br/>　　“那行！”<br/>　　<br/>　　“对了，我想和你说个事情！”<br/>　　<br/>　　“好，你说吧！”陈默点点头，看着袁若珊。<br/>　　<br/>　　“这次过来，一个是顺道送设备的，另外一个就像想问问你，加入特管局怎么样？”袁若珊说道。<br/>　　<br/>　　“加入特管局？”<br/>　　<br/>　　“不错！自从上次在西市和你说了这个事情后，一直没有时间和你好好聊聊。这次正好有机会，所以就想问问你。”<br/>　　<br/>　　“你怎么老想着让我加入特管局？”陈默有种贱人就是矫情的心里在作怪，虽然已经想好怎么做了，但是有的时候就是有些矫情。<br/>　　<br/>　　其实也不能说他矫情，这样问袁若珊，也是为了给自己争取一些好处。他可不想当一个冲锋在一线的队员，那样子还不如不加入呢！<br/>　　<br/>　　“现在就凭借你的医术，你要是加入特管局的话，会有一个好的起点。而且你也知道，在国内生活，有个身份的话还是比较好处理一些事情的，你说是不是？”<br/>　　<br/>　　“可是加入特管局后，打打杀杀的我不是很喜欢！我就是想安安静静的生活而已。”陈默说道。<br/>　　<br/>　　“其实我们的领导已经对你有所安排！你进入特管局后不用去一线，只要能够发挥你的长处就成。”<br/>　　<br/>　　“你看这样可不可以？我就作为特管局的编外人员好了，如果有伤者需要治疗，只要送到我这里就成。平时没有事情的时候，我想做什么就做什么。”<br/>　　<br/>　　“这个，这个……”<br/>　　<br/>　　袁若珊有些挠头，虽然有的时候可以使用点小小的手段来达到目的。但是对于陈默他们不想，因为这个是为医生，现在他的医术是特管局非常需要的。要是耍了小手段，那么陈默进入后出工不出力，那不就抓瞎了么！<br/>　　<br/>　　“这个我需要和上级领导商量一下，要不先等等，明天中午再告诉你！”<br/>　　<br/>　　“好，就这样吧！”陈默点头，那就再等等吧，反正也不差这么一个晚上。<br/>　　<br/>　　由于晚上有袁若珊在，而且这个家伙可以是后天三层的武者了，所以今天晚上就没有去练功房，而是在自己的卧室里面练习雕刻和书法。<br/>　　<br/>　　袁若珊则联系了上市的宁永志，对于陈默的说法和他的一些想法都和宁永志交流了一番。<br/>　　<br/>　　对于能医术已经经过检验，并且能够治疗异种能量的陈默来说，他还是非常的欣赏的，也就和袁若珊讨论了一下，最后定下了一个章程。<br/>　　<br/>　　半夜的时候，陈默听到袁若珊出去了一趟。不过他却没有去查看，想都能想到，是安排的人过来拿走那个小虫子，所以就随她了。<br/>　　<br/>　　幸好今天下午吸收的异种能量已经被陈默捋顺，而且全部都变成了自己的真元。所以晚上的修炼还是照旧。<br/>　　<br/>　　第二天，陈默练拳回来后，就开始准备早餐。虽然很简单，但是农家饭菜自有一种香味，倒是让袁若珊美美的吃了两大碗粥，还有两个锅盔。<br/>　　<br/>　　“没有想到，你这里的饭菜有种小时候的味道。”袁若珊笑着说道。<br/>　　<br/>　　“其实这种饭菜就是农家平时吃的。现在城里吃的东西，哪有不添加东西的，所以味道虽然好，但是香精色素什么的，添加的比较多，反而失去了食物本身的香味。”陈默说道。<br/>　　<br/>　　“不错！现在都这个样子，也不是谁能解决的。”袁若珊有些意兴阑珊的说道。<br/>　　<br/>　　“呵呵！”陈默除了这个以外，还真的说不上什么。大环境如此，自能保证自己吃的好就成。<br/>　　<br/>　　“你说说昨天晚上商量的结果么？”陈默不是那种能等待的人，再说了袁若珊也算是自己的朋友吧，所以有什么说什么，该如何就如何。<br/>　　<br/>　　“昨天晚上商量过了，就按照你说的。不过编外人员是不可能的，就作为正式人员。作为正式人员的后勤好了，你就在这里负责治疗伤者就成，其他事情不用你去完成。”袁若珊说道。<br/>　　<br/>　　“那就好！还有什么好的条件没有，都一起说出来吧！”陈默笑着说道。<br/>　　<br/>　　“好吧！其实因为你的原因，我们也进行了详细的规定。”袁若珊也就细细给陈默说了一下。<br/>　　<br/>　　昨天晚上，她和宁永志说完后，宁永志又和上级领导交流了一下，就定下了基本基调。<br/>　　<br/>　　陈默作为一名医生和武者，可以不去执行任务，就是专业负责治病救人就成。平时根据本人意愿，在哪里都成。但是遇到病人不能移动或者急诊的，那么就需要他亲自出诊。<br/>　　<br/>　　待遇等同特管局正式人员，而起按照特殊人员的待遇参考。另外由于治病救人，可以给陈默开放特管局的特殊医药库，根据贡献度可以换取一些稀有药材等等。<br/>　　<br/>　　至于贡献度，就是每治疗好一个病人，就有一定的贡献度。当然治疗费用就没有陈默平时要的那么高了，要比他收费标准低的多，特管局也是经费紧张，所以只能给他开放特殊医药库作为吸引力。<br/>　　<br/>　　这条，还是袁若珊告诉领导的。因为陈默这里的收费都是十万元起，甚至那个吴老的治疗费用都要五十万。那么作为特管局的病人，有些好花费大量精力，那么要收取都少费用啊！<br/>　　<br/>　　所以将陈默拉进特管局，作为正式编制人员，就能降低一大部分治疗费用。然后再用贡献度来换取稀有药材等等，那么实际诊费就会低很多。<br/>　　<br/>　　她可是调查清楚了，陈默一直在委托百草堂黄老板收购植株活体，种子等药材，所以才有这个提议。<br/>　　<br/>　　至于还有其他一些权利，就是世俗间的一些了，倒是没有多少吸引力了。<br/>　　<br/>　　陈默听着袁若珊的述说，也就点点头，这已经是最好的结果了。所以陈默就同意加入特管局，作为医师加入的。<br/>　　<br/>　　：。：<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
