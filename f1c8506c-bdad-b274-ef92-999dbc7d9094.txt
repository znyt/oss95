        &lt;h3&gt;第三卷第三百零九章  决定&lt;/h3&gt;
<br/>
<br/>        陈子昂打开门，柴叔和春姨笑眯眯的看着母子三人回家，急急忙忙的将拖鞋给他们三个人找到，拉着陈虞和落妍说：“快洗洗去，别感冒了，也让你妈先去洗洗手。”
<br/>
<br/>        陈子昂母子三个人听话的都各自去洗手了。
<br/>
<br/>        财叔对守中说：“你还没有告诉她？”
<br/>
<br/>        “还没有来得及。”
<br/>
<br/>        “有些事情尽早告诉她，让她好做决定，我这里已经收拾好了。”
<br/>
<br/>        “吃完饭吧，让她休息一会，也不迟这一会。”守中将陈子昂的行李箱交给春姨。
<br/>
<br/>        春姨已经将晚饭准备好了，俩个孩子因为陈子昂回来了，感觉特别兴奋，吃饭的时候都忍不住叽叽喳喳的说过不停，按照他们家的规矩，一般吃饭是很少让小孩子说话的，这一次陈子昂破例的让俩个小家伙不停的斗嘴。
<br/>
<br/>        “妈妈，哥哥输给我了，他心太软。”落妍看着妈妈一脸的骄傲。
<br/>
<br/>        “那你是怎么赢的？”陈子昂放下筷子，看着落妍那调皮的样子就知道落妍一定是使诈了。
<br/>
<br/>        “我要输了，我就大喊：我是你妹妹！”落妍冲着陈虞吐了吐舌头。
<br/>
<br/>        “团团你是怕伤着妹妹了吗？”陈子昂含笑问陈虞。
<br/>
<br/>        “是啊，她是女孩子，又是我妹妹，要不然我会输给她？”陈虞不服气。
<br/>
<br/>        “女孩子和妹妹这几个词语表达的意思是不一样的，陈虞，你真的是心太软，而且好美色，这样不好！”落妍一本正经的胡说八道，惹的大家都忍俊不禁。
<br/>
<br/>        “我哪有，你难道不是我妹妹吗？”陈虞好像从小到大都说不过妹妹。
<br/>
<br/>        “那团团，你以后就不要手下留情了，该怎么样就怎么样，打不过你是她没有学到本领，你没有必要让着她啊，你让着她可以，如果换成别人会让着她吗？同样的，你们俩个人都应该学好防身的本领，想要找你麻烦的人是不会给你手下留情的。”陈子昂开始给孩子们上课了。
<br/>
<br/>        “妈妈，前几天我和落妍就遇到了麻烦，还好有守中老师，不过我和落妍也很机警，我们虽然感受到了威胁但是没有危险。”陈虞看着妈妈带着些许的骄傲。
<br/>
<br/>        “怎么回事？”守中并没有给陈子昂说过别的事情。
<br/>
<br/>        “前几天我带他们出去玩，碰到了一点危险，不过我和守中都已经解决了。看情况是陆家动的手，张函找过我一次。问你将会把孩子带到哪里去。”财叔看着陈子昂。
<br/>
<br/>        “要不，你们带着陈虞和落妍走吧，我等耗材上市之后去看你们。”陈子昂的心在滴血。
<br/>
<br/>        “还有两天，这几天他们都不出门，就等你一起走吧，要不然你不放心。”守中看了一眼俩个孩子。
<br/>
<br/>        “他们还是容不下我们？”陈子昂还是把人想得太好了，她一直默默的努力的生活着，为什么就不能让她安宁。
<br/>
<br/>        “落雪一时半会儿是不会回国内的，在落家和吴家没有确定之前，他们要么拉拢，要么除掉，而你是最不确定的因数，再加上，这俩个孩子虽然没有姓张家的姓，却是张家的血脉。”财叔皱着眉头。
<br/>
<br/>        “你见了张函，张函怎么说？”陈子昂还是想知道张函到底是什么态度。
<br/>
<br/>        “张函说他要去甸城，他和张倩楠准备走了。”财叔叹了口气接着说：“张函希望你们母子平安，他说他此生欠你的，只能等来世再还。如果有可能希望你能找到一个爱人的人，幸福的过好下半生。他还说......他还说，江俞轩人不错。”
<br/>
<br/>        “呵，我的事情轮不到他说三道四，他这是已经选择了和我们站在对立面了。”陈子昂的声音冰冷。
<br/>
<br/>        “妈妈，爸爸说他永远爱我们，永远！”落妍看着陈子昂，眼里有泪花在转动。
<br/>
<br/>        “嗯，我知道，他永远爱着我们，只是他身不由己，好了，我们不说爸爸了，虽然我们不能和他在一起，但是我们会祝他幸福的。”陈子昂不想在孩子的心里种下仇恨的种子。
<br/>
<br/>        在这个世界上，无论如何，血缘亲情是无法斩断的，就像是陈天天。
<br/>
<br/>        “你们都吃好了吗？”陈子昂问陈虞和落妍，她需要和他们说一下以后的计划。
<br/>
<br/>        “嗯。”
<br/>
<br/>        “那我们商量一下，我们应该去哪里上学了。”陈子昂看着落妍，她因为和守中锻炼身体的原因，那胖乎乎的婴儿肥已经没有了，小脸已经成了瓜子脸。
<br/>
<br/>        陈子昂看着落妍和陈虞笑了一下，好像陈家的基因比较强大，无论是男孩子还是女孩子都随陈家多一点，但是陈虞和落妍很明显的有着他爸爸那丰满的唇型和高挺的鼻梁。
<br/>
<br/>        “妈妈说去哪里我们就去哪里吧，我们听妈妈的。”落妍跑过来拉着陈子昂的手，一起朝客厅走去。
<br/>
<br/>        “你能不能有点出息，妈妈是要和我们商量，而不是命令我们，你难道没有自己的主意和意见吗？”陈虞终于是逮着机会反驳落妍了。
<br/>
<br/>        “有啊，怎么会没有?不过，哥哥，你的意见有用吗？”落妍嘻嘻的看着陈虞，陈虞气得狠狠的瞪了一眼落妍，不理落妍了。
<br/>
<br/>        “团团，那你说说你的意见，妈妈听听？”陈子昂为了照顾陈虞的情绪，今天落妍可着劲的欺负哥哥，陈虞的情绪有点低落。
<br/>
<br/>        “有用吗？”陈虞抬头问陈子昂。
<br/>
<br/>        “当然有用啊，你不说我怎么知道有没有用？你告诉我了我才知道你的想法啊。”陈子昂也想征求一下俩个孩子的意见，毕竟他们要生活很长的时间。
<br/>
<br/>        “妈妈，我觉得我们不能去欧洲。”陈虞一语惊人。
<br/>
<br/>        “为什么？”陈子昂好奇的问，财叔和守中也很好奇。
<br/>
<br/>        “爸爸他们应该也能想到我们会去欧洲，你既然不想让他们知道我们去的地方，那就不要选择去欧洲，因为大家都知道落爷爷在欧洲有庄园。”陈虞笃定的说。
<br/>
<br/>        “那我们去哪里？”陈子昂问陈虞。
<br/>
<br/>        “落雪小姨现在在美洲，我们去美洲，因为这一段时间看似落家不太平，实际上落雪小姨只是有她的想法，我们去找落雪小姨。”陈虞期盼的看着陈子昂。
<br/>
<br/>        “妈妈，我也想落雪小姨了，我很多年都没有看到她了。”落妍摇着陈子昂的胳膊。
<br/>
<br/>        陈子昂看了一眼守中和财叔，微微一笑：“那就按照陈虞和落妍的意思准备行程？”
<br/>
<br/>        “好！”
<br/>
<br/>        “妈妈，你真的同意了？”陈虞有些不相信的问。
<br/>
<br/>        “你的建议如此好，我为什么不同意呢？”陈子昂摸了摸陈虞的头。
<br/>
<br/>        “不过呢，你们不能给落雪小姨添麻烦，要学会保护自己。我不在你们身边的时候一定一定要保护好自己知道吗？”
<br/>
<br/>        “知道了，我们会好好照顾自己的。”陈虞和落妍因为能见到落雪心里很开心，倒是抵消了要离开妈妈的不舍。
<br/>
<br/>        其实是财叔早就和落雪联系好了，他们明着是去欧洲，其实是去美洲，落雪已经在美洲为俩个孩子安排好了一切。
