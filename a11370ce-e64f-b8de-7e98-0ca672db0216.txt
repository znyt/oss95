坤珠内的，但是这是他最大的秘密，不可能让人很知道。另外就是，这个人大概要快死了，那么自然也不想收到乾坤珠内，他不想让自己感到膈应！<br/>　　<br/>　　    他不知道这个想杀死自己的人叫什么，但是现在还不是探究的时候。直接将这个人身再次细致的检查了一遍后，确定没有什么东西了。这才抓住他的胳膊，想探测一下他的身体，想看看雷暴符箓的攻击力到底有多大，是如何致人死亡的。<br/>　　<br/>　　    利用真元和神识，在这个人身体内转了一圈，这才探查出来，雷暴符箓的攻击，将这个人的内脏伤了一些。<br/>　　<br/>　　    但是并不致命，最主要和致命的是，雷暴将这个人的大脑击伤，大部分脑神经部都被击毁，也就是，这个人即使被救过来，也算是个死人了。<br/>　　<br/>　　    在医学上有个定义，如果是脑死亡，但是身体还有反应，一切都正常，那么这个人可以判定为死亡，医院可以根据这个下死亡通知单。<br/>　　<br/>　　    看来后天武者的身体还是比较强的，基本能抗住雷暴符箓的攻击，但是大脑和神经的组织，还是比较弱的。<br/>　　<br/>　　    体修修炼加强筋骨和脏器的强度和韧性，但是大脑属于非常精密和柔弱的身体组织，被加强的力度却是比较的。<br/>　　<br/>　　    可能高级体修，或者先天体修，才能将大脑防御变高，初级体修和中级体修，大脑防御很低。<br/>　　<br/>　　    都检查完毕后，单手将这个家伙提出皮卡的车厢，然后对车箱施了一个洁净术，直接将这个人的气味还有痕迹，部都消除。<br/>　　<br/>　　    这个时候，陈默却有些赫然！为什呢？因为刚才在公路上清理痕迹的时候，竟然用树枝清理，难道不能使用洁净术么，只要用了这个法术，什么痕迹不能消除啊！是不是傻啊！<br/>　　<br/>　　    晕！陈默有些自责，发现遇到事情后，他自己依然有些智障和紧张了，如果能从容些，那么这些事情自然都能想的起来。<br/>　　<br/>　　    自己现在已经是位修士了，自然有着修士的手段，但是似乎在关键的时刻，他总是下意识的将自己当成一个普通人！也许，这就是首次杀人后的紧张心情所造成的吧！<br/>　　<br/>　　    自责了一番后，这才提溜着这个家伙，朝着山中奔跑过去。对于陈默来，手中抓着一个将近90公斤的人，还是非常轻松的。<br/>　　<br/>　　    等到了一个山坡的位置，陈默寻找了一下，正好找到一个土凹的地方，不是很大，但是有一人多高，里面也不是很大，但是正好斜凹进去，雨水进不去。<br/>　　<br/>　　    陈默将人扔到地上，想了半天，觉得要是不杀死他，后面处理的话可能更加的残忍，所以只能狠下心来，直接照着胸位置一拳。<br/>　　<br/>　　    在没有防御的情况下，他这一拳直接将心脏给打烂，然后就看到这个人：“呃”了几声，吐出几鲜血后，就腿一瞪，咽下了最后一气。<br/>　　<br/>　　    “别怪我心狠！如果今天是我失败的话，那么我也活不了。杀人者，人恒杀之！”叹了一气，感觉自己自从走上修士的道路后，似乎平静的生活也就不可能了，总是伴随着一些惊险刺激的日子。<br/>　　<br/>　　    这些也许是因为，他自身成为修士后，有了些能力，所以在遇到事情就不会躲避，而是迎难而上，那么麻烦有时候自然升级。<br/>　　<br/>　　    陈默掏出一张符箓，真火符箓！真元一激发，然后扔到了王琦的身上。轰然间身起火，熊熊燃烧起来。<br/>　　<br/>　　    害怕火力不足，他又掏出一张，然后都一一激发后扔到了王琦尸体上。当然，他已经退出了山凹，在较远的地方站着，虽然还能闻到隐隐的皮肉糊味，但是在雨水和山风中却也快速消散。<br/>　　<br/>　　    如果感觉不错，请推荐和收藏一下，谢谢了！<br/>　　<br/>　　    5***<br/><br/><br/>Ps:书友们，我是小学嗣业，推荐一款免费小说App，支持小说下载、听书、零广告、多种阅读模式。书友们快关注起来吧！<br/><br/><br/>
