　　第415章
<br/>　　日头从东边露出一半，这证明李行哉纠结的时间并不算特别长。
<br/>　　他直起身子，双手捧拳面向程大雷。
<br/>　　“程当家……”
<br/>　　程大雷同样捧拳，道：“江湖有缘，日后再会。”
<br/>　　程大雷直接打断李行哉的话，李行哉很感激程大雷，他并没有让自己难堪。
<br/>　　他点了点头，道：“再会。”
<br/>　　程大雷竖起一支手掌，合拢又分开，这在蛤蟆城代表的是退兵的讯号。龙字军开始有条不紊的撤退，前队改后队，过程中竟没有发生任何混乱，他们凭空出现，又悄无声息的消失。
<br/>　　宋伯康从马上跃下来，大步跑到李行哉面前，屈膝跪倒，口中道：“臣死罪，请殿下随我回城。”
<br/>　　天亮了，东方的光芒落在李行哉脸上，照亮他一张沾满泥尘的脸。他注视着东方的太阳，无论地上发生什么事，太阳依旧东升西落，眼前又是崭新的一天。
<br/>　　“走，回城。”
<br/>　　队伍开始返回凉州城，又有一座新的院子为李行哉安排下来。有年轻的丫鬟为他沐浴更衣，有城中的大夫给他清理伤口，敷上精心调制的药膏。
<br/>　　宋伯康坐在他对面，保持着毕恭毕敬的态度。
<br/>　　“殿下受了惊吓，要不要先休息休息一下，等殿下醒来，臣再向殿下汇报。”
<br/>　　李行哉奔波了一夜，本已经疲惫不堪，然而奇怪的是，他此刻一点睡意也无。
<br/>　　慵懒的躺在罗汉床上，他摇摇头，道：“不必了，现在告诉我，长安发生了什么事吧？”
<br/>　　宋伯康表情顿了顿，道：“两地相隔太远，现在长安的情况瞬息万变，想要知道确切的消息，我们只有赶到长安城。”
<br/>　　凉州、长安相距千里，消息传到此地至少要七天，宋伯康得到的情报要比程大雷更完整些，然而对于哪里发生的真切情况，他也并不是完全知晓。
<br/>　　想要看破当时的真相，需要回到七天前。
<br/>　　……
<br/>　　长安城下，尸堆如山。
<br/>　　这不是一个形容词，而是当时发生的真实情况。一具具尸体堆积在一起，火油浇在上面，血肉燃烧发出阵阵恶臭。远处的工程机抛弃巨石，恶狠狠的砸向城头。城头的重弩发出一根根重剑，穿透尸体。工程梯搭起又被推倒，护城河内早已因尸堆而断流，河水变成了血水。
<br/>　　这是杀戮和杀戮的对决，死亡与死亡厮杀。
<br/>　　攻城是从七天前开始的，在七天内对长安城城东门展开主攻。世上有人不怕死，但没人不想活，正义教发起如此惨烈的进攻，是源于他们已经断粮。
<br/>　　撤退会因断粮而死，进攻却可能迎来胜利，如今已经是破釜沉舟的一战。
<br/>　　无数人死亡，然后又有更多人涌上来，再次被杀死。正义教十万大军，已经伤亡过半，但林少羽并没有丝毫退兵的意思。
<br/>　　付出如此惨重的伤亡，同样给守城的李乐天带来巨大压力。这已是守城的第七天，长安城究竟能不能继续撑下去，他心中也完全不知道。
<br/>　　战场之上，瞬息万变，谁又会是天生的胜者。
<br/>　　从城头往过去，是层峦叠嶂的山峰，远处的伏虎关距离此地也并不遥远。李乐天明白，在那附近有数不清的诸侯军。
<br/>　　如果他们肯出兵，两头夹击必然会击破正义教的匪贼。
<br/>　　然而，他们选择按兵不动，冷眼旁观。
<br/>　　帝国失去天下人心已久，他们就这样选择冷眼看着帝国覆灭。面对这一幕，李乐天心中竟无多少愤怒。
<br/>　　因为该愤怒的已愤怒过，无意义的诅咒并没有任何意义。
<br/>　　唯一能做的，只有提起武器厮杀。
<br/>　　李乐天拔出身上的天子剑，重重挥下：“杀！”
<br/>　　李家出昏君，出纨绔公子，甚至废物，但从来不出懦夫，从来都不。
<br/>　　在危境，他们抛开诗书礼乐，掀翻珍馐美味，提起武器冲锋，祖先的血在体内复活。
<br/>　　李乐天亲自出现在城头，与士兵共存亡，这激起了士兵的士气。战斗从黎明开始，持续到下一个黎明，依旧没分出胜负，只是城下的尸体越积越多。
<br/>　　林少羽亲自出征。
<br/>　　他身披黑甲，大氅飘在身后，那只丑陋的大蛤蟆迎风招展，狰狞可怖。
<br/>　　在其身后，竖立着一面大旗，上面斗大一个义字，以正义之名，诛天下不义。
<br/>　　在两方战士眼中，林少羽的身形变得无比高大，他身上若有光泽，似乎真的是天将临凡，来这世间就是为替天行道。
<br/>　　己方体力提升
<br/>　　己方士气提升
<br/>　　敌方士气体力同比例削弱。
<br/>　　一瞬间，战神旗的作用发挥到最大，看到这面纹着义字的大旗，诸人就好像看见不败的象征。本来疲弱的正义教战士，体内涌现出新的力量，在林少羽的带领下，发动了最惨烈的一次进攻。
<br/>　　林少羽一马当先，在他的带领下，正义教卷起一阵战场上的黑色旋风，直扑城门。
<br/>　　攻城车重新发动，盯着火焰撞击城门，一次，两次，从城头落下的滚木礌石被长矛挑开。随着咚地一声响，城门突然从里面被开启。
<br/>　　一群锦衣华服的男人站立在城门口，口中高呼道：“恭迎天将入城，恭迎天将入城！”
<br/>　　城头的战士抗住了一波波攻击，然而城内的商贾富户却扛不住，在第七日的黎明，他们终于混到城门口，打开了城门。
<br/>　　这一幕，不仅李乐天没想到，就连林少羽也是大吃一惊。但这终究是好消息，林少羽带队向城内冲锋。
<br/>　　“这群畜生！”李乐天口中恶狠狠的骂了一声：“来人，把他们堵回去！”
<br/>　　正义教的战士发出一阵阵怪叫，如同饿了三天的野兽。看到城门被破，士兵的士气到达一个顶峰，怪叫狂啸着向城内冲锋。
<br/>　　林少羽长枪指到之处，鲜血淋漓。士兵前呼后拥，这群乌合之众，在帝国活不下去的刁民，面对精锐的帝国军，展现了恐怖的杀意。
<br/>　　城门下血流成河，众人踩着尸体前进，也分不清脚下的尸体帝国军还是正义教。最后，密密麻麻的帝国兵生生的被杀出一条路来。
<br/>　　林少羽纵马踏过尸山血海，踏马破长安。
