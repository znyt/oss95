　　失忆了？
<br/>　　叶少阳也怔住了，随后浑身一颤，猛然想到了一件事，原来……自己遇到不是自己的鬼仆美华，而是……这个世界的美华。
<br/>　　叶少阳一时间不知道怎么办了，不知道是该相认还是当没看见，如果相认的话，他有办法让美华相信自己，但是……这样一来，很可能改变她将来的人生轨迹，毕竟她跟萧逸云不一样，她将来的经历还算挺曲折的，自己不想让她提前知道。
<br/>　　叶少阳强忍住了相认的欲望，冲美华笑了笑，“认错人了。”
<br/>　　然后拉过在一旁呆若木鸡的云儿和毛小方，匆匆走过街道。
<br/>　　“那个姑娘你认识？”从街道走出来，三人在一家没开张的店面门外停留下来，毛小方实在忍不住，道出心中的疑惑。
<br/>　　“跟萧郎君一样。”叶少阳在门外的台阶上坐下，喃喃说道，想到跟美华相见不能相认，他心中说不出的酸楚。
<br/>　　他不知道美华为什么会来这个地方，但是他相信自己绝对不会看错，刚那个的的确确就是美华！
<br/>　　面对毛小方的疑问，叶少阳把美华的身份又说了一遍。云儿不了解情况，根本听不懂，不过也知道他们是在说正事，老老实实地呆在旁边，并不插嘴。
<br/>　　“她是你的鬼仆？”毛小方听了叶少阳的介绍，十分吃惊，“那你为什么不跟她相认？”
<br/>　　叶少阳叹道：“我怎么相认？我认识的是百年后的她，又不是现在的她，我现在跟她相认，很容易对她人生轨迹产生改变……她将来还有一场劫难，万一我告诉她了，我不知道是对是错，更不知道这么做，会对时空造成什么样的影响……”
<br/>　　毛小方沉吟不语，过了一会说道：“在百年之后，你们关系很好？”
<br/>　　“我的鬼仆啊大哥，能不好吗？”
<br/>　　两人又聊了一会，叶少阳心中正在百般感慨，身后突然想起一个声音：“你真的认识我？”
<br/>　　叶少阳猛然回头，看到美华从店铺的另一边走过来，一时间呆住了。
<br/>　　“我刚偷听了你们说话，你说的……都是真的？”美华看着叶少阳，也是一脸的震惊。“你真的是百年后来的人？”
<br/>　　既然她都听到了，那也就没法隐瞒了，叶少阳冲她点了点头。
<br/>　　美华道：“你是哪个门派的法师？”
<br/>　　“茅山叶少阳。”虽然知道这是一百年前的美华，但是跟她好像两个陌生人一样说话，给叶少阳的感觉很奇怪。
<br/>　　美华上下打量着他，又问了很多关于他如何从百年之后跑到这个时代来的问题，叶少阳如实作答，又让毛小方给他作证。
<br/>　　美华听完之后，半信半疑，说道：“我怎么知道这是不是你设的局，来诓骗我？”
<br/>　　叶少阳摊开手，给她看那一抹深红色的魂印：“这就是你的魂印，你是我第一个收的鬼仆，我刚才试了一下，对你没用……”
<br/>　　美华抓着他的手掌，凝视着上面的魂印，有点不可思议地说道：“你有这么多鬼仆？”
<br/>　　“鬼仆妖仆都有。我成立了一个组织，你们都是我的门人……你若不信，我可以说出你的来历，你不是人……你是来自鬼域荫水河中的河姬，你是鬼域之魂，只是用了鬼术幻化出人形而已，不同于人间的鬼尸。”
<br/>　　美华浑身一颤，望着他的表情也起了变化，有些半信半疑，喃喃说道：“没有人知道我是从哪里来的……”
<br/>　　叶少阳恍然想起什么，把自己的天师牌从兜里摸出来，说道：“这块天师牌，是你给我的，你认得么？”
<br/>　　美华接过来看了一下，当场怔住，伸手去摸自己的口袋，摸了一会说道：“我的天师牌好像在洞府里没有带来，没错，你这个跟我的一模一样……你不可能有工夫去偷，这……这究竟是怎么回事？”
<br/>　　叶少阳道：“这天师牌的来历，我也知道，是一位天师在经过荫水河的时候，扔进河里的，被你捡到……你说是不是？”
<br/>　　美华的表情更加震惊，说不出话来。
<br/>　　“你……真的是我百年之后的主人？”
<br/>　　叶少阳点点头，“我本来不想跟你相认的，只是……你既然听到了，我也没办法，只能告诉你。”
<br/>　　美华得知了这一切，心里头千头万绪，低着头，双眼迟钝的望着地面，说道：“我要缓一会……”
<br/>　　叶少阳明白这种感受，点点头，也没有说什么。
<br/>　　过了好一会，美华抬头看他，说道：“我为什么要认你做主人？”
<br/>　　“这……”叶少阳犹豫了一下，说道，“关于这些，如果你将来还要再经历一场的话，我不想提前告诉你，以免泄露天机，对你也不好，而且你这中间的经历，我确实不大清楚。”
<br/>　　美华叹了口气，道：“我明白了，原来这一切都是注定的……我来这里，就是为了找你。”
<br/>　　“找我？”叶少阳怔住。
<br/>　　“大约十年前，我在南方大泽中为妖，有一位法师擒住我，没有杀我，告诉我有一场机缘，对我将来非常重要，他要我搬到这一带，在这个镇上，会遇到一个我等待的人……他传授我一套修行之法，算我半个师父，我就听了他的话，搬到这里，在深山里找了一处地下泉眼，做我的洞府，最近十年之约到了，我每天都会来镇上走走，等待那位法师说的有缘人，我想，应该就是你了。”
<br/>　　叶少阳跟毛小方互相看了一眼，都惊得说不出话来。
<br/>　　“那法师叫什么？”叶少阳急忙问道。
<br/>　　美华道：“我不知道他的名号，是一位道士，穿蓝色古风道袍，留着一丛黑须，看上去不是很老，对了，他的贴身法器，是一对判官笔。”
<br/>　　“是他！”叶少阳浑身一颤，连嘴角都忍不住颤抖起来。
<br/>　　“是谁？”毛小方看叶少阳脸色苍白，急忙问道。
<br/>　　叶少阳深吸一口气，道：“徐福……”
<br/>　　美华听到这个名字，也是当场惊呆了，问道：“你不是说，徐福用了山海印把你带到这个世界来的吗，是那个徐福？”
