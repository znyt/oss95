　　听完这个规则，很多人顿时一下子就都兴奋了起来。
<br/>　　尤其是那些，最后进入第七重星域的闯关者们，他们根本就没有来得及修炼，才只堪堪达到元婴境而已，修为比最先进入第七重星域的龙辰等人，要差上一大截。
<br/>　　如今在这种规则之下，劣势反而就变成了优势！
<br/>　　就连叶尘也没有想到，这规则竟然如此坑人，顿时有一种被戏耍的感觉。
<br/>　　一旁的龙辰，也不由得苦笑道：
<br/>　　“看来这一关，是要告诉我们，修炼之道，欲速则不达，修为提升的过快，未必是什么好事！”
<br/>　　说到这里，龙辰微微一顿，从人群中缓步走出，朗声道：
<br/>　　“既然如此，那就由我来为大家打头阵，诸位没有意见吧？”
<br/>　　龙辰无论是实力还是威望，都是众人之首，自然没有人胆敢反对，而且他们巴不得有人，可以在前面为他们探路呢！
<br/>　　轰隆！
<br/>　　龙辰来到通天塔之前，那高达数十米的塔门，顿时发出一声巨响，然后自动缓缓打开。
<br/>　　龙辰没有犹豫，直接踏入其中，塔门便再次缓缓合上。
<br/>　　而与此同时，众人发现，通天塔的第一层，发出一道璀璨的光芒。
<br/>　　又过了片刻之后，
<br/>　　轰！
<br/>　　通天塔第二层陡然亮起，众人顿时发出一阵阵惊呼声。
<br/>　　又过了片刻，第三层也发出一道璀璨的光芒。
<br/>　　紧接着便是第四层，第五层……
<br/>　　大约一个小时之后，通天塔第八层已经亮起。
<br/>　　“龙老大已经闯到第六关了！”
<br/>　　“不愧是天骄榜首啊！”
<br/>　　“看来这通天塔，应该也没有那么难闯！”
<br/>　　“这可不好说，你以为谁都能龙老大相提并论吗？”
<br/>　　……
<br/>　　众人纷纷议论起来，全都死死盯着通天塔的变化。
<br/>　　很快，
<br/>　　龙辰闯关的速度，也开始慢慢降低下来。
<br/>　　第六层足足用了两个小时，才顺利通过。
<br/>　　一转眼，又过去了大半天时间，也才仅仅达到第九层而已。
<br/>　　只又足足用了将近一天的时间，才只堪堪达到第一十层。
<br/>　　又过去了三天时间，达到十四层。
<br/>　　八天后，达到第十五层！
<br/>　　而在这八天的时间里，众人体内的封印，也在慢慢的解封，即便是不修炼，修为也开始逐渐恢复，不想提升修为都不行！
<br/>　　到了这个时候，众人这才恍然明白过来，龙辰为什么要做第一个闯关者。
<br/>　　因为越早闯关，显然优势就越大！
<br/>　　众人反应过来之后，全都不约而同的朝塔门靠近，显然都想尽快闯关。
<br/>　　就在这时，
<br/>　　人群中的绝无敌两眼一转，脸上露出一抹阴毒之色，忽然闪身从人群中跳了出来，朗声道：
<br/>　　“诸位！龙老大已经为我们做了一个表率，接下来我们该如何闯关，我觉得大伙有必要好好商量一下了！”
<br/>　　说到这里，绝无敌有意无意的瞥了一眼叶尘，继续笑道：
<br/>　　“我有一个提议，不如我们接下来的闯关顺序，就按照之前初始星石碑上的顺序来进行，诸位意下如何？”
<br/>　　众人听到这话，顿时全都为之一愣。
<br/>　　“我赞成！”
<br/>　　“我也赞成！”
<br/>　　云破天和燕狂生，率先从人群中站了出来，表示同意。
<br/>　　按照在初始星石碑上的排名顺序，他们的排名都十分靠前，而叶尘却是倒数第一，绝无敌这明显是在针对叶尘，他们自然也乐见其成。
<br/>　　绝无敌得意一笑，转而又望向风华舞和乐无忌，
<br/>　　“风师姐，乐师兄，想必两位应该也没有意见吧？”
<br/>　　风华舞想了想，当即也附和道：
<br/>　　“我也赞同！”
<br/>　　乐无忌默然不语，虽然没有表态，但也算是默认了。
<br/>　　至此，七大天骄之中，除了还在闯关的龙辰，和一直一言不发的洛冰玄，其余五人已经达成了共识。
<br/>　　至于其他众人，眼看几大天骄发话，他们虽然心有不甘，却也知道根本反抗不了，只好纷纷表示赞同。
<br/>　　洛冰玄张了张嘴，似乎想要说些什么，最终却又忍住了。
<br/>　　叶尘将众人的表情看在眼中，只是冷冷一笑，没有多说什么。
<br/>　　说到底，七大天骄在这里，具有绝对的话语权。
<br/>　　绝无敌、云破天和燕狂生，明显已经达成了共识，要联手排挤他。
<br/>　　乐无忌虽然和他有些交情，却未必会帮他，加上洛冰玄似乎也在刻意疏远他。
<br/>　　因而，以他如今的修为，还不足以干掉这三人，所以只能暂时先忍着了。
<br/>　　一晃眼，又过去两天时间，
<br/>　　轰隆！
<br/>　　通天塔猛然一阵剧烈晃动，
<br/>　　继而，
<br/>　　砰！
<br/>　　一道人影，从通天塔第十五层，被直接弹飞了出来，赫然正是龙辰。
<br/>　　只见龙辰脸色惨白，似乎是受了伤，从通天塔之中弹出后，当即便身体一晃，回到自己的修炼道场之中，开始休养调息起来。
<br/>　　与此同时，
<br/>　　那道淡漠的浑厚的声音，再次在九天之上回荡开来，
<br/>　　“龙辰，闯关层数，十五层！下一位！”
<br/>　　轰隆隆！
<br/>　　声音落下之后，通天塔的大门再次缓缓打开，
<br/>　　按照之前众人商定的顺序，自然也就轮到石碑排名第二的洛冰玄了。
<br/>　　然而，洛冰玄却眉头紧蹙，不知道在想些什么，并没有立即踏入其中。
<br/>　　一旁的云破天，顿时有些不耐烦，忍不住催促道：
<br/>　　“冰玄仙子，你若是想放弃的话，那我可就先上了！”
<br/>　　洛冰玄迟疑了片刻，最终冷哼一声，还是缓步迈入了其中。
<br/>　　轰隆！
<br/>　　塔门再次合上，通天塔第一层，也发出一道璀璨的光芒。
<br/>　　很快，
<br/>　　第二层……第三层……第四层……
<br/>　　同样是从第六层开始，洛冰玄的速度，就一下子开始减慢了下来。
<br/>　　“你们有没有发现？龙老大和冰玄仙子，都是从第六层开始，速度开始变慢的，也就是说，从第六层开始，闯关的难度，开始大大提升了！”
<br/>　　不知道谁冒出这么一句话出来，顿时得到众人的纷纷赞同。
<br/>　　这时，又有一道声音陡然从后方响起，
<br/>　　“你这话，并不完全正确！”
<br/>　　众人循声望去，只见这说话之人，赫然正是之前一直在休养调息的龙辰，不知道什么什么时候，已经也来到了通天塔前。
