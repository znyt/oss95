　　徐文长也跟下来，对林三生道：“不然我破开虚空，你去空界求援，将你的大军引来。”
<br/>　　“来不及了。”
<br/>　　林三生望着战场，十分清楚，血池罗刹的布局，在短短几分钟之内就能彻底完善，也就是说，如果叶少阳想突围，只有眼下还有机会，一旦等布局完成，那被困住的所有人，都是死路一条。可如果现在突围，从眼前的局面来看，最多也只能活一半。
<br/>　　叶少阳一心要为秋莹报仇，已经杀红了眼，毫无保留地对穷奇展开猛攻，看上去根本没有理智了。
<br/>　　“无论如何，都是个死局了。”慕寒突然喃喃说道，回头看了林三生一眼，眼中带着一丝难以掩饰地幸灾乐祸。
<br/>　　慈心师太也有点幸灾乐祸，但其余人则更多是紧张和担忧。
<br/>　　“捉鬼联盟，是人间最强的一股力量，如果全军覆没，人间必备荼毒，不是太阴山，就是轩辕山。”林三生望着他们，说道，“我知道你们讨厌叶少阳，但你们若还是人间法师，便应当以大局为重。”
<br/>　　大伙听了这话，都很不理解。
<br/>　　“已是死局，与我们有什么关系？”慕寒不解地问道。
<br/>　　“你们若不搭救，人间涂炭，你们都是罪人。”林三生语气很镇定。
<br/>　　慕寒冷笑，“林大帅，我们这里才多少人，敌人百倍、千倍于我们，就算我们舍命冲锋，又能杀得了多少？根本打不开封锁，还白白牺牲了性命。叶少阳他们为了一己私仇，陷入绝境，怪得了谁？”
<br/>　　说出这番话，他内心也有些庆幸，如果叶少阳等人不是被困得这么死、有机会逃脱的话，他们还真的不好见死不救，幸好他们被重重包围，任谁都看得出，根本没有一线生机，作为旁人的他们，就不用担心被人诟病，反正都是救不了，那便不用奉献无谓的牺牲。
<br/>　　“我不要你们硬拼，只求待会若有变故，你们听我号令，一同冲杀过去，会有死伤，但我保证不全军覆没，并且叶少阳他们也能得救。”
<br/>　　所有人怔住，呆呆地看着他。慕寒失笑道：“林大帅，你不是在开玩笑吧，这等局面，他们哪里还有生还的机会？”
<br/>　　“你看不到，不代表没有。”
<br/>　　“林大帅，我凭什么信你？”
<br/>　　“凭你叫我林大帅！”林三生咬着牙，狠狠说道，“凭我统帅三军，力挽狂澜多次击退尸族，运筹帷幄百战百胜！”
<br/>　　林三生的目光异常坚定，逼视着慕寒，这种极端的自信，终于让慕寒退缩了，他暗暗吸了口气，道：“你有什么计策？”
<br/>　　“我没计策。”
<br/>　　“什么？”慕寒以为自己听错了。
<br/>　　“现在能救他们的，只有叶少阳。”
<br/>　　慈心师太插了一句：“叶少阳不是神，都这样了，还有什么机会？”
<br/>　　“别管这个，我只希望，真到那一步，你们能动手施救，慕寒，你口口声声自己是法术界盟主，你能看着叶少阳死？”
<br/>　　“这……若真有机会，我一定搭救。”慕寒冷笑，“不过我有要求，若万一叶少阳不死，将来他和你，都必须遵我为法术界盟主，不可再同我作对。你答应吗？”
<br/>　　“可以。”林三生不假思索。
<br/>　　“你是大帅，令出不改，我是相信的。”慕寒又看了看战场，摇头说道，“不过虽如此说，我还是看不出有任何机会，简直必死无疑……”
<br/>　　得到他的保证，林三生就不再搭理他，转头又跟冲和子说了类似的话。冲和子很干脆地点头答应。“别说是出了机会，就算是现在，只要你一句话，我等皆奋不顾身！”
<br/>　　他的态度，让慕寒和慈心师太脸上有点尴尬，只当没听见，继续观望着战场。
<br/>　　林三生也观望着，眼中难掩忧色，双拳紧握，心中默默说道：“少阳，但愿我没有猜错，死中求生，这可是你一贯的强项啊，纵然十面埋伏，也不是没有一线生机……”
<br/>　　“老大，不然我们突围吧，我掩护你们！”美华一边用水波攻击穷奇，一边忧心忡忡地对叶少阳说道。
<br/>　　“报仇！”
<br/>　　叶少阳淡淡说出这两个字，再一次挺剑而上。
<br/>　　“老大，再不走，我们都得死了，我不怕死，我愿意留下掩护，只希望能多逃走几个兄弟！”美华急得快要哭出来。
<br/>　　“老大……”橙子也恳切地望着他。
<br/>　　“白眉！你怕死么？”
<br/>　　“啥？”吴嘉伟落在小青的背上，缓了一口气，朝叶少阳看去。
<br/>　　“要死了，怕不怕？”
<br/>　　“哦，没事，死也剁了这个怪物！”
<br/>　　“好。”叶少阳环顾左右，“你们听好了，小青小白美华橙子，你们从四方围攻，使出全力，其余人等候命令，至于外头那些，不用去管他们！白眉，我给你当英雄的机会！”
<br/>　　叶少阳咬牙厉声喝道，“假如这是我们人生最后一仗，也要打好这一仗，管它是谁，剁了他，为秋莹报仇！”
<br/>　　既然叶少阳这么决定了，大伙反而不多想了，抖擞精神，使出了最强手段，再一次发动了猛攻。
<br/>　　叶少阳冷静地观察着。
<br/>　　这一段时间的围攻，并不是盲目的，叶少阳一直在观察，他看出了穷奇的弱点，这个三界之内数得着的强者，也有弱点，而且很直接：不擅水战。
<br/>　　如果在陆地上，纵然他们这么多人，也难以伤到它，但经过一段时间的围攻，穷奇已经被压制在水中，纵然它修为极深，能御水行走，但行动多少受到影响。
<br/>　　而小青小白美华橙子，都有强大的水中战斗的能力，这是他们的优势，也几乎是唯一的优势。
<br/>　　这一次总攻，叶少阳没有盲目出击，他一直在指挥，而大伙也都严格贯彻他的要求，小白四人，在水中兴风作浪，缠住了穷奇。
<br/>　　“动手？”雪魔在血池罗刹耳边问道。
<br/>　　“不，他们纵然能拖住局面，但不可能杀掉教主。”血池罗刹环顾左右，道，“而今我们阵型未稳，他们发动袭击，就是想引我们动手，好趁乱突围，何必中计。”
