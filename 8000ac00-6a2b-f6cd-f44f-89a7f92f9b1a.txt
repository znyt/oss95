　　第8章
<br/>　　程大雷对这个世界的医疗能力有一定误解，电影中大侠受伤后，随便缠几条绷带，没多久就可以完好如初。然而事实上，在没有抗生素，细菌病毒滋生的时代，感冒就能要人命，破伤风就能放倒最强壮的汉子。
<br/>　　林少羽的确没有致命伤，可身上的擦伤已经发炎，从昨天开始他就高烧不止，如今已经深度昏迷。
<br/>　　程大雷将他拖回山寨，几个人帮着给他清理伤口，又灌了些热水，程大雷记得系统奖励中有10枚还魂丹，找到后喂他服下，应该问题不大。
<br/>　　“大当家……”秦蛮面带喜色，一张黑脸露出憨厚的笑容。
<br/>　　“怎么了？”
<br/>　　“刚才与此人交手，我隐隐有所领悟，我的梅花枪又进一层。”秦蛮喜道。
<br/>　　程大雷再次查看秦蛮的信息。
<br/>　　姓名：秦蛮（初出茅庐的顶级山贼）
<br/>　　技能：梅花枪（五蕊）
<br/>　　记得系统有条奖励，随即提升山寨内一人技能等级，原来是落在秦蛮身上。这枪法的提升，也直接令他个人等级从优秀变成顶级。这放到军队中，已经足有百人敌的实力。有秦蛮在，应付黑石城的攻打就多一层把握。
<br/>　　“要是有一柄枪就好了，梅花枪传到我手中，可我连打口枪的钱都付不起，唉……”
<br/>　　说着，秦蛮重重叹息一声，手拍在撑着屋顶的木柱上。只听哐当一声，一物竟然从房顶掉下来，刚巧不巧落在秦蛮面前。
<br/>　　咦！
<br/>　　屋内四人都睁大眼睛，秦蛮刚想说要枪，就有一枪从天而降，这也太巧了些。秦蛮将大枪捡起，漆黑枪身，一人多高，枪尖闪着黑光。
<br/>　　“军师，这枪……怎么在房上？”秦蛮吃惊道。
<br/>　　“我也不知道……难道是当年盖房时，当柱子使了？”徐神机也是一脸错愕。
<br/>　　只有程大雷明白怎么会事，这枪是系统奖励，不过出现的方式，还真是挺别致的。今天刷BUG刷出好多奖励，自己该把他们都找出来。
<br/>　　“这柄枪以后就由秦大哥用。”程大雷道。
<br/>　　“真的！”爱不释手的秦蛮一脸震惊，在得到程大雷肯定的答复后，他忽然手持大枪窜出屋子，来到山寨的校场。
<br/>　　秦蛮本就是用枪的，但一柄大枪要四五十斤铁，他靠打猎根本买不起。在几户人合用一个铁锅的时代，兵器，是上等人才能玩得起的。
<br/>　　别看林少羽好像混得不怎样，但一柄精致铁枪、一匹高头大马、一身丝绸长袍就远超许多人，奢侈的完成梦想。这放在程大雷前世，就相当于一身欧洲私人定制款西服，开着豪华跑车风驰电掣无比风骚的穿过街头。
<br/>　　总之，行走江湖的梦想谁都有，但不是谁都付得起这笔钱。
<br/>　　秦蛮做梦都想拥有一柄自己的铁枪，打猎时可能都盘算着何时能够攒够，实在不行，先打个轻点的也行。而今天，却是直接拥有一柄各方面都符合自己想象的铁枪，秦蛮的幸福感直接爆棚。
<br/>　　他拖枪立在校场上，忽然一挺身，枪身绷直，扎入毒蛇吐信，甩如猛虎摇头，一柄枪在他手中舞得风雨不屑，一朵朵枪花在空气中一闪而逝。
<br/>　　“好！”
<br/>　　徐神机和徐灵儿拍手叫好，他们虽然看不懂，但也能感受到秦蛮舞枪时的杀威。
<br/>　　秦蛮收枪而立，脸上浮现一种喝醉般的红。
<br/>　　正这个时候，程大雷扛着一柄锄头从仓库出来，徐神机奇怪道：“大当家，你要做什么？”
<br/>　　“挖宝。”
<br/>　　“挖什么宝，咱们蛤蟆寨哪有宝，大当家你不要搞破坏。”
<br/>　　程大雷不去理他，系统奖励都在山寨内，自己要一样样找出来。
<br/>　　徐神机小跑跟着程大雷，忽然跌了一跤，脚踢开一块石头。他正捂着脚喊疼，忽然看到石头下压着一个陶罐。
<br/>　　“咦，这是什么？”
<br/>　　程大雷打开后看到陶罐内放着十枚绿色丹丸，这应当就是系统奖励的还血丹，可以治愈一般伤病。
<br/>　　“去，给林少羽吃一颗。”程大雷将还血丹递给徐灵儿，自己继续挖宝。
<br/>　　他来到马圈中，现在山寨一共有三匹马，一匹是山寨原先的瘦马，一匹是劫来的，一匹是林少羽的。因为获得奖励时，林少羽还未加入蛤蟆寨，坐骑未获得升级。
<br/>　　程大雷查看这两匹马的信息。
<br/>　　普通黄马（瘦马）
<br/>　　技能：逃逸（逃跑时获得速度加成）
<br/>　　普通黑马
<br/>　　技能：冲锋（冲锋时获得速度加成）
<br/>　　令程大雷惊喜的时，不仅是两马的等级提升，而且分别都领悟了技能。
<br/>　　“秦大哥，这匹黑马以后就是你的坐骑。”程大雷。
<br/>　　“我的！”秦蛮惊喜道，短短时间，自己又是枪法提升，又是获得武器，现在连马匹都有了，这世界对自己是不是太好了。要是再有一副盔甲，自己驰骋沙场的梦想就一步到位。
<br/>　　徐神机有些晕，他站起来跑到马圈，远远喊着：“大当家……”
<br/>　　噗通一声跌倒，徐神机发现绊倒自己的是埋在土里的一口木箱。他刚站起来，手扶在大树上，啪地一声，从树上落下两幅盔甲。
<br/>　　程大雷，秦蛮，徐灵儿将他围了起来，原来玄机在徐神机身上。
<br/>　　“你们看我干什么，我，我……也不知道怎么会事啊？”
<br/>　　“盯紧军师。”
<br/>　　徐神机哇呀一声向后跑，说也奇怪，他每经过一个地方，总能在那里发现些东西。埋在石头下的战鼓，土里的木箱，桌子下面蒙灰的大旗……
<br/>　　最后徐神机跑到山寨内一口枯井前，心里想着，这应该没事了吧。无意间往井中一看，废弃十多年的枯井开始往外冒水。
<br/>　　“哇，见鬼了。”
<br/>　　里里外外将山寨翻了个遍，该找到的东西都找到，四个人聚在校场的旗杆下，程大雷清点今天的收获。
<br/>　　精致铁枪1柄，普通盔甲2幅，还血丹10枚，战鼓1口（附带技能：鼓舞），大旗1面（附带技能：招募），烈酒30坛，镰刀10柄，青牛山地形图1幅，文士服1套。
<br/>　　农田10亩，水井1口，山泉1口。
<br/>　　山寨获得技能治愈，齐心，坐骑集体升级，获得技能。
<br/>　　更惊人的数字是找到木箱50口。
<br/>　　还真是收获满满呐，当然最后系统给出的奖励也越来越水。四个人都被震惊了，徐神机神秘兮兮道：“大当家，你告诉我，是不是老寨主临走前给你留了什么遗言？”
<br/>　　程大雷不置可否，徐神机抓起那套文士服，沾沾自喜道：“这件衣服简直就是为我量身打造，穿上之后我的谋略肯定翻倍”
<br/>　　“放下，那是我的。”程大雷喝止他，你的智商本就是零，翻十倍也还是零。
<br/>　　不过令程大雷遗憾的是，系统给出这么多奖励，连白袍都有了，却还是一口剑也没有。难道说，自己就没有佩剑江湖的气质么。
<br/>　　目光落在那堆箱子上，程大雷眼睛一亮：“咱们开箱子吧。”
<br/>　　
<br/>(本章完)
