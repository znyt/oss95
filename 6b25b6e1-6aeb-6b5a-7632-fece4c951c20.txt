　　杭山市。
<br/>　　佟千雪已经把这次旅行所需要东西全都收拾妥当，查询过T国当地几天天气后，佟千雪还特意带了几件长袖。佟千雪现在是很忌讳感冒生病的，如果不是佟千雪执意要去T国，家里人根本不会让她外出。
<br/>　　出发前，佟千雪还给慕容飞打了一个电话。
<br/>　　“您好，请问您找谁？”
<br/>　　电话中，传来警通连值班士兵的询问。
<br/>　　“您好，麻烦您帮忙给慕容飞带一句话，告诉他不用担心我，我会照顾好我自己。”佟千雪说道。
<br/>　　“你是谁？”值班士兵问道。
<br/>　　“我姓佟，我是他的妻子。”佟千雪回答道。
<br/>　　“好的，我一定会把你的话转告给无常。”值班士兵说道。
<br/>　　“谢谢。”
<br/>　　佟千雪道了一声谢，挂断了电话，拖着行李箱走出了房间。
<br/>　　……
<br/>　　龙隐部队，警通连值班室。
<br/>　　接听佟千雪电话的值班士兵放下电话，眨巴着眼睛对身边的军士长问道：“班长，无常结婚了吗？”
<br/>　　四级军士长瞥了眼值班士兵，说道：“这已经不是新闻了，你的消息也太不灵通了。据说啊，无常跟他妻子是在一次行动中认识了，无常上演了一出英雄救美，两个人就这么对上眼了，无常妻子好像还是个医生，长得很漂亮……”
<br/>　　四级军士长还准备往下说，他旁边的值班士兵用手捅了捅他，示意他往值班室外看。四级军士长往值班室外看了眼，看到司徒空站在值班室外后，立刻从椅子上站了起来，对着司徒空敬了个礼，喊道：“龙猫班长！”
<br/>　　“编，接着编，我听你刚才故事编的挺好啊。”司徒空拿起放在窗台上那叠信件，一封一封翻看着，从里面找出战术中队的信件，抬眼看了眼一脸尴尬的四级军士长，说道：“怎么不说了？”
<br/>　　四级军士长尴尬的笑着，没有答话。
<br/>　　“对了，刚才是佟医生的电话？”司徒空对值班士兵问道。
<br/>　　“是个女的，对方自称姓‘佟’。”值班士兵回答道：“她让我给无常班长带句话，说让无常班长不要担心她，她会照顾好自己。”
<br/>　　“行，我知道了，我会把话带给无常。”
<br/>　　说着，司徒空就摘下挂在腰间的对讲机，对着对讲机说道：“无常，你老婆给你带话，让你不要担心她，她会照顾好她自己。”
<br/>　　警通连值班室里，刚被司徒空训了一顿的四级军士长和值班士兵嘴角狠狠抽了抽，龙猫班长，你就是这样帮忙带话的呀。
<br/>　　对讲机里并没有传来慕容飞的回答，但是只是想想就能知道，此事慕容飞的表情一定非常精彩。
<br/>　　司徒空拿着战术中队的信件正准备离开，他的对讲机中传来林晓阳的提醒，“龙猫，你赶紧跑吧，无常正朝着警通连值班室过来呢。”
<br/>　　“我去！”
<br/>　　司徒空脸色一变，拔腿就跑，留下值班室里的四级军士长和值班士兵面面相觑。
<br/>　　……
<br/>　　T国，Q市。
<br/>　　经过接近四个小时的长途飞行，佟千雪所乘坐的T661航班降落在Q市国际机场，这次佟千莹是带着学生来跟T国Q市某大学的进行学术上的交流。
<br/>　　地处于东南亚的T国，因为其独特的地理位置和气候环境，使得这里的毒虫、蚊虫繁殖很快，并且数量众多，T国当地有很多使用草药、毒虫、蛇毒制造的膏药具有很好的效果，这次佟千莹带着学生来这边进行的就是有关于这方面的交流。
<br/>　　住宿的酒店是安排好的，一家五星级酒店，距离佟千莹进行交流的大学差不多有半个小时的车程，每天都会有车安排接送。
<br/>　　佟千莹把自己的衣服从旅行箱中拿了出来，放在酒店房间的衣柜中，对佟千雪说道：“千雪，一会儿我就要去那边跟他们进行学术上的交流，你自己一个人待在酒店里不要乱跑，吃饭就去一楼的餐厅。”
<br/>　　佟千雪靠在枕头上，轻轻地“嗯”了一声，长途飞行对普通人来说都是一件很累人的事情，尤其是飞机座位狭小的空间，就算是想要睡一觉都会很不舒服，越睡越累，更何况是对一个已经怀孕快四个月的女人。所以，现在佟千雪什么都不想，只想要好好睡一觉。
<br/>　　佟千莹看着躺在床上昏昏欲睡的妹妹，轻轻摇了摇头，最后跟T国大学方面联系了一下，确定了明确的时间后，拿着学术资料离开了酒店房间。
<br/>　　……
<br/>　　T国，西南方向某靠海村落。
<br/>　　这里就是普提查的老巢，毒窝。
<br/>　　在T国涉及“贩毒”和“造假”会面临重刑，甚至是枪毙，所以很多国家的游客来到T国旅游都会选择带一些T国当地的常用药回去，因为不用担心会买到假药。
<br/>　　T国因为靠近“金三角”的关系，曾经有一段时间贩毒非常猖獗，甚至还有大规模的罂粟种植园；但是上世纪九十年代末到二十一世纪初的这几年时间中，国际刑警组织和T国军警的多次联合行动，打掉了T国当地绝大部分的罂粟种植园和贩毒组织。
<br/>　　禁毒，是一个长期的话题，不会因为几次联合行动就彻底销声匿迹，那些毒贩或是逃到了“金三角”、缅甸这些地方，或是像普提查这样龟缩到了沿海的小村落里。
<br/>　　这里本来是个渔村，可是在普提查来了以后，这里就变了，男人和老人被作为劳动力，女人则是成了普提查和他手下那些毒贩的发泄工具。
<br/>　　普提查嘴里咬着一根雪茄，提着裤腰带从一间屋子里走出来，这时一个背着一把AK的毒贩匆匆朝着他走来，说道：“老大，那个女人入境了。”
<br/>　　“哦？”普提查眼睛轻轻眯了眯，对那个毒贩问道：“她现在住在哪儿？”
<br/>　　“Q市，王权酒店！”毒贩回答道。
<br/>　　“王权酒店？”普提查摸了摸自己的下巴，说道：“这倒是有些难办了，那里可不好动手。”
<br/>　　“老大，我们可以伪装成酒店服务员混入酒店，假装成客房打扫，然后……”
