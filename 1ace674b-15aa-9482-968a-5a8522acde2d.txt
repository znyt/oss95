
<br/>　　<br/>　　    古城的内部远比外面要来的更壮观许多，也更加惊悚了些，外面看的只是轮廓再加上能见度较低，所以根本没办法感受到这座城池的气质，真正进入古城里，身影飘荡在一座座石头砌起来的建筑旁，最关键的是身边还有着一具具的水下尸体，你就能身临其境的感受到，自己仿佛是在一座死城中了。<br/>　　<br/>　　    一进入古城内部，顺着街道的走向渐渐前进，就看见了几具两三千年前的尸体，尸体上穿着麻织的衣服，都是男性前倾着身子，手中拿着青铜的器具，紧闭着眼睛，仿佛雕像一样静止不动，当王惊蛰他们游过来后，几个人带起的水流散开带动着尸体轻轻的晃了晃，王惊蛰顿时感觉后背被人给撞了下，就看见一个苗条的身体紧紧的贴上了他，一缕长发飘在了他的脸前。<br/>　　<br/>　　    也就是王惊蛰和陈三岁对这种场景能有接受能力，耿明是曾经见过一次有点心里准备了，剩下的几个，包括那些蛙人都震惊在了滇国古城这诡异的气氛中。<br/>　　<br/>　　    王惊蛰朝着后面打了个手势，示意人全都停下，他随即来到一具尸体前，古城已经沉入抚仙湖两千多年以上了，也就是说这些尸体被泡在水里也得是两千多年了，但你无论如何都想不通的是，为啥这些尸体还都没有腐烂，就算不腐烂用水泡这么长时间也该泡烂了，更何况水中还有鱼虾一类的生物，会啃食尸体，可不管是什么因素，这些尸体保持的依旧非常完整。<br/>　　<br/>　　    尸体的皮肤表面覆盖了一层水泡，衣服上也挂着一些青绿色的水草一样的生物，但并不太多，王惊蛰举起手里的手电照在了尸体的头部，从脑袋上的头发开始缓缓的朝下移动，来回看了两遍以后，他忽然伸出手轻轻的扶向了尸体的脸，手还没触碰上的时候，李菲儿和文峰快速的游了过来，一把拉住了他的胳膊，同时猛地摇了摇头，示意他不能这么做，人的手是有温度的，并且还有可能带着细菌，从他俩的角度来看，王惊蛰如果接触上尸体的肌肤，是完全有可能破坏掉这些两千多年历史的尸体的。<br/>　　<br/>　　    王惊蛰一把甩开了李菲的手，伸出一根手指直接错在了她的胸口，点了几下后，晃了晃脑袋，那意思是我要干什么有我的道理，你别多管闲事。<br/>　　<br/>　　    在这个沟通全靠手的水下世界，交流起来是挺坑爹的。<br/>　　<br/>　　    李菲儿和文峰仍旧不依不饶的拦在他的身旁，谨慎的盯着他，王惊蛰顿时就不耐烦了，直接伸手推开李菲，然后抬腿一脚就踹了过去，蹬在了文峰的胸膛上，他这一动手几个蛙人还有耿明都过来了慌忙把他给拦住了。<br/>　　<br/>　　    水面的船上，从视频里看见这一幕的人全都皱眉了，张所长咬牙说道：“他能不能有点组织纪律了？这种场合还打人，万一出事了怎么办？”<br/>　　<br/>　　    黄九郎当即跟黄詹说道：“黄教授你的人要是这么不配合，我很难做啊，王惊蛰回来恐怕又得起脾气了”<br/>　　<br/>　　    黄詹皱眉说道：“李菲儿他们也有自己的考虑，王惊蛰用手去碰水下的尸体，是完全有可能出现意想不到的连锁反应的，那些尸体保存了两千多年都没有异样，一旦接触到外物，出现什么状况谁也保证不了，那些可能是世界上仅存的人类死体标本了”<br/>　　<br/>　　    黄九郎淡淡的说道：“尸体有很多具，就算有问题了也没什么，但王惊蛰只有一个……”<br/>　　<br/>　　    黄詹顿时就呆了一下，随即叹了口气，点头说道：“我知道了，回头上来我会交代他们的”<br/>　　<br/>　　    王惊蛰抹开尸体脸上的水泡还有耷拉下来的头发，用手电照着尸体的脸孔，他是想看看这人是如何死的。<br/>　　<br/>　　    人死了，不管是横祸，暴毙还是老死等自然死亡都会在面相上显现出来，但这些尸体就蹊跷了，他们的表情平淡神态安详，完全没有任何挣扎的痕迹，这根本就不可能是抚仙湖的水涌入古城中后淹死的迹象。<br/>　　<br/>　　    王惊蛰深深的望了几眼，心中一片震惊，因为他无法从尸体的脸上看出他们的死因。<br/>　　<br/>　　    这些，完全都不像是一个死人。<br/>　　<br/>　　    观看良久以后，耿明忽然过来，伸手碰了他一下，然后抬起手腕指了指，示意王惊蛰这次下水的时间到了，在不返回去的话，氧气就不够用了。<br/>　　<br/>　　    王惊蛰点了点头，朝着他们摆了下手，然后朝上面指了两下，脚蹬着地面，身子开始朝上游了过去。<br/>　　<br/>　　    “哗啦”十几分钟之后，王惊蛰钻出水面，紧接着后面几人陆续露头，李菲儿和文峰出来后看见他，就一脸愤怒的游了过来。<br/>　　<br/>　　    “你不能擅自去触碰那些尸体，你无权损毁这些古物……”<br/>　　<br/>　　    王惊蛰打量了她几眼，压根就没搭理她，张开两手朝着船游了过去，顺着绳梯爬回到船上。<br/>　　<br/>　　    “你是不是有病？你刚才在水下和我们动手，万一出了什么事，你就是杀人犯了，船上这么多人看着，你还能抵赖不成？”文峰上来后，愤慨的冲着王惊蛰抻着脖子吼道。<br/>　　<br/>　　    黄詹连忙走了过来，瞪了文峰和李菲儿一眼，然后和王惊蛰说道：“以后有什么事，我们可以商量着来，惊蛰你动手有点危险了，万一再出什么事的话，想弥补就来不及了”<br/>　　<br/>　　    王惊蛰摘掉氧气瓶，脱着潜水服说道：“我的办事风格就是，让我参与就一切都听我的，要是不听也别给我当累赘”<br/>　　<br/>　　    张生利皱眉说道：“但你在那种环境下，动手也太过分了吧”<br/>　　<br/>　　    王惊蛰抬起手，指着他道：“下次换你去？”<br/>　　<br/>　　    张生利被他怼的张了张嘴，最后憋出一句还：“我是主管行政的……”<br/>　　<br/>　　    王惊蛰换完衣服，今天的发掘工作也就暂时告一段落了，一共水下作业三个小时左右，进展也不过就是刚展开了一小部分，往后几天还得循序渐进的接着来。<br/>　　<br/>　　    下午，天黑之前船返回到了岸上，晚上的时候考古队也召开了个总结的会议，无非就是讨论问题，研究下一步的计划，王惊蛰也参加了，不过他就在人群外面站着也没插嘴，实事求是的讲，关于学术方面的问题，他这个小本毕业的确实找不到什么牌面。<br/>　　<br/>　　    一天过后，隔天，考古发掘继续，按照由外往内的顺序一点点的往前推，这一天基本没啥太大变故，进行的还算比较顺利，下水两小时回来休息一段，然后再继续下潜，距离逐渐的往古城中那座高达十层楼的金字塔推进着。<br/>　　<br/>　　    两天过去，古城内部绝大部分的区域都已经被探索过了，剩下的就是中间那一块了。<br/>　　<br/>　　    勘察第四天，天气，阴。
