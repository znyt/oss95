　　第28章
<br/>　　这次程大雷计划的工作量很大，大部分建筑都需要推倒重建，不过，在人手充足的情况下，这都不算问题，工程进度以可怖的速度推进。
<br/>　　正在的问题不在这里，而在于这些干活的山贼。他们是山寨，不是善男信女，也不仅仅是为每天五斤粮食，两顿饱饭来打工的打工仔。他们很多人的目的都是吞并这里，占领这里。这些人可是随时会炒掉老板，这里说的『炒』，就是字面意思上的炒。
<br/>　　当工程接近完工的时候，问题越发凸显。首先是越来越多人对黄三元的指挥不屑一顾，再然后就是在吃饭时骂骂咧咧，互相之间动手动脚。
<br/>　　这一天中午，周小树扒拉着一碗面条，话说，面条的味道可是真香呐。
<br/>　　“都快点吃，吃完饭咱们就动手。”高飞豹。
<br/>　　周小树动作顿了顿，他不想动手，每天五斤粮食，两顿饱饭他已经觉得很幸福了。如果这样的日子可以一直继续下去，那么肯定是自己上辈子积了大德，才有这样的福分。
<br/>　　但他只是一个小喽啰，他听二当家的。
<br/>　　“二当家，可是秦大哥……秦蛮的力气你也见到了？我听说蛤蟆寨的大当家，还会五雷法。”
<br/>　　“扯淡。”高飞豹道：“这两天我一直在故意找茬，就是想要试探他们的反应，真有五雷法早就用了，还能这样容忍我。听我的，吃完饭叫我没干活的时候都不要动，看我眼色行事。”
<br/>　　“嗯呐，好吧。”几个人小声答应，他们心里对五雷法还是有些畏惧的。
<br/>　　转眼便过了午时，以前这个时候，大家都已经开始干活，但今天这些山贼三三两两懒在地上，一动不动。
<br/>　　“大家开始干活了，干活了！”
<br/>　　“黄管家，天这么热，怎么干活，晒死了怎么办。”高飞豹懒懒道。
<br/>　　“嘿，弟兄们，咱们抓紧时间干活，晚上我加饭。”
<br/>　　“吃龙肉也干不动喽。”
<br/>　　依旧没有人动，人群发出无声的嘲笑。黄三元目光转了一圈，他知道事情已经开始变得不可控。
<br/>　　“喂，过来几个人，把这块石头砸了，挖出来垒墙。”黄三元走到一块般埋在地上的大石前。
<br/>　　“这石头有上千斤，谁砸得动，不如黄管家试试。”
<br/>　　“要不然程当家试试，他不是有千斤之力么，抱起来就扔了。”
<br/>　　黄三元已经看出来，想要动蛤蟆寨的不仅仅飞虎寨一家，其他山寨怕是在打同样的主意，青牛山五十几股山贼，又有谁是吃素的。
<br/>　　人群懒在地上，嘴角挂着若有若无的嘲笑，像是一群豺狼盯着一只误入狼群的小白兔。
<br/>　　今天来到蛤蟆寨的有上千山贼，如果上千人都以同样的目光看着你，这种目光汇聚成的压力可以直接让心智脆弱者崩溃。
<br/>　　黄三元脸上已经渗出汗珠，他不停擦拭，汗水却是流的越来越多。
<br/>　　高飞豹和身边几个人递个眼色，他周围几个人都站了起来。其他山寨也有人站起来，面露不善笑容。
<br/>　　这个时候，程大雷从屋子里走出来。
<br/>　　所有人的目光都汇聚在他身上，见他穿着宽大的布袍，表情漫不经心。
<br/>　　“怎么了？”程大雷似乎没有感觉到气氛有些不对。
<br/>　　“大当家。”黄三元如同见到救星：“这里有块大石头碍事，想让弟兄们砸了，可石头太大，弟兄们搞不动，不如您用五雷法炸掉他。”
<br/>　　一听这话，所有人都睁大眼睛，对于程大雷会五雷法这件事，有人信，也有人不信。可听黄三元的意思，似乎他真的掌握这种玄门法术。
<br/>　　程大雷皱起眉头，有些不耐烦道：“这点小事还来烦我。”
<br/>　　然后大家见他摊开双臂，合二为一，食指突然并指如剑，剑尖指着巨石，口中一声爆喝。
<br/>　　“我有神法，能请天雷！”
<br/>　　嘭！
<br/>　　巨石爆炸，分成十几瓣碎石，有些碎石被炸飞出去，还擦伤了一些人。
<br/>　　嘶！
<br/>　　一片吸气声，上千斤的巨石，竟然被他一声令下砸成粉碎。现在就说程大雷有千斤力气，也没有人再怀疑，这根本不是一个人，这是个神呐。
<br/>　　做了这么了不起的事情，可程大雷的表情却是没有半点动容，反而像是完成一件微不足道的小事。
<br/>　　“以后这种小事就别烦我。”程大雷拍拍身上的布袍，忽然目光一转，看着懒散待在地上的山贼，好奇问：“你们都不干活，在干嘛呢？”
<br/>　　唰！
<br/>　　被他目光扫到的人都觉得脊背冒汗，每个人都像是上足发条一般，立刻从地上站起来，以极大的干劲投入工作。没有一个人，敢发出一句怨言，在这种状态下，工程进度又得到提升。
<br/>　　终于在十天后，工程全部完工，整个蛤蟆寨焕然一新，被井然有序的分成几个区域。
<br/>　　最外围是防御区，用山石和大门垒成围墙，围墙上有瞭望台，只要有敌人接近，居高临下可以在最短时间内警觉。围墙外的大树已经被砍伐干净，一马平川，而在看似平坦的地面上，是秦蛮亲自设计的陷阱，包括陷马坑，套索，可以困住野猪的夹子……不得不说，秦蛮真的是一个顶级猎人。
<br/>　　防御区后是校场，校场中央耸立着一只大旗，如果有敌人突破外围攻击进来，这里将是缓冲地带。
<br/>　　校场后是工作区，包括聚义厅，马圈，以及为铁匠铺，医馆等必要设施预留的位置。
<br/>　　工作区后是生活区，包括住房、厨房、水井、武器库，粮仓。
<br/>　　另外马圈旁边修建公共厕所，厕所紧挨着农田，在没有化学肥料的时代，土家肥也无比重要。
<br/>　　程大雷对改造好的山寨还是很满意的，不说青牛山没有山寨能攻下这里，但程大雷保证，没有任何山寨能承受得起攻占这里的代价。
<br/>　　当然，改造山寨也付出了很大代价。山寨本来的五万多斤粮食消耗一空，仅存的也不过一千余斤而已，不过山寨没有多少人，这一点倒是不必着急。
<br/>　　嘟，山寨建设完成，奖励成熟辣椒一亩。
<br/>　　嘟，触发任务，完成十次抢劫，目前完成度1|10
