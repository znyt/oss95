　　在苏莫控制了第三重时光神殿的规则之后，他意念一动，时光仿佛倒流，他回到了大殿之外。
<br/>　　“太不可思议了！”
<br/>　　苏莫伫立在第三重时光神殿的门前，神情怔怔，眸中精光熠熠。
<br/>　　这是两个时辰之前，他准备进入第三重时光神殿之时。
<br/>　　苏莫心中大动，若是他继续前进，前去后面的神殿，能否回到数十年前，甚至是百年前、千年前？
<br/>　　若是回到了以前，他就能改变以前，包括芊寻月的命运，青璇的命运等等。
<br/>　　少倾，苏莫深深的吸了口气，再次走进了第三重神殿之中。
<br/>　　因为，已经掌控了第三重神殿的时间规则，大殿内一切都清晰明了。
<br/>　　第三重时光神殿之中，人非常的多，显然，掌控此处大殿的时间规则，颇为困难。
<br/>　　苏莫毫不停留，继续前进，进入了第四重时光神殿。
<br/>　　第四重时光神殿，掌控时间规则的难度，暴增数倍。
<br/>　　苏莫整整用了十个时辰，才掌握了第四重神殿的空间规则。
<br/>　　第四重神殿之中，时间规则同样是时间加速，但远远超过了第一重神殿，达到了六十倍左右。
<br/>　　而后，苏莫同样没有停留，第五重神殿、第六重神殿。
<br/>　　随着前进，掌控时间规则的难度越来越大，第五重神殿苏莫用了约十五个时辰，第六重神殿，他用了接近二十个时辰。
<br/>　　第五重神殿，同样是时间变缓，第六重神殿是时间倒流。
<br/>　　掌控了第六重时光神殿的法则之后，苏莫回到了时空之城的外面，他正准备进入时门之时。
<br/>　　“接近五天！”
<br/>　　望着天空之上，时空神殿的时门和空门开启，很多人鱼贯而入，苏莫低声自喃。
<br/>　　这一幕，和五日之前，一模一样，每一道身影，每一个细节，全部一般无二。
<br/>　　苏莫心潮澎湃，如此下去，岂不是就回到了过去。
<br/>　　那么，能否回到苍穹世界之事时。
<br/>　　不过，这里面又出现了问题，若是真的这样，很多人都能逆转过去，这还不乱了套？
<br/>　　苏莫想不明白，他只能继续前进，去往第时门之中。
<br/>　　进入时门之后，一切都没有变化，苏莫径直来到了第六重时光神殿之中，来到了第七重时光神殿的入口处。
<br/>　　入口处，人非常的多，足有上百人聚集在这里。
<br/>　　褚彦、沁儿和骆妤三人，亦在此地。
<br/>　　“呀，才五天时间，你居然闯到了第七重时光神殿！”
<br/>　　看到苏莫到来，娇小女子沁儿，面带惊讶之色，不可思议的望着苏莫。
<br/>　　而褚彦和骆妤两人，亦是面带惊讶，苏莫区区虚神境三变修为，居然来到了此地？
<br/>　　虽然，来到这里和修为没有直接的关系，但是，却是和精神力有直接的关系。
<br/>　　只有修为越强，精神力才能更强。
<br/>　　“你说什么？过去了五天？”
<br/>　　苏莫闻言一震，随即大步来到了沁儿的身前，再次问道：“姑娘，你说已经过去了五天？”
<br/>　　苏莫心有疑惑，从他第一次进入时空之城到现在，的确是用了接近五日的时间。
<br/>　　但是，适才他回到了五天之前，重新进来，按理说，应该只过去了片刻的功夫而已。
<br/>　　“当然是五天，你莫不是时间混乱了？”沁儿笑着说道，一双水灵灵的大眼睛，充满了惊奇。
<br/>　　他们三人，均是真神之境的修为，而且个个天赋不凡，到现在才来到第七重神殿之前，而眼前的青年，居然不比他们慢多少。
<br/>　　苏莫闻言，心中惊疑不定，难道自己真的时间混乱了？
<br/>　　“你要进入第七重神殿？”骆妤打量了苏莫一番，淡淡的问道。
<br/>　　“嗯！”苏莫点了点头。
<br/>　　“我要提醒你一句，九天时间一到，时门会再次打开，你若是不出去，就会死在这里。”骆妤道。
<br/>　　“我会出去的，多谢提醒。”苏莫点了点头。
<br/>　　“你若是进入第七重神殿，怕是就出不去了！”褚彦目光望着第七重神殿，声音幽幽的说道。
<br/>　　“什么意思？”苏莫问道。
<br/>　　“第七重神殿，时间规则更加的玄奇，一旦在剩下的四日之间之内，你无法掌控，那你就会永远逗留在第七重神殿之中，就算你寿元再多，也要耗尽在其内。”褚彦轻笑一声道。
<br/>　　他不认为苏莫，能掌控第七重时光神殿的时间规则。
<br/>　　毕竟，只有四天的时间，连他们三人，都并没有十足的把握。
<br/>　　“多谢告知！”苏莫点了点头，计算危险，他也要进入第七重神殿。
<br/>　　因为，按照之前的规则，第七重神殿，应该是时间加速。
<br/>　　还剩下四天的时间，就要出去了，他必须要到第七重神殿之中修炼。
<br/>　　现如今，时间不多了，他貌似不太可能进入再后面的神殿了，最多停留在第七重神殿。
<br/>　　“沁儿、骆妤，我们进去吧！”
<br/>　　褚彦不再理会苏莫，招呼两女一声，便向第七重神殿走去。
<br/>　　“段惊天，你小心哦！”沁儿对苏莫眨了眨眼见，嘻嘻一笑，便进入了第七重神殿之中。
<br/>　　苏莫深深的吸了口气，同样没有过多逗留，也走进了第七重神殿之中。
<br/>　　同样的情景再次出现，苏莫盘膝而坐，静静的感知神殿之中的时间法则。
<br/>　　第七重时光神殿之中，规则的确玄奇，更加的难以掌握。
<br/>　　不过，苏莫的精神力和悟性，超过了同阶武者太多太多了。
<br/>　　仅仅是两日时间，他便掌控了第七重神殿的规则。
<br/>　　大殿中的情景，映入眼帘，足有上千人盘坐在大殿之中。
<br/>　　这些人，有的是早已经掌控了第七重的规则，有的则还是在参悟。
<br/>　　不过，就算掌控了的人，也很少会继续前进了，因为在这里时间的流逝加快，适合修炼。
<br/>　　苏莫环视了一圈，褚彦和沁儿、骆妤均在殿中。
<br/>　　只不过，三人身上气息内敛，均沉浸在感悟之中，显然还没有掌控规则。
<br/>　　苏莫稍稍测试了一番，这里的时间规则，可以让时间的流速，加快六百倍。

<br/>　　那么，他在这里，可以修炼三年多。
<br/>　　随后，苏莫不再多想，开始修炼了起来。
<br/>　　时光神殿后面的那些神殿，他会进去，但不是这一次了。
<br/>　　不过，听闻时光神殿，每一次开启，规则都不同，这倒是麻烦事。

<br/>　　虽然他身上基本没有了资源，但是，他冲击虚神四变，已经不需要资源了。

<br/>　　一切都顺利无比，他的四大本命元神，完美的和圣丹之力融炼。
<br/>　　从此，他四大元神得以超脱，威力倍增，实力大增。
<br/>　　突破到了虚神境四变之后，苏莫有些无语了，因为，他已经没有办法修炼了。
<br/>　　他的身上，已经没有资源了，无法修炼。
<br/>　　时光神殿之中，虽然有源气，但是并不浓郁。
<br/>　　而修炼奥义，对于现在的他而言，是可有可无。
<br/>　　吞噬奥义，级别不高，而且对于他的实力来说，没有什么用处。
<br/>　　至于其它奥义，如同不朽奥义、空间奥义，他都只是刚刚入门，距离形成法则的地步，还有遥远的距离。
