　　钦陵的担心并不是多余的，战场之上再怎么小心也不为过，毕竟这是在用人命和自己的荣誉在赌，而且自己输了事小，死了一堆人却没有得到应有的战果才是大事。
<br/>　　黑齿常之虽然年轻，但已经隐隐有大将之风，听了钦陵的嘱咐露齿一笑：“大都督，那些倭人如果在这种时候还会保留实力，那只能说明咱们败的不冤枉！”
<br/>　　“你小子哪来那么多屁话，让你注意你就注意点，快去吧！”钦陵举手在黑齿常之的头盔上拍了一下，没好气的说道。
<br/>　　黑齿常之被拍了之后嘿嘿一笑，片刻后恢复了严肃的神态：“大都督，末将去了！”
<br/>　　钦陵不耐烦的摆摆手，示意他快去。
<br/>　　而在黑齿常之离开之后，钦陵也恢复了冷峻的面孔，转头对身边的传令兵说到：“发信号吧，全军突击！”
<br/>　　“是！”传令兵高声答应着跑开，不多时，已经有几颗信号弹升上天空。
<br/>　　……
<br/>　　于唐军有过数次交手的山背大兄很熟悉这样的信号弹，因为每一次见到这个东西之后他们都会面临唐军疯狂的进攻。
<br/>　　所以当他再次看到空中的信号弹时，整个人都是懵的，脑海中不由想到了昨天秦吉胜说的，此时不宜进攻，以防有诈的见解。
<br/>　　如果当时能够听他的多好，如果能够再坚持几天，估计唐军就会认为自己的计划失败，从而放弃这一次的诱敌之策。
<br/>　　当然，如果刚刚他能注意一下秦吉胜的动向也是一个不错的点子，至少那个家伙很可能已经跳到伏击圈的外面去了，不会像自己一样坐以待毙。
<br/>　　不过这一次的情况好像有些不大对头，唐军的信号明明已经升起来了，可是四面八方却一点动静都没有，而且不但如此，甚至连刚刚还在不断向他们发击攻击的炮火也在同一时间停滞了。
<br/>　　发生了什么？这不是大唐军队的风格啊？大唐军队从来都是像狼一样，不把敌人咬死绝不罢休，为什么这一次竟然一点反应都没有？
<br/>　　“咦？这是什么东西？”就在山背大兄对大唐军队的举动有些疑惑不解的时候，距离他不远处的一个护卫无意间踢开了一块石头，露出了下面类似包裹一样的东西。
<br/>　　“不会是唐人军官私藏的军饷吧？这么一大包应该有不少钱！”攻击停止了，小鬼子们心情大好，其中一个年轻的护卫甚至开起玩笑。
<br/>　　不过，他的冷笑话并没有引起任何的笑声，相信，一些年长的护卫脸上已经露出惊恐的表情，那样子就好像见了鬼一样。
<br/>　　“丢，丢掉，快丢掉，跑，快跑，这里又是一个陷阱！”就在几个年轻的护卫不明所以，还想再问些什么的时候，山背大兄身边的物部氏族长已经歇斯底里的叫了起来，一边叫一边拉起山背大兄就跑。
<br/>　　发生了什么？为什么上头看到这东西那么害怕？几个年轻的小鬼子一头雾水的想不明白。
<br/>　　不过随着那包裹后面拖出长长一条引线之后，就算再傻的鬼子也明白了自己抱的是个什么东西。要命啊！这特么哪里是私藏的军饷，明明就是一个用来炸城墙的药包，前段时间大唐军队没少用这个东西来破坏城墙，怪不得会看着如此眼熟。
<br/>　　看懂了这东西是什么之后，几个护卫哪里还管什么东西南北，把手里的东西一丢，撒腿就开始向远处跑去。
<br/>　　但，这个时候他们的动作已经晚了，就在他们刚刚跑出去十余步的时候，地面上已经开始不断传来阵动之声，而且东西两个方向上已经硝烟弥漫，巨大的爆炸声一声紧似一声，连在一起就好像整个地面都在塌陷一般。
<br/>　　刚刚安定下来的小鬼子军队在这样威力的爆炸面前，全都失了方寸，一个个像没头苍蝇一下四处乱窜，但是随着爆炸的声音越来越近，小鬼子发现他们根本无路可逃。
<br/>　　……
<br/>　　钦陵和黑齿常之在此之前的数日之内，已经每日分批次的将自己的部队从大营中撤出，虽然灶火一直在增加，但那大多都是障眼法罢了，除了一堆半干不湿的木头基本上不用付出任何代价。
<br/>　　另外还有一点就是他们在离开之前将大营中进行了精密的布置，每一条排水渠都是经过精心的设计，而且对营地中的每一个目标也都进行了标注，只要在大营之外的某一个点上安置好炮兵部队，用不着几炮就可以把灌进营地中的洧水引燃。
<br/>　　不过做了这些还是不够的，他们还选出了一些敢死队，穿上沉重的护甲，只要敌人一来进攻，他们的第一个任务就是将营地中几个特定位置的洧水桶打开，然后便是带着留在后营中的战马逃跑，跑的越远越快越好。
<br/>　　至于说为什么会有那么多的战马……，老实说，渔猎游牧民族出征谁会只带一匹马？哪个不是两匹，三匹的！
<br/>　　最后要说的就是马厩了，原本的计划当中并没有在马厩的位置埋爆药这个选择，这只不过是钦陵临时增加的一个点子。
<br/>　　在他看来这里是后营，当前营着火，归路断绝的情况下，倭人一定会向这里集中。
<br/>　　而那些上当追赶那百十个赶死队成员的倭人也会在前营火起之后，犹豫不前，所以他们有一半的机会有可能会退回到后营。
<br/>　　这样一来安置数十万匹战马的后营就成了小鬼子们最集中的地方，如果不在这里埋上一些炸药，用来送小鬼子上天，实在有些对不起钦陵自己的良心。
<br/>　　当然，也有可能这些小鬼子没有在这里聚集，如果那样的话，钦陵也没有什么太好的办法，就多就是等大战过后再把那些炸药再挖出来好了。
<br/>　　而且钦陵的布局也并非只有这一点点，如果刚刚那些小鬼子没有回头，继续向前追的话，用不了一柱香的时间他们就会发现一支两万人左右的骑兵队伍截在他们前面。
<br/>　　而在唐军大营的两侧，其实也各埋伏着两万骑兵，如果前面发生交战，他们的任务就是杀出来，断掉小鬼子的后路，驱赶着小鬼子的后军向前，然后六万大军合兵一处，将小鬼子的二十几万军队全部消灭掉。
