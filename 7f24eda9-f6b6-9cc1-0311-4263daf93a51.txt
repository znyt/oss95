        傅夜沉将童筱颖抱回了床边，顺势让她坐在了自己的大腿上，满屋子都是玫瑰花的香气，正慢慢地将荷尔蒙的气息越催越浓郁。
<br/>
<br/>        “我的吻甜，还是那交杯酒甜？”他暧昧地在她的耳边轻语呢喃，大掌捧着她的脸颊，指腹在她的耳垂上轻轻抚弄，引得她身心上一阵酥颤。
<br/>
<br/>        傅夜沉悄无声息地抚摸着童筱颖那如瀑般倾泻在腰际的长发，秀发将她的颈项衬得肌肤格外雪白。
<br/>
<br/>        童筱颖低垂着头，不敢直视傅夜沉的眼睛，抿唇回答道：“都甜。”而且都快要甜入她的心头了。
<br/>
<br/>        在这之前，她真的从来没有想过，自己会嫁给大哥。
<br/>
<br/>        她身上这袭矜贵的旗袍嫁衣，似火在燃烧着她的脸颊和耳根。
<br/>
<br/>        傅夜沉捧住了童筱颖的脸颊，与她的额头轻轻地碰在一起。
<br/>
<br/>        “筱颖，你是上天，赐给我的最好的礼物。”他轻轻地说。
<br/>
<br/>        “那大哥从今以后，都不要离开我。”
<br/>
<br/>        “我会永远守护着你。”他的指腹，摩挲着她微烫的脸颊，语重心长地允诺。
<br/>
<br/>        童筱颖总觉得傅夜沉的这句话另有含义。
<br/>
<br/>        只是这种含义，让她揣摩不出来。
<br/>
<br/>        他的手从她的脸颊上缓缓地滑落到她身前的衣襟上，然后慢条斯理地解开上面那一粒又一粒的凤凰盘扣。
<br/>
<br/>        火红的旗袍上衣从她的肩头滑落到她的臂弯上，肌肤似雪，软玉温香。
<br/>
<br/>        她里面穿着一件大红色倒三角的鸳鸯肚兜，酥胸半露，脖颈、锁骨、香肩，每一处肤若凝脂，令他垂涎，令他躁动不安。
<br/>
<br/>        童筱颖聋拉着脑袋，双手揪着肚兜的下角，微微缩起香肩，手指不知所措地绕了一圈，松开，然后又重新绕上一圈。
<br/>
<br/>        “筱颖，今晚，你逃不掉了。”他凑到她面前，话语一落便霸道地含住了她的唇瓣，却是极其温柔地吻着她。
<br/>
<br/>        这份柔软的吻中带着一抹湿热，酥麻感席卷了童筱颖的心头。
<br/>
<br/>        她瞪大了眼睛后又缓缓地垂下眼帘，笨拙而青涩地回吻着他的温柔，唇瓣微离，她软糯地应道：“那我就不逃了……”
<br/>
<br/>        对，她不逃了……
<br/>
<br/>        既然已经下定了决心，就不应该再退缩。
<br/>
<br/>        只是，此时此刻，她忽然感觉到有些头晕，有种昏昏欲睡的错觉。
<br/>
<br/>        傅夜沉喜上眉梢，长臂揽她入怀时再次攫住她的红唇，并且大手来至她光滑的后背，轻轻地扯开了那根肚兜上的红色绳结。
<br/>
<br/>        视野突然间一片漆黑……
<br/>
<br/>        童筱颖感觉到自己的身子像是在颠簸，软绵无力，甚至不知身在何处。
<br/>
<br/>        她缓缓地睁开眼帘，入眼看到的是一片米白色的车顶，耳畔还传来车轮与地面的摩擦声。
<br/>
<br/>        车？！
<br/>
<br/>        童筱颖倏然坐起身来，顿时瞪大了眼睛。
<br/>
<br/>        她怎么会在车上？
<br/>
<br/>        米白色的车厢地面铺着昂贵的波斯地毯，一边是一排舒适的杏色真皮沙发，另一边则是一排高档的酒柜，前方与驾驶座相隔的车壁上还挂着一台液晶屏，下方悬着一束新鲜的玫瑰花环。
<br/>
<br/>        这是一辆加长型的豪华轿车。
<br/>
<br/>        童筱颖不明白自己为什么会突然在这辆车上？
<br/>
<br/>        她转过身去，趴在沙发的靠背上，眺望着车窗外，阳光洒过枝繁叶茂的大树，蓝天白云在树干后穿梭。
<br/>
<br/>        这是一条上山的路？！
<br/>
<br/>        回过神来时，童筱颖下意识地低头看了看自己，她身上不再是穿着那件嫁衣，而是不知道被谁替她换了一件淡紫色连衣裙。
<br/>
<br/>        除此之外，她还身无它物。
<br/>
<br/>        而壁钟上的时间，正告知她，这是第二天的上午了。
<br/>
<br/>        昨天晚上，她和大哥喝过交杯酒之后的事情，她一点印象也没有……
<br/>
<br/>        这到底是怎么一回事？
<br/>
<br/>        山顶的古堡里，露天阳台上站着一名浑身散发出浓浓杀气的男人，他手里正拿着望远镜，像是在盯着自己的猎物般，目光追随着那辆加长型的黑色豪华轿车。
<br/>
<br/>        “你确定她就是傅夜沉的女人？”他冷冷地质问。
<br/>
<br/>        候在他身旁的一名西装革履的男子，微微向他欠身颔首，态度十分恭谨地应道：“我确定，手下的人亲自从傅家大庄园里‘带’回来的。”
<br/>
<br/>        “很好！”男人的嘴角勾起一抹邪肆的笑意，顿时他周身的空气都变得更加冷冽，“既然如此，这个女人我要了！”他说得斩钉截铁，“将她带到我的卧室去。”
<br/>
<br/>        “是！”黑衣男得令，立刻转身离去，效率一等一。
<br/>
<br/>        “哼！”男人脸上的笑意顿失，脸孔狰狞的低语，“傅夜沉，冒险者游戏又开始了，这回轮到你女人身上了！你只不过是我的一颗棋子罢了！”
