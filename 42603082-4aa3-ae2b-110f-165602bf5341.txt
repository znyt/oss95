　　“世间的事情，没有那么简单，想要金蚕蛊的血达到容颜不老，就必须让它吸食一百个人的精血，成年后才行”
<br/>　　“想要它的蛇毒制作癌药，必须找个活人做为药引子让金蚕蛊咬上一口，蛇毒在药引子的身体内过滤，在药引子被毒死的前一刻，在他脖子的动脉上划一刀，蛇毒经过人血的稀释后放出来，才能使用”
<br/>　　蛇博士，一辈子都想用蛇毒来救人，有过无数的研究，对任何毒蛇都了然无心。
<br/>　　他幽幽的叹气：“制作癌药是为了救人一命，但救命一命的前提下，要先杀一个人，拿又有何意义”
<br/>　　无论是金蚕蛊的血，还是毒，要想正常使用，都要先铺垫不少人命进去，所以，这种蛇从来不在蛇博士的研究范围内。
<br/>　　无可否认，它或许是金矿，但是却是一座杀人的金矿，想要得到里面的金，就要牺牲大量的人命。
<br/>　　涂小安彻底懂了，这种蛇果然是邪恶的，但蛇博士的话，更是让他感慨万分，暗暗感动，这便是医得，一个真正拥有医得的人，有菩萨之心。
<br/>　　他要研究的是对人类百利而无一害的东西，而不是先害后救的宝药。
<br/>　　这一点，让涂小安不得不敬佩，都说科学家是疯子，为了达到目的，在所不惜，当初，他认为陈默也是如此。
<br/>　　但他疯狂的前提下，还保持着人性的光辉。
<br/>　　这时，涂小安将目光放到了蛇博士那双断腿上，眼神闪烁了下。
<br/>　　“博士，这种蛇虽然阴毒，但还是诞生了，想必这蛇的主人也应该深知这种蛇的作用”涂小安忽然一想。
<br/>　　“这也是我今天来的原因，我就想知道，这蛇出自什么人之手，有什么目的，金蚕蛇要想成长，就要用大量的人命去饲养”
<br/>　　涂小安摇了摇头，看向晕迷的余子童：“这点恐怕要等他醒了之后才能知晓”
<br/>　　陈默点了点头，又细细的大量金蚕蛊蛇，呐呐自语：“金蚕蛇呀金蚕蛇，我对你是又爱又恨”
<br/>　　一辈子都想有大作为的蛇博士，半辈子跟毒蛇接触，但是就是不敢接触这种毒蛇，他怕自己控制不住自己的欲望。
<br/>　　可如今，这种蛇，还是出现在他的面前。
<br/>　　涂小安耳力好，听到翻了翻白眼。
<br/>　　这个期间，涂小安跟蛇博士又聊了一会儿，一个小时后。
<br/>　　咳咳...
<br/>　　晕迷的余子童慢慢的转醒了，了；脸色好了不少，这要归功涂小安的蛇莓身上。
<br/>　　“我余某人还活着？？？”他睁开眼睛，虚弱的说了一声，透着不敢置信。
<br/>　　涂小安连忙走过去，笑道：“余大师，恭喜你，捡回了一条命”
<br/>　　余子童看向涂小安，心中明白，是他救了自己，想站起来感谢，但是身体不允许，连连的咳嗽，非常弱不禁风。
<br/>　　涂小安让他别乱动，另外也同情的看着他现在的样子，金蚕蛊蛇是被他弄出来了，但余子童的身体依然是骨瘦如柴，这点已然是现实。
<br/>　　而且金蚕蛊蛇大量的吸食余子童的生命力，让他的身体遭受了毁灭性的伤害。
<br/>　　换句话说，这个人就算现在性命保住了，但也绝对活不长。
<br/>　　除非有大量的天材地宝续命。
<br/>　　“哎呦，涂大师，余某人我悔呀，我当初就该把你的话听进去，不我应带着我的猫回白家，不回白家，我也不至于落到如此地步”
<br/>　　余子童伤心绝望的哭了起来，边哭边咳嗽，他知道，就算自己捡回了一条命，但他的寿命已经去了十之七八了。
<br/>　　可以说，他的寿命被蛊蛇吸食的差不多了。
<br/>　　“余大师，你说你身体内的蛊蛇是白家的人所为嘛”涂小安其实已经猜测到了这一点。
<br/>　　白家的人，当初用变异蚊子吸食人的精血，造血精，现在又用这种阴邪的蛇出来害人。
<br/>　　为的就是催发这种蛇的成长，已人为炉。
<br/>　　余子童绝望的点了点头：“白家的祖奶奶看上了我的黄金瞳，要我卖与她，但我怎么可能答应，最后他们直接强取豪夺，然后将我赶出了白家”
<br/>　　涂小安淡淡的看着他，这个剧本他已经跟余子童说过，可对方压根不讲自己的话当真，算是活该。
<br/>　　“本奶奶要你一只猫，但也不白要，本奶奶送你一条蛇”
<br/>　　此时余子童的脑海中响起了这样一句话。
<br/>　　现在他明白了。
<br/>　　送蛇？送他去死吧。
<br/>　　“可余某人我从始至终，都没看到蛇的影子，出了白家，我才发现自己的身体变得不对劲”余子童疑惑，不知道对方怎么将一条蛊蛇下到自己的体内。
<br/>　　蛇博士道：“蛊蛇想要进入人的身体，有一百种不让你发现的办法，只要进入人的身体，必死无疑”
<br/>　　或许到目前为止，还没一个人被下了蛊蛇，还能活着的。
<br/>　　余子童哪怕给涂小安跪下，磕几个响头，涂小安也是完全的受得起。
<br/>　　因为蛊蛇入了人的身体，就算是它的主人，也没办法将它唤出来，唯一出来的方式，就是这副身体再无吸食的价值，也就是一具干枯的尸体了。
<br/>　　恐怕只有涂小安有这个本事，用统治能力，强行将蛊蛇给召唤出来。
<br/>　　“如果蛊蛇是出自白家人之手，那么就麻烦了”蛇博士蹙眉了起来，他担心并不仅仅只有这一条蛊蛇存在。
<br/>　　唯有真正成年的蛊蛇，才有价值，它的血，它的毒，才能是宝药。
<br/>　　但成年的蛊蛇要用大量的人命牺牲。
<br/>　　对于一位救死扶伤的博士来说，他不忍心看到这一点。
<br/>　　“余大师，你说白家的祖奶奶莫非是一位老太太吗”
<br/>　　能用祖奶奶来称呼，那么岁数一定很大了。
<br/>　　余子童摇了摇头：“这位祖奶奶好像才是白家真正的掌舵人，但是看样子岁数并不大，好像还不到四十岁的样子”
<br/>　　他的脑海中，出现了一位风韵犹存的美妇。
<br/>　　“博士，你对白家了解吗，知道这位祖奶奶什么来历吗”
<br/>　　博士摇了摇头：“那种隐藏家族向来神秘，外界知道不多”
<br/>　　涂小安道：“一位祖奶奶，才不到四十岁的样子，莫非她是喝了金蚕蛊蛇的血吗”
<br/>　　“很多隐藏家族，为了达到自身的目的，都会用一些不为人知的阴邪手段”蛇博士叹了口气，要是白家，他可没能力管。
<br/>　　“如果真如我想象的那样，说明白家肯定有一条成年级别的金蚕蛊蛇了”涂小安眼睛亮了下。
<br/>　　很显然，从余子童身上出来的金蚕蛇，还没成年，而且看样子距离成年也还早着。
<br/>　　这也是这种蛊蛇的可怕之处，完全还没成年，就已经达到了觉醒级别。
<br/>　　真正的成年金蚕蛊蛇，又该是什么样子，他很好奇，还有余子童说的白家祖奶奶，也透着神秘。
