　　廖云剑面上杀机浓郁，一马当先，冲向十名魔甲人。
<br/>　　“杀！”
<br/>　　一声怒喝震动天地，廖云剑手持一柄银色长剑，身上气息爆开，如同剑神临世，威势极为不凡。
<br/>　　咻！
<br/>　　一剑挥出，一柄巨大的剑影，仿若劈开天地的神剑，裹挟惊天剑威，凌厉的金之奥义，凌空向十名魔甲人杀去。
<br/>　　“杀了他们！”
<br/>　　十名魔甲人也已经看到了苏莫等人，顿时厉喝了起来，而后齐齐出手，迎向廖云剑的攻击。
<br/>　　这十名魔甲人个个手持战刀，身上魔气滔天，如同滚滚狼烟，战刀劈砍，道道刀气浩大无比，斩天裂地。
<br/>　　轰隆隆！！
<br/>　　巨大的剑影一斩而下，和十名魔甲人的刀气硬撼的一起，霎时之间，爆响震天，发生了剧烈的爆炸，毁灭了方圆数百里的空间。
<br/>　　一击碰撞，廖云剑和十名魔甲人平分秋色。
<br/>　　“确实有骄傲的资本！”随后到来的苏莫，见到廖云剑的实力，不禁暗暗心惊，此人虽然嚣张狂妄，但实力的确不是盖的。
<br/>　　他暗暗估算了一番，自己的实力，也就和对方差不多，就算强也强不了多少。
<br/>　　至于廖云剑和十名魔甲人的攻击，只破灭了方圆数百里的空间，这是因为此地的空间极为坚固。
<br/>　　当初苏莫离开苍穹世界之后，就已经发现了，外界的空间比之苍穹世界稳固数十倍。
<br/>　　苍穹世界的空间，脆弱无比，他随手一挥都能破灭方圆万里的空间，但在外界根本不可能。
<br/>　　“你等压阵，我来斩杀他们！”廖云剑对苏莫三人吩咐一声，便再次向十名魔甲人攻击了起来。
<br/>　　“死！”廖云剑身形拔高，一声轻喝，手中长剑连挥，数十道巨大的剑影从天穹之上斩下，铺天盖地的向十名魔甲人杀去。
<br/>　　这一击更为强大，每一道剑影都长达千丈，威势惊天动地，那浓郁的金之奥义，仿佛让剑影拥有斩碎一切的锋芒。
<br/>　　“杀！”十名魔甲人也是不弱，立刻爆发出最强的实力，魔气汹涌，魔威滔天，和廖云剑大战了起来。
<br/>　　轰轰轰！！
<br/>　　巨响不断，漫天的气浪狂涌，滔天的魔气汹涌澎湃。
<br/>　　苏莫和韩盈、聂乾三人没有出手，在一旁压阵，既然对方要独战十名魔甲人，他们自然不会抢着出手。
<br/>　　啊啊啊！！
<br/>　　不多时，十名魔甲人便出现了伤亡，足足被廖云剑斩杀了五人。
<br/>　　死亡的五名魔甲人，均是武帝境一重的魔甲人，实力较弱，而剩下的五人，有四名是武帝境二重，为首的一名是武帝境三重。
<br/>　　“撤退！”
<br/>　　那名武帝境三重的魔甲人，见廖云剑强大，便果断撤退，立刻带着四名魔甲人急速逃离。
<br/>　　“杀！”廖云剑得势不饶人，立刻追杀了上去。
<br/>　　韩盈和聂乾便也立刻跟了上去，紧随廖云剑之后。
<br/>　　苏莫没有急着追赶，待三人远去之后，他立刻催动了吞噬武魂，两具魔影被吞噬之力从海水中拉了出来。
<br/>　　先前苏莫看的真切，五名魔甲人被斩杀，其中有三名魔甲人，身躯连带着魔魂一切被毁灭了。
<br/>　　而有两具魔魂却是并未毁灭，钻入了下方的海水之中。
<br/>　　“可恶！这是什么力量？
<br/>　　两具魔影在吞噬之力下口吐人言，怒吼连连，不断的挣扎。
<br/>　　他们两人并没有彻底死亡，身躯虽毁，但只要魔魂还在，并不会死亡。
<br/>　　但现在，他们感受到了巨大的生死危机。
<br/>　　“不错！”苏莫一边催动吞噬武魂，一边暗暗点头，这魔甲人的魔魂，其实和战魂类似，想必吞噬之后的效果，也不必战魂差。
<br/>　　两具战魂虽然不停的挣扎，但没有了肉身的他们，根本无力反抗吞噬之力，很快便被拉扯到了苏莫的身前。
<br/>　　“杀！”
<br/>　　“杀！”
<br/>　　两具魔魂怒吼一声，纷纷出手攻击苏莫，两道虚幻的拳影向苏莫轰杀而来。
<br/>　　不过，没有了肉身的魔魂，攻击只是魂力和魔之奥义的结合，威力根本不足以威胁苏莫。
<br/>　　苏莫挥手间，便破灭了两人的攻击。
<br/>　　而后，两具魔魂瞬间便被吞噬武魂吞噬了。
<br/>　　苏莫能清晰的感觉到，两股强大的魂力，涌入了吞噬武魂之中，让他的吞噬武魂更强一分。
<br/>　　只不过，距离吞噬武魂晋升到天级七阶，还有一段不小的距离。
<br/>　　随即，苏莫没有耽搁时间，立刻向廖云剑三人离去的方向追了上去。
<br/>　　苏莫并没有追逐多少时间，仅仅是半刻钟的时间，便追上了廖云剑三人。
<br/>　　三人已经没有在追杀魔甲人了，均是放慢了速度。
<br/>　　“怎么这么慢？你刚才去哪了？”廖云剑见苏莫追了上来，顿时冷声问道。
<br/>　　“随手击杀了两具魔魂！”苏莫淡淡的道。
<br/>　　“是吗？”廖云剑闻言嗤笑一声，道：“以你的实力，也对付不了魔甲人，若有漏网的魔魂，就交给你了！”
<br/>　　苏莫瞥了对方一眼，心中冷笑一声，对方虽然看不起他，这个主意倒是不错，反正他的目标就是魔魂。
<br/>　　这时，廖云剑瞥了聂乾和韩盈一眼，目光落在了韩盈身上，微笑道：“韩姑娘，你的修为也不高，注意安全，不过有为兄在，你也不必担心！”
<br/>　　“呵呵！廖兄实力当真不凡，能和廖兄组队历练，是小女子的荣幸！”韩盈轻笑一声，柔声说道。
<br/>　　“哈哈！区区一点实力，不足挂齿！”廖云剑大笑一声，谦虚的说道，不过其言语虽然谦虚，但脸上却是傲气十足，眸中尽是得意之色。
<br/>　　他之所以一人独战十名魔甲人，自然是要表现一番，让得佳人青睐。
<br/>　　苏莫看的暗暗摇头，他就是再笨，也能看出这个廖云剑想泡韩盈。
<br/>　　不过，这些都与他无关，他只要能吞噬魔魂便可。
<br/>　　“廖兄，我们快走吧！刚才那几个魔甲人逃了，我估计他们会请来更强大的魔甲人。”聂乾说道。
<br/>　　“嗯！我们走！”廖云剑点了点头，随即认准方向，率先飞了出去。
<br/>　　苏莫三人，则是紧跟其后。
<br/>　　魔云海的魔甲人奇多，而魔甲人喜食海中的兄兽，这样能快速提升他们实力。
<br/>　　所以，四人在大海之上，不时的能遇到零散的魔甲人。
<br/>　　有韩盈的武魂之助，发现强大的魔甲人，他们便提前避开，不是多强的魔甲人，他们便出手击杀。
