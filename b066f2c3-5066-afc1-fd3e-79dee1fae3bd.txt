　　　　当轿车停下来的时候，迎面就看到了风度不凡的苏羽，面带笑容的为涂小安开车门。
<br/>　　下了车，涂小安放眼望去，前方还真是一栋建筑工地，在施工的状态。
<br/>　　“小安兄弟，你看这里怎么样，我要在这里建筑一栋豪华的小区，如今施工都有一年多了”
<br/>　　苏羽笑着给涂小安介绍起来。
<br/>　　这片是温城新开区的地区，将来会非常的繁华，所以苏羽将这块地花大价钱给投标了下来。
<br/>　　如果，比起这种大手笔，味鲜阁对于苏羽还真是小打小闹。
<br/>　　“我今天在你店里闹事，真的不好意思”
<br/>　　对方不提，涂小安不能不提一下，不然摸不清苏羽到底是什么意思。
<br/>　　带自己来看他的工地是为什么。
<br/>　　苏羽笑了笑：“我听说了，小事而已，小安兄弟不用在意，那小子叫刘昊然，跟我也认识，我已经跟他通了电话，事情解决了”
<br/>　　涂小安根本没在意对方是谁，只是觉得在苏羽的店里主动闹事，有点对不住他。
<br/>　　“那你今天叫我来这里是？”
<br/>　　苏羽道：“小安兄弟是捕蛇人，想必手中一定有不少好蛇吧”
<br/>　　涂小安还是不懂对方葫芦里开卖的什么药。
<br/>　　“莫非你还想弄一条蛇用来当镇店之宝？”
<br/>　　苏羽摇头又点头，道：“小安兄弟我就跟你实话实说了吧”
<br/>　　“之前那条龙蛇丢了，风水师说会影响我个人的气运，这话我之前信个四五分，但我现在是全信了”
<br/>　　“你看我这个工地，从投标到建筑，都一帆风顺，几乎没出什么差错，但是就在龙蛇丢了以后，我这个立马工地就出事了”
<br/>　　“说出来你都不信，我这个工地上居然闹起了鼠灾”苏羽都一脸的苦笑。
<br/>　　“工地闹鼠灾？”涂小安听到也诧异的很，像是听到了什么天方夜谭。
<br/>　　工地又不是田野，有什么东西让老鼠吃的。
<br/>　　吃混泥土？
<br/>　　苏羽日了狗的表情：“工地现在是全面停工，工人都不敢来了，不知道从哪里涌出成千上万的老鼠，在我的工地上面肆虐，这些老鼠的牙齿非常的厉害，连这栋初具规模的工地都被咬的千疮百孔，而且只要有工人在上面，就会立刻不要命的攻击人”
<br/>　　“已经有好几个工人被这些老鼠给活生生的咬死了”
<br/>　　他赔了一大笔安家费。
<br/>　　老鼠咬死人？
<br/>　　莫非工地上藏着一条鼠精霍乱？
<br/>　　涂小安顿时如此联想。
<br/>　　“苏老板莫非是想用蛇来灭鼠？”
<br/>　　苏羽道：“我找人问过了，想要这些老鼠消失，必须需要一条非常厉害的毒蛇镇守，才会让老鼠知难而退”
<br/>　　曾经在古代，有一个村庄常年被鼠灾困扰，农田每年都无法丰收，几乎有一半庄稼是喂了老鼠，后来村民发现一处废弃的庙堂里，住着一条有灵性的蛇，顿时想到了一个办法，村民将这条蛇供奉为此庙的守护神，顶礼膜拜，供奉祭祀品，让它保佑这个村庄不受鼠灾。
<br/>　　这条有灵性的蛇吃了村民的祭祀品，自然要为村民帮事，于是就出去灭鼠，慢慢的，这个村的鼠灾就解决了。
<br/>　　来年秋天来了一次大丰收。
<br/>　　“那天苏某见小安兄弟女友手中的小蛇，想必是觉醒类的蛇种吗”
<br/>　　涂小安知道他指的女友是李清照，小蛇自然是太阴蛇了。
<br/>　　“之前没认出那是一条觉醒蛇，到是苏羽眼拙了”他笑了笑，继续道：“这个世界有隐藏的一面，我知道小安兄弟不是寻常人，是觉醒修武者，又有一条觉醒类蛇种，就想请你来帮这个忙，不知可否愿意？”
<br/>　　苏家是温城大家族，知道这些好像也没什么奇怪的。
<br/>　　而且苏羽身边一直有个黑衣保镖，就是觉醒修武者。
<br/>　　他想用觉醒类蛇种镇守工地，让老鼠不在来，这也算是一个办法。
<br/>　　但是能引起大规模的鼠灾，还是在工地上，恐怕就不是普通的鼠灾了，必定有觉醒类的鼠类领头。
<br/>　　那就要看是觉醒蛇厉害，还是那只觉醒鼠厉害了。
<br/>　　“带我上去看看吧”
<br/>　　真的要有觉醒的鼠精霍乱，涂小安既然来了，不介意帮苏羽这个帮。
<br/>　　况且自己还刚在他店里闹了一场，就全当还他这个人情。
<br/>　　里面不但有觉醒类鼠精，还有成千上万的老鼠，可不是区区修武者可以对付。
<br/>　　不然苏羽的保镖也不是摆设。
<br/>　　“太好了，小安兄弟要是帮我解决了这鼠灾，苏某一定重谢”
<br/>　　施工一半的工地现在无法正常建筑，他每天都要损失一大笔钱，而且还会影响最后建成的进度。
<br/>　　这是新开区的地带，建筑豪华小区要是进度慢了，那么别的小区先比你建成，你就只能干瞪眼了。
<br/>　　损失惨重也。
<br/>　　苏羽带着涂小安朝着工地走去，身后黑衣保镖如影随形。
<br/>　　工地门口，里面静悄悄，非常的安静，苏羽开口：“小安兄弟，你别看现在里面很正常，可一旦有人进去，就会涌出很多肥大的老鼠，非常的凶狠”
<br/>　　“你那条觉醒类的蛇可在”他问。
<br/>　　蛇跟鼠是食物链上的天敌，有非常厉害的震慑作用。
<br/>　　“无需那条蛇”
<br/>　　涂小安笑了笑，开玩笑，他一代登基坐殿的蛇王都来了，还需要用别的觉醒蛇？
<br/>　　忽然，身后保镖接了一个电话，跟苏羽窃窃私语了几句。
<br/>　　苏羽突然面露尴尬，对着涂小安道：“小安兄弟，我差点忘记了，因为工地鼠灾，怀疑是有觉醒鼠精作怪，我之前还特意请了一位白家的捉妖师”
<br/>　　捉妖师，是觉醒修武者中的一种职业，就是专门去对付一些成精了的生物。
<br/>　　就好比道长抓鬼，自成一派。
<br/>　　白家是温城最大的隐世修炼家族，有大量的捉妖师。
<br/>　　捉妖师等级分别一品，二品，三品，对应一阶二阶三阶。
<br/>　　其实苏羽先前没想到涂小安身上，也是临时接到店里的电话，说涂小安在店中跟客人起了争执，他就一下子想起来他手中有一条觉醒蛇。
<br/>　　“你跟白家的捉妖师说说，让他先不要来了”
<br/>　　黑衣保镖开口：“苏总，对方已经到了”
<br/>　　旋即，不远处有一辆轿车行驶而来，直接停到了工地的门口。
<br/>　　又是白家，涂小安郁闷，不得不说白家在温城的势力很大，之前校园群蛇，连一位女警官，都是白家的人。
<br/>　　现在又有白家的捉妖师。
<br/>　　很多大家族，要是碰到什么妖邪的事件，都会花钱请白家的捉妖师来解决。
<br/>　　当然，也不单单只有白家，只是白家的实力最强，捉妖师靠谱点。
<br/>　　对方都来了，让苏羽有点尴尬起来。
<br/>　　涂小安笑了笑：“既然有专门的捉妖师，他要是能解决工地鼠灾，自然最好，我倒还省事了”
<br/>　　他并不在意。
