　　嗖！
<br/>　　太史轩身形如电，飞到了苏莫前方数百里处站定。
<br/>　　而跟来的无数弟子，则是急速分散而开，纷纷落在了十几座岛屿之上。
<br/>　　“苏莫，赵城和郭建青呢？是不是死了？”太史轩目光凝望着苏莫，冷冷的问道。
<br/>　　上次他派赵城和郭剑青去追杀苏莫三人，结果两人一去不复返，近两年了都没有回来。
<br/>　　所以，他猜测两人已经死了。
<br/>　　当然，他不认为苏莫有能力杀死赵城和郭剑青，定是火元圣师从中捣鬼，救了苏莫。
<br/>　　“都死了，你很快就会去陪他们！”苏莫淡淡的说道。
<br/>　　“大言不惭！”
<br/>　　太史轩闻言嗤笑一声，随即微微转头，看向岛屿上的一群太史世家一脉的人，朗声问道：“诸位，苏莫害死赵城和郭剑青，罪不容赦，你们希望他怎么死？”
<br/>　　“哈哈！当然是将他轰成渣！”
<br/>　　“太史轩大哥，轰成渣太便宜他了，应该废了他的修为，然后抽出战魂，活活炼化成灰烬！”
<br/>　　“敢挑衅我们太史世家一脉，必须强势镇杀，方可扬我太史世家一脉的威严！”
<br/>　　“……！”
<br/>　　一众太史世家派系的人，纷纷开口，七嘴八舌，言语却是一个比一个狠毒。
<br/>　　太史轩闻言，嘴角笑意盎然，转头看向苏莫，朗声道：“苏莫，你听到了吗？你不死不足以平众怒啊！”
<br/>　　太史轩完全是在戏耍苏莫，等到戏耍够了，再一举击杀苏莫。
<br/>　　“太史轩，我就问你一句，准备好迎接死亡了吗？”苏莫面不改色，凌厉的目光直视对方。
<br/>　　“狂妄！”太史轩闻言心中怒气喷涌，眸中杀机爆闪，他实在受不了苏莫这个姿态，仿佛一切尽在对方掌控一般。
<br/>　　这种姿态，仿佛他已经败了，让他非常的不舒服。
<br/>　　太史轩正欲动手，就在此刻，蓦地，苏莫的身上，陡然涌出一股淡紫色的雾气。
<br/>　　这股雾气缥缈虚幻，瞬间涌动数百里，将太史轩笼罩在了其内。
<br/>　　本来满脸杀机的太史轩，脸色瞬间呆滞了起来，仿若是石化了一般。
<br/>　　“怎么回事？”
<br/>　　“太史轩怎么了？”
<br/>　　“那淡紫色雾气是什么？难道是毒气？”
<br/>　　众人见到此种情况，先是一怔，随即一片哗然。
<br/>　　这还没战斗，居然就发生了这种情况，让很多人都摸不着头脑。
<br/>　　当然，也不乏一些眼界高明之辈，一眼便看出了，那紫色的雾气蕴含强大的魂力。
<br/>　　这说明这淡紫色的雾气，不是战魂就是精神攻击。
<br/>　　然后，仅仅是一个呼吸的时间，众人便看到太史轩的脸色惨白了下来，身躯摇摇晃晃，仿佛马上就要倒下了一般。
<br/>　　所有人都愣住了，这是什么情况？尼玛，还没开战，太史轩就败了吗？
<br/>　　梦魔空间之中。
<br/>　　啊！啊！啊！！
<br/>　　惨叫声不断，太史轩被绑缚在刑架上，口中不断的发出一声声凄厉的哀嚎。
<br/>　　只见苏莫手拿黑色的长鞭，不断的在太史轩的身上抽击，劈头盖脸，噼里啪啦的声音此起彼伏。
<br/>　　这长鞭之上，布满了一根根尖利的倒刺，每一次抽击都带起一片血雾。
<br/>　　“苏莫，你这个混蛋，快放了我！”太史轩一边惨叫，一边怒吼。
<br/>　　他恨啊！没想到苏莫居然有如此诡异的手段，让他猝不及防，还没来得及出手，就中招了！
<br/>　　“放了你？怎么可能？我还没折磨够呢！”苏莫面无表情，手中动作不停，不断的挥鞭抽打，力道越来越大。
<br/>　　啊啊啊！！
<br/>　　太史轩不断的惨叫，这每一鞭都痛入骨髓，深入灵魂，让他痛苦不已。
<br/>　　不多时，太史轩便奄奄一息，不成人形了，精神意识陷了入浑浑噩噩之中。
<br/>　　外界。
<br/>　　众人见太史轩脸色惨白，心中正在震惊之时，
<br/>　　下一刻，便见到太史轩的身躯，倒了下去，落在了海面上。
<br/>　　众人见此，顿时心中掀起惊涛骇浪。
<br/>　　败了！
<br/>　　彻底的败了！
<br/>　　仅仅两个呼吸的时间，太史轩甚至都来不及出手，便彻底败了。
<br/>　　苏莫，出手居然如此的恐怖和诡异。
<br/>　　就在此时，只见苏莫大手一张，凌空一抓，瞬间便将太史轩的身躯吸了过去，一把抓在了手中。
<br/>　　此刻，太史轩双眼翻白，面带呆滞，精神浑浑噩噩，意识已经完全陷入了混乱之中。
<br/>　　“住手！放了太史轩大哥！”
<br/>　　“快住手！”
<br/>　　“赶快放人！”
<br/>　　一众太史世家派系的人，见到苏莫将太史轩擒拿，立刻纷纷放声大喝。
<br/>　　然而，苏莫面无表情，仿佛像没有听到众人的话语一般。
<br/>　　他手掌如刀，直接插进了太史轩的小腹之中，将对方的神丹挖了出来。
<br/>　　而后，他将神丹和对方的储物戒收起，另一只抓着太史轩脖颈的手掌，喷涌出炙热的火属性玄力，如同熊熊烈焰。
<br/>　　瞬息之间，熊熊烈焰将太史轩焚烧成了灰烬，肉身、战魂全部烟消云散。
<br/>　　所有人都瞪大了双眼，死死的盯着这一幕。
<br/>　　死了！
<br/>　　太史轩死了！
<br/>　　苏莫居然敢将太史轩杀了！
<br/>　　不仅是一众太史世家派系的人震惊了，聂乾和韩盈也震惊了，所有人都震惊了。
<br/>　　太史世家的嫡系子弟，天才人物太史轩，拥有境界优势的情况下，没有任何反抗之力，被苏莫瞬间击杀！
<br/>　　所有人都是神情怔怔，面现不可思议之色。
<br/>　　本来很多人，还认为这是一场大战，必然很精彩，但结果完全相反，居然呼吸之间就结束了！

<br/>　　“太狠了，这个苏莫真是狠辣！”
<br/>　　“他到底是怎么打败太史轩的？太诡异了吧！”
<br/>　　“苏莫，敢与我们太史世家派系为敌，你必死无疑！”
<br/>　　众人议论纷纷，而一些太史世家派系的人，眼睛都红了。
<br/>　　苏莫当场击杀太史轩，这不仅是藐视太史世家派系，更是在狠狠的打他们的脸。
<br/>　　“走！我们去找尤天韩大哥！”
<br/>　　而后，一众太史世家派系的人，纷纷离开了战界。
