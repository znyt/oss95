　　第614章
<br/>　　木钗抵着程大雷的脖子，周桃贴着程大雷的耳垂，用软糯的声音道：“我知道程当家一身本事，小女子可承受不起，这木钗浸着剧毒见血封喉，程当家可不要乱动，吓坏了小女子不要紧，别伤了程当家的千金之躯。”
<br/>　　“你瞧瞧，误会了不是。”程大雷呵呵笑着：“牡丹花下死，做鬼也风流，姑娘如花似玉，在下怎敢莽撞。”
<br/>　　周桃抿嘴一笑，看不见她的模样，但只听她的笑声，也觉得百爪挠心。
<br/>　　柳追风看着程大雷和周桃『打情骂俏』，顿时气不打一处来，道：“小桃，与他废话做什么，直接杀掉算了。”
<br/>　　周桃摇了摇头，但程大雷看不见她摇头的动作。
<br/>　　“先将他们处理掉。”周桃冷冷道，她同程大雷说话时还有几分魅惑，可此时声音却是冷冰冰的。
<br/>　　柳追风对其言听计从，立刻向东方童几人出手。可怜东方童在马上也是大将，但在这样逼仄的地方动手，实在不是他所长，只是转眼间，便有两名兄弟被杀。
<br/>　　程大雷现在也是泥菩萨过江，自身难保，虽然不愿看到这一幕发生，但也帮不了什么忙。
<br/>　　眼看再这样发展下去，东方童一行七人都要丧命在柳追风和孟一刀二人手中。忽然，刘凯扑过来猛地抱住了柳追风。当然，长剑也洞穿了他的肚子，可这样不要命的打法，柳追风委实没有见过，一时间被他抱了个结结实实。
<br/>　　“大人，走哇！”
<br/>　　“你！”
<br/>　　东方童一惊，开口竟不知说什么才好。
<br/>　　“走啊，走啊……走了才能给兄弟们报仇。”
<br/>　　长剑贯穿刘凯的内脏，血从他嘴里流出来。柳追风想要挣脱刘凯，可一时半刻竟也做不到。江湖上的武艺，东方童当然比不上他。可军中的打法，他也不熟悉。军人习惯不要命，用自己的命为兄弟开路，正如此刻刘凯做的。
<br/>　　东方童将牙一咬，提着刀闯出山洞，身后刘凯阵阵惨叫，但此刻他也顾不得太多。军人当习惯不要命，包括自己的命，也包括同伴的命，他不能让同伴死得没有价值。
<br/>　　最后，还是孟一刀出手，出手稳准狠，砍了刘凯一条胳膊。如此，柳追风才摆脱束缚，如他这样的一代宗师，遭遇这样狼狈的事毕竟是第一次。他气得三尸神暴跳，五灵豪气腾空，用剑狠狠在刘凯的尸体上扎了两下，将他的心脏扎得稀巴烂。
<br/>　　此时，东方童一行人，除了东方童拼着命逃出去，其他几人都死在山洞之中。
<br/>　　柳追风看向程大雷，似乎是杀得不过瘾，想要将程大雷也一剑杀了。
<br/>　　“小桃，留他做什么！”
<br/>　　周桃仍旧挟持着程大雷，冷笑道：“你莫忘了他是谁，将他献给官家，能换万户侯，在绿林道杀了他，足够让你名扬天下。”
<br/>　　柳追风眼前一亮，目光看着程大雷，像看着一位十八岁的小媳妇。的确，黑白两道都想要程大雷的性命，杀了他名和利也都有了。
<br/>　　“柳郎，先用绳子将他们两个绑住，我们将他带出去。”
<br/>　　柳追风随身带着小牛筋的绳子，将程大雷和小白狼两个绑得结结实实，当真是不给二人半点逃跑的机会。
<br/>　　此时天已差不多快亮了，在一把剑和一把刀的挟持下，程大雷被押着走出山洞，一直走到官道上。
<br/>　　“柳郎，你去套一架马车，装上这两个贼人，等在天下英雄前诛杀此贼，也是柳郎的名声。”
<br/>　　“好，好。”
<br/>　　柳追风乐不可支，与孟一刀一起上路，找附近的村镇套一架马车，至于是借是买，或者说是抢，那就说不好了。
<br/>　　周桃转身面向程大雷，微微躬身：“程当家，得罪了，希望程当家莫要怪我。”
<br/>　　“好说，好说。”程大雷道：“本当家对姑娘一向都是蛮大度的，尤其是姑娘这样的美女。”
<br/>　　程大雷虽然笑得风轻云淡，但心里对这女子一直很警惕。她对柳追风说话时是一副面孔，对自己说话时完全是另外一副面孔。鬼脸婆婆，百变美人，该不会这外号就是如此来得吧。
<br/>　　此时，小白狼突然向周桃发动攻击。她虽然双手被缚，但腿上可没有绑绳子，这一招鸳鸯连环踢当真是份外凌厉。周桃一丝不妨，竟被她踹倒在地。
<br/>　　程大雷看着这一幕，心道这周桃的本事怕就在下毒易容这样的阴损伎俩上，至于本来本事怕是不强。
<br/>　　她一个鲤鱼翻身，从地上跃起，拔出随身短刀搁在程大雷脖子上，挑衅的看着小白狼。
<br/>　　小白狼也没防备她这一招，一时不知该如何是好。程大雷冲她使了个眼神，她立刻转身向后，重新逃向密林。
<br/>　　“程当家的女人，好像并不怎么在意你的性命，竟抛下你逃了。”
<br/>　　“唉，谁说不是呢，如姑娘这样有情有意的女人可是不好找。”
<br/>　　小白狼重新逃入密林后，也不知该如何是好。刚才程大雷用眼神示意她先走，也就是说程大雷对自己的安全问题还是有把握的。她先在山石上磨破绳索，又找到二人的座骑，之后寻思着如何救程大雷脱身。
<br/>　　此刻官道上，一辆马车行来，周桃将程大雷藏在草丛中，自己楚楚可怜的站在路边。
<br/>　　车上是一富家公子，赶车的马夫二十岁出头。等马车行到周桃身边时，那富家公子从车内探出脑袋，问：“姑娘是要搭车么？”
<br/>　　“是。”
<br/>　　周桃盈盈笑着，正笑得公子哥满面桃花，冷不防周桃突然出手，先杀了那公子哥又杀了马夫，一刀一个，砍瓜切菜一般。
<br/>　　她将二人的尸体丢在一边，又抱着程大雷从草丛出来，为了提防程大雷逃走，这次她是将程大雷的腿脚绑在一起，程大雷整个人如粽子相仿。
<br/>　　程当家就以公主抱的姿势窝在周桃怀里，满足道：“姑娘好大的力气。”
<br/>　　“程当家不会生气吧？”
<br/>　　“当然。当然不会。”程大雷笑着道：“姑娘，咱们不等人了么？”
<br/>　　“我带程当家先走一步。”
<br/>　　“那个，要带我私奔……”
<br/>(本章完)
