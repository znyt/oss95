        傅昼景在一楼单元门口，看到一抹熟悉的白影时，全当没看见，径直走出了单元大门。
<br/>
<br/>        而那抹白影，立即追了上去。
<br/>
<br/>        “阿景——”季思妍柔柔地唤了声。
<br/>
<br/>        傅昼景并未因为她的呼唤，而为她驻足。
<br/>
<br/>        季思妍追上他后，不得不踱步来他的跟前，张开双臂将他给拦住。
<br/>
<br/>        她的身材高挑，比例相当匀称，贴身的银白色旗袍将她婀娜多姿的好身材完全展现出来，米白色的高跟鞋在她美丽纤细的脚踝处缠绕着细带，简单而素雅。
<br/>
<br/>        “阿景，我们找个地方坐下来，好好谈一谈，行吗？”季思妍水汪汪的大眼睛里，透着一丝哀求的眸光，她的声音，幽微地哽咽，像是受了极大的委屈，想要找他倾诉。
<br/>
<br/>        “五年前，我们就已经结束了，不是吗？所以，我们之间，没什么好谈的！”傅昼景淡淡地说，绕开季思妍，继续跨着自己前进的步伐。
<br/>
<br/>        季思妍抿了抿红唇，转过身去，带着哭腔，大声喊道：“五年前，是你爷爷逼我离开你的，但我其实一直都爱着你啊！”
<br/>
<br/>        蓦然，傅昼景身心一怔，刚迈出的步伐，就在这一刻定格在地面上一动不动。
<br/>
<br/>        仅仅只是一两秒的出神惊愕，傅昼景便恢复了脸上的平静。
<br/>
<br/>        “那又如何？已经不重要了！”傅昼景语气依旧冷淡，顿了顿后，他语重心长地回答，“我们已经回不去了。”
<br/>
<br/>        说完，他便走了。
<br/>
<br/>        只留下季思妍一个人楚楚可怜地站在那儿，潸然泪下。
<br/>
<br/>        回不去了吗？
<br/>
<br/>        真的已经回不去了吗？
<br/>
<br/>        季思妍在心底一遍又一遍地质问自己，明明她不这么想，为什么他要这么早就放弃自己呢？
<br/>
<br/>        她思念了他五年啊！
<br/>
<br/>        难道，爱情里，分手后，最痛苦的那一个，只是她吗？
<br/>
<br/>        季思妍一个人在城市的街道边犹如行尸走肉般踽踽独行，她回忆着自己和傅昼景的过去，不知不觉中，来到了两人相恋时的那所高中——滨江一中。
<br/>
<br/>        当年，傅昼景是临海城过来的转校生，因为长得帅，成绩优异，性格温和，在学校里很受女生喜欢。
<br/>
<br/>        从他走进教室的那一刻起，季思妍就对他一见钟情。
<br/>
<br/>        是她倒追他的，同时也是她先抛弃他的。
<br/>
<br/>        追忆似水年华，季思妍站在滨江一中的校门口，触景伤情。
<br/>
<br/>        “当——当——当——当——”
<br/>
<br/>        中午放学的铃声，就像宛转悠扬的竖笛，欢快中又透着一丝忧伤。
<br/>
<br/>        季思妍黯然神伤地伫足在校门口，也不知道过了多久，直到她看到一抹熟悉的身影从校门口走出来的时候，还是忍不住唤了声：“雅姗。”
<br/>
<br/>        季雅姗也没想到，会在校门口邂逅自己那同父异母的姐姐季思妍。
<br/>
<br/>        季思妍的目光无意间落在了季雅姗背后的那个书包上，不由地皱了下眉头。
<br/>
<br/>        “姐。”季雅姗抿了抿唇，双手握紧肩上的书包的背带，犹豫了一下，才朝季思妍走了过去。
<br/>
<br/>        名义上，她叫她一声“姐”，可实质上，她两并没有真正的姐妹亲情。
<br/>
<br/>        在季思妍的眼中，她季雅姗只不过是自己的父亲在外养的情妇所生的见不得光的野种罢了！
<br/>
<br/>        不然，那次，季雅姗被歹徒绑架，父亲就不会为了不破财、不伤家庭合起来，而选择不痛不痒地报警了。
<br/>
<br/>        “上次你被绑架的事情，没给你留下什么阴影吧？”季思妍只是随口问问。
<br/>
<br/>        季雅姗当然听得出这个所谓的“姐姐”对自己毫无关心之意，回答得也很随便：“应该没有吧！”
<br/>
<br/>        就连舍命救她的那个大恩人，他们季家人都没有谁去关问过他一句。就算救命恩人身份特殊，那也得去救命恩人的上级那儿送面锦旗什么的吧！
<br/>
<br/>        结果，季家的人，什么都没做。
<br/>
<br/>        甚至，他们其实都巴不得她死在歹徒手里吧！
<br/>
<br/>        对，她的家人，对她就是这样一种“无关紧要”的态度。
<br/>
<br/>        就连当初，她之所以会来到这个世上，还是她母亲想拿肚子里的她作为筹码，逼她父亲跟原配离婚。
<br/>
<br/>        只可惜，她母亲的如意算盘打错了，她父亲的原配已有一儿一女，娘家后台硬，又岂会在乎她母亲肚子里的孩子呢？
<br/>
<br/>        所以，她是多余的……
<br/>
<br/>        值得庆幸的是，她的父亲还不算渣到无可救药，至少，她母亲把她生下来后，父亲给她取了名字，还上了他们季家的户口，每个月也会给母亲和她生活费。
<br/>
<br/>        季思妍再次将目光落在季雅姗背后背的书包上，若有所思地问：“你的书包，价值三万，而且还是国际限量款。你——哪来那么多钱，买这么贵的书包？”
<br/>
<br/>        季雅姗下意识地斜睨了自己背后的书包一眼，不屑地笑了笑：“哎呀呀，姐姐不愧是在国外待了五年才回来的，还真识货了。”
<br/>
<br/>        “呵？”季思妍更加不屑地冷嘲热讽起来，“雅姗是步入了你妈妈的后尘吗？”
<br/>
<br/>        蓦地，被说中的心思，季雅姗心口一阵抽痛，可爱的眉毛微微蹙动，粉唇紧抿。
<br/>
<br/>        即便如此，季雅姗也要仰起头来，理直气壮地怼回去：“原来姐姐的眼睛是有颜色的呀！妹妹这书包了，是一个追求者送的。姐姐莫非是羡慕嫉妒呢？”
<br/>
<br/>        “到底是不是追求者送的，你心知肚明，何必在这跟我贫嘴。雅姗，我只是想奉劝你一句。当小三，可是没什么好下场的！特别是你这种，还在念高中即将步入大学的女孩子！不要毁了自己，浪费了自己的青春！”季思妍昂头挺胸，十足的御姐风范，恍若居高临下般，蔑视着季雅姗，语气咄咄逼人。
<br/>
<br/>        季雅姗不想跟季思妍浪费口水，只是白了她一眼，二话不说，转身就走了。
<br/>
<br/>        她迅速转身的时候，那长长的马尾辫，就像一道鞭子，狠狠地抽在了季思妍的脸上。
<br/>
<br/>        季思妍吃痛地捂着脸颊，上前追了一步，又放弃了。
<br/>
<br/>        她这是何必多管闲事呢？
<br/>
<br/>        这个差点害得她父母离异，还每个月不要脸地上她家来向她母亲索要生活费的小三的女儿，是被男人包养了也好，被男人玩弄了也好，都与她无关。
