　　第496章
<br/>　　这个女人眼神干净得像冬天湖泊的冰，手中的匕首却像野兽的眼睛，程大雷一瞬间就有种被死亡盯上的感觉。
<br/>　　一开始程大雷就猜测过，她会不会什么十步绝杀之类的功夫。而现在，二人的距离不足十步，空气静得呼吸可闻。
<br/>　　“嗨！”程大雷第一时间捏灭了桌上的油灯。
<br/>　　“程大雷，我替所有惨死在你手上的人来取你狗命！”
<br/>　　房间顿时暗下来，聂隐娘身子扑向程大雷，匕首闪过的寒芒像空中一闪而逝的流星。整个人散发的杀气，填满了整座房间。
<br/>　　程大雷在地上踢了一脚，地上的机关被触发，为了自己的安全问题，程大雷自然不介意多加一道保险。
<br/>　　聂隐娘前扑的姿势生生停下，即便在绝对黑暗的环境下，她的视线也并未收到太大影响。从她所立的位置看过去，面前立起一道道铁栏杆，整座房间被铁栏隔成九个独立的空间，像极了一座九宫格。本来这座房间是一单独的铁笼，现在被隔成了九座。
<br/>　　程大雷和聂隐娘被分别隔在一个九宫格中，二人的距离依旧谈不上远，互相能看得见彼此。
<br/>　　“嗨。”程大雷摆了摆手，说出刚才没说完的台词：“卿本佳人，奈何为贼。”
<br/>　　聂隐娘没有说话，观察着身处的环境，另外抽时间打量了程大雷一眼。这是程大雷第一次看见聂隐娘的真容，聂隐娘又何尝不是初次见到程大雷。
<br/>　　见他站在那里，一脸小人得志的猖狂，果然如画像中一般，面目可憎，令人作呕。
<br/>　　程大雷从身上取出一张连击驽，瞄准了聂隐娘，口中道：“机关发动，我的人就会得到消息，他们很快就会到，但我希望在他们来之前，由我亲自解决你。”
<br/>　　聂隐娘也知道自己目前的状况，如果不能迅速脱身，程大雷的帮手一道，自己必然会陷在此地。
<br/>　　嗖！
<br/>　　一枚驽箭射出，程大雷杀心已起，容这样的女人活在世上，自己是会做噩梦的。每一枚弩箭上都淬有剧毒，见血封喉。
<br/>　　啪！
<br/>　　聂隐娘拂袖打落，向前挥出了匕首。
<br/>　　程大雷心中一惊，这样短的距离，视线又极其差，她还能又如此快的反应，看来自己还是低估了她的实力。
<br/>　　更令程大雷惊讶的是，匕首落在拇指粗细的铁栏杆上，铁栏杆应声而断，断口整齐，甚至没发出太大的声响。
<br/>　　就算她手中握着的是绝世利器，想要切断铁栏也需要极强的腕力，她的强悍，还需要重新估计。
<br/>　　程大雷接连射出几枚弩箭，稍稍减缓了聂隐娘突破的速度，但对她并没有太大影响。以她目前的速度。她很快就可以接近自己，然后杀掉自己。
<br/>　　现在程大雷想走也走不掉，这九宫格困住了聂隐娘，也同样困住了自己。匹夫剑可不是什么神兵利器，程大雷大概能明白作茧自缚是什么道理。
<br/>　　眼看聂隐娘距离自己越来越近，程大雷能感受到扑面而来的杀气，可怖的是，她眼睛依旧干净得没有任何表情。
<br/>　　程大雷拎起地上的马桶，整个泼了出去，一时间，屋子里充满恶臭。饶是聂隐娘心知如石，但毕竟是个女儿身，下意识用袖子去挡。
<br/>　　匹夫剑出鞘，铛地一声，程大雷隔着铁笼刺了出去。
<br/>　　“大当家，大当家！”
<br/>　　张肥率领的张字军鱼贯而入，将屋子团团围住，每个人手中都握着连击驽，在他们后面，是持铁弓的战士。
<br/>　　“先别放箭，莫要伤了大当家。”
<br/>　　咚！
<br/>　　房顶被撞破，一道黑影突破屋顶向外逃去。张肥立刻下了命令：“放箭！”
<br/>　　羽箭纷飞，在接近黑影时，却被轻而易举的击落。
<br/>　　程大雷被从笼子里放了出来，七八个人围着他，道：“大当家，你没事吧？”
<br/>　　程大雷摇摇头，抬起匹夫剑看了一眼，见剑尖挂着鲜血。
<br/>　　“原来最后一剑还是伤到她了。”
<br/>　　他抬眼望着聂隐娘逃走的方向，摆下的埋伏也奈何不了她。张肥气得咬牙跺脚，得到信号后，他第一时间就过来了。本想抓住这刺客雪耻，但还是让她跑了。
<br/>　　张肥虽然在战场上杀人如麻，马上步下都没有问题，可这种江湖上的小巧功夫，蹿房越脊之类的，的确不是他擅长的。
<br/>　　如秦蛮、赵子龙几人也是同样状况，所以今夜他们都没有出现。
<br/>　　如果追，程大雷倒是有可能追上这聂隐娘，但追上做什么呢，难道让他杀了自己么。通过今夜正式接触，程大雷彻底认清了这聂隐娘的实力，单对单厮杀，自己根本不是她对手。
<br/>　　就在所有人面对逃走的聂隐娘都束手无策时，突然见一道黑影凌空而起，向着远处追去。
<br/>　　“咦，山寨竟还有如此高手。”程大雷道：“那人是谁？”
<br/>　　“是刚到山寨的李道姑啊，大当家还没见过面吧。”
<br/>　　程大雷点点头，原来是赤练仙子李莫愁，她能被系统评为绝世，本事自然也不会弱。只不过她怕不是聂隐娘的对手，搞不好别再被杀了。
<br/>　　“等她回来了，让她来见我，当然，如果她能活着回来的话。”程大雷道：“再给我准备一个藏身的地方，然后……先让我洗个澡。”
<br/>　　张肥几个人与程大雷说着话，其实悄悄都掩住了鼻子。
<br/>　　程大雷白了他们一眼，道：“你瞧瞧，活着是件容易的事情么。”
<br/>　　趁着聂隐娘刺杀失败逃离的这段时间，程大雷好好收拾了一下。毕竟，被关在铁笼里足不出户的日子，实在有些难熬。
<br/>　　然后，程大雷又被转移到一个安全屋内，这次知道她藏身位置的人，就少很多了。
<br/>　　程大雷这次藏身的位置，是在军营的一个地牢中，地上是三万千锤百炼的精兵，地牢内只有程大雷一个人。
<br/>　　程大雷一个人住在地牢内，只有一扇气窗能看到外面的情形。
<br/>　　程大雷变得更加谨慎，今天以有心算无心，侥幸活下命来，这还有一定运气的成分。
<br/>　　等下次，自己就未必有这样的好运气了。
<br/>(本章完)
