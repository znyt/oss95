　　苏莫没有理会三人，手臂一挥，一股气浪卷出，便将三人裹到了远处。
<br/>　　随即他眼帘低垂，静静的感受着体内的力量。
<br/>　　他的吞噬武魂，已经成功进化成了战魂，那方圆三千丈的巨大漩涡，便是他的战魂。
<br/>　　因为成功进化战魂，他也顺利进入了武帝境一重的境界。
<br/>　　进入武帝境一重，他的玄力并没有什么变化，但战魂的力量反哺进体内，让得他实力暴增。
<br/>　　他静静的感受着自己的战魂，感觉到了其内玄妙莫测的力量。
<br/>　　只见他的识海之中，缩小了无数倍的吞噬战魂，正悬浮在其中。
<br/>　　战魂，已经入驻识海，控制着他的一切。
<br/>　　因为战魂是神魂和武魂的融合，所以他现在对于自己战魂的了解，更加的清晰透彻了。
<br/>　　武魂进化为战魂，战魂的能力，再一次得到了加强，甚至是有了质的飞跃。
<br/>　　不仅是吞噬灵气，现在简直是能吞噬万物了，而且吞噬之力的加强，简直是直线暴涨。
<br/>　　刚才，若不是他及时停止了催动战魂，这玉镯空间内的一切，包括方圆三百余里的大地，都要被他吞噬了。
<br/>　　而且，他现在有一种超脱之感，战魂让得他有了更高一层次的生命。
<br/>　　现在，就算他肉身毁灭，也不会死亡了。
<br/>　　“呵呵！苏莫，汝之战魂，绝对是超越吞天圣魂的存在！”阎魔鬼圣的笑声，在苏莫脑海中响起，颇有感慨之意。
<br/>　　吞天圣魂，乃是太古武魂榜排名第十九的武魂，号称吞噬天地万物，无数不吞无所不噬。
<br/>　　可现在看来，苏莫的战魂，比这吞噬圣魂还要强大。
<br/>　　因为吞噬圣魂虽然什么都可以吞，但好像并没有掠夺的能力，吞噬了也就消失了。
<br/>　　不像苏莫的战魂，吞噬的血脉和他人的武魂，全部掠夺成了自己的血脉和武魂。
<br/>　　这是巨大的差距！
<br/>　　两者根本不可相提并论！
<br/>　　“或许吧！”苏莫点了点头，超越什么他不在乎，只要战魂够强大就行。
<br/>　　随即，他手臂一挥，少量的下品灵晶和魔晶出现在了他的身前。
<br/>　　这些灵晶足有六百块左右，魔晶则是更多一些。
<br/>　　他共有一千两百多快灵晶，直接取出了一半
<br/>　　“吞噬！”苏莫轻喝一声，便微微催动了吞噬战魂，将身前的灵剑和魔晶全部笼罩。
<br/>　　现在武魂进化为战魂之后，苏莫对于战魂的控制，简直是得心应手。
<br/>　　因为战魂之内融合了他的神魂，他可以随意的控制吞噬之力的强弱和外围，意念所指，吞噬之力便能到达何处。
<br/>　　嘭嘭嘭！！
<br/>　　霎时之间，六百多块灵晶和魔晶全部炸裂，灵气向苏莫的体内涌去。
<br/>　　苏莫立刻开始极力炼化，这些灵晶的灵气极为磅礴，浩瀚如海。
<br/>　　而魔晶，虽然远不如灵晶，但也相当于数十亿极品灵石，只不过其内蕴含驳杂的魔气，炼化起来更加的费力而已。
<br/>　　时间流逝。
<br/>　　苏莫快速炼化灵气，他身上的气息则是越来越浑厚，越来越浩大。
<br/>　　提升到了武帝之境，他的修为玄力，终于能再次快速的提升了。
<br/>　　约莫三个时辰之后，灵气被苏莫炼化一空，他便结束了修炼。
<br/>　　六百多块下品灵晶，加上略多一些的魔晶，已经一举将他的修为，推到了武帝境一重的巅峰。
<br/>　　苏莫没有再继续吞噬灵晶，因为他刚才试着冲击了一下境界，晋阶壁障坚如磐石，很难冲过去。
<br/>　　他还需沉淀一番，方可再次吞噬，冲击武帝境二重境界。
<br/>　　少倾之后，苏莫离开了虚界神图，回到了阁楼中。
<br/>　　回来之后，他便是继续修炼，大混沌五行神诀运转，淬炼修为，凝练玄力。
<br/>　　就这样，苏莫静静的修炼了十数日。
<br/>　　这一日，清晨，他停止了修炼，离开了阁楼。
<br/>　　今日是导师授课的日子，他要去南苑广场听课。
<br/>　　虽然他有阎魔鬼圣的指导，但导师的授课，有时间的话，他还是会听的。
<br/>　　不多时，苏莫便来到了南苑广场。
<br/>　　由于他来的比较早，广场上人不多，只有零零散散的百余人。
<br/>　　“看，那就是苏莫，杀死了北苑的冷江和何传岩！”
<br/>　　“听说他有特殊体质，各大派系的人争相拉拢！”
<br/>　　“不过他却没有加入任何派系！”
<br/>　　“或许是不想陷入各大派系的争斗中吧！”
<br/>　　苏莫的到来，立刻引起了广场上所有人的注意，不少人低声私语。
<br/>　　苏莫没有理会任何人，静静的伫立在广场边缘，等待导师的到来。
<br/>　　距上次战界之事，已经过去了这么久，学府也没有追究他杀死何传岩的责任，现在他已经放心了。
<br/>　　看来只要天赋足够强，在学府中杀人，学府也不会多管。
<br/>　　随着时间的推移，广场上汇聚的弟子越来越多。
<br/>　　就在这时，一名年轻的蓝衣弟子走到了苏莫的身前，冷冷的望着他。
<br/>　　苏莫见此心中疑惑，打量了对方一番，他并不认识此人。
<br/>　　“苏莫，太史轩大哥请你过去！”蓝衣青年开口了，声音淡淡说道。
<br/>　　苏莫闻言眉头微皱，果断的摇了摇头，道：“没空！”
<br/>　　他可没有时间理会太史轩，也不想与对方有什么纠缠。
<br/>　　“你若不去，就准备为聂乾和韩盈收尸吧！”蓝衣青年冷冷的说了一句，便不在多言，转身大步离开。
<br/>　　苏莫脸色一沉，眸中顿时闪过凌厉的寒光，太史轩什么意思？想用聂乾和韩盈的命威胁他加入太史世家一脉吗？
<br/>　　这未免也太霸道了吧？
<br/>　　阴沉着脸，苏莫抬起脚步离开了广场，跟着蓝衣青年而去。
<br/>　　他倒想看看太史轩要耍什么花招，当然，他也没有什么惧意，既然对方想拉拢他，应该不会对他下杀手。
<br/>　　南苑非常的大，比一座雄城还要大，苏莫跟着蓝衣青年快速行走。
<br/>　　片刻之后，两人来到了一座院落之前。
<br/>　　这座院落占地数百丈，不算多大，但建造的极为精美，墙壁仿似白玉铸就，其上还铺盖着精美的琉璃瓦。
<br/>　　院落内还有恢弘的阁楼，精美的凉亭，高高的耸立。
<br/>　　咚！咚！咚！
<br/>　　蓝衣青年轻轻敲了敲院落的大门，便自行推开大门，走进了院落了。
<br/>　　苏莫紧跟其后，也走进了院落中。
