　　　　玻璃的出现，对历史上很多东西的发明都有着巨大的促进作用，镜子只是其中一个极不起眼的应用而以。
<br/>　　而且东宫的玻璃大棚、李承乾制备化工材料所使用的器皿，这些都利用到了玻璃，相信在不远的将来，更多样式各异的玻璃被制造出来之后，将会有更加广泛的用途。
<br/>　　玻璃工坊在有了那个会写汉字的波斯人参与进去之后，透明玻璃很快就被制造出来，而且里面的气泡也开始越来越少。
<br/>　　但是随着不断的试验，这样带着气泡的玻璃越来越多，慢慢的又在仓库里存下一大堆的半废品。
<br/>　　而李承乾自然不会任由这些东西就那么放着，思来想去觉得反正玻璃大棚对玻璃也没有什么特殊要求，索性就开始一件一件的把原本的毛玻璃替换下来，以便让玻璃大棚保证极好的透光度。
<br/>　　李二在知道了镜子是由玻璃制成的以后，也开始表现出对玻璃工坊的关心，时不时的也会参观一下，看到那里满是气泡或者杂色的玻璃之时，则会发出一阵惋惜的叹息。
<br/>　　长乐和孔雯两人也都得到了一个镶着小镜子的梳妆盒，把两个小丫头乐的跟什么似的，就差没抱着李承乾亲一口了。
<br/>　　而没有得到小镜子的小武和程小四则是三天没开晴，一直郁闷的嘟着小嘴生闷气，直到李承乾又用一些边角料给她们两个也一人弄了一个之后，才高兴的喊了一声：“谢谢太子哥哥！”
<br/>　　日子就这样一点一点的过去，平淡的像是一杯温开水。
<br/>　　随着除夕的逐渐接近，西域各国的使节和重臣已经纷纷赶到长安，打算借着大唐完胜******的机会，向李二表示自己的臣服，共尊大唐天子为‘天可汗’。
<br/>　　而在各国的使臣中间，流传最广的，则是仙酒的流言。
<br/>　　化外之民，嗜酒如命，吐蕃、薛延陀、铁勒、回鹘、昭武九姓、契丹等生活条件艰苦的种族，简直是无酒不欢。
<br/>　　这就造成了除夕还没到，迎宾楼的包厢就已经被纷纷预订，订的晚了甚至连包厢都没有情况。
<br/>　　而此时的李承乾则是老老实实的缩在宫里，大门不出二门不迈，一副大家闺秀的派头。
<br/>　　没办法，外面现在太乱了，神头鬼脸什么人物都有，这个时候出去，如果被哪里飞来的一根弩箭取了性命，那特么才叫冤呢。
<br/>　　一天两天，时间继续飞逝，每到了除夕夜，李承乾再爆惊人之举。
<br/>　　整整一筐青翠欲滴的黄瓜，顶花带刺，被抬进太极殿的时候，群臣哑然。
<br/>　　“这，这不是真的吧？”
<br/>　　“怎么可能是新鲜黄瓜？”
<br/>　　“老程，你掐我一把，看看俺是不是在作梦！”
<br/>　　数九寒天中一筐新摘的黄瓜震住了大殿中所有人，无数人在发表着感慨，看向李承乾的眼神满是恐怖。
<br/>　　李二那根已经被李承乾锻炼出来的神经同样狠狠一颤，看着大殿外飘着的雪花，又看看自己这个儿子，这特么是想把神仙弟子的名声坐实么？
<br/>　　皇宫之中秋天存下来的青菜的确是有一些，但那也只是存下来的，和新摘的根本就没有可比性。
<br/>　　所以寒冬腊月的一筐新鲜黄瓜，完全超出李二以及众臣的理解范围，那些被请来的外国使臣更是已经在底下悄悄的议论起坊间的传说。
<br/>　　“父皇，今日除夕，儿臣无甚礼物，只有献上一份小小心意！”忍出心中那份虚荣，李承乾对老头子躬身施礼。
<br/>　　“唔！”李二犹自沉浸在震惊之中没有回过神来。
<br/>　　长孙皇后亦是嘴角直抽，半晌说不出话来。
<br/>　　一个多月前的小镜子已经让她和老头子吓了一跳，不过在知道是一种特殊琉璃制作的之后，发现也就是那么回事儿，那个还是在可以理解的范围之内。
<br/>　　但是冬天的黄瓜算怎么回事？用什么东西可以解释？这完全就不合常理。因为只要是个人就知道，长安的冬天是什么都不长的，所有的青菜都会寒冷的天气杀死，不会有任何例外。
<br/>　　南方现在的确有蔬菜，但如果从南方运到长安，没有十天半个月肯定是不行，也就是任何蔬菜到了长安都会是蔫蔫的，决不会这么新鲜。
<br/>　　“你且一旁坐下吧！”好半晌之后，李二摇摇头，回过神来，不动声色的安排李承乾找个地方坐着。
<br/>　　然后亲自来到御阶之下，从筐里拿出一根黄瓜，认真的打量一番，同时将上面黄色的小花摘去，送入口中咬了一口。
<br/>　　“咔咔”的一阵脆响，黄瓜特有的清香通过鼻吸进入肺里，使得老李一边嚼着，一边露出一个舒畅的表情。
<br/>　　接着，李二又从筐里拿了几根出来，回到御阶之上，递给长孙皇后。
<br/>　　“陛下，那是黄瓜么？”破锣一样的声音在群臣中响起，不用问都知道，是程妖精那个老货。
<br/>　　“是！”李二点点头。
<br/>　　“那……好吃么？”老程咽了一口口水，有些纠结的问道。
<br/>　　“好吃啊！”李二继续点头。
<br/>　　“臣也想尝尝！”老程抽了抽鼻子，咂咂嘴，看样子馋的口水都快流下来了。
<br/>　　“可以。”李二忍住把嘴里黄瓜喷出去的冲动，对站在一边的内侍说道：“去把筐里的黄瓜分一下，每人一根，算是太子殿下的赏赐吧。”
<br/>　　“喏！”内侍同样也馋啊，只是看看那个小筐，估计还不一定够朝臣还是使节分的，自己……还是别想了。
<br/>　　“别‘喏’了，你道是快点下来发啊！”大唐的武疯子们早就已经等的不耐烦了，眼看内侍还在哪里磨叽，忍不住出声催促。反正前面已经有程妖精那个老货丢人现眼，现在催催也没啥大不了的。
<br/>　　“你先去那边发！”李二扫了一眼跃跃欲试的老程和老尉迟，对内侍嘱咐着。
<br/>　　“呃，陛下……”。
<br/>　　“臣等谢过陛下恩典！”与武疯子们的愕然不同，文官一侧纷纷表示谢恩。
<br/>　　现在的情况谁都明白，如果先给那一群老杀才发，估计还没发到一半，这些黄瓜就特么全没了。
