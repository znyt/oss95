　　轰…轰…轰…
<br/>　　一股股无形的气爆声，不断响现于空气中。
<br/>　　那是魂力间的对轰。
<br/>　　“嘶。”萧逸忽然倒吸一口凉气。
<br/>　　他的魂力，显然远远比不过面前这头魂兽。
<br/>　　魂力间的交锋，他完全处于下风。
<br/>　　来自于魂力间的碰撞和反噬，让得他脑海如蒙针刺，自是让他倒吸一口凉气。
<br/>　　想也知道，萧逸当初初初得到‘魂念诀’这门功法，以及‘焚寂’这门武技时，不过是区区地极境修为。
<br/>　　之后，便未怎么修炼过魂师一道。
<br/>　　自然的，他魂力怎么可能强。
<br/>　　而面前这头魂兽，是单靠一身魂力，便有圣皇境实力的妖兽。
<br/>　　二者单论魂力，根本不是一个层次。
<br/>　　不过，此刻明明是这头魂兽占上方，它却瑟瑟发抖。
<br/>　　因为，在它身体周遭，一把把星光利剑，将它牢牢压制。
<br/>　　它是魂兽，甚至比一般的妖兽更为敏感。
<br/>　　它明显感觉到，这些星光利剑，哪怕是一把，也足以轻松取它性命。
<br/>　　更别说现在它身体周遭有密密麻麻不下数千把星光利剑。
<br/>　　它毫不怀疑，这些星光理解，足以在一瞬间将它打成筛子。
<br/>　　“嗯？”萧逸忽然轻嗯一声。
<br/>　　“收回魂力做什么？”
<br/>　　萧逸直视这头魂兽，冷漠的眸子中带着不满。
<br/>　　这头明明占据上风的圣皇境魂兽，霎时打了个冷颤。
<br/>　　凶戾而巨大的眸子上，此时透漏着的，是恐慌。
<br/>　　不远处，顾怜星掩嘴一笑，“我还是第一次见到这么没有骨气的魂兽。”
<br/>　　公孙火舞撇撇嘴，“小贼，你就只会欺负人，不，欺负魂兽？。”
<br/>　　“若单凭你那不入流的魂力，敢这么嚣张，早被这头魂兽吞进肚子了。”
<br/>　　萧逸满头黑线，懒得理会这二人。
<br/>　　无奈的目光，仍旧看向这头魂兽。
<br/>　　“放出魂力，与我一战。”萧逸沉声道。
<br/>　　“若我满意，之后放你离去。”
<br/>　　他想借这两天短暂时间，锻炼一番自己的魂力。
<br/>　　这妖兽不与他交锋，他如何锻炼？
<br/>　　虽然说，杀了妖兽，直接取了魂丹，炼制成的丹药有极大的提升魂力修为的功效。
<br/>　　但魂师一道，同样有着其天地武道规则，并非是单单魂力足够就能晋升的。
<br/>　　如同正常武道，武者除了拥有足够的元力外，还需要足够的武道参悟，方能突破修为。
<br/>　　萧逸在魂师一道方面，境界和武道感悟弱得顶多在地极境、天极境左右。
<br/>　　故他即便炼制出圣皇境级别的魂丹丹药，也不可能突破魂师一道上的修为。
<br/>　　另一方面，若要磨砺他的魂师水准，最好的交战对象，乃是地极境、天极境左右的魂兽。
<br/>　　不过，他可不知道要到哪里去寻这个层次的魂兽。
<br/>　　这幻光险地之内，最弱的魂兽都在圣王境以上。
<br/>　　故他在深处也罢，在外头边缘也罢，每差。
<br/>　　这时，面前这头魂兽，凶戾的眼眸上，露出一丝希冀。
<br/>　　它的本能，自然是不相信人类武者的。
<br/>　　不过，它现今别无选择。
<br/>　　它很清楚，惹怒面前这个年轻人类，下场只是瞬间身死。
<br/>　　恐惧而惊咦下，它只能换换放出一丝魂力。
<br/>　　萧逸满意地点点头，同样放出魂力，与之交锋。
<br/>　　“魂力的程度再高些。”萧逸冷喝一声。
<br/>　　这头魂兽，‘乖巧’地微微增加了魂力的浑厚程度。
<br/>　　半晌，萧逸低喝道，“停。”
<br/>　　空气中提升强度的魂力，霎时停下，停在了这个层次。
<br/>　　在萧逸的感知中，这个层次，恰好处于天极境左右，最是适合他磨砺。
<br/>　　“嘶。”不多时，萧逸再次倒吸一口凉气。
<br/>　　魂师的手段，诡异归诡异，但一受反噬，那等非人的疼痛，还是极其难受。
<br/>　　脑海中，如被万条细针穿刺的疼痛感，绝不好受。
<br/>　　但他也明显感觉到，自己的魂师水准，正不断提升。
<br/>　　对武者而言，历练是提升实力的一个极好办法。
<br/>　　魂师也一样，通过这种极限的磨砺，感受其中魂力玄奥，大有裨益。
<br/>　　萧逸一边承受着疼痛，一边磨砺，一边也在感知参悟着魂师一道。
<br/>　　不远处，顾怜星二人看着萧逸不断露出的痛苦脸色，摇了摇头。
<br/>　　大半天后。
<br/>　　公孙火舞忍不住出言道，“这小贼，是被虐狂吗？”
<br/>　　“我看他脸上那痛苦脸色，不时还露出一丝欣喜，显然是痛并快乐着。”
<br/>　　顾怜星摇了摇头，轻笑道，“萧逸公子在磨砺自己的魂师水准呢。”
<br/>　　“而且，你看真切些，萧逸公子脸上的痛苦之色，正不断减缓。”
<br/>　　“从一开始的倒吸凉气，到后来的仅仅是咬了咬牙便挺过去，再到现在，虽然面露痛苦，却连一声闷哼都不曾传出。”
<br/>　　“这证明萧逸公子正以极快的速度提升着魂师水准。”
<br/>　　“极快？”公孙火舞脸上闪过一丝讶异。
<br/>　　“据我所知，魂师一道的修炼，异常困难。”
<br/>　　“这小贼临阵磨枪，还能以极快的速度提升？”
<br/>　　“那岂不是…？”
<br/>　　“对。”顾怜星点点头，“萧逸公子的领悟力，异常恐怖。”
<br/>　　“绝世妖孽四个字，当之无愧。”
<br/>　　“单论领悟力和参悟力，我还从未见过有如此恐怖程度的武者。”
<br/>　　公孙火舞皱了皱脸，“这等妖孽，还会被两大学院驱逐？莫不是那些老家伙都眼瞎了吧。”
<br/>　　“不过算了。”公孙火舞忽然异色连连，“没想到这小贼还挺厉害的。”
<br/>　　顾怜星看了眼公孙火舞，蓦地掩嘴轻笑。
<br/>　　……
<br/>　　时间，渐渐过去。
<br/>　　萧逸仍旧在拿这头魂兽磨砺自己魂师水准。
<br/>　　两天时间，转瞬便过。
<br/>　　萧逸的魂师水准，不断从磨砺与感悟中提升；那头魂兽亦‘自觉’地不断缓慢提升魂力的强度。
<br/>　　而顾怜星二人，大多时间，都是在闲聊。
<br/>　　公孙火舞，除了一开始的异色连连外；大多数时间，都是面露忧色地看着顾怜星。
<br/>　　“怜星，难道就真的没有办法彻底治好你吗？”公孙火舞的语气，有些低落。
<br/>　　顾怜星笑笑，“有的，不过…”
<br/>　　“有？”公孙火舞美眸一喜。
<br/>　　顾怜星笑笑，没有再说下去。
<br/>　　不远处，空气中的魂力对轰，忽然停下。
<br/>　　萧逸从参悟中醒来，沉声道，“我没猜错的话，现在已过去两日时间了吧。”
<br/>　　对，猜。
<br/>　　魂师与这些魂兽的交锋历练，很多时候会陷入参悟状态。
<br/>　　当然了，参悟时，一身元力与魂力会自动护身，倒不必担心有危险。
<br/>　　只是，这等状态下，时间上的流逝，倒不会太敏感了。
<br/>　　“嗯，过去两天了。”顾怜星点了点头。
<br/>　　萧逸看了眼天空，此时，已然入夜，月亮正圆。
<br/>　　“十五月圆了，也该看个究竟了。”萧逸自语一声。
