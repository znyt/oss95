　　苏莫的目光落在了北海女皇的身上，眸中冷漠无比，等待对方开口。
<br/>　　“见过苏宫主！”
<br/>　　“见过苏宫主！”
<br/>　　北海女皇向苏莫微微躬身，三长老亦是抱拳行礼。
<br/>　　“嗯！”苏莫微微颌首，面色淡淡。
<br/>　　北海女皇打量了苏莫一番，想看出一丝苏莫心中的想法，但非常的失望。
<br/>　　苏莫的脸色冷漠、无情、古板，没有任何的神态留露。
<br/>　　心中组织了一番语言，北海女皇道：“苏宫主，本宫管教无方，让得门下长老得罪了你，还望苏宫主海涵。”
<br/>　　对于三长老得罪了苏莫，北海女皇是又惊又怒，既然苏莫有言，让她带三长老负荆请罪，她自然不敢不从。
<br/>　　“请苏宫主海涵！”三长老亦是出声，向苏莫躬身一拜。
<br/>　　“我的度量很小，不会轻易宽恕得罪我的势力！”苏莫淡淡的说道。
<br/>　　他言语间的意思，是将北海冰宫也拉进来了，就是要告诉对方，得罪他的是北海冰宫，不只是三长老个人。
<br/>　　北海女皇闻言，俏脸微变，嘴角勾起了一丝苦涩的笑容。
<br/>　　“那我们要如何做，苏宫主才会满意？”北海女皇道。
<br/>　　既然苏莫让她们来，肯定是有重罚，她已经有了心理准备。
<br/>　　“五百亿极品灵石吧，或者是等价的资源！”苏莫淡淡的道。
<br/>　　“什么？”北海女皇闻言，顿时惊呼出声，满脸的呆滞之色。
<br/>　　北海冰宫三长老，以及所有在场的苍穹神宫弟子，都被苏莫的话震惊了，个个瞪大了眼眸。
<br/>　　五百亿极品灵石？
<br/>　　这是什么概念？
<br/>　　这绝对超出了苍穹世界任何势力的承受范围，北海冰宫就算再富裕，也拿不出这么多的灵石！
<br/>　　这可是极品灵石，不是上品灵石，更不是中品灵石。
<br/>　　五百亿极品灵石，对于任何势力、任何强者而言，都是天闻数字。
<br/>　　北海女皇和三长老惊呆了，她们想过苏莫可能重重的惩戒她们，用于震慑天下。
<br/>　　但是，她们完全没有想到，苏莫会如此的狮子大开口。
<br/>　　她们北海冰宫，若是真的拿出五百亿极品灵石，或者是等价的资源，怕是整个宗门都要砸锅卖铁了。
<br/>　　一众苍穹神宫的弟子，也是被苏莫的言语惊呆了！
<br/>　　自家宫主未免也太夸张了吧！
<br/>　　很多人身体都颤.抖了起来，他们想放声大笑，却是不敢笑出声来，憋的全身颤抖，别提多难受了！
<br/>　　五百亿极品灵石，这就是天闻数字，宫主真是牛.逼啊！这简直是要了北海冰宫的命啊！
<br/>　　“苏……苏宫主你……你没有搞错吧？”北海女皇眸中闪过怒火，但想到苏莫的实力，她又强制压下了怒火，脸色难看的问道。
<br/>　　“五百亿极品灵石或者等价的资源，一个月内送过来！”
<br/>　　苏莫强势无比，根本不给北海女皇讨价还价的机会，冷声道：“若是不然，北海冰宫也不必存在了，你自己选择吧！”
<br/>　　言罢，苏莫不再多言，一摆衣袖，果断的转身离去。
<br/>　　“苏宫主，你这是强人所难，五百万极品灵石，这根本不可能！”眼见着苏莫即将远去，北海女皇心中大急，大声喝道。
<br/>　　“灵石和灭门，你们自己决断吧！”苏莫幽幽的声音从远处传来，而后他的身形便消失在了众人的视线中。
<br/>　　“这……！”北海女皇脸色铁青，本来娇媚的容颜都有些扭曲了，她心中的怒火几乎快要炸开了！
<br/>　　五百万极品灵石，简直是欺人太甚啊！
<br/>　　最主要的是，苏莫摆明了坑她，连解释或者讨价的机会都不给，说完要求之后就直接离去。
<br/>　　苏莫的强势，让她心中怒火汹涌，她好歹是准帝强者，苍穹大陆的巅峰存在，统御一方的霸主，何时受过这种窝囊气。
<br/>　　不过，想到苏莫的实力，北海女皇还是缓缓的压制住了心中的怒气。
<br/>　　她实在是没有勇气，正面对抗苏莫。
<br/>　　“我们走！”转头看向身边的三长老，北海女皇怒喝一声，仿佛要将心中的怒火呼喊出来。
<br/>　　她真恨不得一掌拍死三长老，你说你什么人不得罪？干嘛去得罪苏莫这个煞星？
<br/>　　三长老吓的身形一颤，心中充满了苦涩，这一次他是真的惨了！
<br/>　　怕是整个北海冰宫，都会怪罪于他！
<br/>　　来的快去的也快，待了不足半刻钟，北海女皇和三长老便离开了苍穹神宫。
<br/>　　两人急速返回北海冰宫，路上，北海女皇将三长老怒骂了一通，才稍稍出了口心中的闷气。
<br/>　　一路无话，回到北海冰宫的北海女皇，立刻召集全宗所有长老议事，商谈如何应对。
<br/>　　当得知苏莫索要五百亿极品灵石之时，所有北海冰宫的高层，气的几乎要炸天了。
<br/>　　“五百亿极品灵石，我看苏莫是想灵石想疯了！”
<br/>　　“这绝对不可能，我们就算是砸锅卖铁，也不一定能凑齐这么多灵石！”
<br/>　　“若是真的拿出这么多灵石，我们北海冰宫还靠什么生存！”
<br/>　　“我们跟他拼了，他苏莫虽强，但我北海冰宫也不是被吓大的！”
<br/>　　“苏莫残暴无道，我等应该联合其它势力，共同反抗！”
<br/>　　一时之间，一众北海冰宫的高层，怒喝震天，各种想法纷纷呈现。
<br/>　　不过，待良久之后，一阵喧嚣的怒火平息之后，冷静下来的众人，都陷入了沉默之中。
<br/>　　反抗苏莫？
<br/>　　这可能吗？
<br/>　　苏莫现在可是堪比武帝的存在，而且还能打败武帝，实力之强，已然逆天。
<br/>　　谁敢反抗？
<br/>　　可以这么说，就算是十几名准帝联手，都不一定是苏莫的对手。
<br/>　　武帝强者，已然是武道神话，一人便可灭万千势力。
<br/>　　反抗便意味着自寻死路啊！
<br/>　　北海冰宫仿佛被阴云覆盖，一众北海冰宫的高层，整整商议了一天，最后才做出了决定。
<br/>　　他们选择了屈从，即便再不甘、再恼怒，但为了性命，为了整个北海冰宫，他们不得不屈从。
<br/>　　随后的时间里，整个北海冰宫都陷入了忙碌之中，各种灵石、灵草、灵药、神兵等等全部紧急收拢。
<br/>　　整整耗费了半个月的时间，北海冰宫聚集了全宗上下所有的财富，又将麾下一些附属势力，挨个敲诈了一遍，才勉强凑齐价值五百亿极品灵石的财富。
<br/>　　很多长老人物，自己私人的财富都拿了出来，北海女皇更是凄惨，除了自己所用的兵器留下了之外，她的所有家底全部拿了出来。
<br/>　　当然，最惨的还是三长老，他是此次事件的罪魁祸首，付出的代价也是最大的。
<br/>　　三长老彻底的一无所有了，连自己的兵器、贴身穿戴的内甲，他所有的一切，都全部贡献了出来。
<br/>　　凑齐了财物之后，北海女皇再次动身，亲自前去苍穹神宫奉上财物。
