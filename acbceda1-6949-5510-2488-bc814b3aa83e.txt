　　　　苏莫和海明江走出了房间，离开了客栈。
<br/>　　但是，当两人刚刚走出客栈的大门，还没来的及离开天海城之时，异变陡生。
<br/>　　只见阴阳神宫上空，伫立在虚空中的阳鼎天，手中立刻结出一个个复杂的印诀。
<br/>　　一个个印诀如同飞驰的流星一般，向四面八方激.射，急速飞射到了天海城的四周。
<br/>　　随着这些印诀飞入天海城的四周，一个浩大的阵法立刻被激活了。
<br/>　　只见天海城的上空，以四周城墙为起点，一个巨大无比的阵法气罩骤然出现，将整个天海城完全笼罩在内。
<br/>　　这阵法呈赤红之色，仿佛其内有火焰在燃烧，惊人无比。
<br/>　　“不好！”苏莫看到这突然出现的阵法，顿时面色一边，心中暗道糟糕。
<br/>　　“坏了！”海明江同样面色一变，心中震动不已，这绝对是埋伏！
<br/>　　而此刻，无数天海城的武者，均是愕然不已，望着天空上巨大的阵法气罩，均是有些不明所以。
<br/>　　“怎么回事？”
<br/>　　“哪来的大阵？”
<br/>　　“难道是阴阴神宫布置的大阵？”
<br/>　　无数天海城的人议论纷纷。
<br/>　　“哈哈哈！！”
<br/>　　一阵狂笑声陡然响起，阳鼎天眼眸扫视四方，面带张狂之色，大笑道：“苏莫、海明江，我知道你们就在天海城，你们逃不掉了！”
<br/>　　阳鼎天的声音，如同滚滚雷霆，响彻在了整个天海城。
<br/>　　所有人闻言，无不变色，原来苏莫和海明江来到了天海城！
<br/>　　这阵法，看来是阳鼎天对付苏莫和天涯海阁阁主之用啊！
<br/>　　苏莫和海明江眉头紧皱了起来，两人对视一眼，都从对方眼眸中看到了凝重之色。
<br/>　　阳鼎天困住他们，看来是有绝杀的手段了！
<br/>　　苏莫和海明江并未现身，两人混进街道上无尽的人.流之中，准备先静观其变。
<br/>　　嗖！嗖！嗖！！
<br/>　　就在此刻，阴阳神宫的深处，又有四道人影冲天而起，飞到了阳鼎天的身边。
<br/>　　这四人，分别是大魔老、二魔老、殷棕和北堂平华。
<br/>　　这一刻，四人都现身了，因为苏莫绝对逃不掉了。
<br/>　　“这是……！”苏莫和海明江两人，看到突然出现的二魔老四人，顿时心中再次一惊。
<br/>　　居然又来了四名准帝强者！
<br/>　　加上阳鼎天的话，总共就是有五人了！
<br/>　　“这下麻烦了！”苏莫眉头紧皱，心中不禁一沉，以他和海明江的实力，二对五，胜算太低了。
<br/>　　天空中，阳鼎天等五人伫立虚空，磅礴的气息弥漫八方。
<br/>　　本来，殷棕是准备离开天海城的，毕竟，殷家被灭，他无论如何也要赶回去看看。
<br/>　　但是，当知道苏莫来了天海城之后，他当然不会离开了。
<br/>　　他要亲手将苏莫挫骨扬灰，为殷家那数百万惨死的族人报仇雪恨，以慰藉这些族人在天之灵。
<br/>　　“苏莫，你这个畜生，快给我滚出来！”
<br/>　　殷棕乍一现身，便满面怒色，放声大吼了起来：“今日我要将你挫骨扬灰，让你永世不得超生！”
<br/>　　殷棕的面色有些狰狞，而他身上的气息猛然狂暴了起来，便的非常恐怖，如同惊涛骇浪，轰隆隆的碾压而下。
<br/>　　磅礴的威压，充斥在整个天海城，让得无数武者口中鲜血狂喷。
<br/>　　一些修为较低的武者，面对殷棕的威压，甚至直接被压的趴在了地上，无法动弹。
<br/>　　殷棕心中的怒火已经炸开了，再也抑制不住，他恨不得现在就大开杀戒。
<br/>　　在阴阳神宫中，殷棕当时从那两名幸存的长老口中，确认了家族被灭的消息时，当场就被气的口吐鲜血，体内玄力控制不住，差点没走火入魔。
<br/>　　当然，别说是殷棕了，就算是换成任何一个人，家族被灭，无数族人惨死，都会疯狂的。
<br/>　　“你给我出来，快给我出来！”
<br/>　　殷棕疯狂怒吼，其大手猛然一拍，一道巨大的掌印轰击而出。
<br/>　　轰隆隆！！
<br/>　　掌印瞬间轰击在远处的街道上，霎时间山崩地裂，大量的建筑灰飞烟灭，大量的无辜之人被轰成了碎片。
<br/>　　“啊……！”
<br/>　　“救命啊！”
<br/>　　“快跑！”
<br/>　　很多武者吓得肝胆俱裂，疯狂的逃窜，急速远离了现场。
<br/>　　而这个时候，阳鼎天手捏印诀，居然稍稍打开了阵法。
<br/>　　只见在阳鼎天等五大强者的头顶上空，随着阳鼎天打出一个个印诀，大阵居然缓缓打开了一个通道。
<br/>　　而后，大量的阴阳神宫弟子，如同潮水一般，通过通道离开了天海城，走出了阵法之外。
<br/>　　“嗯？”让苏莫目光一凝的是，他居然在人群中看到了上官昊。
<br/>　　虽然上官昊模样变化很大，身上的气息也改变了，但他还是一眼认出了对方。
<br/>　　苏莫顿时心中杀机大起，但他还是暂时压制住了，现在显然不是杀上官昊的时候。
<br/>　　“他们这是要干什么？”海明江见此有些疑惑，阴阳神宫的人居然这么着急离开！
<br/>　　难道是怕等会爆发大战，遭池鱼之殃！
<br/>　　“海阁主，不能再等下去了！”
<br/>　　苏莫目光望向海明江，沉声道：“天海城就这么大，我们无法躲藏，待阴阳神宫的弟子离开，他们更无顾忌了！”
<br/>　　苏莫准备动手了，虽然他们现在还未现身，但阳鼎天等人想要找出他们两人，实在太简单了。
<br/>　　大战难以避免！
<br/>　　这天海城上的大阵，虽然不知道具体的威力，但以苏莫的观察，其内蕴含的力量，并不比天殷城的土灵阵差多少。
<br/>　　所以，他们两人想破阵离开，基本上没有多大的可能了！
<br/>　　因为殷棕和二魔老等人，不可能让他们安静破阵。
<br/>　　那现在唯有一战了，或许他们可以通过这个阳鼎天打开的通道先行脱身！
<br/>　　“动手！”
<br/>　　海明江和苏莫对视一眼，立刻明白了苏莫心中的想法。
<br/>　　而后，毫不犹豫，他身形冲天而起，急速向那处通道冲去。
<br/>　　“杀！”
<br/>　　海明江身形刚刚冲起，他手中便出现了一杆长矛，而后猛然暴刺。
<br/>　　咻咻咻！！
<br/>　　霎时之间，矛影翻飞，十数道长达千丈的矛影洞穿空间，急速向殷棕五人杀去。
<br/>　　而苏莫的速度比海明江丝毫不慢，同样身形拔天，向通道冲了过去。
<br/>　　与此同时，九柄本命灵剑如同道道闪电一般，向殷棕五人身上疾刺而去。
