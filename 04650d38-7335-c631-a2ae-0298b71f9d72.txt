　　元释从深层次的悟道之中苏醒过来，却是因为，在内心之中感受到了，这混沌之中，存在一个劫煞之气，而且还越来越多。这让他感受到一个前所未有的巨大危机！
<br/>　　以他如今的实力，能够给他巨大危机的情况，以元释的智慧，哪还不明白，这意味着什么？
<br/>　　那就是盘古开天辟地，混沌大劫将至！
<br/>　　“大劫就要来临了吗？若是再给我一段时间，我未必不能再次突破，更进一步。罢了，既然大劫将至，那我也该多做一些准备了！盘古！我倒要看看，你到底有多强？是不是真的无法超越？！”
<br/>　　双目睁开，眼中透露着幽深的光芒，宇宙生灭，生死幻灭，天地一瞬，都在元释眼中闪过。元释看向混沌深处，目中闪耀着奇艺光芒，他看的那个方向，正是混沌中心，盘古所在之地！
<br/>　　“我的实力虽强，但较之盘古还是差了一些，如今盘古的实力已经混元十二重天了吧，就快要证道了。所以，我还是要为以后的开天多做些准备！”悟道古茶树下，元释思索道。
<br/>　　时间已然不多，距离大劫将近。混沌魔神真身，堪比混元圆满的巅峰力量，更有因果法则，趁着悟道古茶树下这段时间，其余几种法则也有领悟，修为距混元大罗金仙十一重天只差一步之遥，若是几种法则叠加，我的战力，已经无限接近混元十二重天了。这一战，我必须以最强的姿态，才有可能与盘古一战。
<br/>　　除了肉身，吾有两个修行方向，一是吾之本命大道法则——因果之道，另一个就是鸿蒙元始印中记载的本源大道了。此二者相比，虽说目前我的因果之道更强，本源大道吾如今也领悟不深。但，吾之本源大道却潜力无穷。不过，元始印终究是不完善，需要更多的大道法则来完善其内的本源界。
<br/>　　不过，毕竟时间已然不多了，吾已算到，吾之道，更在未来的洪荒世界之中。那么，现在，就要做多重打算了。
<br/>　　想到这里，元释眼神渐渐坚定，带着一种思索，更带着一种决然之芒在内。默默感应着盘古所在方向，良久之后，他深吸一口气，双目猛然一闪，眼色中透露着一股执着与果决，道：
<br/>　　“吾之本源之道潜力无穷，这是记载在鸿蒙元始印之中的一种道，不再大道之内，吾之因果大道已经接近巅峰了，想要突破大道，除非是遇到逆天机缘，不过，这种机缘又是何等珍贵，必然不会轻易出现。故而，我今后的修炼，当以本源之道为主，因果之道为辅！”
<br/>　　“既然确定了今后的路线，那我就要多做一番准备了。如此，我可分出一道真灵，放到鸿蒙元始印的本源界之中。这元始印是鸿蒙至宝，哪怕混沌劫再凶悍，也发现不了他，借此正好躲过这开天之劫！”
<br/>　　念及到此，元释目光一闪，神色一定，咬牙之下，元神之中，灵魂深处，以因果大道凝聚一柄法则之剑，猛然向着其中心的灵魂之火斩去！
<br/>　　“给我斩！
<br/>　　斩！斩！斩！”
<br/>　　三个斩字，可见决心之坚。猛然间，元释身体一震，眉心顿然飞出一朵洁白无瑕的白色火焰，此火恢弘，纯粹而磅礴，一出现，它周围的混沌之气都被点燃，可见温度之高，威力之强。
<br/>　　“哼！这一小团灵魂之火，可是吾这里，灵魂之中最精华、最纯净的一部分，更是分出融入了一丝丝蕴含吾所有记忆真灵。灵魂之火不灭，则真灵不灭，真灵不灭，则吾不灭！如此，吾必将重来！”
<br/>　　这一团灵魂之火飞出，元释闷哼一声，脸色瞬间变得苍白无比，身体颤动之下，那团灵魂之火瞬间没入元始印之中，那里，有着本源界的保护，想必不会受到危害。
<br/>　　身体气息一阵萎靡，但元释却没有在意，反而目光之中，光芒更胜了，透着一股决然和执着。“我还需将吾之真灵与因果线相连，到危急关头，绝命之际，说不定这余下的真灵，也有机会遁入元始印之中。”
<br/>　　“嗡！嗡！”
<br/>　　“吾为因果之主，因果之道，听吾号令，因果之线，灵魂之火，连！喝！”
<br/>　　一瞬间，在元释有意的控制之下，元始印一颤，鸣响之中，顿时一道若有若无的黑白因果线于虚空之中延伸而出，接着延伸至元始印之中，与那团灵魂之火相连，渐渐地，因果之线隐没，直至消失不见。
<br/>　　“因果之道，果然神奇！如此，我便没有了后顾之忧，可以放手一搏了。盘古？呵呵！”做完这一切，元释不管一切，闭上双眼，在混沌悟道古茶树下，修炼了起来。灵魂受损，可不是好修复的。
<br/>　　时间慢慢流逝，百万年，千万年过去了，苍茫依旧，混沌不变，灰蒙蒙一片，亘古如此。而在这混沌之中，劫煞之气已经越来越严重了。在这劫煞之气的影响之下，无尽的生灵，掀起了一场腥风血雨的厮杀、争斗！
<br/>　　无数的魔神血洒苍茫，暗然陨落，作为大道之路上失败的一员，他们注定要退出混沌舞台，消失在这无尽的混沌之中，亿万年之后，又有几人会被铭记？
<br/>　　千万年之中，元释时时刻刻都在修炼，悟道古茶树的树叶已经被他用了一度又一度。他的伤势，已然恢复，浑身散发着大道的韵味，古老而苍茫，玄之又玄！
<br/>　　混沌之中不记年，有是两百万年过去了。
<br/>　　混沌中心的某一处，滚滚混沌气流翻滚间，一朵弥漫着无穷造化之光，蕴含着无尽伟力，光芒四射的三十六品混沌青莲在那里扎根绽放着！
<br/>　　玄妙的道韵无穷无尽，柔和的道光在闪动着，一道道近乎实质的大道法则在其中演化着，一会儿化成一朵莲花，一会儿化作神钟，一会儿又是神龙，一会儿是绿鼎，神凰、道塔、麒麟、大江、神山……无穷无尽，玄之又玄！
<br/>　　于那混沌色青莲之中，一道惊天的身影在里面沉睡着，有亿万丈之高，散发着令乾坤失色的浩瀚伟力。
<br/>　　那仿佛超脱了岁月时空，挣断了因果轮回的无尽气息在其身躯上散发着，更有着一股骇人的不朽青光在其身躯上渐渐升腾。一道无边无尽，如同大道般磅礴浩瀚的道念萦绕在其身边，笼罩了苍茫大地，带着一股永恒的气息！
<br/>　　那道沉睡的身影之中，怀抱着一柄霸气锋芒的巨斧，混沌色的光辉仿佛可以斩断苍茫，划破混沌，此刻它却在不断散发道韵滋养着那沉睡的大汉，融入其身体之内，使之更加强大！
<br/>　　渐渐地，那身影气息越来越磅礴，周身之上，十二道不灭青光弥漫不止，仿佛亘古不朽，散发着破灭一切的伟力！
