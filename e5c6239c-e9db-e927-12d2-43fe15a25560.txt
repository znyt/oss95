　　但反过来，假如自己穿越过来之后，就不再受到原来的生死簿的约束，那自己……原则上只要不出意外，不被人打死，岂不是要长生不老了？
<br/>　　这又是一个矛盾的悖论。
<br/>　　叶少阳当然想不到真相，灵机一动，干脆从背包里找出自己的圆珠笔和本子，把这些自己想不明白的事情都记下来，将来再慢慢想。
<br/>　　在庵子里带到傍晚，幸好雨也彻底停了，三人赶尸上路。
<br/>　　因为下了一场雨的缘故，山路湿滑，人还好，尸体很不好走。陈三拿出了一些牛毛毡纸，上面绑上了一些木棍，给每一具尸体等于都做了一副脚套，然后赶着上山。
<br/>　　走着走着，叶少阳突然招呼陈三站住，说道：“等下！”
<br/>　　陈三回头看去，见叶少阳一脸凝重，心头一紧，说道：“又怎么了！”
<br/>　　叶少阳凝神朝左右望去，缓缓说道：“有邪物跟着。”
<br/>　　“不会吧！”陈三听了这句话，差点跌掉，颤巍巍地说道：“怎么又遇到邪物了……不会这么巧吧。”
<br/>　　“应该不是巧合。”叶少阳恍然大悟，“我明白了，之前我还在好奇，那个被我灭了的鬼，根本不具有冲破鸡血石挂坠的修为……她上翠云姐的身，只有同伴帮忙，他们当时想的应该是里应外合，没想到其中一个被我灭了。”
<br/>　　陈三听他这么一说，也朝左右看去，“那……这个鬼不是来找我们报仇的吧？”
<br/>　　“应该不敢，鬼也不傻，看我灭了另一个，当时没上，就是知道不是我对手，现在不可能来报仇。”叶少阳目光落在那具女尸身上，“我怀疑，他们是奔着这具人形煞来的。不然也不会想方设法进我们庵子。”
<br/>　　“那、那咋办。”
<br/>　　“凉拌！”叶少阳冷笑一声，“既然它不敢现身，我们盯着这些尸体就行，只要它敢来，我就灭了它。”
<br/>　　只要不是突然袭击，叶少阳其实都是有自信的，虽然自己现在实力不济，不过自己手中那些法器，灭一个普通的厉鬼还是没问题的。
<br/>　　三人继续赶路。
<br/>　　叶少阳能够察觉到有东西一直在附近若即若离地跟着，不过也无法确定具体位置，也就不管了，只是在陈三和翠云身上都贴了灵符，提醒他们注意。
<br/>　　走了大半夜，前面听见哗哗的水声，陈三赶过去一看，苦着脸回来，告诉叶少阳，因为下雨，山洪爆发，把路给冲了，前面走不了了，只能绕路。
<br/>　　“绕路就绕呗，怎么了？”叶少阳看他神色不对，纳闷道。
<br/>　　“绕路是没啥，只是从这边过去，只有一条路，叫二龙山，那里有土匪……”
<br/>　　叶少阳嘴角抽了抽，道：“没别的路了？”
<br/>　　“有，绕的远，要多走好几天，还不一定保证道路通畅。”
<br/>　　叶少阳倒吸一口气，自己对付妖魔鬼怪是没问题，但是土匪的话……人家可是有家伙的，叶少阳不知为何想起了之前陈三说的关于点天灯的那一段，有些头皮发麻。
<br/>　　“走你说的那条路，一定会遇到土匪？”
<br/>　　“那不会，就是从山脚下走，不太容易遇到，只是说，有这种可能，而且一般土匪，也不会为难我们赶尸的。”
<br/>　　叶少阳想了想，决定还是走这条路，毕竟绕路走的话，多出几天，会多很多麻烦，而且前途未知。
<br/>　　不过在上那条路之前，叶少阳让翠云化妆了一下，像上次一样，弄的蓬头垢面的，让男人一看就没兴趣，免得万一遇到土匪，被抢走做了压寨夫人。
<br/>　　绕路之后，一直走到快天亮，突然又下雨了，一开始下的不大，僵尸头上都戴着斗笠，影响不大，但叶少阳担心下大雨会冲掉僵尸身上的灵符，问陈三怎么办。
<br/>　　陈三是走过这条路，表示前面山脚下有个破庙，可以暂时避雨，正好天也快亮了，可以在庙里呆上一天，晚上再赶路。
<br/>　　叶少阳一听说破庙，想起上次在破庙里遇到几个傻比逃兵，被抢走了几根金条的事，有些阴影，问陈三那个庙距离土匪窝太近，不会出事吧。
<br/>　　“那没问题的，土匪闲的蛋疼，去破庙干什么。”
<br/>　　叶少阳听了这话一怔，陈三忙问他怎么了。
<br/>　　“没什么。”叶少阳苦笑，原来蛋疼这个词，一百年前就有了。
<br/>　　三人持续赶路，走了半个小时左右，在前面一个山坳里果然找到了一座破庙，看上去还算完整，三人把尸体赶进去，发现大殿里有倒塌的梁木，都腐朽了，于是烧火热饭吃。
<br/>　　叶少阳在破庙里观察起来，发现神像都塌了，里面蛀着一堆白蚁，看上去很恶心。叶少阳在神像前面找到一块腐朽的牌位，上面字迹都没了，因此也看不出这个庙里供的是什么神仙。看房屋结构，应该是民间信仰的那种神，而不是道家或佛家的庙宇。
<br/>　　三个人在火堆上烤了干粮，吃完之后，天也亮了。
<br/>　　三人从行李里拿出铺盖，各自躺下。叶少阳能感觉到那个东西就在附近徘徊，因此不敢睡觉，跟陈三聊起这座二龙山上的土匪，听了一些道听途说来的传闻，叶少阳津津有味地听着，翠云在旁边铺盖上躺着睡着了。
<br/>　　两人闲扯了一会，突然外面传来一阵马叫，两人都是一怔，互相看去。
<br/>　　“不会是……土匪吧。”叶少阳问。
<br/>　　“我不知道，这一代闹土匪之后，好像确实没什么人家了……”陈三脸都绿了。
<br/>　　说话间，已经能够听到马蹄声，踩在泥水里走过来，接着是有人唱小调的声音。
<br/>　　叶少阳和陈三豁的一下站起来，意识到了什么，想要找地方躲起来，但是庙堂就这么大，根本就没地方躲。
<br/>　　正在踌躇之间，外面想起了几个人的对话声，听声音都是粗野汉子：
<br/>　　“我说，这他娘的下雨下的难受，咱哥几个进庙里歇会儿？”
<br/>　　“歇你姥姥的，大当家等着咱去送信。”
<br/>　　“下雨你他么的看不见！就这怎么下山，大当家给咱一天时间，啥时候下山不行！”
<br/>　　“都别吵都别吵，听三当家的！”
