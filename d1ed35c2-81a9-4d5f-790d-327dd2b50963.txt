　　混元大陆南部，这荒凉之地上，一座巨大无比的祭坛，突然布置完成。
<br/>　　这祭坛周围，布置的种种诸天奇物，若是让许多武者看见，只怕都会一个个嫉妒到发狂的地步，纷纷想要夺取。
<br/>　　只是这些诸天奇物，如今全部都是被布置在祭坛周围，当做祭品。
<br/>　　而在祭坛前，两道身影，正站在那里，正是楚风眠跟白萱二人。
<br/>　　“天下苍生，祭祀天意！”
<br/>　　楚风眠长啸一声。
<br/>　　在这祭坛之上，猛然之间，发生了种种变化，在这祭坛上，突然出现了无数古怪纹路。
<br/>　　这些古怪纹路，古老到极致，就算是在博学的人，都无法看出其中的意思。
<br/>　　因为这是在人类都还没有诞生前，从混沌之中，诞生出来的文字。
<br/>　　玄妙，晦涩，每一个字，都蕴含有着天地真理。
<br/>　　“轰隆！”
<br/>　　就在祭坛上方的天空，仿佛都是出现了一个巨大无比的脸，这脸，似人脸，似兽脸，妖兽，魔头，异兽，仿佛是天下一切苍生的面孔，都可以从这张脸中找到。
<br/>　　这张脸，便是天意的脸庞，凝聚整个天下的脸庞。
<br/>　　“祭！”
<br/>　　看到这张脸出现，楚风眠大手一挥，那涅槃龙蛋，造化树枝，龙皇龙爪，一下子都飞入到了祭坛之上去。
<br/>　　飞到了这祭坛之上，一下子都是被融入了无数混沌文字。
<br/>　　随后这三件宝物，全部都是直飞云霄，一下子飞到了天上去，被那张诡异脸庞，直接吞了下来。
<br/>　　“轰隆！”
<br/>　　许久之后，从空中，猛然倾泻而下无数灵力，这灵力倾泻下来，简直是比起海啸都要可怕。
<br/>　　若是轰击在武者的身上，只怕是生死境高阶的武者，都不会有这任何抵挡的可能。
<br/>　　“这是？”
<br/>　　白萱脸上露出了几分紧张，运转起身躯之中的灵力，似乎是想要抵挡。
<br/>　　“不要抵挡，这些灵力是天意的回馈！”
<br/>　　楚风眠长啸一声。
<br/>　　白萱这才将一身运转起来的灵力，全部散开，有些紧张，又有些期待的，看着空中落下的滚滚灵力。
<br/>　　“轰隆隆！”
<br/>　　一股股灵力，倾泻而下，冲刷下来，全部冲刷在楚风眠跟白萱的身体之上。
<br/>　　“我的力量，居然又增长了？怎么可能？”
<br/>　　这一股股灵力，冲刷在白萱身体，融入到了白萱身体之中的一刻，白萱居然是感觉到，她身上的力量，居然是又更进了一步。
<br/>　　白萱身上的灵力，已经是到达了神力境极限的地步，也就是意味着她自己，无论是在吞噬什么诸天奇物，都不可能再进一步。
<br/>　　除非是选择度过生死大劫，踏入生死境，才可能。
<br/>　　可现在，在这天意回馈的灵力之下，她的实力居然又是开始节节攀升起来。
<br/>　　这简直是令白萱自己都不可置信。
<br/>　　一旁的楚风眠，听到白萱的话，心中只是笑了笑。
<br/>　　这力量极限，在天意面前，又算得了什么，天意的力量，足以冲破极限。
<br/>　　在远古时代，一些强者为了将力量达到最为极限的地步，也会利用圣祭，来利用天意的力量，冲击自己的极限。
<br/>　　只是这样做却是奢侈无比的行为，要知道楚风眠举办这圣祭，需要耗费了多少财富，还是从乾元等人的空戒中得到了大量的宝物，才如今能够举办这次圣祭。
<br/>　　要在举办一次的话，只怕是楚风眠找破了头也不一定足以将其中的东西找全。
<br/>　　“也该到我了！”
<br/>　　只见楚风眠的背后，那战龙决所凝聚出来的远古战龙虚影，便是清晰可见，翻腾这异样的光芒。
<br/>　　一股股灵力，也是冲刷在了楚风眠的身躯之中，全部融入到楚风眠的身躯之中去。
<br/>　　只是这灵力在多，融入到楚风眠的身躯之中，也无法再凝聚出一滴远古战龙精血。
<br/>　　现在楚风眠身躯中的远古战龙精血，已经达到极限，这是恒古不变的规则，就算是天意，都无法改变这一切。
<br/>　　只是这一股股灵力，却是都融入到了楚风眠的身躯之中，融入到了楚风眠的每一份血肉之中去。
<br/>　　令楚风眠身躯中的血肉，无数次的重组，凝聚。
<br/>　　每一次的重组，凝聚，都令楚风眠的身躯，要强大一份。
<br/>　　终于在又一次破碎之后，楚风眠身上的血肉，已经不在是那血红色，而是一种金色。
<br/>　　金色血肉，可是只有远古战龙的血肉，才会呈现出来的颜色。
<br/>　　现在楚风眠不单单是身上的血脉，就连身上的血肉，也开始向着远古战龙开始变化这。
<br/>　　“据说传闻中，远古战龙成年后，便是都会凝聚龙身，这龙身的强弱，也决定了一尊远古战龙的未来强弱，不知道我的凝聚出的龙身，会是达到一种什么地步！”
<br/>　　楚风眠一边心中暗自想到，一边也是疯狂的吞噬了无数灵力，统统融入到了楚风眠的身躯之中去。
<br/>　　“碰碰碰碰！”
<br/>　　楚风眠的血肉，一步步都撕裂，一步步重组，瞬息之间，楚风眠的血肉，都化为了金色，成为了真正的战龙身躯。
<br/>　　在这金色血肉之中，还足以清晰可见一些，古老的纹路。
<br/>　　这些纹路，都是龙语，最为古老的语言之一，虽然不如混沌文字的古老，却是蕴含有着龙族的真谛。
<br/>　　这些龙语凝聚在楚风眠的血肉之中，便是都化为了一道道阵眼一般。
<br/>　　将楚风眠的身躯，都是化为了一道阵法，源源不断的加持这楚风眠的力量。
<br/>　　“远古龙王身！这阵法，我的身躯已经达到了远古龙王神身的地步！”
<br/>　　楚风眠已经感觉得到，他现在的力量，已经是比起曾经，还要强大无数倍。
<br/>　　虽然说楚风眠身上的境界，那远古战龙精血，没有发生任何的变化。
<br/>　　但是他的身躯，却是也达到了真正远古战龙的地步，足以将他血脉中的力量，发挥到极致的地步。
<br/>　　力量没有任何变化，但是现在楚风眠的实力，只怕是比起曾经，还要强大无数。
<br/>　　“我的实力，居然达到了这一步！”
<br/>　　站在楚风眠身边的白萱，也是缓缓的张开双眼。
<br/>　　察觉到她自己身上的力量，白萱都是似乎有些不可置信。
<br/>　　她现在的力量，居然是隐隐约约都要超越了乾元，古林这等七大宗门的天才。
