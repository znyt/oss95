　　苏莫不用想也知道，这名高大青年定是太史世家派系的人。
<br/>　　不过，这太史世家派系也真是太不要脸了，居然派了一名武帝境七重的武者来挑战他。
<br/>　　对方怎么不直接派一名武尊呢？这不是必胜的战斗吗？
<br/>　　修为如此悬殊，他当然不会应战。
<br/>　　苏莫的声音从洞府中传出，传到了高大青年的耳中，立刻让得对方脸色一沉。
<br/>　　目光望向苏莫洞府的方向，高大青年眸中冷光闪烁，沉声道：“真是嚣张狂妄，既然你觉得不公平，自然还会有其他人来挑战你！”
<br/>　　言罢，高大青年转身离去。
<br/>　　既然苏莫不应战，他多说也是无益，他也知道以自己的修为，苏莫不太可能应战。
<br/>　　但是，苏莫狂妄的言语，亦是让他心中杀意凛然。
<br/>　　同阶武者，来多少杀多少？
<br/>　　这口气未免太大了，也不怕闪了舌头！
<br/>　　高大青年走后，苏莫继续修炼。
<br/>　　内府空间之中，很多的山峰上，都有人在注意这一幕，均是其它派系的弟子。
<br/>　　他们不是在关注苏莫，而是关注太史世家派系将会如何对付苏莫。
<br/>　　在高大青年走后，约莫盏茶的功夫之后，又有一人来到了火元峰之前。
<br/>　　此人个子不高，皮肤黝黑，双眉如钢刀，给人一种戾气超强之感。
<br/>　　“苏莫，你我同阶修为，敢否一战？”青年伫立在火元峰前，目光望着苏莫的洞府，朗声喝道。
<br/>　　洞府中，苏莫睁开了眼眸，嘴角勾起一丝冷笑。
<br/>　　太史世家派系的人既然不怕死，他自然不会龟缩不出。
<br/>　　唰！
<br/>　　苏莫身形一闪，出现在了洞府之外，眸光直视黝黑青年。
<br/>　　四目相对，虚空中出现了道道裂缝。
<br/>　　此时，聂乾和韩盈，早已出现在了洞府之外。
<br/>　　“苏兄，此人叫雷灿，是外府中非常强大的一个天才，战力超强！”聂乾向苏莫说道。
<br/>　　苏莫微微点了点头，随即轻笑一声，对雷灿道：“走吧！去战界！”
<br/>　　言罢，他身形破空，直接向外府飞去。
<br/>　　雷灿见此，顿时阴狠一笑，这一次他一定要杀了苏莫。
<br/>　　为了杀苏莫，派系中可是拿出了大量的奖励，只要能斩杀苏莫，便能得到这些奖励。
<br/>　　这些奖励可是让很多人眼红，他可是向尤天韩大哥极力自荐，才得到了这个与苏莫一战的机会。
<br/>　　嗖！
<br/>　　毫不迟疑，雷灿身形一闪，追着苏莫急速离开了内府。
<br/>　　随后，聂乾和韩盈两人，也立刻跟了上去。
<br/>　　在几人走了以后，这内府中不少人都离开了内府，同样的随着苏莫而去。
<br/>　　这些人多是其它派系的人，很多都是内府弟子，甚至其中还有一些导师人物。
<br/>　　木璃导师，同样离开了内府，更了上去。
<br/>　　苏莫来到外府之后，发现外府中弟子极少，他也没有停留直接来到了通宇大殿，再次的进入了战界。
<br/>　　“好多人！”来到战界之后，苏莫一眼望去，发现十几座观战的小岛上，布满了密密麻麻的人影，少数也有几万人。
<br/>　　原本嘈杂不堪的战界，在苏莫到来的这一刻，突兀的安静了下来。
<br/>　　无数道的目光，都凝视在了他的身上。
<br/>　　“看来太史世家派系都准备好了！”苏莫心中暗道，此种情况，明显是太史世家派系已经通知了所有人，今日要在战界对付他。
<br/>　　想要当着所有人的面，将他击杀。
<br/>　　苏莫心中沉吟，同阶一战他不惧任何人。
<br/>　　但是，他担心的是，若是他胜了之后，太史世家派系狗急跳墙，派修为远高于他的人强势出手，这就糟糕了。
<br/>　　不过，想来这个可能性不大，首先他背后有火元圣师这个虎皮、其次当着学府中其它各大的派系的面，太史世家应该不会这么无耻吧！
<br/>　　嗖！
<br/>　　随即，苏莫身形破空，呼吸间飞到了十几座岛屿的中央，身形缓缓的落在了海面上。
<br/>　　“哈哈！苏莫来了，这是要硬磕太史世家派系了啊！”
<br/>　　“不知道太史世家派系，派谁出战！”
<br/>　　“太史世家派系天才无数，强者如云，不知道苏莫能否挡住？”
<br/>　　“苏莫上次那种手段太诡异了，我到现在都不清楚太史轩是如何落败的！”
<br/>　　十几座岛屿上，议论声四起，现场再次嘈杂了起来。
<br/>　　很多人都还不知道，要和苏莫交手的人是谁！
<br/>　　但是，仅仅是呼吸之间，众人便立刻知道了答案。
<br/>　　只见雷灿紧随而来，飞到了十几座岛屿中央，与苏莫相隔千里而立。
<br/>　　“居然是雷灿出手了！”
<br/>　　“雷灿的修为虽然不如太史轩，但战力可是远超太史轩啊！”
<br/>　　“这下精彩了，不知道苏莫能否战胜雷灿！”
<br/>　　“雷灿专修雷霆之力，攻击刚猛霸道，无物不破，战力超强！”
<br/>　　众人见到与苏莫一战之人是雷灿，顿时惊讶不已，不过，这也没有让众人意外。
<br/>　　苏莫的实力这么强大，太史世家派系，自然不会派小虾米上场，肯定是派战力强大之辈出战。
<br/>　　一些其它派系之人，均是脸色挂着淡笑，眸中露出了期待之色。
<br/>　　若是雷灿胜了，击杀了苏莫，那还好说。
<br/>　　若是雷灿败了，被苏莫所杀，那太史世家派系的脸，就丢的更大了。
<br/>　　他们都很乐意看到太史世家派系出丑，然后便能不断的打压，让太史世家派系的名声一落千丈。
<br/>　　如此，他们各大派系，就能拉拢更多的天才。
<br/>　　“苏莫，今日生死一战，我会用你的血，祭奠太史轩！”
<br/>　　雷灿眸中杀意沸腾，朗声高喝，他的面色稍稍凝重了下来，显然并没有轻视苏莫。
<br/>　　“可惜你没有这个实力！”
<br/>　　苏莫轻笑一声，心中丝毫无惧，同阶一战？不是他自大，别说这神武学府之内，就算是整个古周星，他都不信有人能是他的对手。
<br/>　　当然，苏莫也没有狂妄到，认为自己有同阶无敌的实力，毕竟宇宙星空无比浩瀚，单是天荒星域中的一个古灵星河，就已经大到无边无际了。
<br/>　　“废话少说，拿命来吧！”
<br/>　　雷灿不再多言，脚下猛然一跺海面，海水瞬间炸开，他整个人立刻冲到了数万丈的高空之上。
<br/>　　雷灿不傻，在去挑战苏莫之前，他已经详细的了解了苏莫，知晓苏莫手段诡异，身上能喷涌出淡紫色的雾气，动用精神类的攻击。
<br/>　　所以，他一直没有离苏莫太近，乍一开战，便直接冲到了天穹之上。
