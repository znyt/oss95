
<br/>　　<br/>　　    地上那人眼神阴霾的棱着眼珠子，那双眼睛里透出来的表情就是不屑，还有就是你别白费力气了。<br/>　　<br/>　　    王惊蛰皱眉的提着棍子，照着他的枪伤就插了过去，抿着嘴唇用树枝的一头还捅咕了两下，这看的郭茜茜都有些叹气了，枪伤加上断骨，在被捅了这么两下，绝对痛入骨髓的那种疼，这比老虎凳和辣椒水可要难熬多了。<br/>　　<br/>　　    “嗯哼”这人咬着嘴唇闷哼了一声，就是忍着没叫出声来。<br/>　　<br/>　　    郭茜茜摇头说道：“你这么上刑没什么用的，他的意志很坚定，肉体上的疼，再疼他都能忍得住，策略不对”<br/>　　<br/>　　    王惊蛰诧异的回头问道：“你又行了？”<br/>　　<br/>　　    “你也不看我是干什么的”郭茜茜白了他一眼，说道：“把他提起来，拎到车那边去”<br/>　　<br/>　　    “那等会的”王惊蛰转过头，用棍子戳着他的脸说道：“脱衣服，全脱了”<br/>　　<br/>　　    郭茜茜顿时惊愕了，她不可置信的看着王惊蛰，一阵恶寒从脚底蔓延到了脑袋顶上，一层层的鸡皮疙瘩往地上掉。<br/>　　<br/>　　    王惊蛰看见她的反应，无语的说道：“你往哪想呢，这人有些手段防不胜防，我得小心着点……脱，快点的，一件不能剩”<br/>　　<br/>　　    这人屈辱的抬着脑袋，眼神里嗷嗷怨毒。<br/>　　<br/>　　    王惊蛰见他没动，只得硬着头皮上前“刺啦”一下就扯开了他的衣服，他是怕对方在身上藏有什么下降头的方式，自己不一定会怕，但这警花估计够呛能防得住，解降太麻烦了点。<br/>　　<br/>　　    对方穿的衣服也不是很多，王惊蛰三两下就将他给扒光了，然后直接拎着他就回到了车旁边，郭茜茜打开后备箱拿出两瓶矿泉水，还有一盒纸巾，抽出几张纸后盖到了对方的脸上，就拧开矿泉水“哗啦”一下，缓缓的倒在了他的脸上。<br/>　　<br/>　　    “这是FBI刑讯逼供的方式之一，看着很简单，其实能将人心底最深处的恐惧逼出来，可以将他的防线很快就全部摧毁了，据我所知除非受过特殊训练的特工，剩下的没谁能挺住这种生不如死的感觉”<br/>　　<br/>　　    “哗啦，哗啦”瓶子里的水一直流着，对方脸上的纸巾湿透了以后，郭茜茜又接着往上面一张一张的放着纸，王惊蛰明显感觉到自己摁着的人在剧烈的挣扎着，这状态看起来确实相当的痛苦，难熬了。<br/>　　<br/>　　    “青竹蛇儿口，黄蜂尾上针，两般皆不毒，最毒妇人心呢……这娘们太狠了”王惊蛰倒吸了一口冷气，在心底腹诽了一句。<br/>　　<br/>　　    “啪”郭茜茜伸手就把对方脸上的纸巾给扫了下去，然后跟王惊蛰说道：“你问吧，他要是再坚持坚持，我继续给他加点量”<br/>　　<br/>　　    王惊蛰沉默的点了点头，然后疑惑的问道：“你是不还没有对象呢？”<br/>　　<br/>　　    “嗯？”<br/>　　<br/>　　    “以后，温柔点吧，不然啥好老爷们都得被你给吓跑了”王惊蛰感慨了一声，蹲在地上盯着对方的脸，说道：“是谁让你给破庙里的那几个孤魂野鬼，下了降头的，我要是没看错的话，你下的是鬼降吧？”<br/>　　<br/>　　    这人眼中诧异的表情一闪即逝，似乎没料到对方会这么了解他的手段，一语就道出了他下的是鬼降头。<br/>　　<br/>　　    “你不说也没关系，我知道是徐少中……”<br/>　　<br/>　　    “唰”对方脸上表情再次变换，王惊蛰直接精准的捕捉到了，继续说道：“我抓你只是要确定一下这个判断罢了，现在看来我们查的没错，但我很奇怪的是徐少中怎么会认识你这种能下降头的人，在内地降头师都已经快要绝种了，我看你的长相也根本不是内地的，我猜猜看你是南洋哪里的，大马，又或者是泰缅两地？看你年纪也不大，想必你不知道，外界的降头师或巫师，轻易都不敢进入华夏内地吧？你这么犯险前来，把自己给漏了的话，就不怕有来无回么？”<br/>　　<br/>　　    一直默不作声的这人，眼神中忽然露出一抹惊慌的神情。<br/>　　<br/>　　    郭茜茜则是突然察觉到，王惊蛰的身上杀机冒了出来，她诧异的愣了愣，但在她还没想清楚时，就见王惊蛰突兀的抬起手，手里的树枝“咔嚓”一下就被他给掰断了，随即尖锐的树枝朝前一松，顿时“噗嗤”一声，居然一下子就插进了对方的喉咙下面。<br/>　　<br/>　　    “呲……”一簇鲜血涌了出来，这人不可置信的身手捂着自己喷血的脖子，一只手惊讶的指了指王惊蛰，嘴里艰难的吐出了几个字：“我死，你也……”<br/>　　<br/>　　    王惊蛰下手太快了，不光对方没有反应过来，就连郭茜茜也没想到，直到那人尸体缓缓倒在地上的血泊里时，她才回过神来。并且下意识的就抓紧了手里的枪。<br/>　　<br/>　　    “你杀人了，杀了他？”郭茜茜震惊的说道：“你不知道旁边站着个警察呢么？”<br/>　　<br/>　　    王惊蛰头都没有抬，拿出符纸放在对上尸体上，说道：“他犯了规矩，自然就得杀了他”<br/>　　<br/>　　    “什么规矩？你这是犯法，是杀人罪！”<br/>　　<br/>　　    王惊蛰布置完，起身拍了拍手，很平淡的说道：“中华大地上有一句话，叫犯我中华者虽远必诛，但还有一句话你们可能就不知道了，大概意思是说番邦异类但凡来犯，就有来无回”<br/>　　<br/>　　    郭茜茜皱眉说道：“什，什么意思？”<br/>　　<br/>　　    “他不是什么普通的人，是一个会下降头的降头师，这种人如果进入中华大地肆意妄为的话，将会很难处理的，所以很久以前我们就有过规矩，内地以外的巫师，降头师，还有小鬼子那些忍者什么的，这一类人是严禁进入内地的，怕的就是他们惹下麻烦后一走了之，我们却很难处理了，所以这个规矩很严，不但不允许他们进入内地，如果来了，那就干脆别回去了……”<br/>　　<br/>　　    郭茜茜听完，自然十分不理解，但她仍旧盯着王惊蛰说道：“我是警察，你却当着我的面杀了人，你触犯了法律，这是事实！”
