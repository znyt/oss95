　　现在，失踪近一年的李春出现在这里，无论气质或者实力，都有天翻地覆的变化。
<br/>　　先前只是一名普通的在不能普通的市民，现在却是练气二重的修士，当真令人好奇。
<br/>　　此时，李春心中满是惊恐，他可是练气二重修为，哪怕来一个连的特种兵，只要给他时间，他都能全部撩倒。
<br/>　　可在面前的青年手中，却犹如手无缚鸡之力的孩童，不对，是连孩童都不如，简直是待宰的雏鸡。
<br/>　　原本，他打算宰了慕清霄，将欧阳紫嫣给强抢过来，现在看来，他的想法简直太天真了。
<br/>　　因为，他一直都引以为傲的实力，在慕清霄面前，连出手的机会都没有。
<br/>　　“你......你们是故意引我来这里的！”
<br/>　　闻言，慕清霄并没有回答他的问题，反倒是一脚踩在李春脸上。
<br/>　　强大的压迫力，令得扎在他脸上的玻璃碎片镶嵌的更深，猩红的血液流淌个不停。
<br/>　　“啊啊...!”
<br/>　　感受到脸上传来的痛楚，李春疼的浑身抽搐起来，心中惊恐的同时又充满了怨恨，这张脸估计是毁了。
<br/>　　“你修为是怎么来的？”
<br/>　　“我不知道你在说什么，快放我离开！”
<br/>　　强忍着恐惧与疼痛，李春竭斯底里的开口道。
<br/>　　闻言，慕清霄更加用力，使得玻璃碎片全都都扎进他的脸颊中，冷笑道：“给我想要的信息，就放你离开。”
<br/>　　“我都说了不知道你在说什么，难道你还敢杀我不成。”
<br/>　　“呵...”
<br/>　　将李春从地上拎了起来，慕清霄道：“你已经消磨掉我为数不多的耐心，既然如此，我就自己动手。”
<br/>　　说罢，瞳孔中就浮现三个黑色勾玉，显得妖异至极。
<br/>　　空中，四目相对，恐怖的精神力量席卷而出，顷刻间，李春那怨毒的眼神就变得暗淡呆泄起来。
<br/>　　“告诉我想要的信息。”
<br/>　　李春的目光依旧呆泄，道：“李春，孤儿，静海市本地人......”
<br/>　　见此，慕清霄剑眉皱在一起，李春只说出了自己的姓名籍贯，没有任何关于他修为的信息。
<br/>　　想到此处，慕清霄中指与食指点在他的额头上，微微闭上双眼，开始强行探查他的记忆。
<br/>　　接着，李春记忆中的一幕幕就如同幻灯片一样，呈现在慕清霄的脑海中。
<br/>　　片刻后，慕清霄睁开双眼，瞥了眼失去生息的李春，眼中满是困惑。
<br/>　　李春的记忆中，只有从小到大的一幕幕，没有一丝关于修炼的信息，而且他失踪期间的记忆，完全处于封印状态。
<br/>　　就在刚才，他想要强行破开封印，却被一股奇异的力量给挡了下来，甚至连李春本人，也彻底死亡。
<br/>　　“真是有意思。”
<br/>　　慕清霄现在都觉得有些不可思议，他练虚九重的修为，后者居然能在他的手中杀人，稍微触动一下封印，李春就死了。
<br/>　　打开车门，将李春的尸体丢进去，一缕黑色火苗落下，慕清霄潇洒转身离去。
<br/>　　车内，欧阳紫嫣望了一眼身后，渐渐化作灰烬的别克商务车，忍不住问道：“问出信息了吗？”
<br/>　　“没有，对方手段不一般。”
<br/>　　微微摇头，慕清霄眯了眯眼睛，不知道在想些什么。
<br/>　　“系统，关于李春被封印的记忆，需要多少掠夺积分。”
<br/>　　“叮，查看被封存的记忆，需要一百万掠夺积分，宿主是否支付一百万掠夺积分进行查看？”
<br/>　　不理会系统的询问，慕清霄将车窗关上，待身后的别克商务车彻底化作灰烬后，掉头向市中心驶去。
<br/>　　使用一百万掠夺积分，去查看李春的信息，除非他脑子里灌了水。
<br/>　　不过，根据系统的回答，看来李春背后的存在非常神秘，否则也不至于需要一百万掠夺积分。
<br/>　　“系统，地球有其他修士的存在吗？”
<br/>　　“叮，回答宿主此问题，需要两千掠夺积分。”
<br/>　　慕清霄微微一笑，道：“扣除。”
<br/>　　“叮，扣除成功，宿主所在的世界，灵气稀薄，不存在修士，但也有属于本世界的体系，华夏国术，岛国忍术......”
<br/>　　闻言，压在他心上的巨石总算是落下，只要不存在修士，他就可以安心了，但李春的修为是怎么回事？
<br/>　　将脑海中复杂的信息抛之脑后，慕清霄看了眼欧阳紫嫣，道：“明天，将一年前，失踪人口的信息收集一下。”
<br/>　　“嗯，我会让人去办的。”
<br/>　　欧阳紫嫣知道事关重大，自然不敢怠慢。
<br/>　　等回到别墅中，已经是深夜，慕清霄也没有做任何出格的事情，搂着欧阳紫嫣沉沉睡去。
