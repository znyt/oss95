　　第122章
<br/>　　耿星汉在知道对方有神射手后，便意识到今天碰到硬茬子。往日抢劫来往货船，对方一般只是象征性的抵抗一阵，便开始缴械投降，然后伸着脖子任自己杀。
<br/>　　然而今天的事情显然不一样，在登船之后，便遭遇顽强的抵抗。说抵抗，倒不如说是杀戮，在面对鲜血时，对方显得比己方还要亢奋，在杀人这件事上，对方显得比自己还要内行。
<br/>　　这究竟是一帮什么人。
<br/>　　他一直小心翼翼的隐蔽自己的身形，并大声呼喝着手下。
<br/>　　“进攻，进攻，进攻！”
<br/>　　嗖！
<br/>　　是羽箭破空的声音，耿星汉几乎是下意识扭过头，一支箭不偏不倚的扎进他的喉咙。
<br/>　　耿星汉倒地，手中的鱼叉落入海中，发出清脆水响。
<br/>　　终究是没有躲过啊。
<br/>　　船头，赵子龙清脆的吹了声口哨，交待给自己的任务总算是完成了。此刻有一海贼从背后冲向他，他在船头跃起，身体在空中360度旋身，弓卡住对方脖子，用力，收紧，对方在弓弦下挣扎，却如同陷入猫爪的老鼠，直到被勒死，再无法动弹。
<br/>　　在杀戮场上还如此大开大合，要么是找死，要么是在炫技。
<br/>　　或许他，就是在炫技。
<br/>　　当耿星汉死后，这场战斗也分出胜负，很难相信一帮海贼会有怎样的战斗意志。在首领被杀，死亡人数超过三十人时，他们的战斗意志便彻底崩溃。
<br/>　　贪生怕死人之常情，没有人会在明知是死的情况还保持向前冲锋的姿态。
<br/>　　一帮人惶惶而逃，有些来不及上船的，在水中噗通噗通挣扎。
<br/>　　战斗过后，水面若血染，海面漂浮着一具具尸体。有海贼的，也有程大雷一方的。
<br/>　　清点战场，程大雷发现己方牺牲了五人，有些人尸体掉进水中，在这种情况下，也没办法捞上来。
<br/>　　战斗便必然会遇到牺牲，死人的事情是在所难免的。以五十人的兵力击溃二百海贼，而且还斩其首领，这场战斗无论如何都算得上胜利，甚至可以引以自豪。
<br/>　　但程大雷还是不开心，直到现在，他还没习惯牺牲。
<br/>　　“耿星河，总有一日，我会亲手杀你。”
<br/>　　程大雷心底浮现这句话。
<br/>　　……
<br/>　　千舟城，渔阳楼。
<br/>　　白天是耿星河休息的时间，他的工作往往在晚上展开。白天不忙的时候，他往往会温一壶老酒，煮锅老肉，一边喝酒一边让胡姬为自己跳舞。
<br/>　　胡姬是个好女人，不同江南女人的柔弱和娇小，胡姬有不输男儿的身高，牛奶般的皮肤，紧实细腻的小腿以及饱满的胸脯。
<br/>　　在千舟城，无论任何人见到耿星河，会发现他脸上总是带着笑，像个和气生财的财主。
<br/>　　如果把任何一人换到耿星河的位置上，相信他也会每天脸上都堆满笑容。每日有酒有肉有全城数一数二的美女，他如何能不高兴。
<br/>　　令耿星河这样开心的还有另外一方面原因，一种智商上的优越感。谁会想到，为祸海上，整个扬州都在通缉的红胡子，此刻就在城里，把一座城的人都得团团转。
<br/>　　咚咚咚
<br/>　　门外急促的敲门声，有个声音问：“大……老爷，您在里面么？”
<br/>　　耿星河正在由胡姬陪着饮酒，喝着喝着胡姬便被耿星河抱在话里，罗裳半解，秀发微乱，耿星河正仔细把玩着胡姬胸前的玉软。
<br/>　　如此被打扰，他心里有几分不开心。
<br/>　　“什么事？”
<br/>　　“海上的事。”
<br/>　　耿星河如此才将胡姬放开，冲她挥挥手道：“你先下去吧。”
<br/>　　胡姬在门口与一伙计擦肩而过，伙计进门后小心翼翼把门关上。
<br/>　　“什么事不能待会再谈。”耿星河冷哼一声。
<br/>　　“大当家，海上传消息来了……”
<br/>　　耿星河微微点头：“算时辰也该有消息来了，老二把事情都办妥了？”
<br/>　　在这密室内，耿星河才稍稍显出一方匪首的峥嵘，在他对面说话的伙计，连头都不敢抬。
<br/>　　“失手了。”
<br/>　　“什么！你再说一遍！”
<br/>　　“折了五十几号弟兄，就连二当家也，二当家也死在当场，兄弟们连他的尸首都没捞着。”
<br/>　　“什么！”
<br/>　　耿星河腾地一下从椅子上坐起，眼中显露出狠戾之色。
<br/>　　无怪耿星河难以置信，他查探过程大雷的船，对方不过有五十几号人，为首还是一个酒囊饭袋。为了那两船粮食，他出动了二百人，可结果……却实在难以预料。
<br/>　　自己亲弟弟死了，骷髅岛也因为这件事元气大伤。海上不仅有骷髅岛一伙海贼，本来是自己弟弟坐镇，海上的事情无用自己担忧。但如今，自己不得不回去了。
<br/>　　唉，好日子到头了。
<br/>　　……
<br/>　　来时用了五天，回去因为逆风的缘故，却需要更多的时间。半路途中遭遇一场风暴，虽然最后平安度过，但却又平白耽误两天时间。
<br/>　　时间就是生命，如今这句话对程大雷来说，不再是虚无缥缈的人生道理，而是真真切切的眼前现实。
<br/>　　扬州总之算是繁华的，可一旦往北走，能见到的繁华就越来越少。等到青州境内，就已能见到逃荒的灾民。程大雷停在码头休整时，就碰到要高价买粮的人，越往北走，粮价也是越贵，到最后高到一个令人难以置信的地步。甚至有人把十几岁的闺女往船上送，所求不过是能换一斗米。
<br/>　　不仅幽州在死人。
<br/>　　老实说，程大雷有些心软，但最后还是把牙咬住了。兼济不了天下，目前能做的也只是独善其身。
<br/>　　终于在二十天后的一个夜里，船在落叶城外的一个野码头靠岸。这次行动程大雷只打了十五天时间，因为山寨的粮食最多只能撑这么长时间。可一来一去，却足足用了一个月时间。
<br/>　　也不知山里的情况如今怎样，他们还撑不撑得住。
<br/>　　船刚靠岸，程大雷就让徐神机和赵子龙连夜回山，让山寨来人取粮。至于程大雷自己，则是抱着斧头守在船上，连眼睛都不敢闭一下。
<br/>　　幽州赤旱千里，多少人食不果腹，把两大船的粮食搁在这里，若是消息走漏出去，会发生怎样的事情，后果不堪设想。
<br/>　　
<br/>(本章完)
