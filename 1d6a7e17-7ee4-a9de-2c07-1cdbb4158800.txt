　　芊寻月从棺椁之中漂浮了起来，缓缓浮上了半空。
<br/>　　这一刻，苏莫心中稍稍有些紧张，芊寻月能否被救活，今日是最关键的时刻。
<br/>　　苏莫和绫裳两人，均是稍稍后退，免得影响了紫霄。
<br/>　　紫霄没将急着施手，而是仔细的打量着眼前的芊寻月，面色显得颇为郑重。
<br/>　　少倾之后，紫霄动手了，只见他的身上，陡然升腾两股气息。
<br/>　　这两股气息，分别是生之气息和死之气息，玄妙无比。
<br/>　　两股气息越来越强势，在紫霄的身上，形成了两股一黑一白的气流。
<br/>　　两股气流彼此交织、缠绕，如同旋转的太极阴阳图，深邃玄妙的气息，不断的弥漫而出。
<br/>　　少倾，两股气流涌到了芊寻月的身上，将其完全包裹了起来。
<br/>　　而后，紫霄缓缓的坐了下去，芊寻月悬浮在紫霄的头顶，在一黑一白两种气流之内，微微沉浮。
<br/>　　紫霄的身上，更是升腾起一种磅礴的大势，这股大势与黑白气流相连，产生出旺盛的生命力。
<br/>　　苏莫和绫裳两人，面色凝重的望着这一幕，期待芊寻月真的能被救活。
<br/>　　苏莫静静的等待，他无法帮忙，也不懂得紫霄是如何救治的，只能耐心等待。
<br/>　　时间不断的流逝，很快便过去了两个时辰，在苏莫的观察之中，芊寻月好似没有任何的变化。
<br/>　　这时，苏莫稍稍有些急躁了起来，这么久了，也没看到显著的成果。
<br/>　　突然，紫霄身上的大势，迅速的收敛了起来，一黑一白两股气流，如同万流归宗一般，回到了他的其内。
<br/>　　芊寻月的身躯，缓缓的落在了棺椁之中，紫霄也长身而起。
<br/>　　唰！
<br/>　　苏莫一步踏出，瞬间便来到了棺椁之前，查看了一眼芊寻月，顿时面色一沉。
<br/>　　芊寻月，根本就没有活过来，而且，依旧是毫无生机的模样。
<br/>　　“紫霄宫主，这是怎么回事？”苏莫阴沉着脸问道。
<br/>　　“她仅存的那丝执念，已经被本宫主用生之力巩固，她可以坚持三百年！”紫霄神王道。
<br/>　　“什么？”
<br/>　　苏莫闻言，脸色一黑，怒声说道：“你不是说能救活她的吗？”
<br/>　　对方之前信誓旦旦，只要绫裳答应其要求，就能救活芊寻月。
<br/>　　但是现在，对方根本没有救活芊寻月，只是延续了芊寻月的‘生命’而已。
<br/>　　本来，对方说芊寻月只能在坚持百年，现在则是增加到了三百年。
<br/>　　“放肆！”
<br/>　　面对苏莫愤怒的喝声，紫霄神王顿时面色一沉，随即厉声道：“你以下犯上，看在绫裳的面子上，本宫主饶你不死，给我滚！”
<br/>　　话音一落，紫霄大袖一挥，一股无形的力量，顿时倾泄而出。
<br/>　　这股力量之强大，无语伦比，这不是修为之力，而是一种天地大势。
<br/>　　嘭！
<br/>　　毫无意外，苏莫瞬间被击飞了出去，飞出了大殿。
<br/>　　蹬蹬蹬！！
<br/>　　苏莫落在地上，连续倒退了数十步，才稳住了身形。
<br/>　　他的眸中，迸射出滔天的怒火，此人果然心机深沉。
<br/>　　如此看来，就算绫裳真心答应了对方的要求，对方都不一定会救芊寻月。
<br/>　　“紫霄，你居然出尔反尔？”绫裳的脸色，布满了寒霜，怒视着紫霄神王。
<br/>　　“绫裳，我并未出尔反尔，只不过，救活她损耗太大，需要慢慢来，等到你恢复到神王之时，差不多就能彻底救活此女了！”紫霄神王微微摇头说道。
<br/>　　“你……！”绫裳心中同样愤怒，他明白对方的意思，但是却无言以对。
<br/>　　很显然，紫霄不是傻子，不可能立刻救活芊寻月。
<br/>　　紫霄的意思很明显，什么时候绫裳恢复到了神王，与紫霄双修助后者突破到高级神王，便彻底救活芊寻月。
<br/>　　绫裳脸色难看，紫霄将芊寻月的‘生命’延续到了三百年，但是她三百年之内，肯定无法恢复到神王之境。
<br/>　　那么，三百年后，必须再次让紫霄出手一次。
<br/>　　而这三百的时间，也算是对方对她的监督，如果她在这三百年之内，故意延缓恢复的速度，对方以后不可能会出手了。
<br/>　　“紫霄，没想到你如此精于算计！”绫裳眸中闪过厉声，心中充满了无奈。
<br/>　　“算计倒是谈不上，我只是希望我们的合作，能圆满结束而已！”紫霄微笑着说道，并没有因为绫裳的语气，而有所动怒。
<br/>　　“希望你言而有信！”
<br/>　　绫裳冷冷的说了一句，便将芊寻月的棺椁收了起来，而后转身离开，来到了殿外苏莫的身前。
<br/>　　“秦陨，我们走！”绫裳向苏莫说道。
<br/>　　苏莫攥了攥拳头，转而又松开了，他无奈的点了点头。
<br/>　　现如今，只能暂时退却了，与紫霄理论无异于找死。
<br/>　　随即，两人大步而行，向紫霄剑宫之外走去。
<br/>　　“对了，忘记告诉你们，由我施手之后，任何强者再也不能对那位女子施救，不然此女的执念立刻就会湮灭！”
<br/>　　就在此刻，紫霄的声音，悠悠的响起，传到了苏莫和绫裳的耳中。
<br/>　　紫霄端坐在殿中的主座之上，面上带着若有若无的笑意，他用生死元神之力，为芊寻月的执念进行了巩固。
<br/>　　而他的生死元神之力，蕴含特殊的意志，除了他之外，任何人都不能再出手救治芊寻月。
<br/>　　不然，芊寻月的执念，就会立刻湮灭，测地的荡然无存。
<br/>　　苏莫闻听此言，顿时心中一震，随即，一股滔天的怒火，在他的心中炸开了。
<br/>　　“可恶！”
<br/>　　苏莫猛然转身，犀利的杀机在他身上升腾而起，他彻底怒了，这个紫霄实在是卑鄙无耻。
<br/>　　“我们走！”绫裳俏脸冷漠，一把拽住苏莫的手臂，飞速向外而去。
<br/>　　她知道，自己若是不阻拦苏莫，苏莫必死无疑。
<br/>　　在绫裳的阻拦之下，苏莫没有出手，按捺住了心中的怒气。
<br/>　　大殿之中，紫霄目光如电，望着飞速离去的苏莫和绫裳，眉头微微一皱。
<br/>　　“秦陨……什么关系呢？”紫霄低声自语，他自然能看出，绫裳和苏莫关系不一般。
<br/>　　他对于绫裳再了解不过了，性格冷淡，根本不会和男子相处。
<br/>　　而如今，绫裳为了这个秦陨的道侣，居然如此热心，更不惜答应他的要求，可见绫裳和这个秦陨，关系非同小可。
<br/>　　不过，紫霄也不在意，绫裳逃不出他的掌心。
<br/>　　虽然说，绫裳就算恢复到了神王之境，与他双修合.体，对于他冲击高级神王，只有一两成的助力加成，但是，一两成已经很逆天了。
<br/>　　修为到了如此地步，哪怕是一成的成功略，都是天大的机遇。
<br/>　　“绫裳，希望你不要怪为兄，这或许就是你的宿命！”
<br/>　　紫霄微微的叹了口气，绫裳的归来，以及拥有纯阴之体，这是冥冥之中天道给予他的造化。
<br/>　　所以，他不会放过这个造化，只能牺牲绫裳了。
<br/>　　不错！是牺牲！
<br/>　　最霸道、最狠辣的双修之术，为了达到最好的效果，双修之后，炉鼎不可能活命，会耗尽一身的精气和神魂之力。
<br/>　　对此，紫霄有些无奈，他也曾喜欢过绫裳，爱慕这个绝美而清纯的女子，所以本不愿看到这种结果。
<br/>　　但是，为了武道，为了生存，为了超脱纪元，他只能舍弃绫裳了。
<br/>　　再者，如今的绫裳，天赋已经不足，不可能比前世更进一步了，早些成全他紫霄，也算是死得其所了。
