        “六十万！”傅昼景淡淡地回答道。

<br/>        “我借你一个亿，你跟我一起创业自己开公司，自己当老板，如何？”季思妍微笑着问，顿了顿后，她又从包包里掏出两份礼物，递给了傅昼景，接着说道，“我替你还清所有的外债！”

<br/>        傅昼景看到季思妍递过来的两样东西，不由地皱起了眉头。

<br/>        这是那日，他被逼得走投无路，拿去当铺典当了的手表和钢笔，他没想到，她竟然帮他全部赎了回来。
<br/>
<br/>        季思妍是季氏集团的大小姐，上头虽然有个哥哥和同父异母的妹妹，但是她爸爸只对他们兄妹两一视同仁。

<br/>        所以，季氏集团，季思妍占有四分之一的股份。

<br/>
<br/>        即便如此，季氏集团却是这滨江城里唯一一个实力能与傅氏集团抗衡的大公司。

<br/>        “你真打算一辈子都给你大哥打工吗？你爷爷压根就不重视你。”季思妍试图动摇傅昼景心里的想法。
<br/>
<br/>        傅昼景在傅家是个什么地位，季思妍一清二楚。
<br/>
<br/>        就连当初，她要求他跟她一起出国留学，都遭到了傅老太爷的反对。
<br/>
<br/>        傅老太爷为什么反对？
<br/>
<br/>        十有八九，傅老太爷是不希望傅昼景太优秀。
<br/>
<br/>        因为，傅昼景倘若太过优秀，就会盖过那位嫡长孙傅夜沉的光芒。
<br/>
<br/>        “阿景，五年前，其实，我是为了你的将来，才去美国留学的啊！”季思妍伸出手来，轻轻地拉住了傅昼景的手，眼神真挚地看着傅昼景。
<br/>
<br/>        傅昼景眉头紧锁，语气冷了几分：“我来，只是想知道我妈妈的事情。”
<br/>
<br/>        季思妍识趣，只好起身，去了自己的卧室。
<br/>
<br/>        当她再次走出房门的时候，手里多了一份牛皮纸袋。
<br/>
<br/>        “阿景，这个资料，我是通过非正规渠道得到的。我知道，你一直就在暗中调查你妈妈当年真正的死因是什么。毕竟，‘殉情’这种说法，确实不靠谱，不是吗？”季思妍话中有话的说道。
<br/>
<br/>        关于阿景的母亲和父亲之间的那点事情，她略知一二。
<br/>
<br/>        只是她不相信，一个男人死了后，原配不殉情，殉情的却是小三？
<br/>
<br/>        噢，不，她不应该在背地里称呼阿景的母亲为“小三”。
<br/>
<br/>        毕竟，阿景的母亲，连个“小三”都算不上。
<br/>
<br/>        她也只不过是道听途说，当年，阿景的母亲是“算计”了他的父亲，才怀上阿景的。
<br/>
<br/>        这件事情，一点都不光彩吧！
<br/>
<br/>        所以，阿景在傅家一直都不受待见。
<br/>
<br/>        傅昼景接过季思妍手中的牛皮纸袋，有一种急速不安的感觉迫使他快速地拆开了牛皮纸袋上的白色棉绳。
<br/>
<br/>        里面不仅有照片，有文字记载资料，还有录音笔。
<br/>
<br/>        傅昼景一一仔细地翻看，一遍又一遍非常认真地听录音笔里的内容，最后悲愤交加，恼火又痛心地将牛皮纸袋拿在手里捏成了纸团。
<br/>
<br/>        “阿景……我其实有点儿怀疑这些资料的真实性……所以才犹豫要不要给你……”季思妍欲言又止。
<br/>
<br/>        傅昼景的眼睛里有着暗暗的血丝，薄唇微微颤抖着，与额角跳动的筋脉一起，像是在向她宣告着他内心的痛苦与愤怒。
<br/>
<br/>        “阿景，其实，我也不敢相信，你的母亲是被你大哥的母亲害死的！只不过，这些资料又不得不让我去怀疑。”季思妍微微垂下了眼帘，娇柔的声音不停地哽咽着，“而且，你大哥和你爷爷一定也知道这件事。你爷爷一直在包庇你大哥的母亲，所以你大哥才会对你格外好，因为，他在为他的母亲赎罪。”
<br/>
<br/>        傅昼景听着季思妍的话，此刻的心情，几乎面临崩溃状态。
<br/>
<br/>        这是一种苍穹轰然倒塌的绝望，死神的来临，恍若将他的灵魂带走，让他痛得生不如死。
<br/>
<br/>        为什么？
<br/>
<br/>        为什么他在暗中调查了这么久，每次快要接近真相的时候，线索总会突然中断，而这中断后的点，就是季思妍这次所带给他的资料。

<br/>        真的是大妈害死他妈妈的吗？
<br/>
<br/>        这件事情，大哥和爷爷真的知道吗？

<br/>        他在傅家不受待见不要紧，但他的妈妈一直死不瞑目啊！
<br/>
<br/>        季思妍见傅昼景痛不欲生的样子，心疼地从自己的座位上起身，一屁股坐在了傅昼景的大腿上。

<br/>        她双手缠住傅昼景的脖子，低头，猝不及防地覆上了傅昼景的唇，舔着他唇上的苍白，弱弱地问道：“阿景，你是不是想为自己的母亲报仇？”

<br/>        他母亲不欠傅家，不是吗？

<br/>        他母亲身为军医，常年陪伴在他父亲身侧，任劳任怨。

<br/>        他傅昼景算什么？
<br/>
<br/>        他傅昼景的母亲又算什么？

<br/>
<br/>        傅昼景微微抬眸看着季思妍的眼睛，她那双黑白分明，坚定如星光般璀璨的眸子里，清晰地倒映出他的影子。
<br/>
<br/>        季思妍继续试探性地安抚着傅昼景的心情，语调抑扬顿挫地接着说道：“我的阿景，天生就有王者之风。何苦委屈自己，臣服于别人呢？我们联手，一起拿下远东城的国防军机部的建设项目。我听说，那个项目，上头拨款下来一共有十个亿，除了建厂子，还得建军事基地。”

<br/>
<br/>        远东城国防军机部的项目，傅氏集团势在必得，不然那次，堂叔冷岩峰也不会抢破头皮都要跟他争地利置业的总经理一职。

<br/>        季思妍将自己的身子，软软地依偎在傅昼景的怀中。
<br/>
<br/>        她的身上有一种让他恋恋不忘的香水味。
<br/>
<br/>        十八岁那年，在他就在她身上闻到过。
<br/>
<br/>        这五年都过去了，她的身上，还是那个令他怀念的味道。

<br/>        “阿景，让我做你的情人好吗？我知道，你纠结在我和你的新婚妻子之间。我不想为难你，但是，我一直都深爱着你啊！我无法接受除了你以外的别的男人！所以，这段日子，我想通了。我不求你能给我什么名分，我只求能这样长期陪伴在你身边就好。无论是在工作上，还是感情上，还有这身体上。我都能完完全全地成为你的贤内助，做你背后的女人！”季思妍一番深情地告白，她想为自己五年前的离开而让他失去他们彼此的这五年来赎罪。
<br/>
<br/>        傅昼景惊怔地看着季思妍，他完全没想到，她会如此善解人意。
<br/>
<br/>        倘若换做是是筱颖，筱颖会接受吗？
<br/>
<br/>        “你愿意接受筱颖？”傅昼景仍旧有些难以置信。
<br/>
<br/>        季思妍抿了抿唇，非常认真地看着傅昼景，点了点头：“嗯！她是你的新婚妻子，你都愿意接受她，我又为何不愿意接受她呢？我爱你，就会打心底的爱屋及乌。阿景，我爱你，永远地爱你。我的心，我的身体，还有我的钱，都是阿景的！我唯你独尊，好不好？阿景来当我的王！筱颖是王后，我甘愿当你的妃！阿景，今晚就要了我，好吗？”
