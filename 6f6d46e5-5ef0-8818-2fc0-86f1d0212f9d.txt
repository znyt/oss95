　　房内，萧逸盘膝修炼着。
<br/>　　对他而言，难得的闲暇时间，几乎也都在修炼中度过。
<br/>　　边修炼，也边思索着。
<br/>　　按他的性格，一般不会出这些风头；即便来参加比拼，也只是独自等待，参加完就会离去。
<br/>　　不曾想，刚落下东离剑宫，东离剑宫宫主，便这般热情相迎。
<br/>　　一般而言，萧逸以往也很少有这般待遇，加之他总是在外行走独来独往，少有与这些势力打交道之时。
<br/>　　这次，想来是他在六龙地域击杀毒袍之事，让得他如今声名极甚。
<br/>　　六龙地域，本来就在东面范围。
<br/>　　自然的，六龙地域距离极东之地这边不近但也不远，那边的事传来这里极快。
<br/>　　能让傲东楼亲自相迎。
<br/>　　这，或许便是强者的待遇了。
<br/>　　在萧逸的感知中，傲东楼的气息，大概和龙啸差不多。
<br/>　　故，傲东楼，必然也是位绝世9999道的强者，距离传奇，仅差最后一步。
<br/>　　不过，对比龙啸的狂霸气息，傲东楼的气息更加凌厉，也更加内敛。
<br/>　　证明，傲东楼的实力，应该比龙啸还高上些许。
<br/>　　毕竟傲东楼乃是剑修。
<br/>　　东离剑宫，位列三大剑宫之一。
<br/>　　而剑修，又是最擅战斗的职业。
<br/>　　不过，若在六龙地域之内，龙啸可是等同无敌。
<br/>　　当然，即便真在外相遇，傲东楼想败龙啸，估计也非易事。
<br/>　　龙啸能有传奇之下最强的称号，本就证明了他的实力。
<br/>　　另外，极东之地这边，事实上根本算不得上是地域。
<br/>　　极东之地，位于中域最东面，往后，便是一片荒凉。
<br/>　　而东离剑宫，本身弟子等等，也极少在外行走。
<br/>　　萧逸捏了捏有力的拳头，淡淡一笑。
<br/>　　若有机会，他倒真想和傲东楼切磋一番。
<br/>　　大半月前，他能败毒袍，且那时他便确定，毒袍，再无机会追上他。
<br/>　　且现今，控火兽的层次更高了，自身修为也提升了，这可都是实质性的提升。
<br/>　　若他没猜错的话，他现今实力，应该能和龙啸、傲东楼等强者，打个不相伯仲。
<br/>　　萧逸边思索着，也想起了刚才大殿之内的痕迹。
<br/>　　他是猎妖师，又是控火武者。
<br/>　　故之前大殿内焚烧的妖火痕迹，瞒不过他的眼睛。
<br/>　　傲东楼的话，也证实了他的猜测。
<br/>　　只是，到底是哪两方势力，竟拥有这般强者？
<br/>　　他几乎一眼就能断定，那股妖火的层次，甚至还超过了灸炎。
<br/>　　还有，两位如此出色的天骄？
<br/>　　就他所知，就算是中域十八府，抑或五大学宫，也不可能有这般可怕的天骄。
<br/>　　介乎绝世9996道至9997道之间，对年轻天骄而言，这可是个可怕的概念。
<br/>　　“看来之前来此的，是两方隐世势力。”萧逸双眼一眯。
<br/>　　隐世势力，对于以前的萧逸来说，还是比较陌生的。
<br/>　　但这些年，游走于风刹殿、修罗殿、猎妖殿等五殿，而且现今头衔也不低了。
<br/>　　自然的，他能了解事情，多了不少。
<br/>　　隐世势力，便是其中之一。
<br/>　　偌大个中域，庞大得可怕。
<br/>　　即便不算各个地域，单单那些险地、杳无人迹的荒芜之地等等，就是一个极其庞大的范围。
<br/>　　还有一些外人难以知晓的古老之地，隐地等等。
<br/>　　整个中域，着实精彩无比，也秘辛无数。
<br/>　　隐世宗门，便是存在于那些外人难以知晓的古老之地，或者隐地之中。
<br/>　　关于这个层次的秘辛，萧逸了解得不多。
<br/>　　但就他听说过的隐世势力，倒也不少。
<br/>　　家族、宗门等等，不下十数个。
<br/>　　他所熟知的冰皇宫就是其中之一。
<br/>　　这些隐世势力，极少行走在中域，他们也很少插手中域之内的事宜。
<br/>　　故对于几乎大多武者和实力来说，这几乎是一些不存在的透明势力。
<br/>　　可毫无疑问，这些隐世势力，尽皆极强。
<br/>　　而且他们传承的岁月，也异常久远，甚至有的可以媲美八殿。
<br/>　　而最重要的一点是，这些古老之地、隐地，无不是洞天福地，一方宝地。
<br/>　　对寻常武者来说，那里简直是修炼圣地。
<br/>　　里头，也有着远比普通地方更适合天材地宝生长之地。
<br/>　　故这些隐世势力的天骄、武者，修炼起来，也比外头的武者快得多，容易得多。
<br/>　　萧逸之所以猜测之前来东离剑宫的两方势力，是隐世势力，便是因为这个原因。
<br/>　　放眼中域，十八府天骄、五大学宫天骄，已经是一等一的绝世天骄。
<br/>　　可刚才的两方势力天骄，却几乎有着碾压他们的修为。
<br/>　　那么，可能只有一个，就是他们来自于隐世势力之内。
<br/>　　想起来，当年在四方域那边夺得冰冥幽火时，连陈星这样的十八府天骄，都对进入冰皇宫修炼而雀跃不已。
<br/>　　便可知晓那些地方，确实远优于外头的寻常地方。
<br/>　　当然，萧逸知晓的，大概也只这些。
<br/>　　“算了，若他日有机会遇到，想来会是不错的对手。”萧逸自信一笑。
<br/>　　下一秒，萧逸收敛心头杂念，全心修炼起来。
<br/>　　……
<br/>　　翌日，清晨。
<br/>　　天蒙蒙亮，萧逸便结束了修炼，撤去房中禁制，缓步走出房间。
<br/>　　“呼。”萧逸深呼吸一口气，看了眼远方初阳。
<br/>　　他是个控火武者，能将火焰完美操控。
<br/>　　但，他仍旧喜欢感受初阳时的温暖。
<br/>　　初阳而生，不灼不冷，带有淡淡温暖之感，着实让人舒服。
<br/>　　而初晨的空气，亦最是让人神清气爽。
<br/>　　“嗯？”萧逸刚深呼吸一口气，便注意到门外站着二人。
<br/>　　看其服饰，应当是剑宫弟子。
<br/>　　“易霄副殿主。”二人手中端着盘子，道，“奉命送早宴过来。”
<br/>　　“之后，我二人回领易霄副殿主去剑宫广场。”
<br/>　　“嗯。”萧逸点点头。
<br/>　　想来，这是傲东楼一番好意，好生招待。
<br/>　　萧逸也没理由拒绝。
<br/>　　随意吃了早饭，便跟随这二人前往剑宫广场。
<br/>　　刚到剑宫广场，周遭，已是人山人海。
<br/>　　“易霄副殿主。”两位剑宫弟子笑道，“各大势力争夺名额，这般盛事，个个都来得很早呢。”
<br/>　　“不过，宫主早就给易霄副殿主准备好了观赛席。”
<br/>　　“有劳。”萧逸点点头。
<br/>　　剑宫广场正中央，有一观赛首席。
<br/>　　刚来到首席处，原本人山人海的喧闹，人声鼎沸的议论，霎时静谧。
<br/>　　“紫炎易霄？”一双双凌厉的目光，几乎一瞬间聚集在萧逸身上。
<br/>　　首席上，傲东楼缓缓走来，苦笑道，“易霄小友，老夫再慎重问你一句，你可是真要参加这名额之争？”
<br/>　　“据老夫所知，猎妖殿等三殿，本身就有名额。”
<br/>　　“以易霄小友的出色，理应本身就拥有名额，所以…”
<br/>　　傲东楼笑笑，道，“不妨和老夫一起当这观赛人，要不当个裁判也行…”
<br/>　　萧逸摇摇头，打断道，“我来此，只为参加比拼。”
<br/>　　此言一出，下方霎时满座哗然。
<br/>　　“什么…”
<br/>　　“紫炎易霄也要参加此次比拼？”
<br/>　　“该死，谁打得过这个变态？”
<br/>　　“堂堂紫炎易霄，也太不要脸了吧。”
<br/>　　议论声、不满声，不绝于耳。
<br/>　　萧逸皱了皱眉，他不过是来参加比拼罢了，何至于如此不满。
<br/>　　一旁，傲东楼满脸苦笑。
<br/>　　事实上，萧逸似乎忘了一事。
<br/>　　他当自己是年轻天骄，可他传出的各种事迹，还有现今实力，在外人眼中，早已是位列成名强者行列。
<br/>　　外人早便将他放到了那些成名数百年以上的强者行列，根本未将他当作年轻天骄。
<br/>　　“给我闭嘴。”
<br/>　　周遭喧闹，吵得萧逸一阵头疼，只得冷喝一声。
<br/>　　“谁不服，可与易某一战。”
<br/>　　滔天气势，瞬间压下，惊人的火焰气息，肆虐全场。
<br/>　　剑宫广场，霎时满座肃静。
<br/>　　“易霄，我来与你一战。”一声暴喝，瞬间响起。
<br/>　　“嗯？”萧逸微微皱眉，他总感觉，这道声音，有些熟耳。
<br/>　　轰…高空中，雷电闪烁，崩腾而下。
<br/>　　一杆长枪，破空而来。
<br/>　　“天枪冉琦。”萧逸自语一声。
